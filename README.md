## データの更新
基本的に[運営からのお知らせ](https://ekimemo.com/news/)で公表される駅情報の更新に応じて同様の作業を実施。  
具体的な内容は[更新の記録](./history.md)に記述。  
なお現在の最新データのバージョン・場所等の情報は[このファイル](./latest_info.json)で定義されています。
