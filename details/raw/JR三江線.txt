<h2 id="w605c1fe">路線の特徴</h2>

<ul>
    <li class="ulist1">江津～三次間を結んでいた路線。路線総延長は108.1km。「<span style="font-weight: bold;">さんこうせん</span>」と読む。<br />
全線が非電化、単線で、キハ120系気動車が1～2両で運行されていた。</li>
</ul>


<br />
<ul>
    <li class="ulist1">JR線で最も儲かっていない路線であった為、JR西日本は2016年9月1日に当路線の廃止を決定。（<a title="http://www.westjr.co.jp/press/article/2016/09/page_9174.html" href="/jump/http/www.westjr.co.jp%2Fpress%2Farticle%2F2016%2F09%2Fpage_9174.html" class="jump">出典</a>）<br />
2018年4月1日に<span style="color:red;" class="font">全線廃止</span>された。駅メモ上では、2018年4月27日に<a title="路線案内：廃止路線" href="/wiki/%E8%B7%AF%E7%B7%9A%E6%A1%88%E5%86%85%EF%BC%9A%E5%BB%83%E6%AD%A2%E8%B7%AF%E7%B7%9A">廃線</a>扱いになった。</li>
    <li class="ulist1">最初の区間が開業してから88年、最終運行日はちょうど桜の満開と重なった。</li>
</ul>


<br />
<ul>
    <li class="ulist1">「<span style="font-weight: bold;">江の川鉄道</span>」の愛称があり、その名の通り<a title="https://www.cgr.mlit.go.jp/cginfo/syokai/busyo/kasen/suikei/gouno.htm" href="/jump/https/www.cgr.mlit.go.jp%2Fcginfo%2Fsyokai%2Fbusyo%2Fkasen%2Fsuikei%2Fgouno.htm" class="jump">江の川</a>（ごうのかわ）に沿ってグネグネと走っていた。<br />
そのため、時速25kmあるいは15kmといった、低速の速度制限がかけられている箇所も少なくなかった。<br />
ただ、景色は新緑の頃も紅葉の頃も見応えは十分である。</li>
</ul>


<br />
<ul>
    <li class="ulist1">沿線は伝統芸能の「<a title="http://gotsu-kanko.jp/iwamikagura/aboutkagura/" href="/jump/http/gotsu-kanko.jp%2Fiwamikagura%2Faboutkagura%2F" class="jump">神楽</a>」で有名であり、各駅の愛称にも神楽の題名が使われていた。（<a title="http://sankousen.com/wp-content/uploads/2012/10/92b2bb6d84b0b7b1dd7c3a8ed4485aba.pdf" href="/jump/http/sankousen.com%2Fwp-content%2Fuploads%2F2012%2F10%2F92b2bb6d84b0b7b1dd7c3a8ed4485aba.pdf" class="jump">詳細</a>）<br />
また、車両の運用状況にもよるが、神楽のラッピングがされた車両が多用されていた。</li>
</ul>


<br />
<ul>
    <li class="ulist1">廃止後の代替バスも区間で区切られる上に本数が少なく、島根県と広島県のマスターオブ称号を取るためには最難関な路線になってしまった。<br />
これから路線制覇をする場合は余裕を持った日程で望むことを強く推奨（レーダー等によるアクセス方法は後述）。</li>
</ul>


<br />

<h3 id="x313ce29">駅メモ路線情報</h3>

<ul>
    <li class="ulist1">江津～宇都井、口羽～作木口：<a title="路線案内：島根県" href="/wiki/%E8%B7%AF%E7%B7%9A%E6%A1%88%E5%86%85%EF%BC%9A%E5%B3%B6%E6%A0%B9%E7%9C%8C">島根県</a>に所在。</li>
    <li class="ulist1">伊賀和志、香淀～三次：<a title="路線案内：広島県" href="/wiki/%E8%B7%AF%E7%B7%9A%E6%A1%88%E5%86%85%EF%BC%9A%E5%BA%83%E5%B3%B6%E7%9C%8C">広島県</a>に所在。</li>
</ul>

<h2 id="s410a659">攻略法(廃止以降)</h2>


<h3 id="jd53edf8">代替バスを使う場合</h3>

<ul>
    <li class="ulist1">廃止後、<a title="http://www.town.ohnan.lg.jp/www/contents/1522197138359/index.html" href="/jump/http/www.town.ohnan.lg.jp%2Fwww%2Fcontents%2F1522197138359%2Findex.html" class="jump" target="_blank">代替バス</a>が走っているが、主として以下の4系統に分割されている。(平日は川本美郷線が浜原駅前を境に分かれる)<ul>
        <li class="ulist2">江津川本線(石見交通)：江津駅前～石見川本</li>
        <li class="ulist2">川本美郷線(大和観光)：石見川本～浜原駅前～道の駅グリーンロード大和</li>
        <li class="ulist2">作木線(備北交通)：道の駅グリーンロード大和～川の駅常清(港別)</li>
        <li class="ulist2">川の駅三次線(君田交通)：川の駅常清(港別)～三次駅前</li>
    </ul></li>
</ul>


<br />
<ul>
    <li class="ulist1">他のバスルートの方が、駅跡の近くを通る場合もある。(江津本町、千金、沢谷、宇都井、式敷～尾関山など)<br />
このうち宇都井駅だけは、上記ルートだとチェックイン困難な場合があるため、レーダーアクセスも考えておこう。</li>
</ul>


<br />
<ul>
    <li class="ulist1">計算上は、平日1往復、土日祝日2往復乗り通すことが可能。<br />
しかし、乗り換え待ちが3時間あったり、逆に乗り換えがシビアな場合もあったりするので、攻略時は下調べをしっかり行ってから臨もう。</li>
</ul>


<br />

<h3 id="f2b53549">周辺の公共交通機関からレーダー取得</h3>

<ul>
    <li class="ulist1">いずれの場合も、<a title="三条なつめ" href="/wiki/%E4%B8%89%E6%9D%A1%E3%81%AA%E3%81%A4%E3%82%81">なつめ</a>(と<a title="立石あやめ" href="/wiki/%E7%AB%8B%E7%9F%B3%E3%81%82%E3%82%84%E3%82%81">あやめ</a>)の力を借りて、レーダー14駅範囲を限界まで使う場合が多く、また12駅範囲でも位置が大変シビア。<ul>
        <li class="ulist2">山陰本線<ul>
            <li class="ulist3">江津駅周辺から江津～川戸間がレーダー12駅範囲</li>
            <li class="ulist3">太田市駅周辺から粕淵、石見簗瀬が12駅、浜原、明塚が14駅範囲</li>
        </ul></li>
        <li class="ulist2">芸備線<ul>
            <li class="ulist3">上川立駅周辺から信木～三次がレーダー12駅、式敷駅が14駅範囲</li>
            <li class="ulist3">三次駅周辺からだと、所木駅までしか届かない。</li>
        </ul></li>
        <li class="ulist2">中国自動車道（広島～松江の高速バスが多数あり）<ul>
            <li class="ulist3">本郷PA（上り方面）から作木口～船佐まで12駅、口羽14駅（下りだと、口羽までレーダーは届かない）</li>
            <li class="ulist3">高田IC上記から口羽がレーダー14駅範囲外になる代わりに江平がレーダー14駅範囲内となる。</li>
            <li class="ulist3">三次IC側で、上記より三次までの各駅を取得可能。チェックインできるのは信木のみとなる</li>
        </ul></li>
        <li class="ulist2">浜田自動車道（広島～浜田の高速バスが多数あり）　※トンネルが多いので位置情報や電波のラグに注意<ul>
            <li class="ulist3">川戸、田津、石見川越、宇都井にチェックイン可能。</li>
            <li class="ulist3">瑞穂ICから千金～竹が12駅範囲内、乙原が14駅範囲</li>
            <li class="ulist3">大朝ICから田津～石見川本と宇都井～式敷の内、江平以外がレーダー12駅範囲</li>
            <li class="ulist3">大朝ICからすこし広島側に移動した時点で江平も取得可能</li>
        </ul></li>
    </ul></li>
</ul>


<br />
<ul>
    <li class="ulist1">沢谷～石見都賀間は、上記の方法では取得不可能だが、「ゴールデンユートピアおおち」まで来れば、この間がレーダー12駅範囲に入るので、一考の価値あり。<br />
<small>ここまで来るなら、結局は潮駅まで訪れて潮温泉に入ったり、いっそ道の駅グリーンロード大和まで行ってしまうのもありかも。</small></li>
</ul>


<br />

<h2 id="wd2b8ae6">廃止直前の状況</h2>

<ul>
    <li class="ulist1">種別が普通しか存在せず、しかも駅数が多く、所要時間も長い。全線乗り通すには、最速でも<span style="font-weight: bold;">3時間28分</span>を要する。<br />
本数も1日に僅か4～5往復（その内全線通し運転は下り2本・上り1本）と、極端に少ない。<br />
路線コンプを狙う場合は緻密な計画、及び相当な根気が必要となる。</li>
</ul>


<br />
<ul>
    <li class="ulist1">途中に山の陰で電波が駅に停車中全く入らなくなる駅が2つある。レーダーかルートビューンを少しは持っていこう。<ul>
        <li class="ulist2">(補足)2016年8月乗車時、docomoでは圏外区間があったもののアイテムなしですべての駅が取れました。</li>
        <li class="ulist2">(補足)2017年11月乗車時、docomo圏外区間は粟屋～式敷でした。駅付近では安定します。</li>
    </ul></li>
</ul>


<br />
<ul>
    <li class="ulist1">夏場の暑い日中は、<span style="font-weight: bold;">線路保護</span>のために<span style="color: red;">運休</span>になることがしばしばある。青春18きっぷ等で路線コンプ計画を立てている場合は注意しよう。</li>
</ul>


<br />
<ul>
    <li class="ulist1">廃止が近く、三次10:02発の列車はツアー団体が乗車することがある。その場合の混雑は相当なものとなるので注意。</li>
</ul>





<br />
<ul>
    <li class="ulist1">廃止直前となる2018年3月17日のダイヤ改正で、全線通しの列車を1往復増やし途中乗り換え含め乗り通せる列車が4往復走ることとなる。(<a title="https://www.westjr.co.jp/press/article/items/1171215_00_yonago_1.pdf" href="/jump/https/www.westjr.co.jp%2Fpress%2Farticle%2Fitems%2F1171215_00_yonago_1.pdf" class="jump">出典</a>)。</li>
</ul>
