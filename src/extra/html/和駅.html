[[石狩沼田駅]]間が[[廃線|廃止]]になるとともに廃駅となった。 == 歴史 == * [[1931年]]（昭和6年）[[10月10日]] - [[札沼線|札沼北線]][[石狩沼田駅|石狩沼田]] - 中徳富（初代、現在の[[新十津川駅|新十津川]]）間開通にともない開業。[[一般駅]]。 * [[1944年]]（昭和19年）[[7月21日]] - [[太平洋戦争]]激化により[[不要不急線]]として[[石狩追分駅|石狩追分]] - 石狩沼田間休止に伴い当駅も休止。 * [[1956年]]（昭和31年）[[11月16日]] - [[雨竜駅|雨竜]] - 石狩沼田間復活により当駅が復活。札沼線が全線復旧。 * [[1972年]]（昭和47年）[[6月19日]] - 札沼線新十津川 - 石狩沼田間廃止に伴い廃駅となる。 === 駅名の由来 === もともと同地はアイヌ語でエタイエベツ（引き出す川）と称されていたが<ref name=":2">{{Cite book|和書|author=|editor=札幌鉄道局|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|page=34|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref>、[[吉植庄一郎]]ら、[[千葉県]][[埜原村]]（やわら：現在の[[印西市]]の一部）出身者が入植し、出身地の「埜原」から音を借り名づけた<ref name=":2" /><ref name=":0">{{Cite book|author=本多 貢|editor=児玉　芳明|title=北海道地名漢字解|url=https://www.worldcat.org/oclc/40491505|date=1995-01-25|year=|accessdate=2018-10-16|publisher=[[北海道新聞社]]|language=日本語|isbn=4893637606|page=114|location=札幌市|last2=|author2=|first2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|oclc=40491505}}</ref>。 「和」の字を用いた理由としては、永久に平和なる村を建設する意味とする説や<ref name=":2" />、住民の一致団結を願ったとする説<ref name=":0" />がある。 == 駅構造 == 駅舎は桑園駅に向かって右手の西側に設置され、駅舎前に単式ホーム1面1線と副本線を有し、副本線から札幌方に車庫線が分岐していた。また、駅舎横石狩沼田側の貨物ホームへ1本の引込線を有していた<ref>昭和7年 線路一覧略図 札幌鉄道局発行。</ref>。 札沼線の部分廃止直前は[[浦臼駅]]と石狩沼田駅の間は1日5往復まで列車本数が削減されていたが、当駅ではこの区間で唯一[[列車交換]]が行われていた。 == 廃駅跡 == 縫製工場が建っている。旧駅舎の建物の一部がこの縫製工場に取り込まれている。 == 脚注 == {{Reflist}} == 隣の駅 == ; 日本国有鉄道 : 札沼線 :: [[渭ノ津駅]] -
