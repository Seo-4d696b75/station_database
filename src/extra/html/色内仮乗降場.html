'''色内仮乗降場'''（いろないかりじょうこうじょう）は、かつて[[北海道]][[小樽市]]色内町にあった[[日本国有鉄道]]（国鉄）[[手宮線]]の[[仮乗降場]]（[[廃駅]]）である。 == 歴史 == * [[1912年]]（[[大正]]元年）[[8月6日]]：[[鉄道省|国有鉄道]][[手宮線]]の'''色内仮停車場'''（いろないかりていしゃじょう）として開設。旅客のみ取扱い。必要な時に限り営業する後にいう[[臨時駅]]であった<ref>[{{NDLDC|2952099/2}} 官報 1912年8月6日 鉄道院告示第2号]。</ref>。 * [[1914年]]（大正3年）[[12月1日]]：色内仮停車場閉鎖（[[休止駅|休止]]）<ref name="chizu35">{{Cite book|和書|author=今尾恵介（監修） |title=[[日本鉄道旅行地図帳]] |publisher=[[新潮社]] |volume=1 北海道 |year=2008 |isbn=978-4-10-790019-7 |page=35}}</ref>。 * [[1920年]]（大正9年）[[6月1日]]：色内仮停車場再開。旅客のみ取扱い<ref name="chizu35"/>。 * [[1943年]]（[[昭和]]18年） ** [[5月1日]]：'''色内駅'''（いろないえき）となる<ref name="chizu35"/><ref>[{{NDLDC|2961390/8}} 官報 1943年4月28日 鉄道省告示第97号]。</ref>。 ** [[10月1日]]：色内駅休止<ref name="chizu35"/>。 * [[1948年]]（昭和23年）[[11月10日]]：色内駅廃止。 * [[1949年]]（昭和24年） ** [[6月1日]]：日本国有鉄道法施行に伴い、手宮線が[[日本国有鉄道]]（国鉄）に継承。 ** [[9月1日]]：'''色内仮乗降場'''（[[鉄道管理局|局]]設定）として開設。旅客のみ取扱い<ref name="chizu35"/>。 * [[1962年]]（昭和37年）[[5月15日]]：手宮線の旅客営業廃止に伴い、色内仮乗降場廃止<ref name="chizu35"/>。 == 駅名の由来 == 当地域の地区名から。「色内」は[[アイヌ語]]のイルエ=ナイ（澄んでいない川）に当て字をしたもの。イルウエ=ナイ（熊道の沢）の説もある<ref>「駅名の起源」 札幌鉄道局 昭和14年発行。</ref>。 == 利用状況 == [[1958年]]（昭和33年）度　乗客 79,179、　降客 102,948 <ref name="shisei34">『小樽市勢概要』 左文字書店 1960年（昭和35年）発行</ref> == 現状 == 仮乗降場跡は、隣接する市立[[小樽文学館]]・[[小樽美術館]]の再整備事業の一環として、[[2011年]]（[[平成]]23年）に当時の駅舎が休憩所にアレンジされた上で[[復元]]されている。 == 隣の駅 == ; 日本国有鉄道 : 手宮線 :: [[南小樽駅]] - '''色内仮乗降場''' - [[手宮駅]] == 備考 == * 当駅は[[太平洋戦争]]後に仮乗降場であったと記録されている。国鉄の規則上、仮乗降場や[[臨時駅]]には[[入場券]]は存在しないが、[[昭和30年代]]の同駅には入場券が存在し、券面に臨時駅を示す（臨）の文字が印刷されている。なぜ規則にない入場券が発売されていたのか、なぜ（臨）の文字が印刷されていたのかは定かではない。 * [[1910年]]（明治43年）頃から1943年（昭和18年）までの間、手宮線は[[複線]]であった。[[1928年]]（昭和3年）の[[鉄道管理局|札幌鉄道局]]発行の線路一覧略図には、[[踏切]]を間に挟んで互いが進行方向に渡った位置に[[プラットホーム#千鳥式ホーム|千鳥状にずれた相対するホーム2面]]が描かれている。駅舎は設置されていない。[[1932年]]（昭和7年）の線路一覧略図になると、手宮方面のホームは消えて南小樽方面のホームだけとなり、駅舎も設けられている。<!--これは昭和6年7月に開催された小樽海港博覧会に合わせて整備されたものと考えられるが、詳細不明。-->なお、[{{NDLDC|976218/7}} 1925年（大正14年）]や[{{NDLDC|1450124/28}} 1937年（昭和12年）]等の小樽市内地図（いずれも国立国会図書館近代デジタルライブラリー）によれば、駅の位置は[[日本銀行]]小樽支店（現・[[日本銀行旧小樽支店金融資料館|金融資料館]]）の横（小樽美術館からは日銀通りを挟んだ斜向かい）。 * 1943年（昭和18年）以降手宮線は[[単線]]区間となる。[[国土地理院]] 地図・空中写真閲覧サービスの1948年4月、廃駅となる半年前の航空写真<ref>米軍撮影 [http://mapps.gsi.go.jp/contentsImage.do?specificationId=233410&dispType=1 USA-R246-154]。</ref>では、日銀通りの踏切手宮側（小樽美術館裏側）すぐ横の山手側に約65 - 70 mのホームと、その手宮側、市街区画の長手方向中央位置に道路に正対して（線路やホームに対して斜めに）駅舎が設けられているのが確認できる。一方、同サービスの[[1961年]]（昭和36年）5月、仮乗降場廃止の1年前の航空写真<ref>国土地理院撮影 [http://mapps.gsi.go.jp/contentsImage.do?specificationId=366845&dispType=1 MHO613C-C1-7981]。</ref>では、駅舎の長さが短縮されて位置も日銀通り踏切側に移動し、線路に正対して（道路に対して斜めに）設置されている。ホームは市街区画長手方向全体の約130 mに拡張され、駅舎を挟んでホーム上に長い上屋の[[待合所]]が2つあり、駅時代より仮乗降場時代のほうが整備されている様に見える。なお、2011年（平成23年）に休憩施設として復元された駅舎の位置はこの仮乗降場時代の位置とほぼ同じだが、ホームに対して正対していない点が異なる。 == 脚注 == {{Reflist}} == 関連項目 == * [[日本の鉄道駅一覧]] * [[廃駅]]
