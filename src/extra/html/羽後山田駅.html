'''雄勝鉄道'''湯沢駅 - 西馬音内駅間開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、2008年6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、2010年4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年8月発行）30,32-33,36ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月16日]] - [[陸上交通事業調整法|交通統合]]に伴い'''横荘鉄道'''の駅となる<ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1944年]]（昭和19年）[[6月1日]] - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''雄勝線'''に制定。それに伴い'''羽後鉄道雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">『私鉄の廃線跡を歩くI 北海道・東北編』106-109ページより。</ref>。 * [[1951年]]（昭和26年）[[7月18日]] - 信号機増設及び第二種連動装置設置を行い交換駅となる<ref name="bunkenKW03"/><ref name="bunkenRML052">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、ネコ・パブリッシング、2003年11月発行）4,14-15,19-21,23,27,31,34,38,46ページより。</ref>（日付は認可日<ref name="bunkenRML052"/>）。 * [[1947年]]（昭和22年） ** 2月 - 変電機械の故障により送電停止、横荘線から[[蒸気機関車]]を回送の上使用された際の、給水の基地となる<ref name="bunkenRML052-2">『RM LIBRARY 52』17-18ページより。</ref>。 ** 3月頃 - 変電機械が復旧し、蒸気機関車を返却する<ref name="bunkenRML052-2"/>。 ** [[7月23日]] - 豪雨による路盤及び橋脚損壊により雄勝線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 ** [[8月14日]] - 当駅 - 梺駅間が復旧、当駅も一時的な始発駅として営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 ** [[12月3日]] - 湯沢駅 - 当駅間が復旧、全線営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 * [[1952年]]（昭和27年）[[2月15日]] - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1971年]]（昭和40年）[[12月9日]] - 交換設備廃止<ref name="bunkenKW03"/><ref name="bunken2008SH-1-R"/>、同時に無人化<ref name="bunken2008SH-1-R"/>。 * [[1973年]]（昭和48年）[[4月1日]] - 雄勝線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。晩年に交換設備が廃止になるまでは島式ホーム1面2線を有する[[列車交換]]可能な交換駅であった<ref name="bunkenRML052"/>。外側（西側）が下り線（梺方面）、駅舎側（東側）が上り線（湯沢方面）となっていた<ref name="bunkenRML052"/>。そのほか[[停車場#側線|側線]]として、上り線から東に分岐し駅舎との間を通過し、駅舎東側の貨物積卸場への[[貨物線]]を1線有していた<ref name="bunkenRML052"/>。閉塞取扱廃止後の側線の状況は不明である。 [[無人駅]]となっていたが、[[日本の鉄道駅#直営駅|職員配置駅]]時代の駅舎が残っていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の北東側に位置し、ホーム北側の[[斜路|スロープ]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML052"/>。 閉塞取扱廃止前までの列車交換の通標は、湯沢駅 - 当駅間は「○」、当駅 - 西馬音内駅間は「□」であった<ref name="bunkenRML052"/>。 貨物側線からの積載品は[[米|米穀]]であったが、戦時中までは[[DOWAホールディングス|藤田組]]松岡鉱山の鉱石輸送も行われていた<ref name="bunken2008SH-1"/>。その関係で構内は広く取られていた<ref name="bunken2008SH-1"/>。貨物積卸場には農業倉庫も設置されていた<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、1999年12月発行）64ページより。</ref>。 1947年（昭和22年）2月に起こった[[変電設備|変電機械]]の故障による[[送電]]停止の際は、横荘線から[[蒸気機関車]]を回送の上使用されたが、車輌基地のある西馬音内駅は地盤が[[洪積層]]で水涸れという現象も起きていた場所にあり、給水の便も悪かったために、当駅が給水の基地となった<ref name="bunkenRML052"/>。但し[[電気鉄道]]である故に[[給水設備]]は設置されておらず、[[消防団]]から手牽きの[[日本の消防#歴史|ガソリンポンプ]]を借用したという<ref name="bunkenRML052"/>。 === 雄物川橋梁 === 湯沢駅 - 当駅間には雄物川が流れており、「雄物川橋梁」<ref name="bunkenRML052-3">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、[[ネコ・パブリッシング]]、2003年11月発行）8-10ページより。</ref>が架けられていた。この鉄橋は地盤の関係で多くの橋梁に用いられる杭打工法ではなく[[ケーソン|オープンケーソン工法]]を用いられていた<ref name="bunkenRML052-3"/>。また、建設費節約のために中古の[[桁橋|デッキガーダー]]を使用していた<ref name="bunkenRML052-3"/>。さらに工期の関係で補強を行わず「仮設」として届出を行っており<ref name="bunkenRML052-3"/>、最終的には「仮設」の状況を「設計変更」として変更せずに完成とした<ref name="bunkenRML052-3"/>。 == 駅周辺 == 当駅は深堀地区の北東側に位置し、南西側に集落がある。東側には[[雄物川]]が流れており、対岸に湯沢市の市街地が広がる。周囲は穀倉地帯であった<ref name="bunkenKW03"/>。 * [[秋田県道311号羽後雄勝線]] * 文月橋 * [[こまち農業協同組合|JAこまちライスセンター]] * 医療法人仁恵会佐藤病院 * 松岡鉱山 - 銀山<ref name="bunkenRML052"/>。休山となる1923年（大正12年）までは松岡[[鉱山鉄道|鉱山軌道]]が敷設されていた<ref name="bunkenRML052"/>。この軌道は鉱石以外に農産物の搬出も行っていた<ref name="bunkenRML052"/>。 == 駅跡 == 路線廃止後、駅跡は[[1980年]]（昭和55年）[[4月1日]]に開園した社会福祉法人湯沢保育会「深堀保育園」の園庭になっている。1999年（平成11年）時点では保育園の前後に橋台が残存していた<ref name="bunkenTouhoku1999"/>。2007年（平成19年）5月時点<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）時点でも同様で<ref name="bunken2010H-1">『新 鉄道廃線跡を歩く1』199ページより。</ref>、この橋台はコンクリート製で2線分のものであった<ref name="bunken2010H-1"/>。 また、雄物川橋梁跡から当駅跡附近の線路跡は[[区画整理]]による水田となっており、1995年（平成7年）時点で線路跡をたどることはできなくなっていた<ref name="bunken1995H">書籍『鉄道廃線跡を歩く』（JTBパブリッシング、1995年11月発行）36-37ページより。</ref>。1999年（平成11年）時点<ref name="bunkenTouhoku1999"/>、2007年（平成19年）5月時点<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1"/>。 そのほか雄物川橋梁は、1995年（平成7年）時点では湯沢方、当駅方双方の橋台が残存していた<ref name="bunken1995H"/>。1999年（平成11年）時点でも同様で、湯沢方は高さ約2m、当駅方は約5m近い立派なものであった<ref name="bunkenTouhoku1999"/>。また堤防上の道路を挟んで小さな橋台と[[溝渠#暗渠|暗渠]]も残存していた<ref name="bunkenTouhoku1999"/>。2007年（平成19年）5月時点では大きかった当駅方の橋台は撤去され、湯沢方のみ残存していた<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1"/>。<!-- 私設ページによりコメントアウト <ref>[http://www.tetsuhai.com/ogachi.html 羽後交通　雄勝線] -
