当駅間開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として開業。 * [[1938年]]（昭和13年）[[10月21日]]：当駅 - 碁盤坂駅（後の[[千軒駅]]）間開通に伴い、中間駅となる。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：福山線が'''[[松前線]]'''に改称。 * [[1982年]]（昭和57年）[[11月15日]]：貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]]：[[チッキ|荷物]]取扱い廃止。 * [[1986年]]（昭和61年）[[11月1日]]：[[日本の鉄道駅#簡易委託駅|簡易委託]]化され、無人駅化。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]に伴い、[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）2月1日：松前線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]片面使用の1面1線を有する[[地上駅]]であった。ホームは、線路の東側（松前方面に向かって左手側）に存在した。駅舎側の1線は撤去されていたが、かつては島式ホーム1面2線を有する[[列車交換]]可能な[[日本の鉄道駅#直営駅|職員配置駅]]であった。1983年（昭和58年）時点では、使われなくなった駅舎側の1線は[[交換駅|交換設備]]運用廃止後も両側の[[分岐器|転轍機]]が撤去された状態で[[停車場#側線|側線]]として残っていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）61ページより。</ref>。そのほか両側の転轍機が撤去された、駅舎隣の短い貨物ホームに接した[[貨物列車|貨物]]側線を1本有していた。 [[無人駅]]（[[日本の鉄道駅#簡易委託駅|簡易委託駅]]）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎はホームの東側に位置し、[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。 == 駅名の由来 == {{See also|知内町#町名の由来}} 町名に[[旧国名#駅名|旧国名]]の「[[渡島国|渡島]]」を冠する。旧国名を冠したのは開業時、[[東北本線]]に同音の尻内駅（現在の[[八戸駅]]）が存在したためである<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=24 }}</ref>。 == 利用状況 == * 1981年度の1日乗降客数は281人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道531号小谷石渡島知内停車場線]]・[[北海道道698号湯の里渡島知内停車場線]] * [[国道228号]] * 知内町役場 * [[木古内警察署]]知内駐在所 * 知内郵便局 * 知内駅前簡易郵便局 * [[道南うみ街信用金庫]]知内支店 * 新函館農業協同組合（JA新はこだて）知内支店 * [[北海道知内高等学校]] * 知内町立知内中学校 * 涌元・小谷石海岸 -
