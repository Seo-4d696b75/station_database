kitamiAioi.JPG|thumb|駅名標（2012年7月）]] '''北見相生駅'''（きたみあいおいえき）は、[[北海道]][[網走郡]][[津別町]]相生にあった[[日本国有鉄道]]（国鉄）[[相生線]]の[[鉄道駅|駅]]（[[廃駅]]）で、同線の[[終着駅]]である。[[電報略号 (鉄道)|電報略号]]は'''キア'''。相生線の[[廃線]]に伴い[[1985年]]（昭和60年）[[4月1日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]で、[[終着駅|終端駅]]となっていた。[[日本の鉄道駅#直営駅|職員配置駅]]であった。ほかに[[機回し線]]などの[[停車場#側線|側線]]を数本有していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）159ページより。</ref>。ホームは線路の東側（北見相生方面に向かって左手側）に存在し、平屋建ての木造駅舎<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]]10月発行）12ページより。</ref>が接していた。駅舎の南側部分には側線が1本入り、[[切欠きホーム]]となっていた。かつては[[給水設備#船舶・車両の給水設備|給水タンク]]や[[転車台|ターンテーブル]]も存在したが、路線廃止前に撤去されている（ターンテーブルは遺構あり<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）37-39ページより。</ref>）。ホームには[[高山植物]]の[[コマクサ]]が植えられていた<ref name="bunken1980"/>。 == 駅名の由来 == 当駅の所在する地名に、既設の他駅との混同を防止するため[[旧国名#駅名|旧国名]]の「北見」を付する<ref name=":02">{{Cite book|和書|author=|editor=札幌鉄道局編|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|page=105|pages=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref>。 『駅名の起源』（1936年版）では、「相生」の由来について、この地を流れるヌプパマナイ川にはかつて「女」「男」の2つの橋が架かっていたが、道路開墾に当たり、その中間に「相生橋」という木橋が架されたことに由来する、としている<ref name=":02" />。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は60人<ref name="bunken1983"/>。 == 駅周辺 == 周辺は山のふもとに開けた小さな集落である。 * 北見相生郵便局 * 津別町営バス車庫 * 津別町営牧場 * さけ・ます孵化場 * [[国道240号]] * [[北海道道768号相生津別停車場線]] * [[網走川]] == 歴史 == * [[1925年]]（大正14年）[[11月15日]] - [[鉄道省]]相生線津別駅 - 当駅間開通に伴い開業。[[一般駅]]。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1979年]]（昭和54年）[[12月11日]] - 貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。 * [[1985年]]（昭和60年）[[4月1日]] - 相生線の廃線に伴い廃止となる。 == 駅跡 ==
