'''[[北海道鉄道 (2代)|北海道鉱業鉄道]]金山線'''生鼈駅（後の旭岡駅） - 辺富内駅（後の富内駅）間延伸開通に伴い'''杵臼駅'''（きなうすえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1924年]]（大正13年）[[3月3日]] - 鉄道会社名を'''北海道鉄道'''（2代目）に改称、それに伴い同鉄道の駅となる。 * [[1943年]]（昭和18年）[[8月1日]] - 北海道鉄道が[[戦時買収私鉄|戦時買収]]により国有化。線路名を'''富内線'''に改称、それに伴い同線の駅となる。同時に'''豊田駅'''に改称。 * [[1960年]]（昭和35年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1977年]]（昭和52年）[[2月1日]] - 貨物・[[チッキ|荷物]]取扱い廃止。 * 時期不詳<ref group="注" name="ekigyoumu">1981年（昭和56年）4月時点では既に無人駅となっている（書籍『コロタン文庫36 国鉄駅名全百科 56.4訂補版』（[[小学館]]、[[1981年]][[10月]]発行）54ページより）。</ref> -無人化。 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。ホームは線路の北西側（日高町方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）107ページより。</ref>。かつては島式ホーム1面2線を有する[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎側の1線は、交換設備運用廃止後も日高町方の[[分岐器|転轍機]]及びホーム近くまでの線路が[[停車場#側線|側線]]として残っていた（但し1983年（昭和58年）時点では転轍機の先の部分に[[車止め]]が設置されていた<ref name="bunken1983"/>）。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の北西側に位置した<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名より。 旧駅名の「杵臼」は、[[アイヌ語]]の「キナウㇱ」（草が・多い）に由来する<ref>{{Cite book|author=本多 貢|editor=児玉　芳明|title=北海道地名漢字解|url=https://www.worldcat.org/oclc/40491505|date=1995-01-25|year=|accessdate=2018-11-14|publisher=[[北海道新聞社]]|language=日本語|isbn=4893637606|page=41|location=札幌市|last2=|author2=|first2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|oclc=40491505}}</ref><ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は8人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道74号穂別鵡川線]] * 胆振豊田郵便局 * [[鵡川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）11ページより。</ref> * キナウス川<ref name="bunken1980M"/> == 駅跡 == 1999年（平成11年）時点では[[消防団]]の建物が建築されていた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）65ページより。</ref>。2011年（平成23年）時点でも同様であった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）84-86ページより。</ref>。 また、1999年（平成11年）時点では駅跡の穂別方を流れるキナウス川にガーダー橋が残存し農道に転用されていた<ref name="bunken2000H"/>。2011年（平成23年）時点でも同様で、そのほか栄方に線路跡の築堤や[[トンネル]]の坑口、[[擁壁]]、[[落石 (自然災害)|落石]]防止用フェンス、[[鉄道標識|標識]]などが残存していた<ref name="bunken2011H"/>。 == 隣の駅 == ; 日本国有鉄道 : 富内線 :: [[栄駅 (北海道)|栄駅]] -
