寺泊駅間[[鉄道の電化|電化]]。 * [[1960年]]（昭和35年）[[10月1日]] [[越後交通]]が発足し、同社の長岡線の駅となる。 * [[1961年]]（昭和36年） ** 8月5日 水害により、当駅 - 寺泊間が不通となる。 ** 10月3日 当駅 - 寺泊間が休止。 * 1966年（昭和41年）5月31日 当駅 - 寺泊間が廃止。それに伴い当駅が'''寺泊駅'''に改称される。 * 1973年（昭和48年）4月16日 大河津 - 当駅間が廃止。 == 駅跡地 == 跡地は国道に整備されて痕跡は残ってはいないが[http://www2s.biglobe.ne.jp/~taka55/tp/wch/nagaokatetudou/cyoutetu-L5.htm リンク先]の写真を参照すると三叉路の山側路線あたりが駅舎跡と推測される。 == 隣の駅 == ; 越後交通 : 長岡線 :: [[法崎駅]] - '''寺泊新道駅''' -
