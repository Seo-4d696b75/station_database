[[日本国有鉄道]]高千穂線の駅として開業{{R|停車場}}。旅客営業のみ{{R|停車場}}。 *[[1981年]]（昭和56年）[[4月14日]] - 駅業務委託先が[[ジェイアールバステック|日本交通観光社]]から[[JR九州サービスサポート|九鉄開発]]大分支社に変更<ref>{{Cite news |title=委託業務を一本化 |newspaper=[[交通新聞]] |publisher=交通協力会 |date=1981-04-26 |page=1 }}</ref>。 *[[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]扱い廃止{{R|停車場}}。 *[[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により九州旅客鉄道（JR九州）に継承{{R|停車場}}。 *[[1989年]]（平成元年）[[4月28日]] - 第三セクター転換により高千穂鉄道の駅となる<ref name="RJ365"/>{{R|停車場}}。 *[[2005年]]（平成17年）[[9月6日]] - [[平成17年台風第14号|台風14号]]による被害のため、高千穂鉄道全線で運転休止。 *[[2008年]]（平成20年）[[12月28日]] - 槇峰駅 - 当駅間（全線）の廃止に伴い廃駅。 == 駅構造 == 島式1面2線[[プラットホーム|ホーム]]の地上駅で、[[延岡駅]]以外では唯一の有人駅<ref name="RJ365">{{Cite journal|和書 |date = 1997-03 |title = ローカル線の実態と問題を現地に見る61 高千穂鉄道 |author = [[鈴木文彦]] |journal = [[鉄道ジャーナル]] |volume = 31 |issue = 3 |pages = 76-83 |publisher = [[鉄道ジャーナル社]] }}</ref>。駅の先端には車止めの代わりに[[車両基地]]がある<ref name="RJ365"/>。この部分は開業当時線路こそ敷かれていなかったものの、[[高森駅|高森]]方面への延伸に備えてすでに切り通しとなっていた。 [[日本国有鉄道|国鉄]]時代は[[夜間滞泊|列車の滞泊]]がなく、当駅に夜間到着した列車は[[日之影温泉駅|日ノ影駅]]まで折り返していたが、高千穂鉄道になって当駅構内に車庫が設置され、滞泊が行われるようになった。 == 駅周辺 == {{要出典範囲|駅前は高千穂町の市街地になっている。[[高千穂峡]]や神社の観光名所であって、[[旅館]]や[[ホテル]]など[[宿泊施設]]が林立している。|date=2015年3月}} *[[国道218号]]・[[国道325号]]（終点） *[[高千穂警察署]] *高千穂郵便局 *[[宮崎県立高千穂高等学校]]<ref name="RJ365"/> *第一高千穂幼稚園 *高千穂町立高千穂小学校 *[[道の駅高千穂]] *高千穂町役場 *高千穂森林管理所 *高千穂法務総合庁舎 == バス == * [[宮崎交通]] ** 高千穂バスセンター行 ** [[延岡駅前バスセンター]]行 ** 南延岡（南営業所前）行 ==利用状況== 1日平均乗車人員は220人であった（2003年度）<!---宮崎県統計年鑑---> ==現況== [[2008年]][[10月20日]]時点<ref>鉄道ジャーナル[[2008年]]12月号</ref>では、駅構内には気動車とトロッコ車両が留置されていた。気動車は製造から20年近く経過し、しかも屋外に雨ざらし状態で放置されているため、傷みが激しくなっていた。 その後、トロッコ車両は[[九州旅客鉄道]]（JR九州）に譲渡され、[[JR九州キハ125形気動車#400番台（特急仕様車）|キハ125形400番台]]に改造後[[2009年]]10月より同じ宮崎県内の[[日南線]]で「[[海幸山幸 (列車)|海幸山幸]]」として再起した。また、気動車のうち1両は四国の[[阿佐海岸鉄道]]に譲渡されている。 [[2010年]]現在、旧駅舎には[[高千穂あまてらす鉄道]]の事務所が置かれ、土日祝日等に同社により旧駅構内が開放されている。同年8月から、スーパーカートによる当駅 -
