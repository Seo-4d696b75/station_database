[[日本国有鉄道]]（国鉄）名寄本線の'''班渓仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年） **[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。同時に[[臨時駅]]に昇格<ref name="bunken2010H-2">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）214ページより。</ref>。'''班渓駅'''となる。 ** [[12月1日]] - 常設駅化<ref name="bunken2010H-2"/>。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == {{節スタブ}} 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。[[仮乗降場]]に出自を持つ[[無人駅]]となっていた。 == 駅名の由来 == 当駅が所在していた地名、「班渓」より。地名は、[[アイヌ語]]の「パンケニウプ」（下流に木のある川）に由来するとされている<ref name="bunken2004N">『北海道の駅878ものがたり』186ページより。</ref>。当駅近くを興部川支流の1つ、パンケ沢（班渓川）が流れる。 == 駅周辺 == * [[国道239号]]（天北国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）18ページより。</ref> * 興部川<ref name="bunken1980M"/> * 班渓川<ref name="bunken1980M"/> * [[名士バス]]「班渓入口」停留所 == 駅跡 == 2011年（平成23年）時点では、当駅の遺構は何も残されていない<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）118ページより。</ref>。駅跡附近は、草木で埋め尽くされている。 また、2011年（平成23年）時点では当駅跡の名寄方に、興部川の支流班渓川に架かっていた「班渓川橋梁」の鉄骨[[桁橋|ガーダー橋]]が残存しており、国道の左側に確認できた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[中興部駅]] -
