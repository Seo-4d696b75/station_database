[[日本国有鉄道]]（国鉄）名寄本線の'''富丘仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、国鉄からJR北海道に継承。同時に駅に昇格し、'''富丘駅'''となる。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、構内の南西側（遠軽方面に向かって右側）に存在した<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=118}}</ref>。 [[仮乗降場]]に出自を持つ[[無人駅]]となっており、駅舎は無かったがホームから少し離れた位置に[[待合所]]を有していた<ref name="bunken2011P"/>。ホームは木造で名寄方に90度の角度で設置された[[斜路|スロープ]]を有し<ref name="bunken2011P"/>、駅施設外に連絡していた。 駅全体が周囲より高台の海岸段丘上に位置し、道路へ出るには防雪覆い付きの階段を下る形で連絡した<ref name="bunken2001H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |page=34}}</ref>。 == 駅名の由来 == 当駅が所在した地名より。元はアイヌ語のオ=シカリ=サロル（川尻が迂回する湿地）またはオ=ム=サロル（川尻が塞がる湿地）から「思沙流」と呼ばれた段丘地帯であったが、1951年（昭和26年）の字名地番改正の際にこの地区の開拓先駆者で自治功労者の名前から「富」の一字を採り、それに「丘」を付けて「富丘」とした<ref>{{Cite book|和書|editor=興部町史編纂委員会 |title=興部町百年史 |date=1993-3 |publisher=興部町 |isbn= |page=P.98,P.256}}</ref>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref> * 思沙留川<ref name="bunken1980M"/> * [[北紋バス]]「富丘」停留所 == 駅跡 == [[2001年]]（平成13年）時点では、道路から駅構内への防雪覆い付きの階段のみが残存していた<ref name="bunken2001H"/>。駅跡前後の線路跡は、農道として再利用されている模様であった<ref name="bunken2001H"/>。[[2011年]]（平成23年）時点でも同様に防雪覆い付きの階段は残存しており、ホーム跡には枕木が外された形で放置され、散乱していた<ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |pages=120-122}}</ref>。線路跡も確認できた<ref name="bunken2011H"/>。築堤上の空き地になっている。 また、旧駅附近の国道に立てられている「農業基盤整備事業」の看板には、当駅跡附近の名寄本線の絵が描かれていた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[沙留駅]] -
