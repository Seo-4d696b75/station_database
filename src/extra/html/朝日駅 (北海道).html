同町滝ノ上から[[志文駅]]まで[[三井物産]]木材部が万字線とほぼ同ルートに[[馬車鉄道|馬鉄]]を敷設。 * [[1914年]]（大正3年） - '''[[日本国有鉄道|国有鉄道]]万字軽便線'''志文駅 - 万字炭山駅間開通<ref name="kanpo1">[https://dl.ndl.go.jp/info:ndljp/pid/2952785/4 『官報』 1914年11月04日　鉄道院告示第102号]（国立国会図書館デジタルコレクション）</ref>。三井物産馬車鉄道撤去。 * [[1919年]]（大正8年） ** 2月 - 北海道採炭会社幌内炭礦（後の朝日炭砿）開坑 ** [[11月11日]] - 当駅開業<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2954290/2 『官報』 1919年11月06日　鉄道院告示第106号]（国立国会図書館デジタルコレクション）</ref>。一般駅。 * [[1922年]]（大正11年）9月2日 - 線路名を'''万字線'''に改称、同線の駅となる。 * [[1970年]]（昭和45年）8月17日 - 専用線発着車貨物以外の貨物取扱い廃止。 * [[1974年]]（昭和49年） - 朝日炭砿が閉山。 * [[1975年]]（昭和50年）7月11日 - 貨物取扱い廃止。 * [[1978年]]（昭和53年）4月6日 - [[チッキ|荷物]]取扱い廃止。[[無人駅|無人]]（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1985年]]（昭和60年）[[4月1日]] - 万字線の廃線に伴い廃止となる。 * [[1999年]]（平成11年） - [[国鉄B20形蒸気機関車|B20形蒸気機関車]]が[[東山公園 (岩見沢市)|東山公園]]から移設、駅舎裏に展示される。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の南側（万字炭山方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）70ページより。</ref>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。 [[無人駅]]（[[日本の鉄道駅#簡易委託駅|簡易委託駅]]）となっていたが、黒く塗装された<ref name="bunken1983" />[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っており[[乗車券]]が駅舎内で発券されていた。駅舎は構内の南側に位置しホーム西側に接していた<ref name="bunken1983" />。 == 駅名の由来 == 当駅の所在する地名より。当駅付近に旭沢が流れており、この方角から朝日が昇るので当地を「朝日」と名付けたとされる。他に朝日のように炭砿が興隆してほしいという願望から付けられたという説もある<ref>岩見澤市史 昭和38年8月発行。</ref>。 == 利用状況 == * [[1981年]]度（昭和56年度）の1日乗降客数は10人<ref name="bunken1983" />。 == 駅周辺 == 朝日町の市街地が広がっている。駅前を通る道はかつて万字線と並行して走る[[北海道道38号夕張岩見沢線]]であったが、駅付近で新道に付け替えられ、現在の駅跡は旧道に面している形となる。 * [[岩見沢市役所]]朝日サービスセンター * 岩見沢朝日簡易郵便局 * 栗沢温泉 - 駅から南西に約2km<ref name="bunken1983" />。 * [[北海道中央バス]]「旧朝日駅前」停留所 == 駅跡 == 駅舎は現役時代の建物が残存し、館内には廃線当時のポスターが貼られた状態で保管されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、2011年9月発行）27ページより。</ref>。また、上志文駅と同じ形状の[[花崗岩]]の[[石碑]]が建立されている。 旧構内は元々廃止当時の構内施設がそのまま残る形で保存されていたが、B20 1号機移設保存の時にホームの一部が撤去されている。 ===万字線鉄道公園=== {{公園 |名称 = 万字線鉄道公園 |英語名称 = |画像 = [[ファイル:JNR_B20type_No1_Steam_Locomotive.jpg|300px]] |画像キャプション = B20形1号機 (2006年7月撮影) |国 = {{JPN}} |都市 = 北海道岩見沢市朝日町176 | 緯度度 = |緯度分 = |緯度秒 = | 経度度 = |経度分 = |経度秒 = |分類 = 公園 |面積 = |前身 = |開園 = |運営者 = 岩見沢市 |年来園者数 = |現況 = 通年開放 |設備・遊具 = |駐車場 = あり（無料約20台） |バリアフリー = |建築家と技術者 = |告示 = |事務所 = |事務所所在地 = |アクセス = |備考 = |公式サイト = }} 旧駅構内に<ref>[http://www.sorachi.pref.hokkaido.jp/so-tssak/html/parts/06asahi_station.html 万字線鉄道公園（旧朝日駅）] -
