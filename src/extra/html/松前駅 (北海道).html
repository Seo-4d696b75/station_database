(北海道)|松前町]]字博多にあった[[北海道旅客鉄道]]（JR北海道）[[松前線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''マエ'''。松前線の[[廃線]]に伴い[[1988年]]（昭和63年）[[2月1日]]に廃駅となった。 [[1980年]]（昭和55年）まで運行されていた急行「[[松前線#年表|松前]]」の発着駅だった。 == 歴史 == * [[1953年]]（[[昭和]]28年）[[11月8日]]：[[日本国有鉄道]]（国鉄）'''福山線'''の[[渡島大沢駅]] - 当駅間延伸に伴い、開業<ref name="bunken1993-R">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）183ページより。</ref><ref name="bunken2008M">書籍『日本鉄道旅行地図帳 全線全駅全廃線 1 北海道』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]5月発行）27ページより。</ref>。同時に、福山線が'''松前線'''に改称し、同線の駅となる<ref name="bunken1993-R"/><ref name="bunken2008M"/>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1963年]]（昭和38年）[[12月1日]]：[[準急列車|準急]]「'''松前'''」運行開始。 * [[1976年]]（昭和51年）：駅舎改築<ref name="bunken1983"/>。 * [[1980年]]（昭和55年）[[10月1日]]：急行「松前」廃止により、[[優等列車]]の停車がなくなる。 * [[1982年]]（昭和57年）[[11月15日]]：貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]]：[[チッキ|荷物]]取扱い廃止。 * [[1987年]]（昭和62年） ** [[3月16日]]：[[瀬棚線]][[瀬棚駅]]の廃止に伴い、北海道最西端の駅となる。 ** [[4月1日]]：[[国鉄分割民営化]]に伴い、[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）2月1日：松前線の廃線に伴い、廃駅<ref name="bunken1993-R"/><ref name="bunken2008M"/>。 === 駅名の由来 === 当駅の所在する地名より。地名は、[[アイヌ語]]の「マツ・オ・マイ」（婦人のいる所）に由来するとされる<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]]2月発行）180ページより。</ref>が、[[江戸時代]]にこの地を支配した[[蠣崎氏|松前氏]]に由来するともいう<ref name="bunken2004N"/>。松前氏の「松前」については松平家康（後の[[徳川家康]]）の「松」と[[前田利家]]の「前」から採られたという説、あるいは城下の海に面した所に大きな[[マツ|松]]の木があり、船がこの木を目当てに出入したことから名付けられたとの説もある<ref name="bunken2004N"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]で、松前線の[[終着駅|終端駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）63ページより。</ref>。ホームは線路の南側（松前方面に向かって左手側）に存在した。旅客列車の発着に使用する駅舎側の1番線の北側にホームを有さない上下[[停車場#本線|副本線]]の2番線、及び3番線を有した。3番線から分岐した[[停車場#側線|側線]]はかつて[[転車台]]も有した炭水線であった<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）9ページより。</ref>（転車台は1983年（昭和58年）時点で既に撤去されている<ref name="bunken1983"/>）。1番線の延長上は[[引き上げ線]]となっており、また駅舎側に戻る形で分岐し駅舎西側の[[切欠きホーム|切欠き]]部分の貨物ホームへの貨物側線を1本有していた<ref name="bunken1983p193">『国鉄全線各駅停車1』193ページより。</ref>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は線路の南側に位置しホーム中央部分に接していた<ref name="bunken1983p193"/><ref name="bunken2010HE"/>。駅舎は1976年（昭和51年）に<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]]10月発行）66ページより。</ref>開業時からの駅舎の出入口部分を増改築した建物であった。ホームは[[アスファルト]]舗装されていた。 [[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 線路は松前町大島地区までの延長計画があり、[[未成線]]の橋脚などが残っている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）166-167ページより。</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は425人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道228号]]（松前国道） * [[北海道道380号松前停車場線]] [[File:JR_Matsumae_sta_001.jpg|松前駅跡地|thumb|right]] 1999年（平成11年）時点では駅跡に、路線廃止前から存在していた「北海道最南端の町 松前駅」の石碑と当時の駅前商店が残存していた<ref>書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、1999年3月発行）53ページより。</ref>。2010年（平成22年）時点では石碑は残り<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、2010年4月発行）170ページより。</ref>、ホームの一部も残存していた<ref name="bunken2010H"/>。2011年（平成23年）時点では旧駅構内は駐車場と広場になっており、旧観光案内所と観光案内板も残存していた<ref name="bunken2011H"/>。 駅の及部方にあった松前トンネルの坑口は、1999年（平成11年）時点では及部方は消えているが、松前方はトンネル周囲の石垣に似た石組みで塞がれていた<ref name="bunken1999H">『鉄道廃線跡を歩くVI』53-54ページより。</ref>。2010年（平成22年）時点でも同様の状況であり<ref name="bunken2010H"/>、松前方は2012年（平成24年）時点でも同様であった<ref name="bunken2012P">書籍『北海道の大地から消えた鉄道風景』（撮影：上田哲郎、エムジー・コーポレーションMG BOOKS、[[2012年]]3月発行）123ページより。</ref>。 == エピソードなど == *
