[[鉄道省]]名雨線名寄 - 初茶志内（後の天塩弥生）間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1941年]]（昭和16年）[[10月10日]] - 線路名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1960年]]（昭和35年）[[9月15日]] - 貨物・[[チッキ|荷物]]取扱い廃止。 * [[1961年]]（昭和36年）[[4月1日]] - 無人化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1995年]]（平成7年）[[9月4日]] - 深名線廃線により廃止。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の北側（名寄方面に向かって左手側）に存在した。[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]][[6月]]発行）77ページより。</ref>。かつては[[列車交換]]可能な[[交換駅]]であった。使われなくなった1線は、交換設備運用廃止後は撤去された。そのほか本線から構内南側に分岐する貨物[[停車場#側線|側線]]を3線有していた<ref name="bunken1993"/>。この貨物側線は木材運搬に使用され、1983年（昭和58年）4月時点では、分岐器が深川方、名寄方の両方向とも維持されていた形で2線が残存していたが<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）206ページより。</ref>、1993年（平成5年）までに撤去された<ref name="bunken1993"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が改築されて残っていた。駅舎は構内の北側に位置しホーム西側とを結ぶ通路で連絡した<ref name="bunken1993"/>。駅舎は無人化後縮小され、待合室部分のみの間口の狭い建物となっていた（1983年（昭和58年）時点ではこの形状<ref name="bunken1983"/>）。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地（名寄市）の「西」に位置することから付けられた。 == 利用状況 == * 1992年度（平成4年度）の1日乗降客数は2人<ref name="bunken1993"/>。 == 駅周辺 == 民家は少なく、周囲は休耕[[田]]が目立っていた<ref name="bunken1983"/>。 * [[北海道道798号西風連名寄線]] * [[天塩川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）15ページより。</ref> * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「稲川商店前」停留所 == 駅跡 == 2000年（平成12年）時点では既に整地され<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）36ページより。</ref>、2011年（平成23年）時点では「上川ライスターミナル」の物流センター倉庫が建築されており、倉庫の前に[[鉄道標識#停車場接近標識|停車場接近標識]]が立っていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）181ページより。</ref>。また、[[駅名標]]のレプリカが、[[天塩弥生駅]]の駅名板と共に名寄市の「[[名寄市北国博物館]]」に、[[キマロキ編成]]とともに保存されている。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[天塩弥生駅]] -
