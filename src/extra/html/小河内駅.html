|備考 = [[ファイル:JR area HIRO.png|15px]] [[特定都区市内|広島市内駅]]<br />路線廃止にともなう廃駅。 |備考全幅 = }} [[ファイル:小河内.jpg|thumb|240px|広島市内が表示された駅名標]] '''小河内駅'''（おがうちえき）は[[広島市]][[安佐北区]][[安佐 (広島市)|安佐町]]小河内に存在した[[西日本旅客鉄道]]（JR西日本）[[可部線]]の[[鉄道駅|駅]]（[[廃駅]]）である。 可部線非電化区間（[[可部駅|可部]] - [[三段峡駅|三段峡]]間）の廃線に伴い[[2003年]]（[[平成]]15年）12月1日に廃止された。また、JRの特定都区市内制度では当駅までが「広島市内」であった。 == 歴史 == * [[1954年]]（[[昭和]]29年）3月30日 - [[日本国有鉄道|国鉄]]可部線 布 - 加計間開通時に開業{{R|停車場}}。[[鉄道駅#一般駅|一般駅]]。 ** 当時の所在地表示は広島県安佐郡小河内村であった。 * [[1955年]]（昭和30年）3月31日 - 安佐町成立に伴い、所在地表示が広島県安佐郡安佐町小河内に変更。 * [[1960年]]（昭和35年）4月1日 - 貨物の取扱を廃止（[[鉄道駅#旅客駅|旅客駅]]となる）{{R|停車場}}。 * [[1971年]]（昭和46年）5月20日 - 安佐町が広島市に編入され、所在地表示が広島市安佐町小河内に変更。 * [[1980年]]（昭和55年）4月1日 - 安佐北区成立に伴い、所在地表示が広島市安佐北区安佐町小河内に変更。 * [[1984年]]（昭和59年）2月1日 - [[チッキ|荷物]]の取扱を廃止{{R|停車場}}。 * [[1985年]]（昭和60年）2月1日 - 無人駅化<ref>{{Cite news |title=通報 |newspaper=[[鉄道公報]] |publisher=[[日本国有鉄道]]総裁室文書課 |date=1985-02-01 |page=2 }}</ref>（[[簡易委託]]駅化）。 * [[1987年]]（昭和62年）4月1日 - [[国鉄分割民営化]]によりJR西日本が継承{{R|停車場}}。 * [[2003年]]（[[平成]]15年）12月1日 - 廃止。 == 駅名の由来 == 開業当時の駅所在地の村名（広島県[[安佐郡]]小河内村）から。 == 駅構造 == [[島式ホーム]]1面2線を持つ[[地上駅]]であったが、晩年は南側（下り線）が撤去され、[[単式ホーム]]となっていた。ホームの南側にコンクリート造り・平屋建ての[[駅舎]]があったが[[無人駅]]であった。駅舎とホームは下り線を渡る通路で結ばれ、ホームの南側に繋がっていた。 == 利用状況 == 以下の情報は、広島市統計書及び広島市勢要覧に基づいたデータである。 {| 駅前には広場があった。駅のすぐ南側を[[国道191号]]が通り、さらに南側を[[太田川]]が流れている。 駅の少し東側にある野冠橋（広島市道）で対岸の広島市安佐北区安佐町久地に渡ることができた。その辺りには[[宇賀ダム]]や[[宇賀峡]]がある。駅と野冠橋の間に国道191号と[[山県郡]][[豊平町 (広島県)|豊平町]]（現：[[北広島町]]）に通じる[[広島県道38号広島豊平線]]の分岐点がある。 * [[広電バス]] 小浜バス停 == 現状 == ホームが残っているものの、駅舎・線路は撤去され、可部駅側の跡地に[[コンビニエンスストア]]が建っている。 == 隣の駅 == ; 西日本旅客鉄道 : 可部線 :: [[布駅]] - '''小河内駅''' - [[安野駅]] == 脚注 ==
