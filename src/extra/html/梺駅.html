梺駅間）に伴い廃駅 }} '''梺駅'''（ふもとえき）は、[[秋田県]][[雄勝郡]][[羽後町]]大字[[西馬音内]]堀回上石地蔵（開業時は旧・雄勝郡[[元西馬音内村]]下石地蔵）にあった[[羽後交通]][[羽後交通雄勝線|雄勝線]]（旧・雄勝鉄道）の[[鉄道駅|駅]]（[[廃駅]]）である。雄勝線の部分[[廃線]]に伴い[[1967年]]（昭和42年）[[12月1日]]に廃駅となった。 == 歴史 == 西馬音内駅 - 当駅間は、利用者僅少であることを理由に、全線廃止に先駆け部分廃止となった<ref name="bunkenRML052">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、[[ネコ・パブリッシング]]、[[2003年]]11月発行）4,12-13,15,19,21,29ページより。</ref>。また、同区間は冬季、雪による抵抗で電圧が低下するために、除雪を人力に頼らざるを得なく、他の区間に比して人件費も高騰している状況であった<ref name="bunkenRML052"/>。 * [[1935年]]（昭和10年）[[2月13日]] - '''雄勝鉄道'''西馬音内駅 - 当駅間延伸開通（雄勝鉄道全通）に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年8月発行）30,32-33,36ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月16日]] - [[陸上交通事業調整法|交通統合]]に伴い'''横荘鉄道'''の駅となる<ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1944年]]（昭和19年）[[6月1日]] - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''雄勝線'''に制定。それに伴い'''羽後鉄道雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">『私鉄の廃線跡を歩くI』106-109ページより。</ref>。 * [[1947年]]（昭和22年） ** [[7月23日]] - 豪雨による路盤及び橋脚損壊により雄勝線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2">『RM LIBRARY 52』17ページより。</ref>。 ** [[8月14日]] - 羽後山田駅 - 当駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 * [[1952年]]（昭和27年）[[2月15日]] - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1967年]]（昭和42年）[[12月1日]] - 雄勝線の部分廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、単式ホーム1面1線を有する[[地上駅]]であった<ref name="bunkenRML052"/>。ホームは線路の南側（梺方面に向かって左手側）に存在した<ref name="bunkenRML052"/>。そのほか[[停車場#側線|側線]]として、本線のホーム手前から北側に分岐した廻り線（機回し線）を1線と、廻り線の梺方から分岐し構内西側に至る行き止まりの貨物1番線、廻り線の横手方から北側に分岐し構内西側に至る行き止まりの貨物2番線を有した<ref name="bunkenRML052"/>。うち貨物2番線には貨物積卸場を有した<ref name="bunkenRML052"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の北西側に位置し、ホーム西側の[[斜路|スロープ]]とを結ぶ長めの通路及び[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML052"/>。 == 利用状況 == * 1966年度（昭和41年度）の1日乗降客数は146人<ref name="bunkenRML052"/>。 == 駅周辺 == 当駅の南西側には羽後町立元西小学校が隣接している。北側には[[雄物川]]支流である西馬音内川が流れている。 * [[秋田県道57号十文字羽後鳥海線]] * 元西馬音内[[郵便局]]（[[湯沢郵便局]]管轄・2015年移転し元西郵便局に改称<ref>[http://www.post.japanpost.jp/notification/storeinformation/detail/index.php?id=2561 「移転・改称：元西馬音内郵便局（秋田県）」日本郵便プレスリリース　2015年10月20日発表]</ref>）- 改称と同時に「車両保存庫」の隣接地へ移転した。 * 元西馬音内交番（[[湯沢警察署]]管轄） * [[こまち農業協同組合]] * 西馬音内城跡 - 車両保存庫近傍の県道沿いに「小野寺家大手門址」の石柱が立つ == 駅跡 == 旧構内は一時[[幼稚園]]が建築されたがその後空地となった<ref name="bunkenKW03"/>。1995年（平成7年）時点では、[[鉄道信号機#機械式信号機|腕木式信号機]]が設置保存され、車庫が建築され庫内には[[羽後交通雄勝線#現在保存されている車両ほか|デハ1形]]'''デハ3'''が復元保存され<ref name="bunken1995H">書籍『鉄道廃線跡を歩く』（[[JTBパブリッシング]]、[[1995年]]11月発行）37ページより。</ref>、ホームと[[駅名標]]が作られていた。1999年（平成11年）時点でも同様で、車庫は羽後町で建てられ、デハ3は電気を流せば動き出すと言われる状態まで復元されていた<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）64ページより。</ref>。2007年（平成19年）5月時点でも同様で、デハ3の整備も行き届いた状態であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1">『新 鉄道廃線跡を歩く1』200-201ページより。</ref>。 このデハ3は、もともと雨ざらし状態であり盗難品もあったため、[[1983年]]（昭和58年）に地元の人により修復され、車庫に保存することになった<ref>[https://common3.pref.akita.lg.jp/genkimura/village/detail.html?cid=39&vid=1&id=530 幻の鉄道・雄勝線の終着「梺（ふもと）駅」] - あきた元気ムラ！（秋田県）</ref>。なお、平日の9:00 - 17:00に車庫内を見学（事前予約）することができる。 また、当駅跡附近の線路跡は1999年（平成11年）時点では何も残っていなかった<ref name="bunkenTouhoku1999"/>。 代替交通としての路線バスは西馬音内川の対岸を運行している。 == 隣の駅 == ; 羽後交通 : 雄勝線 :: [[元西馬音内駅]] -
