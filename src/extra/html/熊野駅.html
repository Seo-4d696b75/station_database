'''富山軽便鉄道'''堀川新駅（後の南富山駅） - 笹津駅間開通（富山軽便鉄道全通）に伴い開業<ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）38ページより。</ref>。 * [[1915年]]（大正4年）[[10月24日]] - 鉄道会社名を'''富山鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2008M-06"/><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）211ページより。</ref>。 * [[1933年]]（昭和8年）[[4月20日]] - 堀川新駅（現・南富山駅） - 笹津駅間部分廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunken2010H-3-R"/><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]][[12月]]発行）43-45,47ページより。</ref>。 * [[1950年]]（昭和25年）[[9月1日]] - '''富山地方鉄道笹津線'''南富山駅 - 大久保町駅間再開通に伴い再開業<ref name="bunken2008M-06"/><ref name="bunken2010H-3-R"/><ref name="bunkenKW07"/><ref name="bunken2008SH-3">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、[[2008年]][[5月]]発行）80-83ページより。</ref>。 * 時期不詳 - 交換設備撤去、無人化<ref name="bunkenRML107"/>。 * [[1974年]]（昭和49年） ** [[7月10日]] - 水害による熊野川鉄橋橋脚傾斜により当駅 - 大久保町駅間運休、当駅が南富山側の一時的な[[終着駅]]となる<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。 ** [[7月30日]] - 運休区間が復旧、当駅も通常営業となる<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。 * [[1975年]]（昭和50年）[[4月1日]] - 笹津線の廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunken2010H-3-R"/><ref name="bunkenKW07"/><ref name="bunken2008SH-3-R">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、2008年5月発行）165ページより。</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。ホームは線路の東側（地鉄笹津方面に向かって左手側）に存在した<ref name="bunkenRML107"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML107"/>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。交換設備運用廃止後は線路は撤去されたが、ホーム前後の線路は転轍機の名残で湾曲していた<ref name="bunkenRML107"/>。 [[無人駅]]となっていた<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。ホーム北側出入口附近に[[屋根#形状による分類|片流れ]]屋根の[[待合所]]を有していた<ref name="bunkenRML107"/>。ホームは舗装されていた<ref name="bunkenRML107"/>。また列車接近時には[[駅自動放送|テープによる案内放送]]がされていた<ref name="bunkenRML107"/>。 構内の東側には回転交流機による[[変電所]]が設置されていた<ref name="bunkenRML107"/>。この変電所は晩年老朽化しており、1973年（昭和48年）夏からは列車の3連運転が不可能になってしまった<ref name="bunkenRML107"/>。 富山軽便鉄道（後の富山鉄道）時代の当駅は、停車場として開業していた<ref name="bunkenRML107"/>。 == 駅周辺 == 上野駅 - 当駅間は田圃の中を走行した<ref name="bunkenRML107"/>。 * [[国道41号]]（越中東街道） * [[熊野川 (富山県)|熊野川]]<ref name="bunkenRML107"/> * 土川 == 駅跡 == 1998年（平成10年）時点では、下熊野交差点附近に位置が確認出来た<ref name="bunken1998H">書籍『鉄道廃線跡を歩くV』（[[JTBパブリッシング]]、[[1998年]][[6月]]発行）80-81ページより。</ref>。2007年（平成19年）9月時点時点では駅跡の特定は困難であったが<ref name="bunken2008SH-3"/>、位置は廃線跡転用の市道と新婦スーパー農道の交差点の南側であった<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、[[2008年]][[7月]]発行）80ページより。</ref>。 また、当駅跡附近の線路跡は、1998年（平成10年）時点では当線廃線後に南富山駅 - 日本繊維前駅の間を東西に走る形で開通した国道359号の取付け道路附近から田村町駅跡南側付近までは2車線の舗装道路となっていた<ref name="bunken1998H"/>。2008年（平成20年）時点でも同様で、当駅跡附近の道路は[[市町村道|市道]]であった<ref name="bunkenToyama2008H"/>が、当駅跡 - 伊豆ノ宮駅間の[[熊野川 (富山県)|熊野川]]を渡る手前附近から[[富山県道318号笹津安養寺線]]となっていた<ref name="bunkenToyama2008H"/>。 == 隣の駅 == ; 富山地方鉄道 : 笹津線 :: [[上野駅 (富山県)|上野駅]] -
