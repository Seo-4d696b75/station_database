江若鉄道の[[近江木戸駅|近江木戸]] - [[近江舞子駅 (江若鉄道)|近江舞子]]間に臨時駅として開業{{Sfn|江若鉄道の思い出|p=74}}{{Sfn|今尾|2008|pp=31&ndash;32}}。 * 時期不詳（1962年以降） - 常設駅化{{Sfn|江若鉄道の思い出|p=74}}{{Sfn|寺田|2010|p=10}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=近江舞子南口駅線路配置図|left=[[比良駅 (江若鉄道)|比良]]}} {{鉄道配線図|voie|voie|voie|voie|voie}} {{鉄道配線図|0|quai|quai|quai|0}} {{駅配線図終了|right=[[近江舞子駅 (江若鉄道)|近江舞子]]|source={{Sfn|竹内|1967}}}} 近江舞子南口駅は旅客のみを取り扱う[[旅客駅]]{{Sfn|寺田|2010|p=10}}。線路の片側（[[近江今津駅 (江若鉄道)|近江今津]]方面に向かって右側{{Sfn|レイル|p=80}}）に[[プラットホーム|ホーム]]1面のみという簡単な造りの駅だった{{Sfn|レイル|p=82}}{{Sfn|寺田|2010|p=15}}。駅員は配置されていない[[無人駅]]である{{Sfn|レイル|p=83}}。 == 駅周辺 == 駅は比良川の近くにあった{{Sfn|吉田|1998|p=190}}。周辺には企業の保養所が建ち並ぶ{{Sfn|寺田|2010|p=25}}。[[国民宿舎]]の近江舞子ロッジは駅に隣接して建っていたが{{Sfn|レイル|p=82}}、1997年（平成9年）に閉鎖されている{{Sfn|ありし日の江若鉄道|p=15}}。 当駅を含む[[比良駅 (江若鉄道)|比良駅]]から[[近江舞子駅 (江若鉄道)|近江舞子駅]]までの区間はカーブが連続し{{Sfn|レイル|p=82}}、当駅から近江舞子駅にかけて線路は内湖の西岸を通っていた{{Sfn|江若鉄道の思い出|p=74}}。江若鉄道の廃線後に開業した[[湖西線]]はこのカーブを避け、山側を短絡している{{Sfn|レイル|p=82}}{{Sfn|寺田|2010|p=25}}。この区間の線路跡は自転車道を併設した県道に転用され、当駅跡も道路になっているが、当駅から近江舞子駅までの区間は線路跡と道路が完全には重なっておらず、路盤の跡を確認することができる{{Sfn|江若鉄道の思い出|p=74}}{{Sfn|寺田|2010|p=25}}{{Sfn|吉田|1998|pp=190&ndash;192}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[比良駅 (江若鉄道)|比良駅]] -
