37</ref>。 乗降客数は[[1984年]]当時、一日平均62人<ref name="rm"/>。 == 歴史 == * [[1926年]]（[[大正]]15年）[[2月1日]] - 金名鉄道の駅として開業。 * [[1943年]]（[[昭和]]18年）[[10月13日]] - 金名鉄道が合併されたことにより、北陸鉄道金名線の駅となる。 * [[1956年]]（昭和31年）[[10月1日]] - 列車交換設備を設置、閉塞取扱駅となる<ref name="rm"/>。 * [[1966年]]（昭和41年）[[12月21日]] - 列車交換設備が廃止、再び[[プラットホーム#単式ホーム|棒線]]化<ref name="rm"/>。 * [[1983年]]（昭和58年）[[10月31日]] - 豪雨による[[大日川]]橋梁の橋脚周囲の岩盤崩壊により、[[大日川駅|大日川]] - [[白山下駅|白山下]]間が休止、それにより当駅も休止。 * [[1984年]]（昭和59年）[[3月11日]] - 復旧により営業を再開。 * 1984年（昭和59年）[[12月12日]] - [[金名橋|手取川橋梁]]が危険な状態となったことにより、[[加賀一の宮駅|加賀一の宮]] - 白山下間の金名線全線が休止、それにより当駅も休止。 * [[1987年]]（昭和62年）[[4月29日]] - 金名線は営業再開することなく全線廃止、それにより廃駅。 == 隣の駅 == ; 北陸鉄道 : 金名線 :: [[釜清水駅]] -
