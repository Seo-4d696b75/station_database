[[日本国有鉄道|国鉄]]明知線 [[岩村駅|岩村]]～[[明智駅 (岐阜県恵那市)|明知]]間開通時に、'''遠山駅'''（とおやまえき）として開業<ref name="sone26"/>。[[鉄道駅#一般駅|一般駅]]。 * [[1956年]]（昭和31年）[[12月20日]] - '''山岡駅'''に改称<ref name="sone26"/>。 * [[1974年]]（昭和49年）[[12月1日]] - [[車扱貨物]]・[[チッキ|荷物]]の取扱を廃止（[[鉄道駅#旅客駅|旅客駅]]となる）。同時に無人駅化。 * [[1985年]]（昭和60年）[[11月16日]] - 明知鉄道に転換<ref name="sone26"/>。 * [[2014年]]（平成26年）[[3月15日]] - ダイヤ改正により急行[[大正ロマン号]]の停車駅となる。 * [[2014年]]（平成26年）[[7月26日]] - 駅前のヘルシーハウス山岡が'''山岡駅 かんてんかん'''としてリニュアルオープン。同時に倉庫として使用されているアケチ1を再塗装。 * [[2016年]]（平成28年）[[7月15日]] - 山岡駅かんてんかん隣の倉庫として使われていた[[明知鉄道アケチ1形気動車|アケチ1形(アケチ1号)]]が改装され、'''山岡駅 かんてんかん　森の列車カフェ'''としてオープン。 == 駅構造 == 1面1線の[[単式ホーム]]を持つ[[地上駅]]。[[無人駅]]。かつては、1面2線の[[島式ホーム]]であったが、片方の線路は北側に一部だけ残され、そこに留置されたアケチ1形が倉庫として使用されている。それ以外は撤去され整備されている。1988年から一時期駅舎が学習塾として使用されていた。 == 利用状況 == {| class="wikitable" style="text-align:right;" |- !rowspan="1"|年度 !colspan="2"|1日平均乗車人員 |- |2011年（平成23年） ||71 |- |2012年（平成24年） ||76 |- |2013年（平成25年） ||65 |- |2014年（平成26年） ||67 |- |2015年（平成27年） ||63 |- |2016年（平成28年） ||55 |- |} == 駅周辺 == * 寒天村パーク * [[国道363号]] === バス === * [[恵那市自主運行バス]] == 隣の駅 == ; 明知鉄道 : 明知線 :: [[花白温泉駅]]（4） -
