[[鉄道省]]名寄線の上興部 - 興部間延伸開通に伴い、'''瀬戸牛駅'''（せとうしえき）として開業<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2954862/2 『官報』 1921年09月27日　鉄道省告示第129号]（国立国会図書館デジタルコレクション）</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称し、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1954年]]（昭和29年） - 駅舎改築<ref name="murashi">西興部村史 昭和52年11月発行 P807。</ref>。 * [[1955年]]（昭和30年）3月19日 - 木工所専用線敷設<ref name="murashi"/>。 * [[1961年]]（昭和36年）[[3月20日]] - '''西興部駅'''に改称。 * [[1984年]]（昭和59年）[[2月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。 * [[1986年]]（昭和61年）[[11月1日]] - [[無人駅]]化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の南側（遠軽方面に向かって右側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）208ページより。</ref>。そのほか1983年（昭和58年）時点では、名寄方から分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切り欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。かつては、[[単式ホーム]]・[[島式ホーム]]複合型の計2面3線を有する[[列車交換]]可能な[[交換駅]]であった。1983年（昭和58年）時点では、使われなくなった島式ホーム側の線路は交換設備運用廃止後も旧本線、副本線共に[[停車場#側線|側線]]として残っていた。[[分岐器|転轍機]]は遠軽方、名寄方の両方向とも維持されていた<ref name="bunken1983"/>。また、副本線の遠軽方から分岐した行き止まりの側線を1線有していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の南側に位置し、ホーム中央部分に接していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名（村名）より。地名は、興部の西に位置するため<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）186ページより。</ref>、「西」を冠した。 旧駅名の瀬戸牛（せとうし）は、[[アイヌ語]]の「セツ・ウシ・ナイ」（鳥の巣の多い沢）に由来する<ref name="bunken2004N"/><ref>西興部村史 昭和52年11月発行 P37 では「かもしれない」の推定文で断定はされていない。</ref>。 名寄線（当時）が延伸した頃この地域は興部村に属し、七重（ななえ）と呼ばれていた。このため当駅名も当初は「七重」とする話があったが、既に函館方面に[[七飯駅]]があったため混乱するとの理由で、地域内の局所的な地名であった「瀬戸牛」を採用して駅名とした。このため地域名も七重から瀬戸牛となった。1925年（大正14年）1月にこの瀬戸牛を中心地として興部村から分村した際に村名を西興部村としたが、当駅周辺の字名は瀬戸牛のままであったため駅名との乖離はなかった。ところが1945年（昭和20年）頃に字名も西興部となったため問題となり、幾度か改名運動を行ったが、実を結んだのは16年後の1961年（昭和36年）であった<ref>西興部村史 P806等。</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は121人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道239号]]（天北国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * 西興部村役場 *西興部郵便局 * [[北見信用金庫]]西興部支店 * オホーツクはまなす農業協同組合西興部支所 * 特別養護老人ホームにしおこっぺ興楽園 * 村立西興部厚生診療所 -
