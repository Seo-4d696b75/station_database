日本国有鉄道白糠線 白糠駅 - 当駅間開通に伴い開業。[[一般駅]]。 * [[1970年]]（昭和45年）4月　雄別炭鉱上茶路鉱閉山 * [[1972年]]（昭和47年）[[9月8日]] - 当駅 - 北進駅間延伸開通に伴い中間駅となる。 * [[1974年]]（昭和49年）[[3月31日]] - [[チッキ|荷物]]取扱い廃止。無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化（近隣の個人）。 * [[1978年]]（昭和53年）[[10月1日]] - 貨物取扱い廃止。 * [[1983年]]（昭和58年）[[10月23日]] - 白糠線の廃線に伴い廃止となる。 === 駅名の由来 === {{See also|茶路駅#駅名の由来|茶路川#川名の由来}} 所在地名より。茶路川の上流にあることによる命名である<ref name=":2">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=151}}</ref>。 == 駅構造 == 廃止時点で、[[島式ホーム]]片面使用の1面1線を有する[[地上駅]]であった。そのほか、貨物取扱の名残である[[停車場#側線|側線]]を有していた（1983時点で駅舎側に3本、うち2本は駅舎とホームをつなぐ[[踏切#構内踏切|構内踏切]]の部分のみ撤去されており、残り1本は踏切手前までの敷設であった。また[[転轍機]]は外されていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）141ページより。</ref>）。ホームは線路の東側（北進方面に向かって右手側）に存在し、上屋を有していた。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。[[乗車券#鉄道|乗車券]]類は[[日本の鉄道駅#簡易委託駅|簡易委託]]化されており、廃止時まで販売が行われていた。 == 利用状況 == 1981年度（昭和56年度）の1日乗降客数は3人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道665号上茶路上茶路停車場線]] * [[国道392号]] == 駅跡 == 2010年時点では、ホーム（上屋及び[[駅名標]]も残存）、線路の一部（駅構内及び構内から上茶路跨線橋附近まで）、[[鉄道信号機|信号機]]、駅前広場が残存し、[[軌道自転車]]も放置されていた<ref>書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）70-71ページより。</ref>。[[2004年]]までは駅舎も残存したが、解体された。駅舎内には2001年時点では信号操作盤も残されていた<ref>書籍『鉄道廃線跡を歩くVII』（JTBパブリッシング、[[2001年]]1月発行）49ページより。</ref>。ホームの周囲にも木が茂り、林の中に還りつつある状況である。軌道自転車は、2011年に[[新得町]]の[[新内駅]]跡にある狩勝高原エコトロッコ鉄道に移設された<ref>{{Cite web|url=http://ecotorocco.jp/carinfo/|title=トロッコの紹介|publisher=狩勝高原エコトロッコ鉄道|accessdate=2019-2-10}}</ref>。 駅前広場と国道392号の間50メートルほどは[[北海道道665号上茶路上茶路停車場線]]により接続している。なおこの区間は幅2メートル程度の未舗装路で、路面に草木が生い茂り、国道側からの接続地点には道路標示等がない。道道に入って左側数メートル地点には道道665号の標識があるが、国道側から見ると木に隠れており極めて判別しにくい状況にある。 駅付近は、荒地・雑木林が広がるほかは、目立つ人工物は[[国道392号]]と[[北海道道665号上茶路上茶路停車場線|道道665号]]の接続地点に建物が1軒あるのみである。 == 隣の駅 == ; 日本国有鉄道 : 白糠線 :: [[縫別駅]] -
