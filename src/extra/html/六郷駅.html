'''六郷駅'''（ろくごうえき）は、かつて[[北海道]]（[[後志支庁]]）[[虻田郡]][[倶知安町]]北4条東8丁目に設置されていた、[[日本国有鉄道]]（国鉄）[[胆振線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ロク'''。胆振線の[[廃線]]に伴い、[[1986年]]（昭和61年）[[11月1日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南西側（倶知安方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）94ページより。</ref>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983"/>。かつては[[相対式ホーム]]2面2線を有する、[[列車交換]]可能な[[交換駅]]であった。使われなくなった1線は交換設備運用廃止後に線路、ホーム共に撤去されていた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の南西側に位置し、ホーム中央部分に接していた<ref name="bunken1983"/>。無人化後は正面の事務室部分の窓は板材が打ち付けられ、閉鎖されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）205,208ページより。</ref>。 == 駅名の由来 == 開設時の所在地が倶知安町6号線であったことに由来する。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は10人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道276号]]（尻別国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）6ページより。</ref> * 倶知安町立東陵中学校 * 倶知安町立東小学校 * 文化福祉センター - 敷地内に[[国鉄9600形蒸気機関車|9600形蒸気機関車]]'''79615号機'''が[[静態保存]]・展示されている<ref name="bunken2011S">書籍『蒸気機関車完全名鑑 ビジュアル改訂版』（[[廣済堂あかつき|廣済堂ベストムック]]、[[2011年]][[1月]]発行）53ページより。</ref>。胆振線を走行した「2つ目キューロク」で<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（[[JTBパブリッシング]]、[[2001年]][[8月]]発行）68-69ページより。</ref>、「2つ目羊蹄」と記載されたイラスト入りの[[方向幕#ヘッドマーク|ヘッドマーク]]が装着されている<ref name="bunken2011S"/><ref name="bunken2001H"/>。 * 倶知安峠 - 駅から西に約5km<ref name="bunken1983"/>。 * [[尻別川]]<ref name="bunken1980M"/> * [[羊蹄山]]（蝦夷富士） - 駅の南<ref name="bunken1980M"/>。 == 歴史 == * [[1919年]]（大正8年）[[11月15日]] - '''[[日本国有鉄道|国有鉄道]]京極軽便線'''倶知安駅 - 京極駅間開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1922年]]（大正11年）[[9月2日]] - 路線名を'''京極線'''に改称し、それに伴い同線の駅となる。 * [[1944年]]（昭和19年）[[7月1日]] - 胆振縦貫鉄道が[[戦時買収私鉄|戦時買収]]により国有化され、路線名を'''胆振線'''に改称。同時に京極線を胆振線に編入し、それに伴い同線の駅となる。 * [[1971年]]（昭和46年）[[10月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。同時に無人化<ref name="bunken1983"/>。 * [[1986年]]（昭和61年）[[11月1日]] - 胆振線の全線廃止に伴い、廃駅となる。 == 駅跡 == 2001年（平成13年）時点では記念公園として整備され<ref name="bunken2011H"/>、[[車止め#第3種車止め|第3種車止め]]付き<ref name="bunken2011H"/>の[[軌条|レール]]とホーム、[[駅名標]]が設置され、ホームに横付けする形で国鉄の旧型[[客車]]である[[国鉄スハ43系客車#オハ46形|オハ46形]]'''オハ46 501'''と[[国鉄ヨ6000形貨車|ヨ6000形7900番台]][[車掌車]]'''ヨ7913'''の2両が[[静態保存]]・展示されている<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（[[JTBパブリッシング]]、[[2001年]][[8月]]発行）68-69ページより。</ref>。また[[国鉄9600形蒸気機関車|9600形蒸気機関車]]'''9669'''の動輪も記念碑として展示されている<ref name="bunken2001H"/>。[[踏切警報機]]付きの[[踏切]]も再現されている<ref name="bunken2011H"/>。車輌は[[パークゴルフ]]の休憩場に利用され、乗車可能である<ref name="bunken2011H"/>。駅舎は残っておらず、ホームは模擬である<ref name="bunken2011H"/>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）152-154ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。胆振線沿線で唯一の記念施設である<ref name="bunken2011H"/>。尚、本当の駅の位置は公園の西側であり、ホームが雑草に埋もれているものの現存している。 ==
