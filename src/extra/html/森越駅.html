[[渡島知内駅]]間開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として開業。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：福山線が'''[[松前線]]'''に改称。 * [[1969年]]（昭和44年）[[10月1日]]：[[チッキ|荷物]]の取り扱いを廃止。 * [[1970年]]（昭和45年）[[12月12日]]：貨物の取り扱いを廃止し、同時に[[無人駅|無人]]（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]により、[[北海道旅客鉄道]]（JR北海道）が承継。 * [[1988年]]（昭和63年）[[2月1日]]：松前線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、構内の東側（松前方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）61ページより。</ref>。 [[無人駅]]（[[日本の鉄道駅#簡易委託駅|簡易委託駅]]）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎はホームの北側に接していた。 == 駅名の由来 == {{節スタブ}} 当駅が所在していた地名、「森越」より。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は35人<ref name="bunken1983"/>。 == 駅周辺 == 水田の中にあった。 * [[国道228号]] -
