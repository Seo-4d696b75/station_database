[[石狩沼田駅]]間が[[廃線|廃止]]になるとともに廃駅となった。 == 歴史 == * [[1931年]]（昭和6年）[[10月10日]] - [[札沼線|札沼北線]][[石狩沼田駅|石狩沼田]] - 中徳富（初代、現在の[[新十津川駅|新十津川]]）間開通により開業。[[一般駅]]。 * [[1943年]]（昭和18年）[[10月1日]] - [[太平洋戦争]]激化により[[石狩月形駅|石狩月形]] - 当駅間が[[不要不急線]]として休止となり、当駅は終着駅となる。 * [[1944年]]（昭和19年）[[7月21日]] - 当駅 - 石狩沼田間休止にともない休止。 * [[1956年]]（昭和31年）[[11月16日]] - [[雨竜駅|雨竜]] - 石狩沼田間復活により駅が復活。札沼線が全線復旧。 * [[1972年]]（昭和47年）[[6月19日]] - 札沼線新十津川 - 石狩沼田間廃止に伴い廃駅となる。 === 駅名の由来 === 当駅付近が昔から[[石狩国]]と[[天塩国]]の交通上重要な分岐点（[[追分]]）であったことに由来する。旧国名の「石狩」を冠したのは、当駅の開業時すでに道内に[[追分駅 (北海道)|同名の駅]]が存在していたためである<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=33 }}</ref>。 == 駅構造 == 駅舎は札幌に向かって左手の東側に設置され、駅裏側利用の島状の単式ホーム1面1線と、駅舎とホーム間に貨物積卸線を1本有していた。貨物ホームは設置されていなかった。また両端の分岐は貨物積卸線側が開いた片開分岐で、本線側が[[1線スルー]]状になっていた。<ref>昭和7年 線路一覧略図 札幌鉄道局発行。</ref>。 == 駅跡 == 設備は一切撤去されたが、駅のあったことを記念する石碑が立っている。 == 隣の駅 == ; 日本国有鉄道 : 札沼線 :: [[雨竜駅]] -
