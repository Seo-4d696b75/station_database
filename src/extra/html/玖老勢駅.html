'''玖老勢駅'''（くろぜえき）は、[[愛知県]][[新城市]]玖老勢にかつて存在した[[豊橋鉄道]][[豊橋鉄道田口線|田口線]]（旧・田口鉄道）の[[鉄道駅|駅]]。[[南設楽郡]][[鳳来寺村]]（1956年以降は[[鳳来町]]、現・新城市）にあった駅の一つである。 == 歴史 == 玖老勢駅は[[1929年]]（[[昭和]]4年）[[5月22日]]、田口鉄道により開設された<ref name="chizucho">[[#chizucho|『日本鉄道旅行地図帳』7号]]、pp37-38</ref>。 田口鉄道は、現在のJR[[飯田線]][[本長篠駅]]と[[北設楽郡]][[設楽町]]を結んだ鉄道である。玖老勢駅は、同鉄道の第一期線として本長篠駅から[[三河海老駅]]までの区間が開通した際の、途中の駅の一つであった<ref>[[#horai-ta|『鳳来町誌』田口鉄道史編]]、p48,64</ref>。駅が置かれた玖老勢は、[[豊橋市|豊橋]]と[[長野県]]の[[飯田市|飯田]]を結んだ伊那街道沿いの農山村で、鳳来寺村役場の所在地でもあった<ref>[[#kadokawa|『角川日本地名大辞典』23]]、p1205,2029</ref>。 [[1956年]]（昭和31年）[[10月1日]]、田口鉄道が豊橋鉄道に[[合併 (企業)|合併]]されたため豊橋鉄道田口線の駅となった。そして田口線の廃線に伴って[[1968年]]（昭和43年）[[9月1日]]、駅も廃止された<ref name="chizucho"/>。 廃線となった田口線は[[バス (交通機関)|バス]]に転換された。2012年現在、田口線があったルートに沿って走るバスは[[豊鉄バス]][[豊鉄バス新城営業所#田口新城線|田口新城線]]である。[[愛知県道32号長篠東栄線]]を走行しており、駅があった玖老勢地区には、駅名と同じ「玖老勢」などの[[バス停留所|バス停]]がある<ref>[http://www.mapion.co.jp/phonebook/M12001/23221/L0237229/ Mapion電話帳 玖老勢バス停]、2012年9月6日閲覧</ref>。一方、駅の跡地には[[消防団]]の詰め所が建設され、駅付近の線路跡は[[サイクリングロード]]として整備されている<ref>[[#new|『新鉄道廃線跡を歩く』3]]、p132</ref>。 == 構造 == 島式1面2線ホームを持つ<ref>{{Cite journal|和書|author = 白井良和|title = 奥三河に咲いたローカル線 田口線の回想|date = 1986-03|publisher = 鉄道図書刊行会 |journal = 鉄道ピクトリアル|number = 461|page = 59}}</ref>[[日本の鉄道駅#有人駅|有人駅]]で、1956年（昭和31年）の時点で[[駅長]]を含めて4人配置されていた<ref>[[#zuroku|『図録』]]、p8</ref>。 == 利用状況 == 1957年度の乗車人員は9万6千人（1日平均262人）で、そのうち6割にあたる5万8千人が[[定期乗車券]]での利用客であった。この乗車人員は田口線全11駅において、[[本長篠駅]]・[[鳳来寺駅]]・[[三河田口駅]]・[[三河海老駅]]に次いで5番目に多い<ref name="tokei">[[#tokei|『愛知県統計年鑑』昭和34年版]]、p386</ref>。 同年度における貨物取扱量は、発送が16,235トン、到着が227トンであった。田口線の貨物取扱駅7駅の中で、取扱量は本長篠駅に次いで2番目に多い<ref name="tokei" />。駅で扱っていた貨物の中には、鳳来寺村の西隣にあたる[[作手村]]の岩波（現・新城市作手岩波）で産出された[[珪石]]もあった<ref>『鳳来町誌』田口鉄道史編、p138</ref>。 == 隣の駅 == ;
