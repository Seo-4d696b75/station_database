金名鉄道の駅として開業。開業時より[[閉塞 (鉄道)|閉塞]]取扱駅であり、[[駅長]]が常駐した<ref name="rm"/>。 * [[1937年]]（昭和12年）[[10月25日]]（届出）- 貨物量の増加から、構内配線と駅舎など建造物を変更<ref name="rm"/>。 * [[1943年]]（[[昭和]]18年）[[10月13日]] - 金名鉄道が合併されたことにより、北陸鉄道金名線の駅となる。 * [[1970年]]（昭和45年）[[4月1日]] - この日より[[釜清水駅|釜清水]] - 当駅間が[[スタフ閉塞]]式に変更（同時に昼間時間帯の列車運行休止）、数日後に当駅無人化<ref name="rm"/>。 * [[1983年]]（昭和58年）[[10月31日]] - 豪雨による[[大日川]]橋梁の橋脚周囲の岩盤崩壊により、[[大日川駅|大日川]] - 当駅間が休止。 * [[1984年]]（昭和59年）[[3月11日]] - 復旧により営業を再開。 * 1984年（昭和59年）[[12月12日]] - [[金名橋|手取川橋梁]]が危険な状態となったことにより、[[加賀一の宮駅|加賀一の宮]] - 当駅間の金名線全線が休止。 * [[1987年]]（昭和62年）[[4月29日]] 金名線は営業再開することなく全線廃止、それにより廃駅。 == 現状 == 駅舎は金名線廃止後も残され、路線バス乗務員の夜間滞泊に使われていたが、路線延長を機に無人状態となっていた。線路跡には自転車道[[石川県道302号手取川自転車道線|手取キャニオンロード]]が作られており、[[2007年]]（平成19年）5月よりこの駅舎を利用して『自転車屋・白山下駅』として5月から10月の週末・祝日に地元有志による[[レンタサイクル]]事業が行われていたが、2008年（平成20年）には旧駅舎が取り壊され、翌2009年（平成21年）からは新築された「白山下サイクリングターミナル」（または白山下サイクリングパーク）として5月3日から営業を開始している。 == 隣の駅 == ; 北陸鉄道 : 金名線 :: [[三ツ屋野駅]] -
