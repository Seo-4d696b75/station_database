'''天塩栄駅'''（てしおさかええき）は、[[北海道]][[苫前郡]][[初山別村]]字栄にかつて存在した[[日本国有鉄道]]（国鉄）[[羽幌線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''テサ'''。羽幌線の[[廃線]]に伴い、1987年（昭和62年）[[3月30日]]に廃駅となった。 一部の普通列車は通過した（1986年（昭和61年）11月1日改定の時刻（廃止時の時刻表）で、下りのみ1本（急行「はぼろ」後継の主要駅停車列車。上りは停車）<ref>時刻表『JNR編集 時刻表 1987年4月号』（[[交通新聞社|弘済出版社]]、[[1987年]][[4月]]発行）JRニュース13ページより。</ref>）。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]と線路を有する[[地上駅]]であった。ホームは、線路の西側（幌延方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）201ページより。</ref>。 [[無人駅]]となっており、ホーム上の[[待合所]]のみで駅舎は存在しなかった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。開設時、既に[[富内線]]に[[栄駅 (北海道)|栄駅]]が存在していたため、[[旧国名#%E9%A7%85%E5%90%8D|旧国名]]の「天塩」を付した<ref name=":0">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=108}}</ref>。 この地名は[[1941年]]（昭和16年）の字名改正に際して、一般より募集したもので、「『栄える』という希望を込めた」ものとされている<ref name=":0" />。 == 駅周辺 == * [[国道232号]]（天売国道/[[日本海オロロンライン]]） * 栄海岸 - 駅から約1km。[[スズラン]]群生地<ref name="bunken1983"/>。 * モセタキナイ川 == 歴史
