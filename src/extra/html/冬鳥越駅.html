100</ref>。線路を挟んだ両側には蒲原鉄道直営の[[スキー場]]「冬鳥越スキー場」（現・冬鳥越スキーガーデン）が隣接し<ref name="suzuki1998" />、当駅は冬季[[スキー]]客輸送に対応するため2両編成以上が停車可能なホーム[[有効長]]が確保されていた<ref name="suzuki1998" />。ホームの[[村松駅 (新潟県)|村松]]寄りには小さな駅舎が存在した<ref name="suzuki1998" />。 == 歴史 == [[1932年]]（[[昭和]]7年）11月<ref name="miyawaki-ii_1">宮脇 (1996) p.181</ref>に開業した[[停車場#停車場の定義|停留所]]である<ref name="miyawaki-ii_1" />。前述のように当駅に隣接して冬鳥越スキー場が所在し、冬季はスキー客で賑わったが<ref name="endoh">遠藤 (2003) p.30</ref>、周辺人口が希薄であったことからオフシーズンの利用客は極めて少なかった<ref name="suzuki1998" />。 [[1985年]]（昭和60年）4月1日付で実施された[[加茂駅 (新潟県)|加茂]] - 村松間[[廃線|廃止]]・路線縮小に伴って当駅も廃止となった<ref name="miyawaki-ii_1" />。廃止後、駅舎ならびにホーム跡は冬鳥越スキー場の駐車場スペースに転用された<ref name="miyawaki-ii_2">宮脇 (1996) pp.72 - 73</ref>。 === 年表 === * [[1932年]]（[[昭和]]7年）11月20日<ref name="miyawaki-ii_1" /> - 蒲原鉄道線[[七谷駅|七谷]] - [[土倉駅|土倉]]間に新設された駅（停留所）として開業。 * [[1985年]]（昭和60年）4月1日<ref name="miyawaki-ii_1" /> - 加茂 - 村松間 (17.7 km) [[廃線|廃止]]に伴って廃駅。 == 乗客の推移 == {| class="wikitable" border="1" cellspacing="0" cellpadding="2" style="font-size:100%; text-align:center; width:50%;" |- !年度 !乗車人員（人） !降車人員（人） |- |1934||6,399（17）||2,541（7） |- |1935||5,008（14）||5,005（14） |- |1937||6,190（17）||5,994（16） |- |1938||6,914（19）||7,513（21） |- |} * 新潟県統計書各年度版より（ ）内は1日平均 == 冬鳥越スキー場 == 沿線唯一の誘客施設として1932年（昭和7年）末に蒲原鉄道直営で約1万6000坪の[[スキー場]]を開設した。最盛期（[[昭和30年代]]）の日曜日には約2,500人のスキー場利用客があったという。[[1967年]]（昭和42年）スキートロイカを設置。翌年には[[チェアリフト|リフト]]に置き換えし、7月には共同で出資した冬鳥越リフト株式会社を設立した。やがて降雪が不安定なことや他地域のスキー場開発により利用客は下降線をたどっていった。 == 保存車両 == 蒲原鉄道線は[[1999年]]（[[平成]]11年）10月4日付<ref name="miyawaki-ix">宮脇 (2003) pp.78 - 79</ref>で全線廃止となったが、路線全廃時まで在籍した旅客用[[電車]]の1両である[[蒲原鉄道の車両形式#モハ61形|モハ61形]]61と、蒲原鉄道線開業に際して新製された[[蒲原鉄道の車両形式#デ1形|モハ1形]]1が、[[2002年]]（平成13年）より冬鳥越スキーガーデン入口付近において[[静態保存]]されている<ref>[http://www.kenoh.com/2002/20020410moha/ 旧蒲原鉄道の主役モハ1号、63号が修復、再生] - ケンオー・ドットコム.2002年4月10日付.2019年4月16日閲覧。</ref>。モハ1形1（旧デ1形2）については路線全廃当時、村松車庫において作業員詰所として車体のみが転用されていたものを、保存に際して内外装とも現役当時に近付けるよう修復工事が実施されたものであり<ref name="natori_blog">[http://rail.hobidas.com/blog/natori09/archives/2007/11/post_652.html 編集長敬白アーカイブ：蒲原のモハ1を見る。] -
