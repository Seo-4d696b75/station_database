駅の西<ref name="bunkenKW05"/>。 * [[国道152号]] * 丸子郵便局 * 丸子中学校 * 依田川 == 駅跡 == 2007年（平成19年）8月時点では、遺構は無くなっていた<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点では、空地になっていた<ref name="bunkenKW05">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）29,34-35ページより。</ref>。 また、1996年（平成8年）時点では、丸子町駅附近から電鉄大屋駅附近までの線路跡は丸子[[市町村道|町道]]（後に上田市道）となっていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]][[9月]]発行）88ページより。</ref>。舗装道路となって整備されているため、2007年（平成19年）8月時点では遺構は無かった<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点でも道路は健在であった<ref name="bunkenKW05"/><ref name="bunken2010H-3">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）71ページより。</ref>。 == その他 == === 西丸子線連絡線計画 === 上田丸子電鉄としての車輛運用の合理化を計るため<ref name="bunkenRML073-R"/>、当駅と西丸子線[[寿町駅 (長野県)|寿町駅]]とを連絡する0.8kmの路線を作って別所線と丸子線を結ぶ計画を立て、1951年（昭和26年）7月31日に[[運輸省]]（現・[[国土交通省]]）から路線敷設認可が交付された<ref name="bunkenRML073-R"/><ref>[{{NDLDC|2963935/6}} 「運輸審議会の決定」『官報』1951年8月20日]（国立国会図書館デジタル化資料）</ref>。しかし用地買収が困難になり計画変更、八日堂駅 - 別所線上田駅間を連絡する（八田線<ref name="bunkenRML073-R"/>/八日堂線<ref name="bunkenKW05"/>）案を進めることとなった<ref name="bunkenRML073-R"/>。当駅 - 寿町駅間の免許は1956年（昭和31年）8月17日に失効、[[未成線]]となった<ref name="bunkenRML073-R"/>。 == 隣の駅 == ; 上田丸子電鉄 : 丸子線 :: [[中丸子駅]] -
