'''越中電気軌道'''富山北口駅 - 当駅間開通に伴い開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref>。 * [[1926年]]（大正15年）[[7月21日]] - 当駅 - 打出浜駅（後の打出駅（2代目））間延伸開通に伴い中間駅となる<ref name="bunken2008M-6"/>。 * [[1927年]]（昭和2年）[[2月13日]] - 鉄道会社名を'''越中鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]に伴い'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * 時期不詳 - 旅客業務を業務委託化<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）38,41ページより。</ref>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[相対式ホーム]]2面2線を有する[[地上駅]]で、廃止直前は途中駅で唯一[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML107-2">『RM LIBRARY 107』38-39,41ページより。</ref>。互いのホームは駅舎側ホーム北側と対向側ホーム北側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunkenRML107-2"/>。ポイントは[[分岐器#手動転轍器|スプリングポイント]]となっていた<ref name="bunken1981TMS">雑誌『[[鉄道模型趣味]]』（[[機芸出版社]]）1981年3月号59ページより。</ref>。 職員配置駅となっていた<ref name="bunken1981TMS"/>（但し旅客業務は[[日本の鉄道駅#業務委託駅|業務委託]]化されていた<ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）52-53ページより。</ref>）。駅舎は構内の北側に位置し、上り線ホーム北側に接していた<ref name="bunkenRML107"/>。正面から向かって左側の一部が2階建てで、ホーム側は屋根の一部が張り出して[[待合所]]の屋根となる形状であった<ref name="bunken1981TMS"/>。対向側ホームにも待合所代わりの上屋を有した<ref name="bunkenRML107"/>。駅舎とは別棟で入母屋風のトイレ棟を有した<ref name="bunken1981TMS"/>。 乗務員区<ref name="bunkenKW07"/>及び南富山車両区四方支区が設置され<ref name="bunkenRML107"/>、射水線における運転関係の拠点駅となっていた<ref name="bunkenKW07"/>。本線の新富山方・交換設備ポイントの手前から[[停車場#側線|側線]]が分岐し、さらに4線に分岐、うち3線が研修庫を兼ねた木造の車庫へ、残り1線は洗車用の線となっていた<ref name="bunken1981TMS"/>。建屋は車庫のほかに職員用の詰所と風呂場を有した<ref name="bunken1981TMS"/>。 射水線には除雪用の車両として、[[富岩鉄道]]が発注した[[富岩鉄道の電車#ボ1形|ボ1形]]'''ボ2'''が在籍しており<ref name="bunken1981TMS"/>、当駅の車庫に配備され廃線まで所属した<ref name="bunkenRML107-3">『RM LIBRARY 107』21ページより。</ref>。 == 駅周辺 == * [[国道415号]] * [[富山県道207号四方新中茶屋線]] * 市立図書館分館 * 四方郵便局 * 四方小学校 * 四方漁港 == 駅跡 == 当駅より鯰鉱泉前方の線路跡は農道となっている（八ヶ山駅前後は富山地鉄のバス専用道）。打出方の線路跡はサイクリングロードとして整備され、道路の下には[[伏木富山港|富山新港]]と[[日本海石油]]富山製油所（[[富山火力発電所]]に隣接）を結ぶ同社の[[パイプライン輸送|パイプライン]]が埋設されている。 [[1997年]]（平成9年）時点では路盤跡地はそのサイクリングロードに、駅舎跡地は駐車場の一部に、ホーム跡地は花壇になっていた<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]）108ページより。</ref>。[[2006年]]（平成18年）時点では車庫跡地は保育所が建設されており、ホーム跡地は引き続き花壇となっていた<ref>書籍『富山廃線紀行』（著：[[草卓人]]、[[桂書房]]、2008年）66ページより。</ref>。[[2010年]]（平成22年）時点でも保育所は存続していた<ref name="bunkenKW07"/>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[鯰鉱泉前駅]] -
