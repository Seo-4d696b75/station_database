'''丸子鉄道'''上田東駅 - 大屋駅（後の電鉄大屋駅）間延伸開通に伴い開業<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）210ページより。</ref><ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）42ページより。</ref><ref name="bunkenKW05-R">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）27-28ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1943年]]（昭和18年）[[10月21日]] - [[陸上交通事業調整法|交通統合]]に伴い'''上田丸子電鉄丸子線'''の駅になる<ref name="bunken2010H-3-R"/><ref name="bunkenKW05-R"/>。 * [[1955年]]（昭和30年）[[5月4日]] - 役員会にて当駅 - [[上田電鉄別所線|別所線]]、国鉄線[[上田駅]]間（八田線）鉄道敷設出願を決定<ref name="bunkenRML073-R">書籍『RM LIBRARY 73 上田丸子電鉄（上）』（著：宮田道一、諸河久、[[ネコ・パブリッシング]]、[[2005年]][[9月]]発行）34,47ページより。</ref>。 * [[1960年]]（昭和35年）[[3月29日]] - 当駅 - 上田駅間（八田線）鉄道敷設免許申請取下げを決定<ref name="bunkenRML073-R"/>。 * [[1969年]]（昭和44年）[[4月20日]] - 丸子線の廃線に伴い廃止となる<ref name="bunken2010H-3-R"/><ref name="bunken2008M-06"/><ref name="bunkenKW05-R"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML073">書籍『RM LIBRARY 73 上田丸子電鉄（上）』（著：宮田道一、諸河久、[[ネコ・パブリッシング]]、[[2005年]][[9月]]発行）5,11,22-24,30ページより。</ref>。駅舎側（南西側）が下り線（上田東方面）、外側（北東側）が上り線（丸子町方面）となっていた<ref name="bunkenRML073"/>。[[分岐器|転轍機]]の形状は上田東方、丸子町方ともに片開き分岐であった<ref name="bunkenRML073"/>。そのほか下り線の丸子町方から分岐した行き止りの[[停車場#側線|側線]]を1線有した<ref name="bunkenRML073"/>。この側線は八日堂縁日用の福[[だるま]]を積んだ貨車が留置されたり、1963年（昭和38年）2月時点では[[山梨交通]][[山梨交通電車線|電車線]]から入線したモハ7（当線運行時にモハ2341に改番）が改造前のしばらくの間、留置されていた<ref name="bunkenRML073"/>。また、島式ホームの下り線側の向かい、駅舎の北側に八日堂縁日（毎年1月7、8日）用の臨時ホームである[[単式ホーム]]1面を有した<ref name="bunkenRML073"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-2">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）64-67ページより。</ref>。駅舎は構内の南西側に位置し、ホーム南側の[[階段]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML073"/>。島式ホームは上屋を有した<ref name="bunkenRML073"/>。 == 駅周辺 == [[信濃国分寺]]跡にて毎年1月7、8日に八日堂縁日が立ち、その日は大変賑わったという<ref name="bunkenRML073"/>。当線廃止後の[[2002年]]（平成14年）[[3月29日]]に、当駅跡から神川方に[[しなの鉄道]][[しなの鉄道しなの鉄道線|しなの鉄道線]][[信濃国分寺駅]]が開業した。 * [[北陸新幹線]][[高架橋]] - 当線廃止後の建設。 * [[国道18号]] * [[信濃国分寺|国分寺]]史跡公園 - 駅跡は当公園の敷地に取り込まれている<ref name="bunkenKW05">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）28-29,34ページより。</ref>。 * [[千曲川]] == 駅跡 == 1996年（平成8年）時点では、確認は困難になっていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]][[9月]]発行）89ページより。</ref>が、駅跡は国分寺史跡公園の敷地として整備されていた<ref name="bunkenKW05"/>。 電鉄大屋駅附近から上田東方面、当駅跡附近までの線路跡は、信越本線の複線化用地となり消滅し<ref name="bunken2010H-3"/>、しなの鉄道下り線の敷地となっていた<ref name="bunken2008SH-2"/><ref name="bunkenKW05"/>。 また、当駅 - 上堀駅間にて信越本線をオーバークロスしていた部分は、信越本線を跨いでいた橋梁の丸子町方の線路跡の、カーブして地平に降りていた部分が2010年（平成22年）10月時点では道路に転用されていた<ref name="bunkenKW05"/>。橋梁部分は2007年（平成19年）8月時点では上田東方に橋台が残存し<ref name="bunken2008SH-2"/>、2010年（平成22年）10月時点でも同様であった<ref name="bunkenKW05"/><ref name="bunken2010H-3">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）71ページより。</ref>。橋台の先、上田東方は2010年（平成22年）10月時点では住宅地となっていた<ref name="bunkenKW05"/>。その先の線路跡の一部が切り通しを埋めた細長い[[駐車場]]の敷地として確認出来た<ref name="bunken2010H-3"/>。 == その他 == === 上田駅連絡線（八田線/八日堂線）計画 === 当線は同社の他路線と接続していない唯一の路線であり、上田駅から当線の各駅まで乗るには、必ず大屋駅まで国鉄線に乗ってそこから当線電車に乗り換えるという方法しかなかったが、方向によっては不便であった。また上田丸子電鉄としての車輛運用の合理化を計るため<ref name="bunkenRML073-R"/>の当線上丸子駅と[[上田丸子電鉄西丸子線|西丸子線]][[寿町駅 (長野県)|寿町駅]]とを連絡する計画が用地買収などの問題により困難になった後に<ref name="bunkenRML073-R"/>、計画変更として1955年（昭和30年）5月4日の役員会にて当駅 - 上田駅間の鉄道敷設（八田線<ref name="bunkenRML073-R"/>/八日堂線<ref name="bunkenKW05"/>）出願を決定<ref name="bunkenRML073-R"/>、この案を進めることとなった。しかし結局1960年（昭和35年）3月29日に鉄道敷設免許申請取下げを決定、実を結ぶことはなかった<ref name="bunkenRML073-R"/>。 == 隣の駅 == ; 上田丸子電鉄 : 丸子線 :: [[上堀駅 (長野県)|上堀駅]] -
