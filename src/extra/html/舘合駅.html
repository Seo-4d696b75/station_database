'''横荘鉄道'''沼館駅 - 当駅間延伸開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]]8月発行）25-28,30-31ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="bunken2008SH-1-R"/>。敷地の購入費は地元の素封家で横荘鉄道創設委員長の[[土田万助]]が寄付した<ref>『秋田人名大事典（第二版）』秋田魁新報社、2000年、391-392頁。</ref>。 * [[1920年]]（大正9年）3月24日 - 当駅 - 羽後大森駅間延伸開通に伴い中間駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1944年]]（昭和19年）6月1日 - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1947年]]（昭和22年） ** 7月23日 - 豪雨による路盤及び橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）16-17ページより。</ref>。 ** 7月29日 - 横手駅 - 当駅間が復旧、当駅が一時的な[[終着駅]]となり営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1948年]]（昭和23年）11月8日 - 運休区間（当駅 - 老方駅間）が復旧、営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1952年]]（昭和27年）2月15日 - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1965年]]（昭和40年） ** 7月9日 - 水害による雄物川橋梁橋脚洗掘により当駅 - 二井山駅間運休、当駅が一時的な終着駅となる<ref name="bunkenRML061-3">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）19-20ページより。</ref><ref name="bunkenKW03"/>。 ** 10月21日 - 当駅 - 二井山駅間が正式に営業休止となる<ref name="bunkenKW03"/>。 * [[1966年]]（昭和41年）6月15日 - 当駅 - 二井山駅間部分廃線に伴い終着駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1969年]]（昭和44年）[[1月16日]] - 沼館駅 - 当駅間部分廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML061">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）6,11,14-15ページより。</ref>。ホームは線路の東側（老方方面に向かって右手側）に存在した<ref name="bunkenRML061"/>。そのほか[[停車場#側線|側線]]として、本線から西に分岐した[[機回し線]]を1線と、本線横手方機回し線[[分岐器|転轍機]]の手前から東に分岐し駅舎南側に至る行き止りの側線を1線、機回し線の老方方本線合流部分の手前から分岐する行き止りの側線を1線有していた<ref name="bunkenRML061"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の東側に位置し、ホーム中央部分に接していた<ref name="bunkenRML061"/>。 1966年（昭和41年）2月時点では雄物川河川敷まで延びる砂利採取線を有し、1947年（昭和22年）の水害による営業休止時の、羽後大森駅残存車輌の回送時に利用された<ref name="bunkenRML061"/><ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）60-61ページより。</ref>。 駅名は、羽後交通の資料には「'''舘'''合」と表記されていた<ref name="bunkenRML061"/>。 === 雄物川橋梁 === 当駅 - 羽後大森駅間には雄物川が流れており、「雄物川橋梁」<ref name="bunkenRML061-3"/>が架けられていた。この鉄橋は延長428mで、23基の橋脚と24連の橋桁を有する構造となっていた<ref name="bunkenRML061-3"/>。 晩年は架橋時に比べ、雄物川が川砂利の濫獲及び堤防工事に伴い河床が低下しており、1965年（昭和40年）7月9日の集中豪雨によって九号橋脚の[[侵食|洗掘]]が著しく進行（同日より列車運行休止）、また調査の結果他の橋脚も老朽化が進んでいることが発覚、復旧は絶望的と目された<ref name="bunkenRML061-3"/>。そして翌1966年（昭和41年）2月4日、職員の目の前で九号橋脚が倒壊<ref name="bunkenRML061-3"/>、復旧の道は絶たれ、横荘線廃止の大きな要因の一つとなったとされる<ref name="bunkenRML061-3"/>。 == 駅周辺 == * [[秋田県道13号湯沢雄物川大曲線]] - 一部区間には当線の線路跡が転用されていた。 * 館合郵便局 * [[雄物川]] == 駅跡 == 2002年（平成14年）3月には、駅跡の[[農業協同組合|農協]]敷地内に、雄物川町（当時）により「横荘線 '''館'''合駅跡」と記載され、沿革も記された白い[[石碑|記念碑]]が建立された<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2007年（平成19年）5月時点<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）10月時点でも同様であった<ref name="bunken2010H-2">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）201-203ページより。</ref>。 また、当駅跡附近の線路跡は道路となっており、2007年（平成19年）5月時点では横手方は往時を偲ぶことは不可能に近い状況であったが<ref name="bunkenKW03"/>、当駅跡から雄物川堤防に至る道路は道床跡が残り、鉄道の雰囲気も残っていた<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H-2"/>。 雄物川橋梁は、1996年（平成8年）時点では、橋脚は[[発破]]により取り壊されてしまっていたが、羽後大森方には橋台と丸い橋脚の基礎が中州に残存していた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]]9月発行）34-35ページより。</ref>。1999年（平成11年）時点<ref name="bunkenTouhoku1999"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）203ページより。</ref>。発破による橋脚の撤去は河川改修のためであった<ref name="bunkenKW03"/>。 == 隣の駅 == ; 羽後交通 : 横荘線 :: [[船沼駅]] -
