'''[[日本国有鉄道|国有鉄道]]（[[鉄道省]]）広尾線'''中札内駅 - 大樹駅間の延伸開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として新設。 * [[1982年]]（昭和57年）[[9月10日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[2月2日]] - 広尾線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の南東側（広尾方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、1983年7月発行）139ページより。</ref>。かつては2面2線の[[相対式ホーム]]を有する、[[列車交換]]が可能な[[交換駅]]であった。使われなくなった駅舎の反対側の1線は、交換設備運用廃止後も[[停車場#側線|側線]]として残っていた（但し[[1983年]]（昭和58年）時点では、プラットホームは撤去されていた<ref name="bunken1983" />）。そのほか、本線広尾方から駅舎側に分岐し駅舎南側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、また上記旧交換線である側線の帯広方から分岐し広尾方も接続する側線を1線有していた<ref name="bunken1983" />。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南東側に位置しホーム中央部分に接していた<ref name="bunken1983" />。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「チウ・ルイ・トープツ」（流れの速い川）に由来する<ref name="bunken1983" />。 == 利用状況 == [[1981年]]度（昭和56年度）の1日当たりの乗降客数は106人<ref name="bunken1983" />。 == 駅周辺 == * [[北海道道657号美成忠類停車場線]] * [[国道236号]]（広尾国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）13ページより。</ref> * [[道の駅]][[道の駅忠類|忠類]] * 幕別町忠類総合支所 * 忠類郵便局 * 忠類支署 * 忠類中学校 * 忠類小学校 * 洞雲寺 * 白銀台スキー場 * [[十勝バス]]忠類案内所、「忠類」停留所 == 駅跡 == 廃駅後は忠類村により旧駅構内が買い取られ、[[1988年]]（昭和63年）に「交通公園」<ref>[http://www.town.makubetsu.lg.jp/makubetsucho/kikakusitsu/sisetuannai/kotukoen.jsp 交通公園] - 幕別町役場企画室</ref>として整備され、駅舎は「忠類鉄道資料館」となった<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]]11月発行）67ページより。</ref>。しばらくの間は構内が当時の姿のまま残されていた。その後村が老朽化を理由に公園を廃止し宅地とする、という計画が決定されたが、存続を望む声が多数寄せられ<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（JTBパブリッシング、1999年3月発行）40ページより。</ref>、その結果、駅舎周辺部に規模を縮小しながらも存続している<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、2010年4月発行）88ページより。</ref>。 [[1999年]]（平成11年）時点<ref name="bunken1999H" />、[[2010年]]（平成22年）時点<ref name="bunken2010H" />、[[2011年]]（平成23年）時点<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、2011年9月発行）189-190ページより。</ref>でも継続して存続している。駅舎、ホーム、[[軌条|レール]]、[[駅名標]]などが当時の状態に修復されて保存されている<ref name="bunken2011H" />。旧駅舎内には料金表や備品が保存・展示され、線路（側線）上に帯広方から順に[[国鉄ワム80000形貨車|ワム80000形]][[有蓋車]]'''ワム187865'''、[[国鉄トラ70000形貨車|トラ70000形]][[無蓋車]]'''トラ74718'''、[[国鉄ヨ6000形貨車|ヨ6000形]][[車掌車]]'''ヨ7908'''、の3両が<ref name="bunken1998S" />連結して[[静態保存]]・展示されている<ref name="bunken2011H" />。上記の規模縮小の結果、レールはホーム端の部分で終わっているが、線路跡が続いている様子も判別できる<ref name="bunken2011H" />。なお、現在はホームが多少老朽化しており近づく際には注意が必要である。 == 隣の駅 == ; 日本国有鉄道 : 広尾線 :: [[上更別駅]] -
