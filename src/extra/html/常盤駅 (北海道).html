[[日本国有鉄道]]（国鉄）北見線の'''常盤仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1961年]]（昭和36年）[[4月1日]] - 路線名を天北線に改称し、それに伴い同線の仮乗降場となる。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、北海道旅客鉄道（JR北海道）の駅となると共に旅客駅に昇格。'''常盤駅'''となる。 * [[1989年]]（平成元年）[[5月1日]] - 天北線の全線廃止に伴い、廃駅となる。 == 駅構造 == {{節スタブ}} 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。 [[仮乗降場]]に出自を持つ[[無人駅]]となっており、駅舎は無いが[[待合所]]を有していた。 == 駅名の由来 == {{節スタブ}} 当駅が所在していた地名より。 == 駅周辺 == 広大な原野が広がる。 * [[国道275号]]（頓別国道） * 頓別川<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）17ページより。</ref> * [[宗谷バス]][[天北線 (宗谷バス)|天北宗谷岬線]]「常盤」停留所 == 駅跡 == 2011年（平成23年）時点で当駅の遺構は何も残されていないが、旧駅横の[[踏切]]跡には線路の跡が残っている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）248-249ページより。</ref>。 また、2011年（平成23年）時点で、駅跡より音威子府方に2kmほど戻った場所にある四号線川に、橋梁上に[[枕木]]が残った鉄骨ガーダー橋が残存している<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 天北線 :: [[下頓別駅]] -
