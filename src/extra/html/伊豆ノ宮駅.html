'''富山鉄道'''熊野駅 - 大久保町駅間に'''上栗山駅'''（かみくりやまえき）として新設開業<ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）38ページより。</ref>。 * [[1933年]]（昭和8年）[[4月20日]] - 堀川新駅（現・南富山駅） - 笹津駅間部分廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]][[12月]]発行）43-45,47ページより。</ref><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）211ページより。</ref>。 * [[1950年]]（昭和25年）[[9月1日]] - '''富山地方鉄道笹津線'''南富山駅 - 大久保町駅間再開通に伴い'''伊豆ノ宮駅'''として再開業<ref name="bunken2008M-06"/><ref name="bunkenKW07"/><ref name="bunken2010H-3-R"/><ref name="bunken2008SH-3-R">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、2008年5月発行）165ページより。</ref>。旅客のみ取扱い<ref name="bunken2008SH-3-R"/>。 * [[1974年]]（昭和49年） ** [[7月10日]] - 水害による熊野川鉄橋橋脚傾斜により熊野駅 - 大久保町駅間運休、当駅も[[休止駅|営業休止]]となる<ref name="bunkenKW07"/><ref name="bunkenRML107_1">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、[[2008年]][[7月]]発行）28-29,47ページより。</ref>。 ** [[7月30日]] - 運休区間が復旧、当駅も営業再開となる<ref name="bunkenKW07"/><ref name="bunkenRML107_1"/>。 * [[1975年]]（昭和50年）[[4月1日]] - 笹津線の廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW07"/><ref name="bunken2010H-3-R"/><ref name="bunken2008SH-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、[[2008年]][[7月]]発行）28-29ページより。</ref>。ホームは線路の西側（地鉄笹津方面に向かって右手側）に存在した<ref name="bunkenRML107"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML107"/>。 [[無人駅]]となっていた<ref name="bunkenRML107"/>。駅舎は無いがホーム中央部分に[[待合所]]を有した<ref name="bunkenToyama2006H">書籍『鉄道の記憶』（編著：草卓人、[[桂書房]]、[[2006年]][[2月]]発行）301ページより。</ref>。 当駅の熊野方には[[神通川]]の支流である[[熊野川 (富山県)|熊野川]]が流れており<ref name="bunkenRML107"/>、当線で唯一の長い[[鉄橋]]である<ref name="bunkenRML107"/>「熊野川鉄橋<ref name="bunkenRML107"/>（熊野川橋梁<ref name="bunkenKW07"/>）」が架橋されていた。鉄橋の前後は半径300mの曲線となっていた<ref name="bunkenRML107"/>。 == 駅周辺 == 熊野川鉄橋附近からの[[立山連峰]]の風景が素晴らしく、笹津線の撮影の名所として知られていた<ref name="bunkenRML107"/>。 * [[国道41号]]（越中東街道） * [[富山県道188号東猪谷富山線]] * [[熊野川 (富山県)|熊野川]]<ref name="bunkenRML107"/> == 駅跡 == 1991年（平成3年）7月まではホーム跡が残存し、鉄道があった当時の雰囲気も残っていた<ref name="bunkenKW07"/>が、その後に開始された道路整備によって失われた<ref name="bunkenKW07"/>。2007年（平成19年）9月時点では駅跡の特定は困難となっていた<ref name="bunken2008SH-3">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、[[2008年]][[5月]]発行）80-83ページより。</ref>。2009年（平成21年）11月時点でも同様であった<ref name="bunkenKW07"/>。 また、当駅跡附近の線路跡は、1998年（平成10年）時点では当線廃線後に南富山駅 - 日本繊維前駅の間を東西に走る形で開通した国道359号の取付け道路附近から田村町駅跡南側付近までは2車線の舗装道路となっていた<ref name="bunken1998H">書籍『鉄道廃線跡を歩くV』（[[JTBパブリッシング]]、[[1998年]][[6月]]発行）80ページより。</ref>。2008年（平成20年）時点でも同様で、当駅跡附近の道路は[[富山県道318号笹津安養寺線]]となっていた<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、[[2008年]][[7月]]発行）80ページより。</ref>。 熊野川鉄橋（熊野川橋梁）も、1991年（平成3年）7月までは橋台が残存していた<ref name="bunkenKW07"/>が、「興南大橋」という道路橋に架け替えられた<ref name="bunkenKW07"/><ref name="bunkenToyama2008H"/>。この橋の欄干には鉄道の車輪をデザインされた装飾が施されており<ref name="bunken1998H"/>、2007年（平成19年）9月時点<ref name="bunken2008SH-3"/>、2008年（平成20年）時点<ref name="bunkenToyama2008H"/>、2009年（平成21年）11月時点でも同様であった<ref name="bunkenKW07"/>。但し笹津線時代の面影は残っていなかった<ref name="bunkenKW07"/>。 == 隣の駅 == ; 富山地方鉄道 : 笹津線 :: [[熊野駅]] -
