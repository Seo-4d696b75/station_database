三河鉄道により[[岩津駅|岩津]] - [[細川駅 (三河鉄道)|細川]]間に開業。同時に当駅 - [[上挙母駅|上挙母]]間が開業し、[[岡崎井田駅|岡崎井田]] - 上挙母間が岡崎線に、当駅 - 門立間が門立支線となり、両線の接続駅となる。 * [[1938年]]（昭和13年）[[5月1日]] - 門立支線休止。 * [[1939年]]（昭和14年）[[10月3日]] - 門立支線廃止により岡崎線の単独駅となる<ref>[{{NDLDC|2960323/16}} 「鉄道営業廃止」『官報』1939年10月7日]（国立国会図書館デジタル化資料）</ref>。 * [[1941年]]（昭和16年）[[6月1日]] - 三河鉄道が名古屋鉄道に吸収合併される。 * [[1948年]]（昭和23年）[[5月16日]] - 岡崎線が挙母線に改称、同線の駅となる。 * [[1969年]]（昭和44年）[[8月1日]] - 無人化<ref>{{Cite book|和書|author=名古屋鉄道広報宣伝部（編）|year=1994|title=名古屋鉄道百年史|publisher=名古屋鉄道|page=887}}</ref>。この頃交換設備が撤去される。 * [[1973年]]（昭和48年）[[3月4日]] - 挙母線の全線廃止により廃駅。 == その他 == * 駅跡は空き地となっている。ホーム跡が長らく存在していたが撤去され、駅の痕跡は無い。 == 隣の駅 == ; 名古屋鉄道 : 挙母線 :: [[細川駅 (愛知県)|細川駅]] - '''三河岩脇駅''' - [[岩津駅]] ::: かつて三河岩脇 - 岩津間に八ツ木駅があった（[[1969年]][[4月5日]]に廃止）。 ; 三河鉄道 : 門立支線 :: '''三河岩脇駅''' -
