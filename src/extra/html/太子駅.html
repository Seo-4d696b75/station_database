'''太子駅'''（おおしえき）は、かつて[[群馬県]][[吾妻郡]][[六合村 (群馬県)|六合村]]（現在の[[中之条町]]）[[大字]]太子にあった[[日本国有鉄道]]（国鉄）[[吾妻線]]（太子支線）の[[鉄道駅|駅]]（[[廃駅]]）である。 == 歴史 == [[File:Chatsubomigoke Park 2017-08 1.jpg|thumb|群馬鉄山跡（穴地獄・チャツボミゴケ公園）]] [[File:Oshi Station museum.jpg|thumb|太子駅跡復元駅舎]] [[File:Oshi Station freight cars.jpg|thumb|貨車類]] [[File:Oshi Station 2017-08 4.jpg|thumb|ホッパー跡（2017年8月撮影）]] [[File:Naganohara-Oshi discontinued line.jpg|thumb|廃線区間の橋梁(右の赤いガーダー橋)、左の緑色の橋は現用の吾妻線、真新しいコンクリート柱は工事中の群馬県道376号線]] [[日本鋼管]][[群馬鉄山]]の[[専用鉄道|専用線]]として[[1945年]]（[[昭和]]20年）に開業した。[[プラットホーム#形状と配置|1面2線]]の[[旅客]]用ホームのほか、[[停車場#側線|側線]]と鉱石積み出し用の[[ホッパー (鉱業)|ホッパー]]を持つ駅であった。[[京浜]]地区に向けて鉱石を発送していたが、鉱石輸送の廃止と吾妻線の[[大前駅|大前]]方面への延伸により[[1970年]]（昭和45年）に休止、翌年[[廃線|廃止]]された。廃止時の所属[[鉄道路線|路線]]は吾妻線だが、長野原線から吾妻線に路線名称が変更されてから[[列車]]が運転されたことはなかった。 === 年表 === * [[1945年]]（[[昭和]]20年）[[1月2日]] - [[日本鋼管]][[群馬鉄山]][[専用鉄道|専用線]]・長野原駅 - 当駅間が開業。同時に[[運輸通信省 (日本)|運輸通信省]]長野原線・渋川駅 - 長野原駅間も開業。 * [[1952年]]（昭和27年）[[10月1日]] - 日本鋼管専用線を国鉄に移管し、国鉄長野原線長野原駅 - 当駅間の[[貨物線|貨物]][[支線]]が開業。[[車扱貨物]]の運輸営業を開始し、当駅を開業<ref>1952年（昭和27年）10月1日日本国有鉄道公示第320号</ref>。 * [[1954年]]（昭和29年）[[6月21日]] - 長野原駅 - 当駅間の[[営業キロ]]程を5.8 kmへ改正する<ref>1954年（昭和29年）6月15日日本国有鉄道公示第157号</ref>。 * [[1961年]]（昭和36年）[[9月1日]] - 営業範囲を[[旅客輸送|旅客]]、[[チッキ|手荷物、小荷物]]および車扱貨物へ改める<ref>1961年（昭和36年）8月29日日本国有鉄道公示第399号「飯田線鳥居停車場等の営業範囲を改正する件」</ref>。 * 1961年（昭和36年）10月1日 - 手荷物および小荷物の配達の取扱を廃止する<ref>1961年（昭和36年）9月29日日本国有鉄道公示第474号「八高線明覚停車場等における手荷物及び小荷物の配達の取扱は廃止する件」</ref>。 * [[1966年]]（昭和41年）10月1日 - 長野原駅 - 当駅間の貨物運輸営業を廃止する<ref>1966年（昭和41年）9月19日日本国有鉄道公示第569号「一般運輸営業を廃止」</ref>。 * [[1967年]]（昭和42年）[[11月1日]] - 営業範囲を旅客へ改正する<ref>1967年（昭和42年）10月25日日本国有鉄道公示第510号「停車場の営業範囲を改正」</ref>。 * [[1970年]]（昭和45年）11月1日 - 長野原駅 - 当駅間の鉄道運輸営業を、1971年（昭和46年）4月30日まで休止する<ref>1970年（昭和45年）10月22日日本国有鉄道公示456号「鉄道運輸営業の休止の件」</ref>。 * [[1971年]]（昭和46年） ** [[3月7日]] - 長野原線の線路名称を吾妻線へ改正する<ref>1971年（昭和46年）2月26日日本国有鉄道公示第76号「日本国有鉄道線路名称の一部改正」</ref>。 ** [[5月1日]] - 長野原駅 - 当駅間の旅客運輸営業および当駅を[[廃駅|廃止]]する<ref>1971年（昭和46年）4月30日日本国有鉄道公示第203号「旅客運輸営業の廃止の件」</ref>。 == 運行形態 == 1967年（昭和42年）6月10日の長野原線[[渋川駅]] ‐ 長野原駅（現[[長野原草津口駅]]）[[鉄道の電化|電化]]後も[[非電化]]のまま取り残され、[[気動車|ディーゼルカー]]が長野原駅と当駅の間を一日5往復するのみであった。電化区間は[[急行列車]]も合わせて[[定期列車|定期]]で15往復の運行があり、接続は概ね良かった<ref>国鉄監修 交通公社の時刻表 1969年9月</ref>。 == 代替交通 == 廃止後は並行していた[[国鉄バス]][[花敷線]]の本数が若干増やされ[[バス代行|代替]]とされたが、[[2009年]]（[[平成]]21年）に廃止され、沿線自治体の[[六合村 (群馬県)|六合村]]へ路線が移管された。現在は[[日本の市町村の廃置分合|市町村合併]]により[[中之条町営バス]]として運行されている。 == 現在 == 当駅跡は[[車止め]]や鉱石積み出しのホッパーなどが[[遺構]]として残っており、また当時を偲ばせる[[駅名標]]等が[[2013年]]（平成25年）に設置し直されている。また当地は中之条町や地域の住民により[[公園]]として整備され、子供用遊具、[[花壇]]、[[公衆便所|公衆トイレ]]などが設置されていた。 これらの整備を行った際に遺構が多数残されていることから、[[2014年]]（平成26年）4月中之条町は当駅を本格的に[[復元]]することを決定し、埋まっている遺構を掘り出し、ホッパー跡を修復することになった。2016年度（平成28年度）にクラウドファンディングにて寄付を募集<ref>旧太子駅跡地復元プロジェクト｢つなげ未来へのレール｣　～戦後日本の復興を支えた旧太子駅の復元～[https://www.furusato-tax.jp/gcf/117]</ref>し、約215メートル分のレール敷設や約71メートルのホーム復元を実施した<ref name = "読売20180331">{{Cite web | url = http://www.yomiuri.co.jp/local/gunma/news/20180330-OYTNT50062.html | title = 太子駅舎 思い出の姿に | date = 2018-03-30 | publisher = 読売新聞 | accessdate = 2018-04-04 |archiveurl = http://web.archive.org/web/20180330032413/http://www.yomiuri.co.jp/local/gunma/news/20180330-OYTNT50062.html |archivedate = 2018-03-30}}</ref>。 木造平屋建て約96平方メートルの復元駅舎は2018年（平成30年）に完成し、鉄鉱石を積んだバケットなども展示された。4月から一般公開され、5月には2[[大井川鐵道井川線]]で使用されていた[[大井川鉄道cワフ0形貨車|ワフ2・ワフ3]]が中之条町に譲渡され、当駅跡に搬入された。総事業費は約1億4000万円であった<ref name = "読売20180331" />。 また復元に合わせて長野原線で使用されていた[[蒸気機関車]]と同形の車両を展示する予定で、車両を保有している[[東日本旅客鉄道]]（JR東日本）と交渉が行われている。<ref>[https://web.archive.org/web/20150402181900/http://mainichi.jp/feature/news/20140404k0000m040130000c.html 旧国鉄・長野原線太子駅:地中のホームも掘り出し復元へ]、毎日新聞 (Internet Archive) </ref> 長野原草津口駅から当駅までの間の廃線跡は、長野原草津口駅を出てすぐにある上路[[橋桁|プレートガーダー]][[橋]]、山間部を抜ける[[トンネル|隧道]]等がそのまま残されている。上述の橋梁から[[国道292号]]と合流するまでの区間の隧道は内部崩落のため通行止めとなっている。国道292号と合流し再度分岐してから先の北側は当駅までの区間を地元の[[抜け道|裏道]]として活用している。該当区間に2箇所ある隧道は[[単線]]規格のため、内部での[[自動車]]の離合はできない。
