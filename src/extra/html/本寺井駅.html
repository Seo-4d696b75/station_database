39</ref>。 == 歴史 == * [[1925年]]（[[大正]]14年）[[3月21日]] - 能美電気鉄道の駅として開業。開業当初から[[閉塞 (鉄道)|閉塞]]取扱駅で、[[駅長]]が常駐していた<ref name="rm"/>。 * [[1926年]]（大正15年）[[6月20日]]（届出）- 構内に貨物上屋を設置<ref name="rm"/>。 * [[1928年]]（[[昭和]]3年）[[11月30日]]（届出）- ホームを砂利敷きから[[コンクリート]]敷きに変更<ref name="rm"/>。 * [[1938年]]（昭和13年）[[10月27日]]（届出）- 構内の車庫を増設<ref name="rm"/>。 * [[1939年]]（昭和14年）[[8月1日]] - 金沢電気軌道の駅となる。 * [[1973年]]（昭和48年）[[8月1日]] - 閉塞の取り扱い及び[[交換設備]]を廃止して[[プラットホーム#単式ホーム|棒線]]化、駅も無人となる。後に単式ホームを線路の北側へ移す<ref name="rm"/>。 * [[1980年]]（昭和55年）[[9月14日]] - 能美線廃止に伴い廃駅。 == 隣の駅 == ; 北陸鉄道 : 能美線 :: [[末信牛島駅]] -
