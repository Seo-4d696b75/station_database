'''越中電気軌道'''富山北口駅 - 四方駅間開通に伴い'''御廟口駅'''（ごびょうくちえき）（初代）として開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref>。新富山駅起点2.1km地点<ref name="bunken2010H-3-R"/>。 * [[1927年]]（昭和2年）[[2月13日]] - 鉄道会社名を'''越中鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1929年]]（昭和4年）[[6月15日]] - 御廟口駅（初代）廃止。0.7km四方方に移設の上御廟口駅（2代目）再開業<ref name="bunken2008M-6"/>（新富山駅起点2.8km地点<ref name="bunken2010H-3-R"/>）。 * [[1929年]]（昭和4年）- [[1930年]]（昭和5年）頃 - '''八ヶ山駅'''に改称<ref name="bunken2008M-6"/>。累計キロを新富山駅起点2.7kmに変更<ref name="bunken2008M-6"/>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]に伴い'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1974年]]（昭和49年）[[4月5日]] - 交換設備廃止<ref name="bunkenRML107-R">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）35ページより。</ref><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）49,51-52ページより。</ref>。 * 時期不詳 - [[日本の鉄道駅#業務委託駅|業務委託]]化<ref name="bunkenRML107">『RM LIBRARY 107』38,41ページより。</ref>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/>。ホームは線路の西側（新港東口方面に向かって左手側、旧下り線）に存在した<ref name="bunkenRML107"/>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。駅舎側ホーム（西側）が下り線、対向側ホーム（東側）が上り線となっていた。使われなくなった旧上り線は、交換設備運用廃止後も[[停車場#側線|側線]]として旧ホームと共に残存していた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた<ref name="bunkenRML107"/>。駅舎は構内の西側に位置し、築堤上にあるためにホームまでは屋根付きのコンクリート製の階段を下りる形で連絡した<ref name="bunkenRML107"/>。三角屋根を持つ[[西洋館|洋館]]風の駅舎であった<ref name="bunken1981TMS">雑誌『[[鉄道模型趣味]]』（[[機芸出版社]]）1981年3月号57ページより。</ref>。ホームは掘割式で旧上り線側ホームに[[待合所]]代わりの上屋を有した<ref name="bunkenRML107"/>。 昭和30年代まで、附近に「八ヶ山遊園地」が営業しており、行楽シーズンなどには当駅発着の[[臨時列車]]も運行されたことがあったという<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）63-64ページより。</ref>。南方には[[呉羽山温泉|富山ヘルスセンター]]があり、学校の遠足などでもよく利用された。 駅の北、八町方には[[変電所#鉄道変電所|変電所]]が存在した<ref name="bunken1997H"/>。 == 駅周辺 == * [[富山県道7号富山八尾線]] * 八ヶ山公園 - 後に廃園。[[サクラ|桜]]の名所であった<ref name="bunkenRML107"/>。 * 呉羽山公園 - [[阿羅漢|五百羅漢]]がある<ref name="bunkenRML107"/>。 * [[神通川]] == 駅跡 == 附近の線路跡は[[富山地方鉄道#バス事業|富山地鉄バス]]専用道路となっており、当駅附近が富山方面への入口となっている<ref name="bunkenKW07"/>。この専用道路は鉄道時代の面影を良く残していた<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]5月発行）108-109ページより。</ref>。1997年（平成9年）時点ではホームは路盤部分が盛り土された形で<ref name="bunkenToyama2008H"/>残存し、旧上り線側ホームの上屋はバス用に転用されている<ref name="bunken1997H"/>。駅舎は撤去されたが（時期不詳）、ホームから駅舎への階段は残存し（但し屋根は撤去された）使用されていた<ref name="bunken1997H"/>。2006年（平成18年）5月時点でも同様で、駅舎跡の改札口も確認できた<ref name="bunkenToyama2008H"/>。2010年（平成22年）時点でも同様であった<ref name="bunkenKW07"/><ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』49-51ページより。</ref>。 2018年(平成30年)時点駅舎の撤去されたものの、2010年(平成22年)同様改札口は確認できた 駅の北方にあった[[変電所#鉄道変電所|変電所]]は1997年（平成9年）時点では残存していたが<ref name="bunken1997H"/>、その後2006年（平成18年）5月から2008年（平成20年）の間までに解体された<ref name="bunkenToyama2008H"/>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[富山北口駅]] -
