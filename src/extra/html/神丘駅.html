[[日本国有鉄道|国有鉄道]]瀬棚線の今金駅 - 丹羽駅間に、無人駅として開業<ref name=":022">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=25}}</ref><ref name="haruka">『道南鉄道100年史 遥』 [[北海道旅客鉄道函館支社]] 2003年2月発行。</ref>。旅客のみ取扱い。 *[[1987年]]（昭和62年）[[3月16日]] - 瀬棚線の全線廃止に伴い、廃駅となる<ref name="haruka" />。 === 駅名の由来 === 所在地名より<ref name=":022" />。 [[1892年]]（明治25年）より[[キリスト教徒]]が入植・開拓した集落が近隣にあり、当初は地名も「[[インマヌエル]]」と称したが、[[1933年]]（昭和8年）に現在の「神丘」に改名を余儀なくされたとされる<ref>[http://nskk-hokkaido.jp/church/imakane.html 今金インマヌエル教会] - [[日本聖公会]]　北海道教区</ref>。現在の地名は「神のいる丘」の意から名づけられた<ref name=":022" /><ref name=":0">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/3140P.pdf|title=アイヌ語地名リスト オニシベ～キタ　P31-40P|accessdate=2017-10-19|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）65ページより。</ref>。ホームは、線路の南側（瀬棚方面に向かって左手側）に存在した。瀬棚方がスロープになっており、道路に連絡した<ref name="bunken1983"/>。 開業時からの[[無人駅]]であった。駅舎は存在せず、地元住人が建築した小さな[[待合所|待合室]]を有していた<ref name="bunken1983"/>。ホームは荒い砂敷きであった<ref name="bunken1983"/>。 == 利用状況 == * 1981年度の1日乗降客数は8人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道936号丹羽今金線]] * 今金町立神丘小学校 * [[後志利別川]] == 駅跡 == 2010年（平成22年）時点でホーム跡が土盛りとして残り、ホームに植えられていた樹木も残っていた。また瀬棚寄りには25キロの速度制限表も残存していた<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）164ページより。</ref>。2011年（平成23年）時点でも同様であった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）77ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 瀬棚線 :: [[今金駅]] -　'''神丘駅''' -
