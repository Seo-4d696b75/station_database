|備考 = }} '''長良橋駅'''（ながらばしえき）は、[[岐阜県]][[岐阜市]]湊町にあった、[[名古屋鉄道]][[名鉄岐阜市内線|岐阜市内線本線]]（長良線<ref>{{Cite book|和書|title=岐阜のチンチン電車 岐阜市内線と美濃町・揖斐・谷汲線の85年 |year=1997 |publisher=[[郷土出版社]] |isbn=4-87670-097-4 |page=31}}</ref>）の[[鉄道駅|駅]]（[[路面電車停留場|停留場]]）である。 駅名は[[長良橋]]に由来する。岐阜市内線本線は起点から当駅までが[[長良川]]の南側（左岸）になる。 == 歴史 == 1912年に岐阜市内線の前身である[[美濃電気軌道]]の駅として開業した。1911年に市内線は[[岐阜駅前駅]]から[[本町駅 (岐阜県)|本町駅]]までを開通させており、当駅はそこから長良川の南岸へ路線を伸ばしたのに合わせて開設された駅である。当初は終着駅であったが、1915年に路面電車の走行に対応した新しい[[長良橋]]の架橋が成ると<ref>{{Cite web |url=http://www.meitetsu.co.jp/recommend/library/exhibition/detail/1241594_5073.html |title=廃止から10年・岐阜の電車（平成27年 春季特別展） |accessdate=2016-07-09 |work=名鉄のオススメ |publisher=名古屋鉄道}}</ref>、当駅から橋を渡ってさらに延伸、すでに川の北岸で開通していた[[名鉄高富線|高富線]]と[[長良北町駅]]にて接続した<ref name=kyodo21>{{Cite book|和書|title=岐阜のチンチン電車 岐阜市内線と美濃町・揖斐・谷汲線の85年 |year=1997 |publisher=[[郷土出版社]] |isbn=4-87670-097-4 |page=21-24}}</ref>。 当駅からこの長良北町駅までの区間は単線であったが、戦後に長良橋が下流に新設されると線路を移設し[[複線]]化。これをもって岐阜市内線は全線で複線化が完了した<ref name=kyodo21/>。しかし自動車の普及により市内線は窮地に立たされ、不振が続いていた[[徹明町駅]] - 長良北町駅間は1988年に廃止される。当駅もこれに合わせて[[廃駅]]となった。 * [[1912年]]（大正元年）[[8月28日]] - 美濃電気軌道市内線の[[本町駅 (岐阜県)|本町駅]] - 当駅間の開通と同時に開業<ref name=kyodo218>{{Cite book|和書|editor= |title=岐阜のチンチン電車 岐阜市内線と美濃町・揖斐・谷汲線の85年 |year=1997 |publisher=[[郷土出版社]] |isbn=4-87670-097-4 |pages=218-230}}</ref>。 * [[1915年]]（大正4年）[[11月20日]] - 市内線が当駅から[[長良北町駅]]まで延伸<ref name=kyodo218/>。 * [[1957年]]（昭和32年）[[3月31日]] - 当駅から長良北町駅までの複線化が完了<ref name=kyodo218/>（1956年の[[長良橋]]架け替え完成による）。 * [[1988年]]（昭和63年）[[6月1日]] - 岐阜市内線の徹明町駅 - 長良北町駅間の廃止に伴い廃駅となる<ref name=kyodo218/>。 == 駅構造 == *
