当駅間が廃止となり、これにより当駅も廃駅となった<ref name=rp7011/>。 越前本線の勝山側と大野側の間には下荒井トンネルがあり、このトンネルは幅が狭いことから通過できる車両が限られていた。勝山 - 当駅間は、[[京福電気鉄道#京福電気鉄道設立以後入線|251形]]電車などの比較的小型の車両しか使用されなかった<ref>勝山 - 当駅間廃止直前に導入が始まった[[京福電気鉄道モハ3001形電車|モハ3001形電車]]も、下荒井トンネルの通過が不可能だった。廃線が決まったために導入に踏み切ったといういきさつがあるとされる（『鉄道ピクトリアル』2001年5月臨時増刊号（通巻第701号） 北陸地方のローカル私鉄特集 p.112）。</ref>。 == 駅構造 == 3面3線の頭端式ホームを有していた<ref name=rp7013>『鉄道ピクトリアル』2001年5月臨時増刊号（通巻第701号） 北陸地方のローカル私鉄特集 p.177</ref>。1番線には、付随車牽引時の電動車付け替えに使用するため、2番線との間に機回し用渡り線があった<ref name=rp7013/>。駅舎は昭和31(1956)年に鉄筋コンクリート造りに建て替えられ、2階はスーパー京福マートであった。 == 歴史 == * [[1918年]]（[[大正]]7年）[[9月1日]] -
