[[日本国有鉄道]]湧網東線の常呂駅 - 当駅間延伸開通に伴い、'''下佐呂間駅'''（しもさろまえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1953年]]（昭和28年）[[10月22日]] - 佐呂間駅 - 当駅間の延伸開通に伴い、中間駅となる。また、中湧別駅 - 網走駅間全通に伴い路線名を'''湧網線'''に改称し、それにより同線の駅となる。 * [[1963年]]（昭和38年）[[10月1日]] - '''浜佐呂間駅'''に改称。 * [[1972年]]（昭和47年）[[2月8日]] - 貨物の取り扱いを廃止。 * [[1982年]]（昭和57年）[[3月29日]] - [[チッキ|荷物]]の取り扱いを廃止し、同時に無人駅（[[日本の鉄道駅#簡易委託駅|簡易委託]]駅）化（駅近隣の個人）。 * [[1987年]]（昭和62年）[[3月20日]] - 湧網線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは線路の西側（網走方面に向かって左手側）に存在した<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=161}}</ref>。そのほか、本線の網走方から分岐し駅舎北側への[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。かつては2面2線の[[相対式ホーム]]を有する、[[列車交換]]可能な[[交換駅]]であった。1983年（昭和58年）時点では使われなくなった対向ホーム側の1線は交換設備運用廃止後も[[分岐器|転轍機]]が中湧別方、網走方の両方向とも撤去された形で側線として残っていた（ホームも撤去されていた）<ref name="bunken1983"/>。ホーム前後の線路は、転轍機の名残で湾曲していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の東側に位置し、ホームに接していた<ref name="bunken1983"/>。[[木構造 (建築)|木造]][[モルタル]]で白く[[塗装]]された駅舎であった<ref name="bunken1983"/>。駅自体は完全無人であったが、駅近隣の個人が[[乗車券#鉄道|乗車券]]を取り扱う簡易委託駅となっていた。受託者は駅附近の[[民宿]]経営者で、駅舎内で宿泊案内を行う場合もあったという<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在していた地名より。地名は、佐呂間より浜に近いために<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=169}}</ref>「浜」を冠した。 旧駅名の下佐呂間（しもさろま）は、佐呂間より佐呂間別の下流に位置していたため<ref name="bunken2004N"/>、「下」を冠した。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は79人<ref name="bunken1983"/>。 == 駅周辺 == 駅前には農協ストアや商店街があった<ref name="bunken1983"/>。 * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=19}}</ref> * [[北海道道103号留辺蘂浜佐呂間線]] * [[北海道道858号キムアネップ岬浜佐呂間線]]<ref name="bunken1980M"/> * 佐呂間大橋（国道238号） * 郷土資料館 - 駅から北東に約5.7km<ref name="bunken1983"/>。 * 佐呂間別川<ref name="bunken1980M"/> * [[サロマ湖]] - 駅から北に約0.7km<ref name="bunken1983"/><ref name="bunken1980M"/>。 * キムアネップ岬 - 駅から北西に約6.5km<ref name="bunken1983"/><ref name="bunken1980M"/>。[[ハマナス]]や[[アッケシソウ|サンゴ草]]の[[原生花園]]。 * [[佐呂間町ふれあいバス]]「浜佐呂間」停留所 == 駅跡 == 駅舎と駅構内施設は早々と撤去され、駅跡地には幾つかの町有施設が新たに整備されており、駅や周辺の線路跡は見る影もなくなっている。 [[1999年]]（平成11年）時点では当駅跡から仁倉駅寄りのイワケシュケコマナイ川に架かっていた橋のコンクリートの橋台が残存しており<ref name="bunken1999H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVI |date=1999-3 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533031502 |pages=31-32}}</ref>、[[2010年]]（平成22年）時点<ref name="bunken2010H">{{Cite book|和書|editor=今尾恵介 |title=新 鉄道廃線跡を歩く1 北海道・北東北編 |date=2010-3 |publisher=JTBパブリッシング |isbn=978-4533078583 |pages=52-53}}</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H">{{Cite book|和書|author=本久公洋|title=北海道の鉄道廃線跡|date=2011-9|publisher=[[北海道新聞社]]|pages=104-105|isbn=978-4894536128}}</ref>。理由は不明だが片側のみであった<ref name="bunken2011H"/>。 == 隣の駅 == ; 日本国有鉄道 : 湧網線 :: [[仁倉駅]] -
