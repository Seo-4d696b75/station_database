'''越中電気軌道'''富山北口駅 - 四方駅間開通に伴い'''今市駅'''（いまいちえき）として開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref>。 * [[1927年]]（昭和2年）[[2月13日]] - 鉄道会社名を'''越中鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1929年]]（昭和4年）頃 - '''鯰鉱泉前駅'''に改称<ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年12月発行）49,52ページより。</ref>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]に伴い'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、ネコ・パブリッシング、2008年7月発行）38-41ページより。</ref>。ホームは線路の東側（新港東口方面に向かって右手側）に存在した<ref name="bunkenRML107"/>。 [[無人駅]]となっていた<ref name="bunkenRML107"/>。駅舎はないがホーム中央部分に[[待合所]]を有した<ref name="bunkenRML107"/>。 == 駅周辺 == * [[富山県道207号四方新中茶屋線]] * [[鯰温泉]] - 駅を挟んで<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）64ページより。</ref>2軒の[[温泉宿|温泉旅館]]が並んでいた<ref name="bunkenRML107"/>。 * 富山市立倉垣小学校 == 駅跡 == 1997年（平成9年）時点では、ホームの残骸らしき土盛りが残存していた<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]5月発行）108-109ページより。</ref>。2006年（平成18年）5月時点でも同様で、[[待合所]]のコンクリートの基礎も確認できた<ref name="bunkenToyama2008H"/>。2010年（平成22年）時点でも同様であった<ref name="bunkenKW07"/><ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』49-51ページより。</ref>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[布目駅]] -
