[[日本国有鉄道]]名寄本線の'''四号線仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。同時に駅に昇格。'''四号線駅'''となる。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の北西側（湧別方面に向かって左手側）に存在した<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=138}}</ref>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。 [[仮乗降場]]に出自を持つ[[無人駅]]で、駅舎および[[待合所]]も存在しなかった<ref name="bunken2011P"/><ref>{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |page=114}}</ref>。ホームは遠軽方に[[斜路|スロープ]]を有し<ref name="bunken2011P"/>駅施設外に連絡していた。板張りのホームであった。 == 駅名の由来 == 当駅の所在地の昔の地域名より<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=188}}</ref>。「四号線」の名は、[[開拓使|開拓]]期に碁盤状に設けられた道路の名に因む。また、当駅遠軽方を東西に通る道路が「東四号線」「西四号線」と名付けられていた<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M"/> * 湧別小学校<ref name="bunken1980M"/> * 湧別中学校<ref name="bunken1980M"/> * 湧別錦町簡易郵便局<ref name="bunken1980M"/> * [[エーコープ|Aコープ]]ゆうべつ店 * 湧別大橋<ref name="bunken1980M"/> * [[湧別川]]<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「湧別4号線」、[[湧別町営バス]]「4号線」[[バス停留所]] == 駅跡 == [[2001年]]（平成13年）時点で[[サイクリング]]の休憩場所である「リララの広場」となっていた<ref name="bunken2001H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |pages=36}}</ref>。[[2011年]]（平成23年）時点でも同様で、散歩者用の休憩場所でもあった<ref>{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |page=128}}</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線（支線） :: [[中湧別駅]] -
