江若鉄道の大溝 - 安曇間の延伸開業に伴い、高島郡水尾村に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957195/6}} 「地方鉄道運輸開始」『官報』1929年6月6日]（国立国会図書館デジタル化資料）</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=水尾駅線路配置図|left=[[高島町駅 (滋賀県)|高島町]]}} {{鉄道配線図|voie|voie|voie|voie|voie|voie|voie|voie|voie}} {{鉄道配線図|0|Dvoie|0|quai|quai|quai|0|Mvoie|0}} {{鉄道配線図|0|0|courbehg|voie|voie|voie|courbehd|0|0}} {{鉄道配線図|0|0|0|0|0|0|0|BV|0}} {{駅配線図終了|right=[[安曇川駅 (江若鉄道)|安曇川]]|source={{Sfn|竹内|1967}}{{Sfn|レイル|p=81}}|note=本線とつながっていない線路は撤去済みを示す}} 水尾駅は旅客と貨物の両方を取り扱うことができた[[一般駅]]{{Sfn|寺田|2010|p=10}}。[[プラットホーム|ホーム]]は線路の片側（[[近江今津駅 (江若鉄道)|近江今津]]方面に向かって右側）に1面のみ設けられていた（[[単式ホーム]]）{{Sfn|レイル|p=81}}{{Sfn|レイル|p=85}}。当初は[[列車交換]]が可能な駅だったが、交換設備が撤去された結果棒線駅となっている{{Sfn|寺田|2010|p=10}}{{Sfn|寺田|2010|p=15}}。 駅に設置されていた[[駅名標]]では、駅名が「み'''づ'''お」と表記された{{Sfn|レイル|p=84}}。 == 利用状況 == 初期の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1931年||15,300人||15,114人||115トン||729トン||<ref>[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料）</ref> |- |1934年||17,608人||17,567人||111トン||600トン||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||16,507人||16,555人||149トン||608トン||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||15,456人||15,650人||90トン||415トン||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅は田園地帯の中にあり{{Sfn|レイル|p=85}}、開業当時は民家が3軒あった{{Sfn|江若鉄道の思い出|p=98}}。駅があった場所は石材店に変わり{{Sfn|ありし日の江若鉄道|p=22}}、周囲には工場やアパートが立地している{{Sfn|江若鉄道の思い出|p=98}}。 当駅付近の線路跡は江若鉄道の廃線後に開通した[[湖西線]]の建設に利用され、新しく高架橋が築かれた{{Sfn|江若鉄道の思い出|p=99}}。この高架橋はもとあった江若鉄道のものより高さがあったため、周辺の住宅では[[電波障害]]が発生したという{{Sfn|江若鉄道の思い出|p=99}}。湖西線は線路跡をなぞるように伸びていて、江若鉄道の遺構はほぼ残っていない{{Sfn|ありし日の江若鉄道|p=22}}{{Sfn|寺田|2010|p=26}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[高島町駅 (滋賀県)|高島町駅]] -
