のと鉄道七尾線3]</ref>。構内には長らく昔ながらの[[鉄道信号機#機械式信号機|腕木式信号機]]が残され、運転士が[[閉塞 (鉄道)#タブレット|タブレット]]を交換する光景が見られたが<ref name="A">寺田裕一『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』、JTBパブリッシング、2008年、139頁。ISBN 978-4-533-07145-4</ref>、[[1998年]]12月のダイヤ改正で列車交換を廃して[[単式ホーム]]1面1線のみを使用するようになり、同時に駅は無人化された。ちなみに、信号機が自動化されたのは列車交換が廃止される直前で、新しく設置された色灯式の信号機はほとんど使われることのないまま使用停止となっている<ref>[http://www.kysd.net/now.html ---The Ruins of Rail---廃線跡を旅する - 地方鉄道の現実]</ref>。廃止時まで使用されなくなったもう片側のホームおよび線路はそのまま残されていた。 == 駅廃止後 == 廃止後しばらくはレールが残され、地元住民がそれを活用して[[トロッコ]]を走らせて話題になったが、[[2004年]]にレールは撤去され<ref>[http://www15.ocn.ne.jp/~nanao/special/noto_railway.html 能登畠山氏七尾の歴史/不定期特集/能登の鉄道網]</ref>、保存活動もいつの間にか行われなくなった。[[2008年]]にはホームも撤去され<ref>[http://ameblo.jp/notohanto/entry-10166050161.html 奥能登の魅力発見！！ ～石川県輪島市三井町を中心に～The charm discovery of okunoto!!　奥能登移住生活 - 能登三井駅]</ref>、現在残るのは駅舎のみとなってしまっている。 駅舎には[[JR]]時代から[[喫茶店]]「サロン社の駅」が入居しており、廃線後も営業していたが、[[2007年]][[3月20日]]をもって閉店<ref name=A/>。現在駅舎はバス待合所として使用されているほか、残りのスペースを新たに喫茶店「駅カフェみどり」が入居し、営業している。 駅前には、かつて駅前[[商店街]]があり昭和40年代には駅前の通りに30軒以上の店舗が並んでいたが、駅廃止後は多くの店が閉店し2020年現在は数店舗が営業するのみである<ref>『三井駅 にぎわいもう一度 輪島 「廃線20年」前に軽トラ市』(2020年11月14日付け北國新聞朝刊25面)2020年11月14日閲覧</ref>。 == 路線バス == {| cellspacing="0" cellpadding="3" frame="hsides" rules="rows" style="border-top:double 3px black; border-bottom:double 3px black;" |- |colspan="3"|'''三井駅前''' |- |rowspan="3"|[[北鉄奥能登バス]] |三井線: |上与呂見行 |- |穴水輪島線: |[[道の駅輪島|輪島駅前]]・塚田行き、[[穴水駅|穴水]]・穴水病院口行き |- |[[能登方面特急バス|輪島特急線]]: |輪島駅前・輪島マリンタウン行き、[[金沢駅]]西口・[[兼六園|兼六園下]]行き |- |} *輪島特急線は、旧能登三井駅前のバス停ではなく、[[石川県道1号七尾輪島線|県道七尾輪島線]]三井バイパス沿いに設置されたバス停（駅跡から徒歩数分）に発着する。 *穴水輪島線と輪島特急線の一部は[[能登空港]]を経由する。 == 歴史 == * [[1935年]]（[[昭和]]10年）[[7月30日]] -
