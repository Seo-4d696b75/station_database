神威駅-歌志内駅間に、仮乗降場として開設？<ref name="shinshishi">新歌志内市史 P1084。</ref>。 * [[1960年]]（昭和35年）[[12月26日]] - 民間委託駅として<ref name="shinshishi"/>正式開業<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年10月 JTB編集・発行。</ref>。旅客駅<ref name="hensen"/>。 * [[1979年]]（昭和54年）11月 - 駅舎改築<ref name="shinshishi"/>。 * [[1986年]]（昭和61年）11月 - [[乗車券]]の発売を取りやめ、無人駅化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承<ref name="hensen"/>。 * [[1988年]]（昭和63年）[[4月25日]] - 歌志内線の[[廃線]]（全線廃止）に伴い、[[廃駅]]となる<ref name="hensen"/>。 == 駅構造 == 廃止時点では、単式ホーム1面1線を有する[[無人駅]]であった。当初は踏切側に2階建ての駅舎を有していた<ref>歌志内市史 P628 写真参照。</ref>が、後に片流れ屋根の平屋の駅舎を踏切から少し奥まった位置に建て直している<ref>新歌志内市史 P1084 写真参照。及び[http://mapps.gsi.go.jp/contentsImageDisplay.do?specificationId=327617&isDetail=true 1987年撮影航空写真（国土地理院 地図・空中写真閲覧サービス）]参照。</ref>。近隣の商店に乗車券の販売を委託した[[日本の鉄道駅#簡易委託駅|簡易委託駅]]であった。 == 駅名の由来 == 当駅が置かれていた地区名を採用した。「歌神」は、1903年（明治36年）4月に歌神炭礦<ref group="注">ちなみにこの炭鉱は、歌志内駅の積込み場まで約600間（およそ1km）の馬車軌道を敷設して運炭していた。</ref>がこの地に開坑したことが始まり。この炭砿名は、歌志内と神威の中間にあることにより名付けられた<ref>歌志内市史 P132。</ref><ref group="注">明治39年からの二級町村制時代の部長制度下では歌神炭礦部、大正8年7月の一級町村制以降から歌神地区となった（歌志内市史 P154-190。）。</ref>。 == 周辺施設 == * [[北海道道114号赤平奈井江線]] * [[かもい岳国際スキー場]] * [[北海道中央バス]]「歌神」停留所 == 隣駅 == ; 北海道旅客鉄道 : 歌志内線 :: [[神威駅]] -
