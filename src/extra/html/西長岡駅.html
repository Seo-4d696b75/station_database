与板間の開通時に、長岡鉄道の駅として開業。 * [[1921年]]（大正10年）[[11月18日]] : 来迎寺 - 当駅間延伸開業。 * [[1951年]]（昭和26年）12月 : 当駅 - 寺泊間[[鉄道の電化|電化]]。 * [[1952年]]（昭和27年） : 来迎寺 - 当駅間が電化され、全線電化。 * [[1960年]]（昭和35年） : [[日産化学工業]]長岡工場へ向かう[[専用鉄道|専用線]]が運輸開始。 * 1960年（昭和35年）[[10月1日]] : 越後交通が発足し、同社の長岡線の駅となる。 * [[1972年]]（昭和47年） : 長岡工場閉鎖に伴い、日産化学工業専用線廃止。 * 1972年（昭和47年）[[4月16日]] : 来迎寺 - 当駅間の旅客営業廃止。 * [[1975年]]（昭和50年）[[4月1日]] : 当駅 - 越後関原間の旅客営業廃止に伴い、駅の旅客営業を廃止。[[鉄道駅#貨物駅|貨物駅]]に。 * [[1992年]]（平成4年）4月1日 : 当駅 - 越後関原間休止。 * [[1993年]]（平成5年）[[3月31日]] : 当駅 - 越後関原間廃線。 * [[1995年]]（平成7年）4月1日 : 来迎寺 - 当駅間廃線に伴い、廃止。 == 駅跡地 == 駅舎や車庫は全て取り壊され、レールや架線柱も撤去された。跡地にはクスリのアオキ大島店や住宅等が建設されている。 == 隣の駅 == ; 越後交通 : 長岡線 :: [[才津駅]] - '''西長岡駅''' - [[日越駅]] : 1973年4月16日までは、才津駅との間に[[有栗駅]]があった。 : 1975年4月1日までは、日越駅との間に[[西中学校駅]]があった。 == 関連項目 == * [[廃駅]] == 外部リンク == *[{{NDLDC|948350/15}} 『長岡鉄道案内』]（国立国会図書館デジタルコレクション）大正5年の長岡市地図　駅は市街とは信濃川を隔てている {{越後交通長岡線}} {{Eki-stub}} {{DEFAULTSORT:にしなかおか}} [[Category:長岡市の廃駅]] [[Category:日本の鉄道駅 に|しなかおか]] [[Category:越後交通の鉄道駅]] [[Category:1916年開業の鉄道駅]] [[Category:1995年廃止の鉄道駅]] lsdi7lph0xhq07rjlek6amjpgmc8hl9
