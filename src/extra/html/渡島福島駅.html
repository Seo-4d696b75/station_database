[[渡島吉岡駅]]間の開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として開業。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：福山線が、路線名を'''[[松前線]]'''に変更。 * [[1963年]]（昭和38年）[[12月1日]]：[[準急列車|準急]]「'''松前'''」運行開始。 * 時期不詳<ref group="注" name="ekigyoumu">[[1981年]]（昭和56年）[[4月]]時点では既に業務委託化されている（書籍『コロタン文庫36 国鉄駅名全百科 56.4訂補版』（[[小学館]]、[[1981年]][[10月]]発行）61ページより）。</ref>：[[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1980年]]（昭和55年）[[10月1日]]：急行「松前」廃止により、[[優等列車]]の停車がなくなる。 * [[1982年]]（昭和57年）[[11月15日]]：貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]]：[[チッキ|荷物]]取扱い廃止。 * [[1986年]]（昭和61年）11月1日：業務委託を廃止し、[[日本の鉄道駅#簡易委託駅|簡易委託駅]]となる。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]に伴い、[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）[[2月1日]]：松前線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]片面使用の1面1線を有する[[地上駅]]であった。使われなくなった駅舎側の1線は、[[交換駅|交換設備]]運用廃止後も[[分岐器|転轍機]]が撤去された状態で[[停車場#側線|側線]]として残っていた。ホームは、線路の南側（松前方面に向かって左手側）に存在した。かつては島式ホーム1面2線を有する、[[列車交換]]可能な[[日本の鉄道駅#直営駅|職員配置駅]]であった。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた。駅舎はホームの南側に位置し、[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。古い木造駅舎で、屋根は[[津軽海峡]]からの[[海風]]により[[錆]]だらけであったという<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）62ページより。</ref>。 == 駅名の由来 == {{節スタブ}} 当駅が所在する地名に、[[旧国名#駅名|旧国名]]の「渡島」を付する。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は392人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道532号岩部渡島福島停車場線]] * [[国道228号]] * 福島町役場 * [[松前警察署]]福島交番 * 福島郵便局 * [[道南うみ街信用金庫]]福島支店 * 福島町立福島小学校 * 福島町立福島中学校 * [[福島町青函トンネル記念館]] * [[道の駅横綱の里ふくしま]] * 福島大神宮 -
