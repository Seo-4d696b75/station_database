老方駅間は、冬季は[[雪]]害による運転休止が多く、戦後の燃料費高騰の影響もあり不採算区間とされ、営業休止を経て廃止となった<ref name="bunkenRML061">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）11,18-19,24-25ページより。</ref>。 * [[1930年]]（昭和5年）[[10月5日]] - '''横荘鉄道'''二井山駅 - 老方駅間延伸開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]]8月発行）25-28,30-31ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1944年]]（昭和19年）6月1日 - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1946年]]（昭和21年）夏 - [[日本国有鉄道|国鉄]]から借入の[[国鉄C12形蒸気機関車|C12形蒸気機関車]]C12 229号機牽引の老方駅発試運転列車が当駅発車後に過速暴走、八沢木駅手前で停車する<ref name="bunkenRML061"/>。 * [[1947年]]（昭和22年）7月23日 - 豪雨による路盤および橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）16-17ページより。</ref>。 * [[1948年]]（昭和23年）11月8日 - 運休区間（舘合駅 - 老方駅間）が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1949年]]（昭和24年）12月22日 - 二井山駅 - 老方駅間営業休止に伴い[[休止駅]]となる<ref name="bunkenKW03"/>。 * [[1952年]]（昭和27年）2月15日 - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通横荘線'''の駅となる（営業休止中）<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1953年]]（昭和28年）8月5日 - 二井山駅 - 老方駅間部分廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML061"/>。ホームは線路の北側（老方方面に向かって右手側）に存在した<ref name="bunkenRML061"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML061"/>。 [[無人駅]]となっており<ref name="bunkenRML061"/><ref name="bunken2008SH-1-R"/>、駅舎は無いがホーム中央部分に[[待合所]]を有した<ref name="bunkenRML061"/>。 旅客のみ取扱いであったとされる<ref name="bunken2008SH-1-R"/>が、かつては[[森林鉄道]]を思わせる木材輸送の「途中積み」の中心駅であったと推定されている<ref name="bunkenRML061"/>。線路脇に伐採された原木を積み上げ、[[無蓋車]]に積載していたとされる<ref name="bunkenRML061"/>。作業は夕刻から始まり、場合によっては夜通しとなったらしい<ref name="bunkenRML061"/>。二井山駅 - 老方駅間の運行休止に伴い、この作業も終焉を迎えたと考察されている<ref name="bunkenRML061"/>。 == 駅周辺 == 二井山駅 - 老方駅間は人家のほとんど無い山間部を縫うように通っており、廃止後では同区間転用の道路沿線において初めての人家のある地域であった<ref name="bunken2008SH-1"/>。 * [[秋田県道48号横手東由利線]] - 同線の二井山地区 - 老方地区の大部分は横荘線の線路跡がそのまま転用されていた<ref name="bunken2008SH-1"/>。 == 駅跡 == 2007年（平成19年）5月時点では、道路に転用された浮蓋トンネルを老方方に抜けた附近に駅跡が推定できた<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 また、二井山駅跡附近から当駅跡附近を含む老方の集落入口附近までの線路跡は県道として再利用され、1996年（平成8年）時点では[[盛土|築堤]]や[[切通し]]がほぼ当時の姿を保っていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]]9月発行）34-35ページより。</ref>。1999年（平成11年）時点でも同様であった<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）61ページより。</ref>。この県道は2007年（平成19年）5月時点では[[秋田県道48号横手東由利線]]であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。道床をそのまま舗装したような道路であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）202-203ページより。</ref>。 なお、二井山駅 - 老方駅間には5か所の[[トンネル]]があり、1996年（平成8年）時点では崩落の危険のある最も二井山駅方の「二井山トンネル」（全長190m）は通行禁止となっていたが、そのほかの4つ、「御岳トンネル」（全長92m）、「金屋トンネル」（全長86m）、「浮蓋トンネル」（全長140m）、「第一柴倉トンネル」（全長200m）の各トンネルは道路トンネルとしてそのまま再利用されていた<ref name="bunken1996H"/>。「二井山トンネル」も坑口が木々に覆われた形で残存していた<ref name="bunken1996H"/>。1999年（平成11年）時点でも同様<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）61ページより。</ref>、2007年（平成19年）5月時点でも道路トンネルとなった4つは同様で<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）10月時点でも5つとも同様であった<ref name="bunken2010H-1"/>。 == 隣の駅 == ; 羽後交通 : 横荘線 :: [[二井山駅]] -
