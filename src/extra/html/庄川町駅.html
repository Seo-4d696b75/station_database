[[砺波鉄道]]の開業により'''青島町駅'''として開業<ref name=":0">『官報』（13頁）、1915年（大正4年）8月2日、内閣印刷局</ref>。 * [[1919年]]（大正8年）[[9月17日]] - [[加越鉄道]]の駅となる<ref name=":1">今尾恵介監修、『日本鉄道旅行地図帳 全線・全駅・全廃線 6号』（33頁）、2008年（平成20年）10月、新潮社</ref>。 * [[1943年]]（昭和18年）[[1月1日]] - [[富山地方鉄道]]の駅となる<ref name=":1" />。 * [[1950年]]（昭和25年）[[10月23日]] - 加越能鉄道の駅となる<ref name=":1" />。 * [[1955年]]（昭和30年） ** 1月1日 - '''庄川町駅'''に改称<ref name=":0" />。 ** 1月12日 - 構内の機関庫で火災が発生。気動車1両が焼失<ref name="tyo">『庄川町史 下巻』1975年、384-385頁</ref>。 * [[1957年]]（昭和32年）1月26日 - 構内の機関庫で火災が発生。機関車1両気動車2両が焼失<ref name="tyo"/>。 * [[1958年]]（昭和33年） ** 2月12日 - 機関庫を復旧する<ref name=":2">加越線資料保存会編、『[http://www.sankyo-son.jp/open_imgs/info/0000000515.pdf 加越線物語～あの頃加越線が走っていた～]』（17頁）、2013年（平成25年）3月、となみ野田園空間博物館推進協議会</ref>。 ** 9月28日 - 富山国体開催（剣道会場）を機に駅舎を鉄筋コンクリート造りに改築<ref name="tyo" /><ref name=":2" />。 *[[1965年]]（昭和40年）8月2日 - 加越線管理事務所を福野駅前から当駅へ移転する<ref name=":3">服部重敬、『RM LIBRARY 219 加越能鉄道加越線―庄川水力電気専用鉄道―』（21及び47頁）、2017年（平成29年）11月、ネコ・パブリッシング</ref><ref name=":4">富山地方鉄道株式会社編、『富山地方鉄道五十年史』（610頁）、1983年（昭和58年）3月、富山地方鉄道</ref>。 *[[1967年]]（昭和42年）5月 - 車扱貨物の取扱を廃止する<ref>服部重敬、『RM LIBRARY 219 加越能鉄道加越線―庄川水力電気専用鉄道―』（12頁）、2017年（平成29年）11月、ネコ・パブリッシング</ref>。 *[[1972年]]（昭和47年）[[9月16日]] - 加越線の廃止により廃駅<ref name=":1" />。 == 駅構造 == 2面3線を有する地上駅であったが、利用客の減少とともに1面のホームの片側のみが使われるようになった<ref name=":5">服部重敬、『RM LIBRARY 219 加越能鉄道加越線―庄川水力電気専用鉄道―』（21頁）、2017年（平成29年）11月、ネコ・パブリッシング </ref>。駅構内には機関庫があり、1965年（昭和40年）からは管理事務所も置かれていた<ref name=":5" /><ref name=":4" />。駅舎は1958年（昭和33年）にブロック積みのコンクリート造りに改築された<ref name=":2" /><ref name=":3" />。構内には1953年（昭和28年）11月に廃車届が出されたハフ10形11やハフ20形21が留置され、倉庫代わりに用いられていた<ref>服部重敬、『RM LIBRARY 219 加越能鉄道加越線―庄川水力電気専用鉄道―』（41頁）、2017年（平成29年）11月、ネコ・パブリッシング</ref>。 == 庄川水力電気専用線 == 1926年（大正15年）1月15日に庄川水力電気が小牧発電所建設の為に青島町駅より発電所工事現場まで専用線の運転を開始した<ref name=":1" />。この資材輸送により青島町駅は活況を呈していた<ref>[{{NDLDC|1087216/6}} 『ダム式発電所写真帖』]（国立国会図書館デジタルコレクション）</ref>。小牧発電所は1930年（昭和5年）11月に発電を開始したが<ref>富山県編、『富山県史 年表』（308頁）、1987年（昭和62年）3月、富山県</ref>、専用線は1939年（昭和14年）10月8日まで使用されていた<ref name=":1" />。1951年（昭和26年）12月15日に加越能鉄道はこの廃線敷を使用して延長線の免許を取得したが<ref>[{{NDLDC|2964072/8}} 「運輸省告示第27号」『官報』1952年2月2日]（国立国会図書館デジタルコレクション）</ref>、実現せずに1960年（昭和35年）9月20日に免許失効となった<ref name=":2" />。 <gallery> 庄川水力電気株式会社小牧発電所.jpg|小牧発電所 Aoshimamachi Station in 1926.jpg|青島町駅における庄川水力専用鉄道と材料倉庫 Aoshimamachi Station in 1926(1).jpg|青島町駅からの専用線と貯木場 </gallery> == 乗客・貨物の推移 == {| class="wikitable" border="1" cellspacing="0" cellpadding="2" style="font-size:90%; text-align:center; width:60%;" |- !年度 !乗車人員 !降車人員 !貨物発送トン !貨物到着トン |- |1918||20,389||19,294||7,576||1,663 |- |1922||34,342||27,287||9,576||2,410 |- |1923||37,821||27,428||10,673||2,721 |- |1926||78,747||58,956||9,266||27,878 |- |1936||76,017||49,593||2,166||4,935 |- |} *富山県統計書各年度版より == 隣の駅 == ; 加越能鉄道 : 加越線 :: [[東山見駅]] -
