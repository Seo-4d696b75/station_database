開業{{Sfn|今尾|2008|p=32}}。 * [[1959年]]（昭和34年）[[8月9日]] - 饗庭野で開かれた第2回ボーイスカウト[[日本ジャンボリー]]の視察のため、当時の[[皇太子明仁親王]]が宿泊地の[[琵琶湖ホテル]]の最寄りであった当駅を利用する{{Sfn|江若鉄道の思い出|p=22}}{{Sfn|レイル|p=82}}。特別列車を当駅と[[近江今津駅 (江若鉄道)|近江今津駅]]との間で運転{{Sfn|レイル|p=82}}。 * [[1964年]]（昭和39年）[[3月20日]] - 駅廃止{{Sfn|今尾|2008|p=32}}。 == 駅構造 == 当駅は旅客のみを扱う[[旅客駅]]で{{Sfn|寺田|2010|p=10}}、びわこ競輪の開催日のみ取り扱いを行う[[臨時駅]]だった{{Sfn|寺田|2010|p=15}}。[[プラットホーム|ホーム]]は[[枕木]]などを土台とする簡単な造りで{{Sfn|江若鉄道の思い出|p=22}}、線路の西側（[[近江今津駅 (江若鉄道)|近江今津]]方面に向かって左側）に設けられていた{{Sfn|江若鉄道の思い出|p=23}}。駅舎のような設備は設けられなかったとされる{{Sfn|江若鉄道の思い出|p=22}}。 == 駅周辺 == 駅があった場所は[[京都府道・滋賀県道30号下鴨大津線|滋賀県道30号]]（[[近江神宮|神宮]]道）の二本松交差点の北側に相当する{{Sfn|江若鉄道の思い出|p=22}}。沿線は田園が広がり、[[大津びわこ競輪場]]は駅の東側{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=79}}{{Sfn|江若鉄道の思い出|p=23}}。{{要出典範囲|当駅の廃止後、競輪が開催される際にはJR[[琵琶湖線]]の[[大津駅]]および[[京阪大津線]]の[[びわ湖浜大津駅|浜大津駅]]から、競輪場最寄りの柳が崎まで[[京阪バス]]（もしくは[[江若交通]]）による[[無料送迎バス]]が運行されていた|date=2017年9月}}。競輪場については2011年（平成23年）3月に開催廃止となっている{{Sfn|江若鉄道の思い出|p=23}}。 当駅を含む[[三井寺下駅]]から[[滋賀駅]]までの線路跡は道路に転用されていて、大津市南志賀四丁目付近で江若鉄道の廃止後に開業した[[湖西線]]の高架橋に行き当たる{{Sfn|江若鉄道の思い出|p=23}}{{Sfn|寺田|2010|p=24}}{{Sfn|吉田|1998|p=186}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[三井寺下駅]] -
