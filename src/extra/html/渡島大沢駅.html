当駅間開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として開業。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：当駅 - [[松前駅 (北海道)|松前駅]]間開通に伴い、中間駅となる。同時に、福山線が'''[[松前線]]'''に改称。 * [[1961年]]（昭和36年）[[11月10日]]：貨物の取り扱いを廃止。 * [[1962年]]（昭和37年）[[7月1日]]：[[日本の鉄道駅#業務委託駅|業務委託駅]]となる。 * [[1970年]]（昭和45年）[[12月12日]]：[[日本の鉄道駅#簡易委託駅|簡易委託駅]]となり、事実上、無人化。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]に伴い、[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）[[2月1日]]：松前線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の東側（松前方面に向かって右側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）63ページより。</ref>。 [[無人駅]]（[[日本の鉄道駅#簡易委託駅|簡易委託駅]]）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた<ref name="bunken1983"/>。駅舎はホームの東側に接していた。 松前線廃止前は北海道最南端の駅であったが、無人駅であったこともありそれを示す標識のようなものはなかった。 （2016年現在の最南端の駅は[[木古内駅]]。） == 駅名の由来 == {{節スタブ}} 当駅が所在した地の近辺の地名（大沢）に、[[旧国名#駅名|旧国名]]の「[[渡島国|渡島]]」を付した。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は44人<ref name="bunken1983"/>。 == 駅周辺 == [[津軽海峡]]の展望が良く、天候の良い日は[[竜飛岬]]を望むことができた<ref name="bunken1983"/>。 * [[国道228号]] * 松前町役場大沢支所 * 白神郵便局 * 松前町立白神小学校 * 白神岳 * 白神岬 -
