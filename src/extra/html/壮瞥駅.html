'''[[胆振縦貫鉄道]]'''紋別駅 - 徳舜瞥駅（後の新大滝駅）間の開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1943年]]（昭和18年）から[[1945年]]（昭和20年）にかけて、[[昭和新山]]の火山活動により、上長流駅 - 当駅間はたびたび線路が崩壊して不通となり、路線変更が繰り返される。 * [[1944年]]（昭和19年）[[7月1日]] - 胆振縦貫鉄道が[[戦時買収私鉄|戦時買収]]により国有化され、路線名を'''胆振線'''に改称、それに伴い同線の駅となる。 * [[1971年]]（昭和46年）[[10月1日]] - 貨物の取り扱いを廃止。 * 時期不詳<ref group="注" name="ekimei">1981年（昭和56年）4月時点では業務委託駅であった（書籍『コロタン文庫36 国鉄駅名全百科 56.4訂補版』（[[小学館]]、[[1981年]]10月発行）55ページより）。</ref> - [[日本の鉄道駅#業務委託駅|業務委託]]駅化。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 胆振線の全線廃止に伴い、廃駅となる。 === 駅名の由来 === 当駅が所在した地名より。地名は、[[アイヌ語]]の「ソーペツ」（滝の川）に由来する。この滝とは、[[洞爺湖]]から[[壮瞥川]]が流れ出す地点にかかる壮瞥滝のことである。 == 駅構造 == 廃止時点で、[[島式ホーム]]の片面を使用する1面1線を有した[[地上駅]]であった。ホームは、線路の北東側（倶知安方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）92ページより。</ref>。かつては、島式ホーム1面2線を有する[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎側の1線は、交換設備運用廃止後も倶知安方の[[分岐器|転轍機]]と一部の線路が撤去された状態で[[停車場#側線|側線]]として残っており<ref name="bunken1983"/>、保線用[[モーターカー]]の留置に使用されていた<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]]12月発行）280ページより。</ref>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっており、駅舎は構内の北東側に位置し、ホームとは側線を渡る[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。 駅前広場にはオンコ（[[イチイ|櫟]]）の木を囲んだロータリーがあった<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は215人<ref name="bunken1983"/>。 == 駅跡 == 1997年（平成9年）時点では2本のホームが残存しており、旧構内は雪捨て場の広場となっていた<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（[[JTBパブリッシング]]、[[1997年]]5月発行）40-41ページより。</ref>。2010年（平成22年）時点ではホームや線路跡は整地され道路に転用されており、[[道南バス]]の「壮瞥役場前」[[バス停留所]]にも利用されている<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）154-156ページより。</ref>。 また、当駅跡から伊達紋別方に行った昭和新山の山麓近くに、上長和駅 - 当駅間の線路付け替えの名残として壮瞥川に架かっていた鉄橋の橋台が残存している。1997年（平成9年）時点では現地までの道がなく木々に囲まれており探訪が難しい状況であったが<ref name="bunken1997H"/>、2011年（平成23年）時点では「昭和新山鉄橋遺構公園」として整備された<ref name="bunken2010H"/>。 そのほか駅跡の伊達紋別方にある「紫明苑」バス停近く<ref name="bunken1997H"/>の国道沿いに、線路付け替え前の旧線跡を示す「国鉄胆振線跡」と記された案内塔が建立されている<ref name="bunken2010H"/>。 駅舎は、[[1987年]]（昭和62年）3月に[[日本テレビ放送網|日本テレビ]]で放送された『[[さよなら大放送 おもしろ国鉄スペシャル]]』においてオークションに掛けられ、[[奄美大島]]の[[奄美アイランド]]に移設された。 == 駅周辺 == * [[北海道道233号伊達洞爺線]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）41ページより。</ref> -
