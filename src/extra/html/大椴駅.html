'''大椴駅'''（おおとどえき）は、かつて[[北海道]]（[[留萌支庁|留萌管内]]）[[留萌郡]][[小平町]]字大椴に設置されていた、[[日本国有鉄道]]（国鉄）[[羽幌線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''オト'''。羽幌線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[3月30日]]に廃駅となった。 一部の普通列車は通過した（1986年（昭和61年）11月1日改定の時刻（廃止時の時刻表）で、上下各1本（急行「はぼろ」後継の主要駅停車列車）<ref>時刻表『JNR編集 時刻表 1987年4月号』（[[交通新聞社|弘済出版社]]、[[1987年]][[4月]]発行）JRニュース13ページより。</ref>）。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南側（幌延方面に向かって左手側）に存在した。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）199ページより。</ref>。駅舎は構内の西側に位置し、ホームに接していた。 == 駅名の由来 == 当駅が所在した地名より。地名はもともと「大椴子（おおとどこ）」と称したが、その上部が駅名となった<ref name=":3">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/2130P.pdf|title=アイヌ語地名リスト エン～オニシ　P21-30P|accessdate=2018-12-24|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref><ref name=":02">{{Cite book|和書|author=|editor=札幌鉄道局編|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|page=55|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref>。 「大椴子」は、[[アイヌ語]]の「ポロトトコ」という語を半訳したことに由来すると考えられる<ref name=":3" />。「ポロ（poro）」は「大きい」の意であるが、「トトコ」が何であるかについては、例えば[[上原熊次郎]]や『駅名の起源（1939年版）』は「トゥトゥㇰ（tu-tuk）」（岬・出っ張っている）の意としているが、このほか「トゥエトコ（tu-etoko）」（山・端）、「トトコ」（上に沼がある）などの諸説がある<ref name=":3" /><ref name=":02" />。 == 駅周辺 == * [[北海道道958号大椴線]] * [[国道232号]] * 大椴子川 == 歴史 == * [[1927年]]（昭和2年）[[10月25日]] - [[鉄道省]]留萠線の留萠駅 - 当駅間の開通に伴い、新設。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1928年]]（昭和3年）[[10月10日]] - 当駅 - 鬼鹿駅間の延伸開通に伴い、中間駅となる。 * [[1931年]]（昭和6年）[[10月10日]] - 留萠駅 - 古丹別駅間を留萠線から分離し路線名を'''羽幌線'''に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1960年]]（昭和35年）[[9月15日]] - 貨物の取り扱いを廃止。 * [[1972年]]（昭和47年）[[2月8日]] - [[チッキ|荷物]]取り扱いを廃止し、同時に無人駅化。 * [[1987年]]（昭和62年）[[3月30日]] - 羽幌線の全線廃止に伴い、廃駅となる。 == 駅跡 == [[File:Haborosen ootodo.jpg|thumb|大椴駅跡（2011年7月26日）]] 駅舎や駅構内施設は撤去され、1999年（平成11年）時点で駅跡地にはダム建設事務所が建てられており、ホームらしい地形と農協の古い倉庫が残っていた<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）25ページより。</ref>。駅前広場の木がそのまま残り、あとは草むらとなっている。 また、2011年（平成23年）時点では駅跡附近に築堤が残存し、コンクリート造りのカルバートも残存している<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）211ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 羽幌線 :: [[小平駅 (北海道)|小平駅]] -
