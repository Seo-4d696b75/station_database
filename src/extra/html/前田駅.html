'''前田駅'''（まえだえき）は、[[北海道]][[岩内郡]][[共和町]]前田にかつて設置されていた、[[日本国有鉄道]]（国鉄）[[岩内線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''マヘ'''。岩内線の[[廃線]]に伴い、[[1985年]]（昭和60年）[[7月1日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[島式ホーム]]（※片面使用）1面1線を有した[[地上駅]]であった。ホームは、線路の南側（岩内方面に向かって左側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）60ページより。</ref>。かつては島式ホーム1面2線を有する、[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎側の1線は、交換設備運用廃止後に撤去された<ref name="bunken1983"/>。そのほか、小沢方から分岐した[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]（業務委託駅）で、駅舎は構内の南側に位置し、ホームとは通路で連絡した<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名（開業当時の、岩内郡[[前田村 (北海道)|前田村]]）より。地名は、この地の開拓を計画した旧金沢藩主[[前田利嗣]]の姓に由来する<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=29 }}</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は203人<ref name="bunken1983"/>。 == 駅周辺 == *
