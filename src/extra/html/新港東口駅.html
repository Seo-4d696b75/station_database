'''越中鉄道'''堀岡駅 - 越ノ潟駅間に'''堀切駅'''（ほりきりえき）として新設開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]により'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1944年]]（昭和19年）[[5月18日]] - 廃止<ref name="bunken2008M-6"/>。 * [[1946年]]（昭和21年）頃 - 営業再開<ref name="bunken2008M-6"/>。 * [[1966年]]（昭和41年） ** [[4月5日]] - 当駅 - 越ノ潟駅間の線路分断・廃止に伴い[[休止駅]]となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 ** [[12月1日]] - 0.1km延伸・移転、同時に'''新港東口駅'''に改称の上営業再開<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>（新設開業とも<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）41ページより。</ref><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）49,53ページより。</ref>）。 * 時期不詳 - [[日本の鉄道駅#業務委託駅|業務委託]]化<ref name="bunkenRML107"/>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]で、同線の[[終着駅|終端駅]]であった。ホームは線路の南側（新港東口方面に向かって左手側）に存在した<ref name="bunkenRML107"/>。かつては島式ホーム1面2線を有した<ref name="bunkenRML107"/>。使われなくなったホーム南側の2番線は廃止され、線路は撤去されていた<ref name="bunkenRML107"/>。稼働線の1番線の番線表示は継続使用されていた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた<ref name="bunkenRML107"/>。駅舎はホームの南西側に位置しホーム西側に接していた<ref name="bunkenRML107"/>。ホームは上屋を有した<ref name="bunkenRML107"/>。 == 駅周辺 == * [[富山県営渡船]]堀岡発着場 * [[富山新港火力発電所]] == 駅跡 == 線路跡は四方駅跡南方附近から当駅附近まで[[サイクリングロード]]となっている<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]5月発行）108-109ページより。</ref>。1997年（平成9年）時点では、線路跡地はそのサイクリングロードであった<ref name="bunken1997H"/>。2006年（平成18年）時点<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）66-67ページより。</ref>、2010年（平成22年）時点でも同様であった<ref name="bunkenKW07"/><ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』50-51ページより。</ref>。 駅跡地は1997年（平成9年）時点では縫製工場、サイクリングロード、並木のある緑地となっていた<ref name="bunken1997H"/>。2006年（平成18年）時点<ref name="bunkenToyama2008H"/>、2010年（平成22年）時点でも同様であった<ref name="bunkenKW07"/><ref name="bunken2010H-3"/>。尚、駅舎跡地が縫製工場、旧2番線及びホーム跡地がサイクリングロード、線路跡地が並木のある緑地という位置関係であった<ref name="bunkenToyama2008H"/>。 また、2006年（平成18年）時点では、駅跡の西側、海岸附近に「堀切鉄橋」の橋台が残存し、防波堤を越えた海中にも鉄橋の橋脚が1基残存していた<ref name="bunkenToyama2008H"/>、2010年（平成22年）時点でも同様であったが2013年7月に隣接する堀切橋跡とともに撤去された。<ref name="bunkenKW07"/><ref name="bunken2010H-3"/>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[堀岡駅]] -
