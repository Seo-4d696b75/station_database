'''寺津駅''' - [[西一色駅]] == その他 == === 北寺津駅 === 寺津駅は当初0.6km三河楠寄りにあり、1927年（昭和2年）3月25日に廃止時の位置へと移転した<ref name="chizucho_teiho"/>。その後、同年[[7月25日]]には移転後の寺津駅から三河楠寄り0.5kmの位置に'''北寺津駅'''が設置されている<ref>鉄道省 編 『[{{NDLDC|1207554/202}} 鉄道停車場一覧 昭和12年10月1日現在]（国立国会図書館デジタルコレクション）』、川口印刷所出版部、1937年、355頁</ref>。 北寺津駅は[[1944年]]に休止され[[1969年]][[4月25日]]正式に廃止となるが、三河鉄道時代のホームは廃止後もそのまま残されていた<ref name="MHS-151">{{Cite book|和書|author=新實守|chapter=三鉄ものがたり|editor =徳田耕一|year =2001|title = 名鉄の廃線を歩く|publisher =JTB|isbn = 978-4533039232|page=151}}</ref>。 === 寺津高架橋 === 西三河南部地域から[[衣浦トンネル]]を通り、[[中部国際空港]]への主要アクセス道路となる[[愛知県道41号西尾幸田線|都市計画道路衣浦岡崎線]]への踏切設置を避けるため、寺津・三河楠間におよそ21億円の費用をかけて鉄道[[高架橋]]が建設され、[[1998年]]11月に供用開始となった。しかし、間もなくして名鉄は三河線の区間廃止を表明したため、わずかな期間しか利用されなかったので批判が噴出した。 <gallery widths="170px"> MT-Kita Terazu Station-Platform.JPG|北寺津駅の遺構<ref name="MHS-151"/> MT-Terazu Viaduct.JPG|寺津高架橋 MT-Terazu elevated bridge.jpg|県道41号線をオーバークロスしている寺津高架橋（2016年） </gallery> == 脚注 == {{脚注ヘルプ}} {{Reflist}} == 関連項目 == *[[廃駅]] {{名鉄三河線|mode=1}} {{リダイレクトの所属カテゴリ |redirect1=北寺津駅 |1-1=日本の鉄道駅 き |1-2=1927年開業の鉄道駅 |1-3=1969年廃止の鉄道駅 }} {{DEFAULTSORT:てらつ}} [[Category:西尾市の廃駅]] [[Category:日本の鉄道駅 て|らつ]] [[Category:名古屋鉄道の廃駅]] [[Category:三河鉄道|てらつえき]] [[Category:1926年開業の鉄道駅]] [[Category:2004年廃止の鉄道駅]] 15ivdvxj4cc1r94j9q22y15cujjnk24
