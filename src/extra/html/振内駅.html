'''[[日本国有鉄道|国有鉄道]]富内線'''富内駅 - 当駅間の延伸開通に伴い、開業<ref name="hokkaido-np-1958-11-16" />。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1959年]]（昭和34年） - 振内営林署所轄貯木場へ、専用線500m敷設<ref>北海道営林局（直轄）50年史 平成9年11月発行 P310。</ref>。 * [[1964年]]（昭和39年）[[11月5日]] - 当駅 - 日高町駅間の延伸開通に伴い、中間駅となる。　 * [[1982年]]（昭和57年）[[11月15日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の全線廃止に伴い、廃駅となる<ref name="hokkaido-np-1986-10-31" />。 == 駅構造 == 廃止時点で、1面2線の[[島式ホーム]]を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）107ページより。</ref>。駅舎側（南側）が上り線、外側（北側）が下り線となっていた（番線表示なし）。その他上下本線それぞれの外側に、[[安全側線]]を持った[[停車場#側線|側線]]を各1線ずつ有し、上り線側の側線の駅舎傍には貨物用の短い[[単式ホーム]]を有した<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南側に位置し、ホーム東側とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「フレ・ナイ」（赤い川）に由来する。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は106人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道237号]]（日高国道） * [[北海道道638号宿志別振内停車場線]] * [[北海道道797号貫気別振内線]] * 平取町役場振内支所 * [[門別警察署]]振内駐在所 * 振内郵便局 * 平取町立振内中学校 * 平取町立振内小学校 * [[苫小牧信用金庫]]振内代理店 * びらとり農業協同組合（JAびらとり）振内支所 * [[沙流川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）11ページより。</ref> * 振内山 - 駅から北東に約7.5km<ref name="bunken1983"/>。 * [[道南バス]]「振内案内所」停留所 == 駅跡 == 旧駅構内は1986年（昭和61年）11月から平取町により、「振内鉄道記念館」として整備されている<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]][[11月]]発行）27ページより。</ref>。駅舎は撤去されたが新たにバスの[[待合所]]兼用の建物が建築され、館内1階に[[国鉄C58形蒸気機関車|C58形蒸気機関車]]の動輪や、当時使用していた[[閉塞 (鉄道)|閉塞器]]、備品、[[保線]]用具、[[駅スタンプ]]、写真パネルなどが保存・展示されている<ref name="bunken1998S"/>。館外には当時の状態のままに<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）90-91ページより。</ref>側線を含む[[軌条|レール]]とホーム、[[鉄道信号機#機械式信号機|腕木式信号機]]が保存され、旧上り線上には旧[[ソビエト連邦|ソ連]]向けに新製された、[[国鉄D51形蒸気機関車]]と同型機である'''D51-23号機'''が[[サハリン州|サハリン]]（旧・樺太）から戻って来て[[静態保存]]・展示されている<ref name="bunken2011S">書籍『蒸気機関車完全名鑑 ビジュアル改訂版』（[[廣済堂あかつき|廣済堂ベストムック]]、[[2011年]][[1月]]発行）54ページより。</ref>。また、旧下り線ホームに横付けする形で<ref name="bunken2011H"/>国鉄の旧型[[客車]]である[[国鉄スハ43系客車#スハ45形|スハ45形]]'''スハ45 37'''と[[国鉄スハ43系客車#スハフ42形|スハフ42形]]'''スハフ42 519'''の2両も静態保存・展示されている<ref name="bunken1998S"/>。客車は[[ライダーハウス]]としても利用されており<ref name="bunken1998S"/>、内装は宿泊施設用に変更されている<ref name="bunken2011H"/>。ホームには当駅のほか幌毛志駅、仁世宇駅、岩知志駅の[[駅名標]]が移設保存されている<ref name="bunken2011H"/>。 == その他 == 当駅を発着駅とする区間列車が上下1本（下り：鵡川駅 -
