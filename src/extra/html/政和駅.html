'''政和駅'''（せいわえき）は、[[北海道]]（[[空知支庁]]）[[雨竜郡]][[幌加内町]]字政和第二にあった[[北海道旅客鉄道]]（JR北海道）[[深名線]]の[[鉄道駅|駅]]（[[廃駅]]）である。深名線の[[廃線]]に伴い[[1995年]]（平成7年）[[9月4日]]に廃駅となった。 == 歴史 == * [[1931年]]（昭和6年）[[9月15日]] - [[鉄道省]]雨龍線幌加内 - 添牛内間開通に伴い開業<ref name=":0222">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=114}}</ref><ref>『北海道鉄道百年史 下巻』1981年3月 日本国有鉄道北海道総局 編集・発行。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年10月 JTB編集発行。</ref>。 * [[1931年]]（昭和6年）[[10月10日]] - 線路名を幌加内線に改称、それに伴い同線の駅となる。 * [[1941年]]（昭和16年）10月10日 - 線路名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1982年]]（昭和57年）[[3月29日]] - 貨物取扱い廃止<ref name="hensen"/>。 * [[1984年]]（昭和59年） ** [[2月1日]] - [[チッキ|荷物]]取扱い廃止<ref name="hensen"/>。 ** [[11月10日]] - 旅客扱い[[無人駅|無人化]]。運転要員は1986年まで引き続き配置。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承<ref name="hensen"/>。 * [[1995年]]（平成7年）[[9月4日]] - 深名線の廃線に伴い廃止となる<ref name="hensen"/>。 === 駅名の由来 === 所在地名より。元々当地は「添牛内五線」と呼ばれていたが、呼びにくいために[[1922年]]（大正11年）に正和（しょうわ）という地名が付けられていた<ref name=":0222" /><ref name=":02">{{Cite book|和書|author=|editor=札幌鉄道局編|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|page=36|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref>（命名は1927年（昭和2年）とも<ref>幌加内町史 P69,127,197。</ref>）。しかし開駅時、[[留萠鉄道]]に「昭和駅」があり紛らわしいため政和（せいわ）とし<ref name=":0222" /><ref name=":02" />、後年地名も改められた<ref group="注">当駅開業の5年後の1936年（昭和11年）に地区名も政和に変更されたが、幌加内村史（1958年11月発行）も幌加内町史も変更理由を「他に正和村が多くあるため」とし、駅名からの影響については言及していない。</ref>。 == 駅構造 == 駅舎は構内の西側に位置し、開業当初は駅舎前から深川方に設けられた[[単式ホーム]]1面1線と副本線1本を有した[[地上駅]]であった<ref>昭和7年度線路一覧略図 札幌鉄道局発行「幌加内線」。</ref>。 その後1947年までに、本線が上下2本に増やされて対向ホーム側が名寄側にずれた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）205ページより。</ref>相対式ホーム2面2線を有する[[列車交換]]可能な[[交換駅]]となり、駅舎横には切り欠きホーム状の貨物ホームが作られて名寄方から貨物線用[[停車場#側線|側線]]が分岐して引き込まれ、対向ホーム裏には留置線1本が敷かれた<ref name="choshi">幌加内町史 1971年9月発行、P720。</ref>。また、留置線の深川方から駅裏土場前へ営林署専用線が分岐して敷かれた<ref name="choshi"/><ref group="注">外部リンク 1947年撮影航空写真参照。幌加内町史によれば営林署専用線は延長120m。</ref>。営林署の専用線は1957年から1964年の間に廃止されている<ref>昭和32年版、昭和39年版全国専用線一覧。</ref>。 貨物取扱と交換設備運用の廃止後となった1983年（昭和58年）時点では、乗降設備は単式ホーム1面1線に戻り、使われなくなった対向ホーム側の旧上り本線は副本線に戻っていた（ホームは残存していた）<ref name="bunken1983"/>。またその副本線の深川方から対向ホーム外側に、かつての留置線が名寄方向の分岐を切られて行き止まりの短い側線となって残っていた<ref name="bunken1983"/>。 廃止時点で、副本線と旧対向ホーム、駅裏側側線は撤去され、単式ホーム1面1線と旧貨物ホームへの側線1線だけとなっていた<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）75ページより。</ref>。ホーム前後の線路は分岐の名残で湾曲していた<ref name="bunken1993"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた<!-- ホーム南側に接していた（※この箇所コメントアウト　理由：bunken1993の図は誤り。駅舎はホームの名寄方の北側に接していた。つまり開業当初から変わっていない。当時の映像により確認。）-->。事務室部分だけがアルミサッシに変わった小さな木造駅舎で<ref name="bunken1983"/>、有人駅時代には待合室に近所の主婦が作った[[シャクナゲ]]の[[造花]]が飾られていた<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は21人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は4人<ref name="bunken1993"/>。 == 駅周辺 == 駅前広場には[[ニレ|楡]]の木と、並んで「幌加内町開基七十周年」記念塔が建てられていた<ref name="bunken1983"/><ref name="bunken1993"/>。 * [[国道275号]]（空知国道） * [[北海道道938号伊文政和線]] * [[道の駅森と湖の里ほろかない]] * [[政和温泉|せいわ温泉]]ルオント * 政和簡易郵便局 * 幌加内町[[公民館]] 政和コミュニティセンター * [[雨竜川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> * 釜尻山 - 駅の西<ref name="bunken1980M"/>。 * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「政和」停留所 == 駅跡 == 廃駅後、駅舎は当初の住居から空き家、そして物置として利用されていた。2000年（平成12年）時点では、駅舎は閉鎖状態で残存し、駅前の「幌加内町開基七十周年」の記念塔も残存していた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）35-36ページより。</ref>。2002年（平成14年）時点でも同様で、外壁が多少劣化した程度であり、ホーム側は物置として再利用されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）178-179ページより。</ref>。再利用者が転々とし<ref name="bunken2011H"/>、2003年（平成15年）夏には改装され食堂となった。2010年（平成22年）時点では食堂は夏期に営業が確認されている<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）41-43ページより。</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[上幌加内駅]] - '''政和駅''' - [[添牛内駅]] :::
