[[1969年]]（昭和44年）[[11月1日]] |乗車人員 = |乗降人員 = |統計年度 = |乗換 = |備考 = }} '''近江今津駅'''（おうみいまづえき）は、かつて[[滋賀県]][[高島郡 (滋賀県)|高島郡]][[今津町 (滋賀県)|今津町]]今津住吉（現在の[[高島市]]今津町住吉{{Sfn|江若鉄道の思い出|p=116}}）にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。同鉄道の[[終着駅]]であった。 == 歴史 == 近江今津駅は1931年（昭和6年）1月1日、江若鉄道の[[安曇川駅 (江若鉄道)|安曇]] - 近江今津間の開通に合わせて開業した{{Sfn|江若鉄道の思い出|p=116}}。江若鉄道の最初の開通区間である[[三井寺下駅|三井寺]] - [[叡山駅 (江若鉄道)|叡山]]間の開通から10年ほどの年月が経過しており、今津町の住民は長らく期待していた鉄道の開通を地域を挙げて歓迎したことが当時の新聞などによって報じられている{{Sfn|ありし日の江若鉄道|p=26}}{{Sfn|レイル|p=73}}。開通を祝して1月16日には町内の小学校で記念式典を開催、余興として花火の打ち上げや[[活動写真]]の上映が行われ、駅前はアーチや提灯で飾り立てられた{{Sfn|ありし日の江若鉄道|p=26}}{{Sfn|江若鉄道の思い出|p=118}}。 江若鉄道はその名の通り近'''江'''（[[滋賀県]]）と'''若'''狭（[[福井県]]）とを結ぶべく建設されてきた鉄道であり{{Sfn|レイル|p=70}}、鉄道敷設の免許も今津町より先、福井県の[[三宅村 (福井県)|三宅村]]（現在の三方上中郡[[若狭町]]）まで得ていた{{Sfn|レイル|p=72}}{{Sfn|寺田|2010|p=9}}。しかし沿線住民が負担に協力した建設資金が枯渇し、三宅村への延伸は断念{{Refnest|1932年（昭和7年）9月の時点で三宅村への延伸は不可能と考えられていて、今津町長が[[鉄道省]]に今津 - 小浜間のバス路線開通を陳情している{{Sfn|田中|宇田|西藤|1998|p=394}}{{Sfn|今津町史第3巻|p=213}}。}}、結果として当駅が路線廃止まで終着駅であった{{Sfn|江若鉄道の思い出|p=116}}{{Sfn|レイル|p=73}}{{Sfn|レイル|p=85}}。鉄道で果たせなかった当駅から三宅村へ至る路線はのちに[[路線バス|自動車路線]]が受け継ぎ、1937年（昭和12年）に[[国鉄バス|省営自動車]]の[[若江線]]が開通している（現在も[[西日本ジェイアールバス]]の路線として存続）{{Sfn|江若鉄道の思い出|p=117}}。 町内に[[大日本帝国陸軍|陸軍]]の饗庭野[[演習場]]（現在の[[陸上自衛隊]]饗庭野演習場）があったことから戦中は当駅から兵士が出征していくこともあり、その際には駅のホームや駅前の広場が見送りの場となった{{Sfn|江若鉄道の思い出|p=118}}。また戦前には演習場の一部が饗庭野スキー場として開放されており、鉄道を利用して訪れるスキー客もいた{{Sfn|江若鉄道の思い出|p=115}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1930年]]（昭和5年）12月 - 駅舎が完成{{Sfn|江若鉄道の思い出|p=116}}。 * [[1931年]]（昭和6年） ** [[1月1日]] - 江若鉄道の安曇 - 当駅間の延伸開業と同時に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957723/4}} 「地方鉄道運輸開始」『官報』1931年3月9日]（国立国会図書館デジタル化資料）</ref>。 ** [[1月16日]] - 町内の今津小学校で開通記念式典が挙行される{{Sfn|江若鉄道の思い出|p=118}}。 * 近江今津間が開通<ref>[{{NDLDC|2959776/8}} 「鉄道省告示第484号」『官報』1937年12月18日]（国立国会図書館デジタル化資料）</ref>。近江今津 - [[小浜駅|小浜]]間がバスでつながる。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == 近江今津駅は旅客と貨物の双方を取り扱う[[一般駅]]{{Sfn|寺田|2010|p=10}}。[[プラットホーム|ホーム]]は1面あり、両側に線路が接する[[島式ホーム]]であった{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=81}}。このほかにも貨物用のホームや[[機関庫]]{{Sfn|竹内|1967}}、多数の[[停車場#側線|側線]]{{Sfn|寺田|2010|p=15}}や[[蒸気機関車]]の水の補給に使われた給水設備{{Sfn|江若鉄道の思い出|p=65}}などを有し、駅構内は広かった{{Sfn|レイル|p=85}}。 駅舎は江若鉄道では珍しい三角屋根が乗る洋風のデザインで、先に開業していた[[高島町駅 (滋賀県)|大溝駅]]とともに目を引く存在であった{{Sfn|江若鉄道の思い出|p=116}}。駅舎の東側が入口、西側が改札口である{{Sfn|今津町史第3巻|p=208}}。 駅に設置されていた[[駅名標]]では、駅名が「お'''お'''みいまづ」と表記された{{Sfn|江若鉄道の思い出|p=116}}。 == 駅前からは[[小浜市|小浜]]方面・[[マキノ町]]方面へ向かうバスが発着していた{{Sfn|ありし日の江若鉄道|p=26}}{{Sfn|レイル|p=85}}。饗庭野演習場、およびその一部に開かれた饗庭野スキー場は徒歩圏内である{{Sfn|江若鉄道の思い出|p=115}}。 駅のあった場所は江若鉄道の廃線後に開業した[[湖西線]]の[[近江今津駅]]の西側にあたり{{Sfn|レイル|p=85}}、[[エーコープ|Aコープ]]今津店になっている{{Sfn|吉田|1998|pp=195&ndash;196}}{{Sfn|寺田|2010|p=26}}。江若鉄道の駅の中で唯一駅舎が現存しており{{Sfn|江若鉄道の思い出|p=116}}{{Sfn|今津町史第3巻|p=208}}、駅舎は[[農業協同組合|農協]]関連施設（JA旅行センター{{Sfn|吉田|1998|pp=195&ndash;196}}）に転用されたのちに温浴施設、スーパーの現場建築事務所とその用途を変え、2012年の時点では無人となっている{{Sfn|レイル|p=97}}。駅構内、ホームがあった場所は道路に変わった{{Sfn|今津町史第3巻|p=208}}{{Sfn|レイル|p=97}}。 当駅付近の線路跡については、湖西線近江今津駅の南に広がる[[留置線]]付近で確認することができ、{{読み仮名|天川|あまかわ}}の土手に橋台や[[築堤]]が残されている{{Sfn|吉田|1998|pp=195&ndash;196}}{{Sfn|今津町史第3巻|p=209}}。このほかに痕跡はほとんどなく、線路跡は南を流れる庄堺川付近で湖西線に取り込まれている{{Sfn|今津町史第3巻|p=209}}。
