(2007年10月撮影)]] '''恋路駅'''（こいじえき）は、かつて[[石川県]][[鳳珠郡]][[能登町]]恋路にあった、[[のと鉄道]][[のと鉄道能登線|能登線]]の[[鉄道駅|駅]]（[[廃駅]]）。 == 概要 == [[急行列車|急行]]停車駅（ただし「のと恋路号」のみ）。[[2005年]]（平成17年）、能登線の[[廃線|廃止]]により[[廃駅]]となった。全国的にも珍しい「恋」のつく駅であるため<ref>他に「恋」のつく駅として、[[母恋駅]]、[[恋山形駅]]、[[恋ヶ窪駅]]、[[恋し浜駅]]がある。</ref>、廃止された今でも多くの訪問者がある。 [[東日本大震災]]後、地元の[[宗玄酒造]]が駅に近い宗玄隧道（宗玄[[トンネル]]、約130メートル）を買い取り、[[津波]]に備えた製品保護と味の熟成のため、酒蔵として活用している<ref>廃線めぐり(3)恋路に続く酒蔵トンネル『[[読売新聞]]』朝刊2017年5月5日</ref><ref name="isico">{{Cite web|url = https://www.isico.or.jp/i-maga/journal/i110247.html|title = 北陸初のトンネル貯蔵酒を販売 低温熟成で味がまろやかに|publisher = 石川県産業創出支援機構|date = 2013-04-05|accessdate = 2020-05-03}}</ref>。旧恋路駅への観光客が多いことから、同社は宗玄トンネルと旧恋路駅間の約300メートルに[[線路]]を再敷設して、[[2013年]][[4月1日]]から奥のとトロッコ鉄道（[[愛称]]「のトロ」）と呼ばれる[[トロッコ]]（[[軌道自転車]]）を走らせている<ref name="isico"/><ref>{{Cite web|url = https://j-town.net/tokyo/column/gotochicolumn/239112.html?p=all|title = 廃線の線路をトロッコで走ろう! 石川・奥のとトロッコ鉄道が楽しそう|publisher = Jタウンネット|date = 2017-02-16|accessdate = 2020-05-03}}</ref><ref>{{Cite news|url = https://news.mynavi.jp/article/20130405-a008/|title = 石川県珠洲市で「奥のとトロッコ鉄道」開業! 足こぎトロッコで奥能登を走る|publisher = [[マイナビニュース]]|date = 2013-04-05|accessdate = 2020-05-03}}</ref>。また、この「のトロ」は開設した2013年度の[[グッドデザイン賞]]を受賞している<ref>[https://www.g-mark.org/award/describe/40419 奥のとエリア活性化「トロッコ鉄道事業＆日本酒のトンネル貯蔵をはじめとする酒蔵ツーリズムによる地域振興」] - グッドデザイン賞公式サイト</ref>。 駅名の縁起のよさから恋路駅行き記念[[切符]]の需要があったが、当駅は[[無人駅]]のため隣の[[松波駅]]で発行されていた<ref>寺田裕一『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』、JTBパブリッシング、2008年、151頁。ISBN 978-4-533-07145-4</ref>。現在は宗玄酒造で入手できるほか<ref name=journal>『鉄道ジャーナル』通巻550号（2012年8月号）、81 - 83頁。</ref>、のと鉄道でも恋路行き切符を封入した[[キーホルダー]]を販売している<ref>{{Cite web|url=http://www.nototetsu.co.jp/html/orijinaru.html|title=記念入場券|publisher=のと鉄道|accessdate=2015年8月4日}}</ref>。 == 駅構内== *[[単式ホーム]]1面1線を有する[[地上駅]]。 *[[駅舎]]はなくホーム上に[[待合所]]があるのみ。
