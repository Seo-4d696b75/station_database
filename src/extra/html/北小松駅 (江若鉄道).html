|統計年度 = |乗換 = |備考 = }} '''北小松駅'''（きたこまつえき）は、かつて[[滋賀県]][[滋賀郡]][[志賀町 (滋賀県)|志賀町]]大字北小松（現在の[[大津市]]北小松{{Sfn|江若鉄道の思い出|p=80}}）にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 == 歴史 == 当駅は1927年（昭和2年）4月、江若鉄道の[[近江舞子駅 (江若鉄道)|雄松駅]]から北小松駅までの区間が開通したのに合わせて開業した{{Sfn|江若鉄道の思い出|p=80}}。同年12月には北小松より先、[[高島町駅 (滋賀県)|大溝駅]]まで路線が延伸する{{Sfn|寺田|2010|p=10}}{{Sfn|田中|宇田|西藤|1998|p=394}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1927年]]（昭和2年） ** [[4月1日]] - 雄松駅から当駅までの延伸開通に伴い、滋賀郡[[小松村 (滋賀県)|小松村]]に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956537/6}} 「地方鉄道運輸開始」『官報』1927年4月6日]（国立国会図書館デジタル化資料）</ref>。 ** [[12月15日]] - 当駅から大溝駅までが延伸開通{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956758/11}} 「地方鉄道運輸開始」『官報』1927年12月24日]（国立国会図書館デジタル化資料）</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == 山側には[[楊梅の滝]]がある{{Sfn|江若鉄道の思い出|p=80}}。かつては駅から滝までのおよそ1キロメートルにわたって[[桜並木]]が続き「[[千本桜]]」と称されていたが、[[湖西線]]の建設や農機具の通行の妨げになることなどを理由に多くが伐採され失われた{{Sfn|江若鉄道の思い出|p=81}}{{Sfn|ありし日の江若鉄道|p=17}}。また1950年（昭和25年）ころからは水泳客が北小松に訪れるようになり、1962年（昭和37年）には地域住民によって[[琵琶湖]]岸に北小松水泳場が開かれている{{Sfn|江若鉄道の思い出|p=80}}。 駅のあった場所は、江若鉄道の廃線後に開通した[[湖西線]]の[[北小松駅]]の駅前広場にあたる{{Sfn|江若鉄道の思い出|p=80}}。同駅の広い駅前広場は、江若鉄道の駅があったことの名残である{{Sfn|吉田|1998|p=192}}。その北側には[[高島町駅 (滋賀県)|高島町駅]]方面に向かってカーブを描いて伸びていた江若鉄道の線路跡が[[築堤]]もそのままに残され、江若鉄道が運行されていたころの様子を色濃く留めていたが{{Sfn|吉田|1998|p=192}}、2003年（平成15年）に舗装され道路に変わり、{{Sfn|寺田|2010|pp=25&ndash;26}}その後国道161号バイパスの工事のため大半が取り壊された。その先、鎧岩以北の線路跡は[[国道161号]]に転用されている{{Sfn|江若鉄道の思い出|p=81}}。
