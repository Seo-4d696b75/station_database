'''越中電気軌道'''当駅 - 四方駅間開通に伴い開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1926年]]（大正15年）[[7月21日]] - 聯隊橋駅（後の新富山駅） - 当駅間延伸開通に伴い中間駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1927年]]（昭和2年）[[2月13日]] - 鉄道会社名を'''越中鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2010H-3-R"/>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]に伴い'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1969年]]（昭和44年）[[4月15日]] - 交換設備撤去<ref name="bunkenRML107-R">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）35,45ページより。</ref><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）49-52ページより。</ref>。 * 時期不詳 - [[日本の鉄道駅#業務委託駅|業務委託]]化<ref name="bunkenRML107">『RM LIBRARY 107』36,41ページより。</ref>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/>。ホームは線路の西側（新港東口方面に向かって左手側）に存在した<ref name="bunkenRML107"/>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML107"/>。駅舎側ホーム（西側）が下り線、対向側ホーム（東側）が上り線となっていた。使われなくなった対向ホーム側の1線は、交換設備運用廃止後は撤去されたが、ホームは残存していた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた<ref name="bunkenRML107"/>。駅舎は構内の南西側に位置しホーム南側に接していた<ref name="bunkenRML107"/>。周囲より若干高台にあり、駅舎入口へは階段を上る必要があった<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）62ページより。</ref>。入口はホームに対し90度の角度で駅舎の南側に設置されていた<ref name="bunkenToyama2008H"/>。ホームに[[待合所]]代わりの上屋を有した<ref name="bunkenRML107"/>。 == 駅周辺 == 附近に学校があり、通学客が多く利用した<ref name="bunkenRML107"/>。 * [[富山県道7号富山八尾線]] * [[富山県道208号小竹諏訪川原線]] * 神通大橋<ref name="bunkenToyama2008H"/> * 富山桜谷郵便局 * [[神通川]] == 駅跡 == 1997年（平成9年）時点では、附近の線路跡は[[富山地方鉄道#バス事業|富山地鉄バス]]専用道路となっており、上下線の2つのホームが、ともに道路脇に雑草が生い茂った、朽ち果てた姿で残存していた<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]5月発行）108ページより。</ref>。その後当駅跡附近の地鉄バス専用道路は廃止され、2006年（平成18年）5月時点では駅跡は[[駐車場]]となっており<ref name="bunkenToyama2008H"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』48-49ページより。</ref>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[富山トヨペット本社前（五福末広町）停留場#射水線新富山駅|新富山停留場]] -
