武岡軽便鉄道の駅として開業。 * [[1918年]]（大正7年）[[3月19日]] - 社名変更により武岡鉄道の駅となる。 * [[1924年]]（大正13年）[[3月7日]] - 今立鉄道（未開業）との合併により、南越鉄道の駅となる。 * [[戦中|戦時中]]（時期不詳） - 一度廃止される。 * [[1951年]]（昭和26年）[[6月15日]] - 復活開業。 * [[1972年]]（昭和47年）[[10月12日]] - 交換設備が撤去される。<ref name="haisen3_p166">[[寺田裕一]] 「私鉄の廃線跡を歩くⅢ 北陸・上越・近畿編」、[[JTBパブリッシング]]、2008年5月、p.166、ISBN 978-4-533-07145-4</ref> * [[1981年]]（昭和56年）[[4月1日]] - 社武生 - 粟田部間の廃止により廃駅となる。 == 駅構造 == [[相対式ホーム]]2面2線を有する駅だった。晩年は[[駅舎|駅舎側]]のホームのみを使用していた。 [[駅舎]]とは反対側の[[待合室]]にはもと[[南海電気鉄道|南海電鉄]]の[[福井鉄道80形電車|旧南海モハ81形]]が使われていた。 == 駅跡 == [[ファイル:Fukutetsu 161-1 01.jpg|thumb|161-1保存車]] 駅の名をとどめるものは残っていない。跡地には現在個人私設の鉄道博物館があり、かつて[[福井鉄道鯖浦線|鯖浦線]]で使用されていて、後に南越線と[[福井鉄道福武線|福武線]]で使用されていた[[福武電気鉄道デハ20形電車|160形]]電車の1両が保存されている。 == 隣の駅 == ; 福井鉄道 : 南越線（廃止） :: [[北府駅 (南越線)|北府駅]] -
