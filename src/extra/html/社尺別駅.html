尺別炭礦[[専用鉄道]]の駅として開業。旅客駅。起点0.26。 * [[1952年]]（昭和27年）[[3月4日]] - [[十勝沖地震]]により駅舎損壊。 * ？ - 駅舎改築。 * [[1962年]]（昭和37年）[[1月1日]] - [[地方鉄道]]の尺別鉄道の駅となる。 * [[1970年]]（昭和45年）[[4月16日]] - 当鉄道廃止により廃駅。 ==構造== 土盛の単式ホーム1面を新尺別に向かって左側に有し、駅裏側に機回し線を有していた。ホーム上に当初は15坪、十勝沖地震後の改築後は25坪の小屋風の駅舎が設置されていた。 == 隣の駅 == ; 雄別炭礦 : 尺別鉄道線 :: [[尺別駅]] -
