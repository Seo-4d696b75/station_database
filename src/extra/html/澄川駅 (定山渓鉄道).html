'''北茨木停留所'''として開業<ref name="Sapporo no Eki185">[[#sapporo|さっぽろ文庫 11『札幌の駅』p.185]]</ref>。 * [[1949年]]（昭和24年）8月 - 南側へ移転、交換可能な側線を設け<ref name="kubo p75"/>駅に昇格<ref name="Sapporo no Eki185"/>。 * [[1957年]]（昭和32年）12月 - '''澄川駅'''に改称<ref name="Sapporo no Eki185"/><ref name="kubo p76">[[#kubo|久保 『定山渓鉄道』p.76]]</ref>。 * [[1963年]]（昭和38年） - 側線を撤去<ref name="kubo p76"/>。 * [[1969年]]（昭和44年）[[11月1日]] - 定山渓鉄道線廃線に伴い廃駅。 == 駅構造 == 設置当時は「機械場通り」と斜めに交差する踏切のすぐ南側に単式ホーム1面1線の構造であった<ref name="kubo p74">[[#kubo|久保 『定山渓鉄道』p.74]]</ref><ref name="kubo p75"/>。 1949年に輸送力増強のため、南側に移転するとともに、列車交換のための側線を備えた島式ホーム1面2線となった<ref name="kubo p74"/><ref name="kubo p75"/>。当初の停留所敷地には1953年に電気容量不足に対応するため「北茨木変電所」が設置された。1963年に側線が撤去され、再び単式ホーム1面1線の構造となり廃止時までその状態であった<ref name="kubo p76"/>。 == 駅跡 == 現在は、[[札幌市営地下鉄南北線]][[澄川駅]]になっている。 == 隣の駅 == ; [[じょうてつ|定山渓鉄道]] : 定山渓鉄道線 :: [[豊平駅]] -
