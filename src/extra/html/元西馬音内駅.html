鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1967年]]（昭和42年）[[12月1日]] - 雄勝線の部分廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML052">『RM LIBRARY 52』4,15ページより。</ref>。ホームは線路の北側（梺方面に向かって右手側）に存在した<ref name="bunkenRML052"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML052"/>。 [[無人駅]]となっており<ref name="bunken2008SH-1-R"/><ref name="bunkenRML052"/>、駅舎は無いがホーム北側に[[待合所]]を有した<ref name="bunkenRML052"/>。 == 駅周辺 == 集落から南に外れており、周囲は水田であった<ref name="bunkenRML052"/>。 * [[秋田県道57号十文字羽後鳥海線]] -
