新得 - 鹿追間開業に伴い開設。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1930年]]（昭和5年）[[9月30日]] - 機関庫増設。 * [[1933年]]（昭和8年）[[6月3日]] - 中音更駅の機関庫をガソリン気動車用客車庫として当駅に移設<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047927&ID=M0000000000000394533&TYPE=&NO= 南新得停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1934年]]（昭和9年）[[1月11日]] - 中音更駅の炭水設備を当駅に移設<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047927&ID=M0000000000000394542&TYPE=&NO= 中音更及南新得停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1968年]]（昭和43年）[[10月1日]] - 北海道拓殖鉄道廃止に伴い廃駅。 == 駅名の由来 == 当地の地域名である新得町の市街地南端に位置する。「新得」の由来については[[新得駅]]参照のこと。 == 駅構造 == * [[列車交換|交換設備]]を持つ一般駅<ref>「私鉄の廃線跡を歩く I」寺田裕一著 JTBパブリッシング P163。</ref>。 * 駅舎は新得に向かって右側（東側）にあった。 * 単式ホーム1面、駅舎横の佐幌側（南側）に貨物ホームと引込線2本を有していた。 * 構内の佐幌側端部に[[転車台]]と機関庫を有していた。 == 駅周辺 == 周辺は市街地から外れていたため、当鉄道職員用の共同浴場や娯楽用の集会場（クラブ）が設けられた。 * [[佐幌川]] * ペンケオタソイ川 * [[国道38号]] * [[北海道新得高等学校]] == 廃止後の状況 == * 敷地は拓鉄公園になっている。 == 隣の駅 == ; 北海道拓殖鉄道 : 北海道拓殖鉄道線 :: [[新得駅]] -
