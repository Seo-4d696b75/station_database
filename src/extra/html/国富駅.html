|乗車人員 = |乗降人員 = |統計年度 = |乗換 = |備考 = 岩内線廃線に伴い廃駅 }} '''国富駅'''（くにとみえき）は、かつて[[北海道]][[岩内郡]][[共和町]]国富にあった[[日本国有鉄道]]（国鉄）[[岩内線]]の[[鉄道駅|駅]]である。[[電報略号 (鉄道)|電報略号]]は'''トミ'''。岩内線の[[廃線]]に伴い[[1985年]]（昭和60年）[[7月1日]]に[[廃駅]]となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の南側（岩内方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）60ページより。</ref>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の南側に位置しホーム中央部分に接していた<ref name="bunken1983"/>。 かつては国富鉱業所精錬所への[[専用鉄道|専用線]]も分岐していた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）12ページより。</ref>。 == 駅名の由来 == 駅付近にある[[国富鉱山]]からの輸送のために設置された駅であり、駅名はこの鉱山の名前に由来する<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=29 }}</ref>。 縁起のよい駅名としても知られていた<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]][[10月]]発行）60ページより。</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は43人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道276号]]（尻別国道） * [[北海道道269号蕨岱国富停車場線]] *
