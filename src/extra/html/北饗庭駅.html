[[近江今津駅 (江若鉄道)|近江今津]]間が開業した際、[[新旭駅 (江若鉄道)|新儀]] - 近江今津間に設置された[[饗庭駅]]は用地の問題もあって両駅の中間地点より南（新儀寄り）にずれた位置に設けられた{{Sfn|江若鉄道の思い出|p=110}}。このため饗庭 - 近江今津間に位置する[[饗庭村]]の岡・{{読み仮名|木津|こうつ}}地区の住民は駅が遠くなり、不便を強いられていた{{Sfn|江若鉄道の思い出|p=114}}。この不便を解消するべく、両地区の住民の要望を反映して饗庭 - 近江今津間に設置されたのが当駅である{{Sfn|江若鉄道の思い出|p=114}}。安曇 - 近江今津間開通の翌年、1932年（昭和7年）に開業した{{Sfn|江若鉄道の思い出|p=114}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1932年]]（昭和7年） ** [[1月28日]] - 饗庭駅の北方1.8 [[キロメートル|km]]地点に停車場を新設する旨を[[鉄道省]]に申請、許可を受ける{{Sfn|江若鉄道の思い出|p=114}}。 ** [[4月1日]] - 江若鉄道の饗庭 - 近江今津間に開業{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=北饗庭駅線路配置図|left=[[饗庭駅|饗庭]]}} {{鉄道配線図|0|quai|quai|quai|0}} {{鉄道配線図|voie|voie|voie|voie|voie}} {{駅配線図終了|right=[[近江今津駅 (江若鉄道)|近江今津]]|source={{Sfn|竹内|1967}}}} 北饗庭駅は旅客の乗降のみを扱う[[旅客駅]]で{{Sfn|寺田|2010|p=10}}、[[プラットホーム|ホーム]]は線路の片側（近江今津方面に向かって左側）に1面のみ（[[単式ホーム]]）{{Sfn|レイル|p=81}}{{Sfn|レイル|p=85}}。ホームには瓦葺きの待合室が設置されていた{{Sfn|江若鉄道の思い出|p=114}}。当初より駅員は配置されていない[[無人駅]]である{{Sfn|江若鉄道の思い出|p=114}}{{Sfn|寺田|2010|p=15}}。 == 利用状況 == 初期の年間乗降客数の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=4|年間乗降客数の推移 |- !rowspan=2|年!!colspan=2|旅客!!rowspan=2|出典 |- !乗車!!降車 |- |1934年||4,132人||4,380人||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||4,842人||5,155人||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||6,229人||6,533人||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅は田園地帯の中にあった{{Sfn|レイル|p=85}}。ただ、近くを旧国道が通っていたため利用する住民は多かった{{Sfn|ありし日の江若鉄道|p=25}}。 駅があった場所は旧国道と[[国道161号]]が交わる饗庭交差点のやや南に相当する{{Sfn|江若鉄道の思い出|p=114}}。当駅付近の線路跡については、江若鉄道の廃線後に開業した[[湖西線]]の線路の東側に一般道としてわずかに残されている区間があるものの、それ以外はおおむね湖西線に取り込まれている{{Sfn|吉田|1998|p=195}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[饗庭駅]] -
