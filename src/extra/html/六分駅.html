新潟電鉄の駅として開業。開業当初から交換可能駅だった<ref name="shogakukan">『JR・私鉄全線 各駅停車6 中央・上信越590駅』小学館、1993年、184頁。</ref>。 * [[1943年]]（昭和18年）[[12月31日]] - [[新潟交通]]の発足により同社の駅となる。 * [[1975年]]（昭和50年）[[2月1日]] - 貨物営業が廃止される<ref name="shogakukan" />。 * [[1981年]]（昭和56年）[[10月1日]] - 請負化される。7時から11時まで人員が配置された<ref name="shogakukan" />。 * [[1993年]]（平成5年）[[8月1日]] - 電車線[[月潟駅]] - [[燕駅]]間廃線により廃駅{{R|kotsu19930707}}。 {{-}} == 駅跡 == {{Multiple image|align=left|image1=Station_signboard,_Remains_of_Rokubu_Station,_Niigata_Kotsu_Railway,_April_2014.JPG|width1=250|caption1=六分駅の跡地から堤防側へ道路を隔てた場所に建てられた駅名標（[[2014年]]4月5日）。|image2=Residential_area,_Remains_of_Rokubu_Station,_Niigata_Kotsu_Railway.jpg|width2=250|caption2=六分駅の跡地は住宅地として再開発されている。庭木や塀のある辺りが線路の位置。新飯田側を望む（2007年9月15日）。|image3=Park, Remains of Rokubu Station, Niigata Kotsu Railway.JPG|width3=250|caption3=公園として整備された現在の六分駅跡地。月潟側を望む（2014年4月5日）。}} {{-}} == 隣の駅 == ; 新潟交通 : 電車線 :: [[月潟駅]] -
