[[日本国有鉄道|国有鉄道]]の駅（[[鉄道駅#一般駅|一般駅]]）として開業。 *[[1909年]]（明治42年）[[10月12日]] - [[鉄道路線の名称|線路名称]]制定、北陸本線所属駅となる。 *[[1957年]]（[[昭和]]32年）[[10月1日]] - 北陸本線[[木ノ本駅|木ノ本]] - [[敦賀駅|敦賀]]間線路付替えにより、柳ヶ瀬線の所属となる。 *[[1960年]]（昭和35年）8月1日 - 貨物の取扱を廃止、[[鉄道駅#旅客駅|旅客駅]]となる。[[チッキ|荷物]]の取扱は継続。 *[[1964年]]（昭和39年）[[5月11日]] - 柳ヶ瀬線廃線に伴い廃止。 == 隣の駅 == ;日本国有鉄道 :柳ヶ瀬線 ::[[木ノ本駅]] - '''中ノ郷駅''' - [[柳ヶ瀬駅]] == 駅跡地 == 路盤は国道に転用。駅跡は余呉町役場（現在は長浜市役所余呉支所）になった。 ホーム跡（一部のみ）には駅跡を示す駅名標のレプリカが立っている。周囲の施設跡は一旦農地に戻された後、売却され建物が建っている。駅名は正式には「中ノ郷」であるが地名の「中之郷」としばしば表記されることも多く、レプリカも初代が「中の郷」、その後「中之郷」、現役版も「中之郷」で、これは地名に合わせてそうされていると思われる。しかし、そのレプリカの駅名をそのまま記した出版物が今も昔も後を絶たない(以下参照）。ちなみに、レプリカの次駅の案内表記も地名に合わせて木之本で、正式な駅名表記の木ノ本にされていないので、取材者は注意が必要である。 誤記例：[[敦賀市立博物館]]『敦賀長浜鉄道物語』2006年 　　　　[[宮脇俊三]]「木ノ本―今庄間の廃線跡を行く」『日本鉄道名所6 北陸線 関西線 紀勢線』所収 [[小学館]] 1987年 　　　　[[所澤秀樹]]「失われた鉄道を求めて七つの廃線跡」別冊太陽『鉄道に魅せられた旅人 宮脇俊三』所収 [[平凡社]] 2007年 　　　　宮脇俊三編著『鉄道廃線跡を歩く』[[徳田耕一]]・太田和年「北陸本線旧線」[[JTB]] 1995年 　　　　吉田恭一『地形図で辿る廃線跡』心交社 1998年→木ノ本駅を木之本駅と記す誤記もある。 このように、鉄道を専門に扱う書籍でさえ誤記が後を絶たない。上記誤記例は、いずれもレプリカ駅名標（駅名ではなく、地名の中之郷表記）の写真が入っているもので、写真入りの書籍や記事での間違いの割合が上記のように非常に高い。昔の時刻表や資料で確認せずに、地名表記のレプリカ駅名標の表記をそのまま記していると言えよう。それを物語るように、レプリカ駅名標の写真が入っていない文献では正しく中ノ郷、木ノ本になっているケースが多い。一方で、写真入りでも正確な駅名を記したものは、当地を熟知している筆者によるものなどでは見られる。 == 脚注 == {{reflist}} == 外部リンク == *[http://inaviss.shiga-ec.ed.jp/data/heiwa/kokai/anime2.html プラットホームは知っている -
