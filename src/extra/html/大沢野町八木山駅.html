'''富山鉄道'''大沢野駅（後の大沢野北口駅） - 笹津駅間に'''八木山臨時駅'''（やきやまりんじえき）として新設開業<ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）38ページより。</ref>（堀川新駅（現・南富山駅）起点10.3km地点<ref name="bunken2008M-06"/>）。 * [[1926年]]（大正15年）[[7月12日]] - 常設駅に昇格。'''八木山駅'''となる<ref name="bunken2008M-06"/>。 * [[1933年]]（昭和8年）[[4月20日]] - 堀川新駅（現・南富山駅） - 笹津駅間部分廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]][[12月]]発行）44-45,47ページより。</ref><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）211ページより。</ref>。 * [[1952年]]（昭和27年）[[8月15日]] - '''富山地方鉄道笹津線'''大久保町駅 - 地鉄笹津駅間延伸開通（笹津線全通）に伴い、0.2km地鉄笹津方に移設（南富山駅起点10.5km地点<ref name="bunken2008M-06"/>）の上、'''大沢野町八木山駅'''として再開業<ref name="bunken2008M-06"/><ref name="bunkenKW07"/><ref name="bunken2010H-3-R"/><ref name="bunken2008SH-3-R">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、2008年5月発行）165ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="bunken2008SH-3-R"/>。 * [[1975年]]（昭和50年）[[4月1日]] - 笹津線の廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW07"/><ref name="bunken2010H-3-R"/><ref name="bunken2008SH-3-R"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML107"/>。[[踏切#構内踏切|構内踏切]]の位置の関係で右側通行であり、西側（駅舎側）が下り線、東側が上り線となっていた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunkenRML107"/>。駅舎は構内の西側に位置し、ホームとは構内踏切で連絡していた<ref name="bunkenRML107"/>。駅舎は駅前広場に比べ一段高い位置にあり、入口へは階段を上る必要があった<ref name="bunkenRML107"/><ref name="bunkenToyama2006H">書籍『鉄道の記憶』（編著：草卓人、[[桂書房]]、[[2006年]][[2月]]発行）269,301ページより。</ref>。またホームはさらに駅舎より一段高い位置に存在し、こちらも階段を上る必要があった<ref name="bunkenRML107"/>。 列車交換は主に朝ラッシュ時の列車が行った<ref name="bunken2008SH-3">書籍『私鉄の廃線跡を歩くIII 北陸・上越・近畿編』（著：寺田裕一、JTBパブリッシング、[[2008年]][[5月]]発行）80-83ページより。</ref>。列車交換の通標は大久保町駅 - 当駅間が「□」、当駅 - 地鉄笹津駅間が「○」であった<ref name="bunkenRML107"/>。 駅前広場には[[噴水]]も設置されていた<ref name="bunkenRML107"/>。 == 駅周辺 == 旧・[[上新川郡]][[大沢野町]]の中心駅であった<ref name="bunkenToyama2008H"/>。当駅 - 敷紡前駅間は[[河岸段丘]]の下を通っており、山間の雰囲気があった<ref name="bunkenRML107"/>。 * [[国道41号]]（越中東街道） * 大沢野町役場 - 現・富山市役所大沢野総合行政センター * 大沢野図書館 * 大沢野小学校 * 八木山鉱泉 - 富山鉄道時代の八木山駅は八木山鉱泉の南側に位置していた<ref name="bunkenToyama2008H"/>。 * 八木山の滝<ref name="bunkenToyama2008H"/> - 避暑地として有名であった<ref name="bunkenToyama2006H"/>。 == 駅跡 == 1998年（平成10年）時点では、駅舎入口への階段が残存しており<ref name="bunken1998H">書籍『鉄道廃線跡を歩くV』（[[JTBパブリッシング]]、[[1998年]][[6月]]発行）80-81ページより。</ref>、駅舎の基礎とホームの一部も残存していた<ref name="bunkenToyama2008H"/>が、2000年（平成12年）7月に撤去された<ref name="bunkenToyama2008H"/>。道路の拡幅工事に伴う撤去であった<ref name="bunkenKW07"/><ref name="bunken2008SH-3"/>。2007年（平成19年）9月時点では駅跡地には文具店が建築されていた<ref name="bunkenKW07"/><ref name="bunken2008SH-3"/>。 富山鉄道時代の八木山駅跡は、2008年（平成20年）時点では八木山鉱泉南側の「八木山鉱泉専用[[駐車場]]」となっていた<ref name="bunkenToyama2008H"/>。 また、当駅跡附近の線路跡は、1998年（平成10年）時点では南富山方の一部は[[バラスト軌道|バラスト]]も残存していた[[盛土|築堤]]として残存し<ref name="bunken1998H"/>、地鉄笹津方は大沢野町八木山駅跡から先の部分が[[サイクリングロード]]に転用されていた<ref name="bunken1998H"/>。2008年（平成20年）時点では南富山方は県道になり、地鉄笹津方は変わらずサイクリングロードで、県道とサイクリングロードの分岐点の手前部分が駅跡地であった<ref name="bunkenToyama2008H"/>。このサイクリングロードは道床をそのまま転用したような感じの<ref name="bunken2008SH-3"/>、鉄道の雰囲気が感じられるものであった<ref name="bunkenKW07"/>。 == 隣の駅 == ; 富山地方鉄道 : 笹津線 :: [[田村町駅]] -
