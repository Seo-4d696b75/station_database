[[鉄道省|鉄道院]]名寄線の名寄 - 下川間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]<ref name="hensen2">『停車場変遷大事典 国鉄・JR編 II』1998年JTB発行、P908-913。</ref>。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1960年]]（昭和35年）[[9月15日]] - 貨物取扱い廃止<ref name="hensen2"/>。 * [[1978年]]（昭和53年）[[12月1日]] - [[チッキ|荷物]]取扱い廃止<ref name="hensen2"/>。同時に出札・改札業務を停止し、旅客業務について無人化。但し、閉塞扱いの運転要員は継続配置。 * [[1986年]]（昭和61年）[[11月1日]] - 交換設備を廃止し、完全無人化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南側（遠軽方面に向かって右手側）に存在した。 国鉄時代末期に無人化されるまでは、[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム中央部分と対向側ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）207ページより。</ref>。駅舎側（南側）ホームが上り線、対向側ホームが下り線となっていた<ref name="bunken1983"/>。そのほか、下り線の遠軽方から北側に分岐し対向側ホーム手前附近で行き止まりとなっていた[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。 [[無人駅]]扱いの運転取扱い要員のみが配置されていた時期を経て完全無人駅となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の南側（遠軽方面に向かって右側）に位置し、上り線ホーム中央部分に接していた。ホームの長さは50mで、2輌編成の列車がやっと停車できる長さであった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在する地名より。地名は、名寄川の上流に位置したため<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）185ページより。</ref>、「名寄」に「上」を冠する。 == 駅周辺 == * [[国道239号]]（下川国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）15ページより。</ref> * 上名寄郵便局 * [[名寄川]]<ref name="bunken1980M"/> * [[名士バス]]「上名寄12線」停留所 - [[駅名標]]が現存する == 駅跡 == 2000年（平成12年）時点では貯木場となっており、跡形も無くなっていた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）39ページより。</ref>。2010年（平成22年）時点では、駅跡付近の国道沿いの[[バス停留所]]に[[駅名標]]が設置されているのみであった<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）33ページより。</ref>。2011年（平成23年）時点でも同様であるが、この駅名標は新しく設置されたレプリカで、上部に「上名寄駅跡」と記載されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）113ページより。</ref>。また駅跡地は仮囲いで囲われており、建築資材などが保管されている模様であった<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[中名寄駅]] -
