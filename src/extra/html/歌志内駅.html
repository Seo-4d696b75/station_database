[[北海道炭礦鉄道]]の駅（[[日本の鉄道駅#一般駅|一般駅]]）として開業<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年JTB編集・発行。</ref>。 ** - 北海道炭礦鉄道空知砿開坑<ref name="shinshishi">新歌志内市史 1994年3月発行。</ref>。下ノ沢貯炭場までの下ノ沢専用線（1.45km）運用開始<ref name="shinshishi"/><ref name="sen">延長距離は札幌鉄道局発行昭和3年度線路一覧略図。</ref>。 * [[1899年]]（明治32年）下期 - 転車台設置<ref>『北海道鉄道百年史 上巻』1976年3月 日本国有鉄道北海道総局 編集・発行。</ref>。 * [[1906年]]（明治39年）[[6月1日]] - 北海道炭礦鉄道の鉄道路線[[鉄道国有法|国有化]]により、[[官設鉄道]]に移管。 * [[1909年]]（明治42年）[[10月12日]] - [[鉄道路線の名称|線路名称]]制定、歌志内線の駅となる。 * [[1914年]]（[[大正]]3年） - 坂市太郎（後に坂炭礦（株）設立、1925年（大正14年）に住友と合併）上歌志内砿開坑<ref name="shinshishi"/>。サックシ選炭場までの上歌志内砿専用線（約2km）運用開始<ref name="shinshishi"/><ref name="sen"/>。 * 不明 - 開北炭礦（後の北炭神威砿開北坑）の選炭工場積込みホッパーへ本線から分岐する専用線を敷設<ref name="shinshishi"/><ref group="注">積込ホッパーは神威駅から北に10町（約1.1km）の後の歌神駅の南側、本線横の砂川駅に向かって右手にあった。専用線の分岐位置は昭和3年度線路一覧略図によれば起点約13.1km。大正12年版全国専用線一覧で1哩以下、昭和5年版全国専用線一覧にて作業距離最大1.3km（「開化炭礦線」と書かれているが「開北炭礦線」の誤り）。当坑は明治41年開坑の臼威炭礦が始まりで、本格的操業は大二神威炭礦を名乗った大正2年以降。その後大正7年から開北炭礦を名乗り、経営者も南昌洋行、金沢鉱業などを経て昭和18年より北炭空知鉱業所と変遷した。</ref>。 * [[1934年]]（[[昭和]]9年）[[12月3日]] - 駅舎改築<ref name="shinshishi"/>。 * [[1947年]]（昭和22年）3月 - 北炭神威砿開北坑閉坑<ref name="shinshishi"/>。 * [[1953年]]（昭和28年）[[1月1日]] - 住友石炭鉱業上歌志内砿が、合理化のため坑内で繋がった住友赤平砿へ統合<ref name="shinshishi"/>。上歌志内の設備を廃止し、赤平側の設備へ一本化<ref name="shinshishi"/>。上歌志内砿専用線も廃止となる。 * [[1963年]]（昭和38年） - 北海道炭礦汽船が空知砿を分離し、空知炭礦（株）設立<ref name="shinshishi"/>。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止<ref name="hensen"/>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、北海道旅客鉄道・日本貨物鉄道が継承。 * [[1988年]]（昭和63年） **[[4月10日]] - 石炭出荷の最終日。 ** [[4月25日]] - 歌志内線の全線廃止に伴い、廃駅となる。 == 駅構造 == [[単式ホーム]]1面1線を有していた[[地上駅]]。ホームに接して[[駅舎]]があった。数本の[[停車場#側線|側線]]が敷設されており、空知炭礦株式会社の[[空知炭鉱]]へ続く[[専用鉄道|専用線]]も存在した。 == 駅名の由来 == 当鉄道敷設時までこの地には正式な地名がなく、[[ベンジャミン・スミス・ライマン|ライマン]]の空知炭田調査以来「オタシナイ炭山」と呼ばれていた。「オタシナイ」（オタ・ウシ・ナイ）とはアイヌ語で「砂の多い川」という意味で、当地を流れるペンケウタシナイ川やパンケウタシナイ川から来ており、本来砂川駅周辺の地名であった。鉄道開業時になって当駅側にオタシナイに当て字をした「歌志内」の駅名が決定され、砂川駅は「オタシナイ」の意訳の方を宛てられて、それぞれ駅名がその地の地域名となった<ref>歌志内市史 P126-131。</ref>。 == 隣の駅 == ; 北海道旅客鉄道（JR北海道） : 歌志内線 :: [[歌神駅]] -
