'''極楽駅'''（ごくらくえき）は、[[岐阜県]][[恵那市]][[岩村町]][[飯羽間村|飯羽間]]にある[[明知鉄道]][[明知鉄道明知線|明知線]]の[[鉄道駅|駅]]である<ref name="chunichi-np-2008-12-26">“「極楽駅」開業で乗客増に期待 明知鉄道に新駅”『[[中日新聞]]』2008年12月26日（中日新聞社）</ref>。[[駅ナンバリング|駅番号]]は'''6'''。 == 概要・駅名 == 2008年に開業した、明知線で最も新しい駅で、[[無人駅]]<ref name="朝日20131213"/>。駅名は公募に寄せられた案のうち、駅所在地付近の[[小字|字]]名と、駅近くに[[鎌倉時代]]から[[南北朝時代 (日本)|南北朝時代]]にかけて存在した[[寺院]]「極楽寺」<ref name="朝日20131213"/>に由来して決められた。[[極楽寺駅]]や[[極楽橋駅]]は他の鉄道で先例があったが、極楽駅は日本初であることも理由であった<ref name="朝日20131213"/>。 [[仏教]]において[[浄土]]を意味する[[縁起]]が良い駅名であることから、駅自体を目的に訪れたり、[[硬券]]の[[切符]]を買い求めたりする旅行者も多い。極楽行き切符購入者は年4千～5千人、盛時は1万9千人にも達した<reF name="朝日20191224">[https://www.asahi.com/articles/ASMDR45K5MDROHGB006.html 金色輝く觔斗雲 より「極楽駅」感 明知鉄道、改修完成式]『[[朝日新聞]]』朝刊2019年12月24日（岐阜面）2020年1月5日閲覧</ref>。[[プラットホーム]]には[[地蔵菩薩像]]や、地元[[岩村藩]]出身の[[儒者]]・[[佐藤一斎]]の『[[言志四録]]』の一文を刻んだ碑文などが置かれている<ref name="朝日20131213">[http://www.asahi.com/special/kotoba/archive2015/moji/2013120500001.html 【観字紀行】極楽へ行ってきます(2)][[朝日新聞デジタル]]・ことばマガジン（2013年12月13日）2020年1月5日閲覧</ref>。 駅舎に極楽駅らしさが足りないという声を受けて改修され、2019年12月23日に完成式が開かれた。[[觔斗雲]]をいただいた[[ベンチ]]付き待合所、[[後光]]が差すように写真撮影できるスポットが置かれた。費用には岐阜県地方鉄道利用促進事業による補助（約350万円）のほか、[[クラウドファンディング]]による寄付金（12月23日時点で約325万円）が充てられた<reF name="朝日20191224"/>。 建設費は約2160万円<ref name="chunichi-np-2008-12-26" />。地元岐阜県に本社がある[[スーパーマーケット]]の[[バロー (チェーンストア)|バロー]]から1,080万円（費用の半額に相当する）の寄付金が寄せられた<ref>[http://www.city.ena.lg.jp/news/2008/200810/200810162/ 明知鉄道の新駅名は「極楽駅」]恵那市ウェブサイト（2008年10月16日）2018年5月6日閲覧。</ref>。バローは新駅設置に前後して、駅前に[[ショッピングセンター]]を開業している（「[[#駅周辺|駅周辺]]」参照）。 == 駅構造 == 単式1面1線の[[無人駅]]。ホームは[[2005年日本国際博覧会|愛知万博]]開催時に[[愛知環状鉄道]] [[八草駅|万博八草駅]]（現・八草駅）のホーム拡張に使われたものを再利用している。 == 歴史 == *[[2008年]]（[[平成]]20年）[[12月25日]] : '''開業'''<ref name="chunichi-np-2008-12-26" />。 *[[2012年]]（[[平成]]24年）[[3月17日]] - ダイヤ改正により急行「[[大正ロマン号]]」の停車駅（下り列車のみ）となる、。 *[[2017年]]（[[平成]]29年）[[3月4日]] - ダイヤ改正により上り列車も急行「大正ロマン号」の停車駅に。これにより上下列車とも停車。 *[[2018年]]（[[平成]]30年）[[3月17日]] - ダイヤ改正により平日上り最終快速列車が通過となる。 *[[2019年]]（[[令和]]元年）[[12月23日]] - 駅舎の改修完成式が行われた<ref>{{YouTube|eYxvO20Ytrw|見た目に「極楽」感　極楽駅がリニューアル　岐阜・明知鉄道}}（朝日新聞社提供、2019年12月24日公開）</ref>。 == 駅構造 == 1面1線の[[単式ホーム]]のみを持つ[[地上駅]]。無人駅である。 == 利用状況 == {|
