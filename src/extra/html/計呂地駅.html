'''計呂地駅'''（けろちえき）は、かつて[[北海道]]（[[網走支庁]]）[[紋別郡]][[湧別町]]字計呂地に設置されていた、[[日本国有鉄道]]（国鉄）[[湧網線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ケロ'''。湧網線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[3月20日]]に廃駅となった。 == 歴史 == * [[1935年]]（昭和10年）[[10月20日]] - [[鉄道省]]湧網西線、中湧別駅 - 当駅間の開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1936年]]（昭和11年）[[10月17日]] - 当駅 - 中佐呂間駅（後の佐呂間駅）間の延伸開通に伴い、中間駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1953年]]（昭和28年）[[10月22日]] - 中湧別駅 - 網走駅間全通により路線名を湧網線に改称、それに伴い同線の駅となる。 * [[1958年]]（昭和33年）[[11月30日]] - 駅舎改築。 * [[1972年]]（昭和47年）[[2月8日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[3月20日]] - 湧網線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面2線の[[島式ホーム]]を有する[[地上駅]]で、[[列車交換]]が可能な[[交換駅]]であった<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=160}}</ref>。駅舎側（東側）が下り線、外側（西側）が上り線となっていた（番線表示なし）。そのほか、上り線の外側に[[停車場#側線|側線]]（[[停車場#本線|副本線]]）を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南東側に位置しホーム南側とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。ホームは砂利敷きであった<ref name="bunken1983"/>。 当駅では、手作りの[[ホタテガイ|帆立]][[貝殻]]付きの[[入場券]]が発売されていた<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=163}}</ref>。 == 駅名の由来 == 当駅が所在していた地名より。地名は、[[アイヌ語]]の「ケイ・ラツ」（[[サケ]]の皮で作った[[靴]]を忘れた所）、「ケレオチ」（非常に削られた所）など、由来には諸説ある<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=168}}</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は32人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図
