当駅間の開通に伴い、'''碁盤坂駅'''（ごばんざかえき）として開業<ref name="bunken1993-R">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]][[6月]]発行）183ページより。</ref><ref name="bunken2008M">書籍『日本鉄道旅行地図帳 全線全駅全廃線 1 北海道』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[5月]]発行）27ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1942年]]（昭和17年）[[11月1日]]：当駅 - [[渡島吉岡駅]]間の開通に伴い、中間駅となる<ref name="bunken1993-R"/><ref name="bunken2008M"/>。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：福山線が'''[[松前線]]'''に改称<ref name="bunken1993-R"/><ref name="bunken2008M"/>。 * [[1970年]]（昭和45年）[[12月12日]]：出札・改札業務を停止し、旅客業務については無人化され、[[日本の鉄道駅#簡易委託駅|簡易委託駅]]となる。但し、運転要員は継続配置。 * [[1971年]]（昭和46年）[[4月26日]]：貨物の取り扱いを廃止。 * [[1972年]]（昭和47年）[[3月15日]]：'''千軒駅'''に改称<ref name="bunken2008M"/><ref>日本国有鉄道コンピューター部『鉄道要覧 昭和48年度』、日本国有鉄道、1975年1月20日、p280。</ref>。 * [[1984年]]（昭和59年）[[2月1日]]：[[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]に伴い、日本国有鉄道から[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）[[2月1日]]：松前線の全線廃止に伴い、廃駅となる<ref name="bunken1993-R"/><ref name="bunken2008M"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。その他に上り線側に副本線（[[停車場#側線|側線]]扱い）、下り線側に松前方から分岐して[[軌条|レール]]末端部分には[[枕木]]による[[車止め]]が設置された<ref name="bunken2012P">書籍『北海道の大地から消えた鉄道風景』（撮影：上田哲郎、エムジー・コーポレーションMG BOOKS、[[2012年]][[3月]]発行）125ページより。</ref>行き止まりの側線を1線有していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）62ページより。</ref>。この側線に接して、旧貨物ホームが残存していた<ref name="bunken2012P"/>。 最終営業日まで、[[無人駅]]扱いの運転取り扱い要員のみが配置されていた駅であった。[[乗車券]]類の販売は、[[日本の鉄道駅#簡易委託駅|簡易委託]]化されていた。駅舎はホームの東南側に位置し、[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。ホームは砂利敷きであった。 == 駅名の由来 == 当駅が所在した地名より。地名は、近辺に存在する山「大千軒岳」の名前に由来する。[[1628年]]（寛永5年）、この山に[[鉱山|金鉱]]が開かれ、開発で千軒の人家があったためにこの名になったという<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は43人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道228号]] * 千軒簡易郵便局 * [[大千軒岳]] -
