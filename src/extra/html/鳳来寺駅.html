'''鳳来寺駅'''（ほうらいじえき）は、[[愛知県]][[新城市]]門谷字笠川にかつて存在した[[豊橋鉄道]][[豊橋鉄道田口線|田口線]]の[[鉄道駅|駅]]。[[鳳来寺]]の最寄り駅であった。 [[1929年]]（[[昭和]]4年）に田口鉄道により開業。[[1956年]]（昭和31年）に豊橋鉄道の駅となるが、[[1968年]]（昭和43年）に路線の廃線に伴って[[廃駅|廃止]]された。[[南設楽郡]][[鳳来寺村]]（1956年以降は[[鳳来町]]、現・新城市）にあった駅の一つである。 == 歴史 == === 開業までの経緯 === 鳳来寺駅は1929年（昭和4年）[[5月22日]]、田口鉄道の第一期線として[[本長篠駅]]（当時は鳳来寺口駅）から[[三河海老駅]]までの区間が開通した際、あわせて開業した<ref name="horai-ta48">[[#horai-ta|『鳳来町誌』田口鉄道史編]]、p48</ref><ref name="chizucho">[[#chizucho|『日本鉄道旅行地図帳』7号]]、pp37-38</ref>。駅名については、当初は[[大字]]と同じ「門谷」が予定されていた<ref name="horai-ta48"/>。駅のできた門谷は[[鳳来寺山]]山麓にあり、鳳来寺の[[門前町]]として栄えた地域である<ref>[[#kadokawa|『角川日本地名大辞典』23]]、p388</ref>。同駅は、鳳来寺の[[参道|表参道]]の入り口に置かれた<ref name="horai-ko324">[[#horai-ko|『鳳来町史』交通史編]]、p324</ref>。 駅を開設した田口鉄道は、[[北設楽郡]][[田口町]]（現・[[設楽町]]）まで鉄道を敷設すべく[[1927年]]（昭和2年）11月に設立された会社である。会社設立前の構想では、[[豊川鉄道]]の終点長篠駅（現・JR[[飯田線]][[大海駅]]）を起点とし、玖老勢までは[[豊川]]（寒狭川）に沿うルートとする予定であった。しかし会社設立段階になり、長篠駅より先の区間を開通させていた[[鳳来寺鉄道]]（豊川鉄道と同様にJR飯田線の前身）や鳳来寺からの出資を得るために<ref>大株主に鳳来寺の名がみられる[{{NDLDC|1234547/179}} 『地方鉄道軌道営業年鑑』]（国立国会図書館近代デジタルライブラリー）</ref>、鳳来寺鉄道鳳来寺口駅へ起点を移し、かつ鳳来寺の前を通すルートとすることになった<ref>『鳳来町誌』田口鉄道史編、pp29-31,33-37</ref>。 === 開業後の推移 === 駅ができると、駅前には十数件の民家が集まり一つの[[集落]]が形成された<ref name="horai-ko324"/>。[[1935年]]（昭和10年）には門谷に鳳来寺女子高等学園（のちの[[愛知県立鳳来寺高等学校]]、2011年閉校）が開校する。同校は田口鉄道によるアクセスを前提として開設されたともいえた<ref>『鳳来町誌』田口鉄道史編、p67</ref>。 [[太平洋戦争]]中の[[1945年]]（昭和20年）[[7月27日]]より、駅に田口鉄道の[[本社]]が入った。もともと本社は[[豊橋市]]内にあったが、同年の[[豊橋空襲]]で被災したため移転してきたものである<ref>『鳳来町誌』田口鉄道史編、pp125-126</ref>。 戦後の[[1949年]]（昭和24年）[[9月23日]]、駅前に「田口鉄道自然科学博物館」が開館した。「東三河の地質と鉱物の会」に田口鉄道が協力して創設したもので、鳳来寺山を中心とする資料を展示した。この施設は[[1963年]]（昭和38年）に開館した町立[[鳳来寺山自然科学博物館]]の基礎となっている<ref>『鳳来町誌』田口鉄道史編、pp148-149,187</ref><ref>[[#shizen|『鳳来町自然と文化』]]</ref>。 鳳来寺の最寄り駅として毎年[[11月23日]]の鳳来寺山もみじ祭の際には、飯田線から[[臨時列車]]が乗り入れていた<ref name="shitetsu">[[#shitetsu|『私鉄廃線跡を歩く』]]2、p55</ref>。また[[1950年]]（昭和25年）から[[1954年]]（昭和29年）にかけて、[[名古屋鉄道]]（名鉄）の[[名鉄小坂井支線|小坂井支線]]と飯田線を介して、名鉄の[[団体専用列車|団体用臨時列車]]が新名古屋駅（現・[[名鉄名古屋駅]]）方面から乗り入れていたこともある<ref>『鳳来町誌』田口鉄道史編、p151</ref> [[1956年]]（昭和31年）[[10月1日]]、田口鉄道は[[豊橋鉄道]]に[[合併 (企業)|合併]]された。これに伴い豊橋鉄道田口線の駅となった<ref>『鳳来町誌』田口鉄道史編、p152</ref>。 === 廃止とその後 === 合併から8年後の[[1964年]]（昭和39年）、豊橋鉄道は[[赤字]]の拡大に伴って田口線の廃止を決定<ref>『鳳来町誌』田口鉄道史編、pp161-162</ref>、4年後の[[1968年]]（昭和43年）8月末を以って全線廃止、[[9月1日]]よりバスへの転換が実施され<ref>『鳳来町誌』田口鉄道史編、pp172-173</ref>、駅もあわせて廃止された<ref name="chizucho"/>。 [[2012年]]（平成24年）現在、本長篠駅と設楽町田口を結ぶのは[[豊鉄バス]][[豊鉄バス新城営業所#田口新城線|田口新城線]]で、[[新城駅]]近くの[[新城市民病院]](以前の名称は新城病院前)[[バス停留所|バス停]]から本長篠駅前経由で田口まで運行されている<ref>「{{PDFlink|[http://www.toyotetsu.jp/routemap/img/shinshiro.pdf 豊鉄バス路線系統図 新城営業所管内]}}」（豊鉄バス公式サイト）、2015年10月22日閲覧</ref>。門谷地内ではバスは[[愛知県道32号長篠東栄線]]を走行しており、駅のあった字笠川に「鳳来寺」バス停が設置されている<ref>[http://www.mapion.co.jp/phonebook/M12001/23221/L0289919/ Mapion電話帳 鳳来寺バス停]、2012年7月29日閲覧</ref>。 また、当駅にあった腕木式信号機は愛知県春日井市にある[[中部大学]]に移設され、保存されている。 == 構造 == 相対式2面2線ホームと貨物ホーム片面1線を持つ<ref>{{Cite journal|和書|author = 白井良和|title = 奥三河に咲いたローカル線 田口線の回想|date = 1986-03|publisher = 鉄道図書刊行会 |journal = 鉄道ピクトリアル|number = 461|page = 59}}</ref>[[日本の鉄道駅#有人駅|有人駅]]で、1956年（昭和31年）の時点で[[駅長]]を含めて4人配置されていた<ref>[[#zuroku|『図録』]]、p8</ref>。 == 利用状況 == 1957年度の乗車人員は20万4千人（1日平均558人）で、そのうち4分の3にあたる15万4千人が[[定期乗車券]]での利用客であった。この乗車人員は田口線全11駅において、[[本長篠駅]]に次いで2番目に多い<ref name="tokei">[[#tokei|『愛知県統計年鑑』昭和34年版]]、p386</ref>。鳳来寺高校の学生の利用が多かった<ref name="shitetsu"/>。 同年度における貨物取扱量は、発送が49トン、到着が71トンであった。田口線の貨物取扱駅は7駅あったが、取扱量はその中で最少である<ref name="tokei" />。 == 駅の跡地 == 駅の跡地は上記の鳳来寺バス停付近である。駅舎跡は[[愛知県道32号長篠東栄線|愛知県道32号]]と鳳来寺の参道（[[愛知県道441号鳳来寺山公園線|愛知県道441号]]）の交差点付近にあたる。線路跡は県道32号に吸収された<ref>[[#new|『新鉄道廃線跡を歩く』3]]、p132</ref>。 駅舎は駅の廃止後も解体されず、改装の上[[1969年]]（昭和44年）[[10月10日]]より「豊鉄ほうらいじ食堂」として営業を開始した<ref>[[#toyo|『豊橋鉄道50年史』]]、p162</ref>。道路改修工事によりこの食堂は撤去されるものの隣地に新たな食堂を建設して営業を続けた<ref>[[#haisen|『鉄道廃線跡を歩く』]]、p116</ref>が、{{要出典範囲|[[2004年]]（平成16年）に閉鎖され、[[2009年]]（平成21年）に|date=2012年8月}}取り壊された。
