[[日本国有鉄道]]（国鉄）名寄本線の駅として開業。旅客のみ取り扱い。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の北西側（遠軽方面に向かって左手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=210}}</ref>。 [[無人駅]]となっており、駅舎及び[[待合所]]は存在しなかった。ホームは名寄方に90度の角度で設置された[[階段]]を有し<ref name="bunken1983"/>、駅施設外に連絡していた。簡易型の単式ホームであった。駅自体は完全無人駅であったが、近隣の商店に[[乗車券]]の販売を委託した[[日本の鉄道駅#簡易委託駅|簡易委託駅]]であった。 == 駅名の由来 == 当駅の所在地近辺の地名より<ref name="bunken1980M-2">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=35}}</ref>。所在する真砂町と駅名由来の潮見町は隣接しており、当駅及び駅前後の線路は2町のほぼ境界線に沿って通っていた<ref name="bunken1980M-2"/>。 潮見町の地名は、紋別市内で海が最もよく見えることから付けられた<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=187}}</ref>。 == 駅周辺 == 旧[[北海道紋別北高等学校]]などの通勤通学の最寄り駅として利用された。 * [[国道238号]]（オホーツク国道）<ref name="bunken1980M-2"/> * 北海道紋別北高等学校 - 現在は閉校。 * 紋別市立潮見中学校 * 紋別市立潮見小学校 * モリタン本社兼工場 * 落石団地<ref name="bunken1980M-2"/> * ウェンヒラリ岬 - 駅から北に約1km<ref name="bunken1983"/> * [[北紋バス]] ** 「潮見町3丁目」[[バス停留所]] ** 「北浜1丁目」停留所 == 駅跡 == [[2001年]]（平成13年）時点ではホームと線路は撤去され、駅跡地にはコンクリート製のホームに上る階段のみが残存していた<ref name="bunken2001H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |page=35}}</ref>。当駅跡から遠軽方の線路跡も、旧潮見坂踏切まで砂利敷きの道床が残存していた<ref name="bunken2001H"/>。[[2011年]]（平成23年）時点でも同様で階段のみが残存し<ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=[[北海道新聞社]] |isbn=978-4894536128 |page=123}}</ref>、遠軽方の築堤上には「L」の記載や「15[[パーミル|‰]]」の記載が残る[[鉄道標識#勾配標|勾配標]]が残存していた<ref name="bunken2011H"/>。この築堤は道路として再利用されている模様であった<ref name="bunken2011H"/>。駅跡は空き地として放置されたあと、並行市道の拡張用地として道路化された。その後、駅跡に残されていた階段も撤去された。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[渚滑駅]] -
