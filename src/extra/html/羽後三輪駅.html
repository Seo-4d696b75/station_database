'''雄勝鉄道'''湯沢駅 - 西馬音内駅間開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年8月発行）30,32-33,36ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月16日]] - [[陸上交通事業調整法|交通統合]]に伴い'''横荘鉄道'''の駅となる<ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1944年]]（昭和19年）[[6月1日]] - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''雄勝線'''に制定。それに伴い'''羽後鉄道雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">『私鉄の廃線跡を歩くI』106-109ページより。</ref>。 * [[1947年]]（昭和22年） ** [[7月23日]] - 豪雨による路盤及び橋脚損壊により雄勝線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、ネコ・パブリッシング、2003年11月発行）17ページより。</ref>。 ** [[8月14日]] - 羽後山田駅 - 梺駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 * [[1952年]]（昭和27年）[[2月15日]] - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1970年]]（昭和45年）[[10月1日]] - 車扱貨物営業が休止となる<ref name="bunkenRML052">『RM LIBRARY 52』（4,15,19,20-21,27,30,32-33,37ページより。</ref>。 * [[1971年]]（昭和46年）[[7月26日]] - 閉塞取扱が廃止となる<ref name="bunkenKW03"/><ref name="bunken2008SH-1-R"/>。 * [[1973年]]（昭和48年）[[4月1日]] - 雄勝線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。晩年に閉塞取扱が廃止になるまでは島式ホーム1面2線を有する[[列車交換]]可能な交換駅であった<ref name="bunkenRML052"/>。外側（西側）が下り線（梺方面）、駅舎側（東側）が上り線（湯沢方面）となっていた<ref name="bunkenRML052"/>。そのほか[[停車場#側線|側線]]として、上り線から東に分岐し駅舎との間を通過し、駅舎東側の貨物積卸場への[[貨物線]]を1線有していた<ref name="bunkenRML052"/>。貨物積卸場には農業倉庫も設置されていた<ref name="bunken2008SH-1"/>。閉塞取扱廃止後の側線の状況は不明である。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の北東側に位置し、ホーム北側の[[斜路|スロープ]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML052"/>。 閉塞取扱廃止前までの列車交換の通標は、湯沢駅 - 当駅間は「○」、当駅 - 西馬音内駅間は「□」であった<ref name="bunkenRML052"/>。また、1965年（昭和40年）12月8日<ref name="bunken2008SH-1-R"/>までの列車交換の通標は、羽後山田駅 - 当駅間は「○」、当駅 - 西馬音内駅間は「□」であった<ref name="bunkenRML052"/>。 == 駅周辺 == 旧・雄勝郡三輪村の中心駅であった。 * [[秋田県道276号下開清水線]] -
