'''佐呂間駅'''（さろまえき）は、[[北海道]]（[[網走支庁]]）[[常呂郡]][[佐呂間町]]字永代町にかつて存在した、[[日本国有鉄道]]（国鉄）[[湧網線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ロマ'''。湧網線の[[廃線]]（廃止）に伴い、[[1987年]]（昭和62年）[[3月20日]]に廃駅となった。 == 歴史 == * [[1936年]]（昭和11年）[[10月17日]] - [[鉄道省]]湧網西線、計呂地駅 - 当駅間の延伸開通に伴い、'''中佐呂間駅'''（なかさろまえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1953年]]（昭和28年）[[10月22日]] - 当駅 - 下佐呂間駅（後の浜佐呂間駅）間の延伸開通（全通）により路線名が湧網線に改称され、それに伴い同線の駅となる。同時に'''佐呂間駅'''に改称。 * [[1982年]]（昭和57年）[[3月29日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[3月20日]] - 湧網線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）を複合した計2面2線のホームと線路を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム東側と対向側ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=160}}</ref>。駅舎側ホーム（北側）が上りの1番線、対向側ホーム（南側）が下りの2番線となっていた<ref name="bunken1983"/>。島式ホームの外側の1線は[[停車場#本線|副本線]]として[[停車場#側線|側線]]扱いで残っていた<ref name="bunken1983"/>。この副本線の外側に側線を3線（うち1線は行き止まり）、副本線中途から分岐する行き止まりの短い側線を東西に各1線有した<ref name="bunken1983"/>。そのほか、1番線の中湧別方から駅舎側に分岐し駅舎東側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、本線の網走方・1、2番線の収束部分の[[分岐器|転轍機]]手前から駅舎側に分岐する側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の北側に位置し単式ホーム中央部分に接していた<ref name="bunken1983"/>。駅舎とは別棟でトイレ棟を有した<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=165}}</ref>。トイレ棟は、駅前広場から向かって駅舎の右側にあった<ref name="bunken2011P"/>。 == 駅名の由来 == 当駅が所在していた地名より。地名は、[[アイヌ語]]の「サル・オマッ・ペッ」（[[ヨシ|芦]]原にある川）の転訛した「サロマペッ」の前半部分に由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=169}}</ref>。佐呂間川を指す<ref name="bunken2004N"/>。 旧駅名の中佐呂間（なかさろま）は、佐呂間の中央部に位置していたため<ref name="bunken2004N"/>、「中」を冠した。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は220人<ref
