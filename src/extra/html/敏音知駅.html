'''敏音知駅'''（ぴんねしりえき）は、[[北海道]]（[[宗谷支庁]]）[[枝幸郡]][[中頓別町]]字敏音知にあった[[北海道旅客鉄道]]（JR北海道）[[天北線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ヒネ'''。天北線の[[廃線]]に伴い[[1989年]]（平成元年）[[5月1日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]2面2線を有する[[地上駅]]であった。ホームが千鳥式に配置された[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）188ページより。</ref>。互いのホームは駅舎側ホーム北側と対向ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側ホーム（南西側）が下り線、対向ホーム（北東側）が上り線となっていた<ref name="bunken1983"/>。そのほか上り線の南稚内方から分岐し構内外側への行き止まりの[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南西側に位置し両ホームとは通路及び構内踏切で連絡した<ref name="bunken1983"/>。古い木造ながら、チョコレート色の化粧ベニヤ張りの駅舎であった<ref name="bunken1983"/>。ホームの長さは50メートルで、[[サルビア]]が植えられている花壇があった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅の所在する地名より。地名は、[[アイヌ語]]の「ピン・ネ・シリ」（男の山）に由来する。附近に二つの目立つ山があり、それぞれアイヌ語で「ピン・ネ・シリ」「マツ・ネ・シリ」（女の山）と呼ばれていた。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は12人<ref name="bunken1983"/>。 {| class="wikitable" style="margin:
