[[日本国有鉄道]]美幸線美深駅 - 当駅間開通に伴い開業<ref name=":0">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=184}}</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1966年]]（昭和41年）[[12月21日]] - 駅舎全焼。 * [[1967年]]（昭和42年）[[11月6日]] - 新駅舎落成。 * [[1984年]]（昭和59年）[[2月1日]] - 貨物・[[チッキ|荷物]]取扱い廃止。 * [[1985年]]（昭和60年）[[9月17日]] - 美幸線の廃線に伴い廃止となる。 === 駅名の由来 === 地名より<ref name=":0" />。[[アイヌ語]]の「ニウプ（niupu）」に由来するが、諸説あり、[[更科源蔵]]は「森林がある川」の意である、としている<ref name=":1">{{Cite book|author=本多 貢|editor=児玉　芳明|title=北海道地名漢字解|url=https://www.worldcat.org/oclc/40491505|date=1995-01-25|year=|accessdate=2018-10-16|publisher=[[北海道新聞社]]|language=日本語|isbn=4893637606|page=69|location=札幌市|last2=|author2=|first2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|oclc=40491505}}</ref>。国鉄北海道総局が発行した『北海道　駅名の起源』には「『ペンケニウプ』（小川の上流の森）<ref name=":0" />」から、とされている。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。ホームは線路の北東側（仁宇布方面に向かって左手側）に存在した。列車の発着に使用する駅舎側の1番線だけではなく、駅舎と反対側の乗り場（2番線）の線路も敷設され[[停車場#側線|側線]]扱いとなっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）192ページより。</ref>。2線はホームの先で収束し、路線延長時には[[列車交換]]が可能な構造となっていた。ほかに2番線の外側に貨物用の側線（3番線）を1本有した<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）36ページより。</ref>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、木造[[プレハブ構造|プレハブ]]造りの駅舎が存在した。駅舎は構内の南東側に位置し、ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983" />。1981年（昭和56年）時点で職員は合計6人が配置されていた<ref name="bunken1983" />。 また、1980年（昭和55年）時点では、駅裏に[[日本鉄道建設公団]]の線路敷設基地が置かれており<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]]10月発行）28ページより。</ref>、構内にはレールや枕木が積まれていた<ref name="bunken1983" />。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は20人<ref name="bunken1983" />。 == 駅周辺 == * 仁宇布簡易郵便局 *[[松山湿原]] - 駅から南に約5km<ref name="bunken1983" />。 *[[トロッコ王国美深]] *[[スバル]]耐寒テストコース　日本最北の自動車テストコース * 付近の農場は[[村上春樹]]の[[羊をめぐる冒険]]のモデルのひとつではないかといわれている[http://matsuyama-farm.com/archives/2004/12/29221229.html][http://matsuyama-farm.com/archives/2008/05/09022317.html]。 *[[名士バス]]「仁宇布待合室」停留所 *美深町立仁宇布小中学校 == 駅跡 == [[1998年]]（平成10年）より美深町のNPO法人'''トロッコ王国美深'''の施設として使われている。2000年（平成12年）時点で駅舎は既に撤去されているが<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）27-29ページより。</ref>、2010年（平成22年）時点でホームや線路は残存している。駅舎跡にログハウス風の事務所が置かれ、当駅跡から辺渓駅方、高広の滝附近へ約5kmの線路が再利用され、[[エンジン]]付き保線用[[軌道自転車]]の運転体験ができる<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、2010年4月発行）29-30ページより。</ref>。 施設内には[[国鉄583系電車]]の中間車両が置かれている（但し、状態は良くない）。また、廃駅となった[[智東駅]]の[[貨車]]駅舎（[[国鉄ヨ3500形貨車|ヨ3500形]][[車掌車]]を改造）が2006年（平成18年）7月3日に移動されて設置されている<ref name="bunken2007">書籍『ダルマ駅へ行こう!』（著：笹田昌宏、[[小学館文庫]]、[[2007年]]5月発行）29-30ページより。</ref>。沿線に残存している保線小屋が資材置き場に転用されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）258ページより。</ref>。 また、旧駅前広場部分には[[蒸気機関車]]の[[車輪]]<ref name="bunken2011H"/>のモニュメント付きの[[石碑]]、「美幸線記念碑」が建立されている<ref name="bunken2000H"/>。2000年（平成12年）時点では北見枝幸方へも、北へ0.1kmほどの線路が残存していたが<ref name="bunken2000H"/>、2010年（平成22年）時点では駅跡の北側までで終わっていた。路盤は2010年時点でもさらに北に続いている<ref name="bunken2010H"/>。 == 隣の駅 == ; 日本国有鉄道 : 美幸線 ::[[辺渓駅]] -
