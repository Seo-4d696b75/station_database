三段峡間）の廃線に伴い、[[2003年]]（平成15年）[[12月1日]]に廃止された。 == 歴史 == * [[1969年]]（昭和44年）[[7月27日]] - [[日本国有鉄道|国鉄]]可部線[[加計駅|加計]] - [[三段峡駅|三段峡]]間開通時に開業{{R|停車場}}。[[鉄道駅#一般駅|一般駅]]。 ** 当時の所在地表示は広島県山県郡戸河内町上本郷であり、廃止まで変わらなかった。 * [[1978年]]（昭和53年）[[10月1日]] - [[車扱貨物]]の取扱を廃止（[[鉄道駅#旅客駅|旅客駅]]となる）{{R|停車場}} * [[1985年]]（昭和60年）[[2月1日]] - 無人駅化<ref>{{Cite news |title=通報 |newspaper=[[鉄道公報]] |publisher=[[日本国有鉄道]]総裁室文書課 |date=1985-02-01 |page=2 }}</ref>（[[加計駅]]管理による乗車券類発売は継続） * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR西日本が継承{{R|停車場}}。 * [[2003年]]（平成15年）[[12月1日]] - 廃止。 == 駅構造 == [[ファイル:Togouchi stn.jpg|thumb|200px|ホーム（2003年8月）]] 1面2線の[[島式ホーム]]を持つ[[地上駅]]。当初は交換設備を使用していたが、晩年は加計 - 三段峡間のスタフ閉塞化により実質棒線化され（島式ホームの南側はレールは残されていたがポイントが固定され、構内の信号機はすべて使用停止・消灯）、島式ホームの北側のみ使用されていた。[[駅舎]]があり、[[日本の鉄道駅#簡易委託駅|簡易委託駅]]であった。駅舎とホームは下り線路と平面交差する通路で結ばれ、東側からホームに上がっていた。 == 駅周辺 == 旧戸河内町の中心部に当たり、戸河内町役場（現：安芸太田町役場）と戸河内町立国民健康保険病院（現：安芸太田町戸河内病院）があった。[[太田川]]が駅の裏側（北側）を流れ、駅のすぐ南側を[[国道191号]]が通っている。 [[ファイル:Togochi st3.JPG|thumb|200px|駅跡]] == 現状 == 駅舎、プラットホーム、駅付近のレールなどはすべて解体・撤去され、跡地にはバス待合所（戸河内役場（現：安芸太田町役場）バス停）が建設された。 == その他 == * 駅名は、駅がある自治体名（広島県山県郡戸河内町）に由来。 * 可部線のうち加計 -
