'''北海鋼機前駅'''（ほっかいこうきまええき）は、[[北海道]][[江別市]]東野幌町にあった[[北海道炭礦汽船夕張鉄道線|夕張鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）である。自社による札幌への連絡バス発着のために野幌バス停留所前駅として[[1956年]]12月に開設され、夕張鉄道線の廃止に伴い[[1975年]]に廃止された。 == 駅構造 == * [[単式ホーム]]1面1線の[[地上駅]]だった。 * ホーム上から連絡バスが発着し、バス車庫も設置されていた。 == 歴史 == * [[1956年]]（昭和31年）[[12月1日]]　'''野幌バス停留所駅'''設置、当駅-札幌大通間のバス連絡輸送開始。従来の上江別駅でのバス連絡廃止。 * [[1964年]]（昭和39年） ** [[7月20日]]　北海鋼機の工場進出に伴い'''北海鋼機前駅'''に改称。 ** [[9月14日]]　北海鋼機専用鉄道運輸開始。　 * [[1974年]]（昭和49年）[[4月1日]]　旅客営業休止。 * [[1975年]]（昭和50年）4月1日　廃止。 == 駅周辺 == * [[夕張鉄道|夕鉄バス]]野幌営業所（バスターミナル併設） == 隣の駅 == : 夕張鉄道線 :: [[野幌駅]] - '''北海鋼機前駅''' - [[上江別駅]] == 接続路線 == * 北海鋼機専用鉄道 == その他 == * 北海鋼機工場横のきらら街道（広域農道）に信号機を利用した、夕張鉄道線の歴史を記した記念碑がある。 * バス連絡に使われていたホームは、駅廃止後の1990年代まで同社のバスが乗り入れていた。 *
