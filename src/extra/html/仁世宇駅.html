'''[[日本国有鉄道|国有鉄道]]富内線'''振内駅 - 日高町駅間の延伸開通に伴い、開業。旅客のみ取り扱い。　 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の南側（日高町方面に向かって右手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）108ページより。</ref>。 開業時からの[[無人駅]]で駅舎は無かったが、ホーム東側出入口附近にホームに屋根が掛けられた形の[[待合所]]を有していた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）91-92ページより。</ref>。 == 駅名の由来 == 当駅の所在地近辺の地名より<ref group="注" name="chimei">駅所在地の北に「仁世宇」の地名あり（書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）12ページより）。</ref>。地名は、[[アイヌ語]]の「ニセウウㇱイ（niseu-us-i）」（ドングリ・多い・ところ）に由来する<ref>{{Cite book|author=本多 貢|editor=児玉　芳明|title=北海道地名漢字解|url=https://www.worldcat.org/oclc/40491505|date=1995-01-25|year=|accessdate=2018-11-14|publisher=[[北海道新聞社]]|language=日本語|isbn=4893637606|page=70|location=札幌市|last2=|author2=|first2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|oclc=40491505}}</ref>。 当駅所在地の集落名は「瑞穂」であったが、同名の地名が各所にあるという理由により近くの集落名を駅名に採用したという<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は14人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道237号]]（日高国道） * [[北海道道638号宿志別振内停車場線]] * [[沙流川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）11ページより。</ref> * ニセウ川<ref name="bunken1980M"/> == 駅跡 == 2011年（平成23年）時点では更地になっているが、線路跡は築堤となって残存しており<ref name="bunken2011H"/>、近くの[[都道府県道|道道]]にも[[踏切]]跡が残存する<ref name="bunken2011H"/>。また、1999年（平成11年）時点では附近の線路跡に[[バラスト軌道|バラスト]]が残存していた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）66ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 富内線 :: [[振内駅]] -
