岡山臨港鉄道 : 岡山臨港鉄道線 :: [[岡南泉田駅]] - '''岡南福田駅''' - [[並木町駅]] == 宇野への延伸計画 == かつて、当駅から[[宇野]]まで路線を延伸する計画があった。[[1952年]][[2月15日]]、当駅（当時、臨港福田）から臨港宇野への路線延伸が取締役会で決議された。[[児島湾]]を横断する[[児島湾締切堤防]]の建設が[[1951年]]から始まり、それを経由して宇野に至る計画であり、1953年4月14日に運輸大臣に申請を行った。しかし、[[1966年]]に、[[瀬戸大橋]]の経路が[[坂出]] - [[下津井]]間に一本化されたため、延伸の必要性が薄まり、1966年12月7日に申請を取り下げた{{Sfn|寺田裕一|2016|p=13}}。 == 脚注 == {{Reflist}} == 参考文献 == * {{Cite book|和書|author = 寺田裕一||editor = 名取紀之|title =岡山臨海鐡道|edition =初版|date = 2016-01-01|publisher = ネコ・パブリッシング |location = [[東京都]][[目黒区]]|series = RM LIBRARY 197|isbn = 978-4-7770-5391-9|pages = 28-33|chapter = 線路・駅|ref = harv}} == 関連項目 == * [[廃駅]] == 外部リンク == * {{Cite web |author=岡山臨海グループ |authorlink= |coauthors= |date= |url= http://www.okarin.co.jp/museum/top.html|title= 岡山臨海鐡道博物館|format= |doi= |work= |publisher= |page= |pages= |language= |archiveurl= |archivedate= |accessdate=2016-03-20 |quote= |ref=}} {{岡山臨港鉄道}} {{Eki-stub}} {{DEFAULTSORT:こうなんふくた}} [[Category:日本の鉄道駅 こ|うなんふくた]] [[Category:岡山市の廃駅]] [[Category:岡山市南区の鉄道駅|廃こうなんふくた]] [[Category:岡山臨港鉄道]] [[Category:1951年開業の鉄道駅]] [[Category:1984年廃止の鉄道駅]] [[Category:岡山市南区の歴史|こうなんふくたえき]] 0x4pz5bl32mz2wlu12eoxadkh19ngvi
