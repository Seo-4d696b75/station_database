'''東鶉駅'''（ひがしうずらえき）は、[[北海道]][[空知郡]][[上砂川町]]字鶉にあった[[北海道旅客鉄道]]（JR北海道）[[函館本線]]（[[上砂川支線]]）の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ヒス'''。上砂川支線の[[廃線]]に伴い[[1994年]]（平成6年）[[5月16日]]に廃駅となった。 == 歴史 == [[ファイル:HigashiUzura eki.jpg|right|thumb|1976年の東鶉駅と周囲500m範囲。左が砂川方面。{{国土航空写真}}]] ディーゼルカー導入に伴い、上砂川町と三井鉱山が設置費の一部を負担することにより開設された<ref>上砂川町史 昭和63年3月発行。</ref>。 === 年表 === * [[1959年]]（昭和34年）[[12月18日]]：'''[[日本国有鉄道|国有鉄道]]函館本線（上砂川支線）'''の'''東鶉駅'''として開業。旅客のみ取扱い。 * [[1960年]]（昭和35年）7月：無人化<ref>上砂川町史 昭和63年3月発行 P1521。なお、この時点で簡易委託化されたかは不明。</ref>。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]によりJR北海道に継承。 * 時期不詳<ref group="注" name="ekigyoumu">1990年7月1日時点では既に完全無人化されている（小冊子『HANDBOOK 1990』「駅は旅の出発点」（発行：北海道旅客鉄道、[[1990年]]発行）より）。</ref>：簡易委託廃止、完全無人化。 * [[1994年]]（平成6年）[[5月16日]]：上砂川支線の廃線に伴い廃止となる。 === 駅名の由来 === 当駅の所在する地の「東」に位置することから付けられた。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の南側（上砂川方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）48ページより。</ref>。上砂川方に[[斜路|スロープ]]を有し踏切際の道路に接していた。 [[無人駅]]となっており、駅舎及び[[待合所|待合室]]は存在しなかった<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]][[6月]]発行）59ページより。</ref>が、[[日本の鉄道駅#簡易委託駅|簡易委託駅]]として[[乗車券]]の販売が行われていた時期もあった（[[1962年]]（昭和37年）8月発行の乗車券が確認されている<ref name="bunken2011">書籍『無人駅探訪』（監修：西崎さいき、[[文芸社]]、[[2011年]][[6月]]発行）20ページより。</ref>）。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は16人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は4人<ref name="bunken1993"/>。 ==
