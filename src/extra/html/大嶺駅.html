大嶺間が開通した[[1905年]][[9月13日]]に開設されたが、沿線の過疎化や[[大嶺炭田]]の炭鉱の閉山などで需要が減り、JR美祢線大嶺支線（[[南大嶺駅|南大嶺]] - 大嶺間、2.8&nbsp;km）の廃止に伴い[[1997年]][[4月1日]]に廃駅になった。 == 歴史 == {{see also|大嶺炭田}} [[大嶺炭田|大嶺炭鉱]]の最寄り駅として繁栄した。[[1953年]]には駅の貨物収入が全国で25位となる黄金期を迎えたが、[[1970年]]に炭鉱が閉鎖されると一転して貨物も旅客も激減。C58蒸気機関車が1両の客車を引いて南大嶺駅間を往復する閑散路線となった。1971年10月からは、一般貨物の取り扱いが外部委託となっていた<ref>「石炭とともに消長 キップ客は1日10人」『中國新聞』昭和46年9月10日</ref>。 * [[1905年]]（[[明治]]38年）9月13日 - [[山陽鉄道]]の駅として開業。[[旅客]]・[[貨物]]の取扱を開始{{R|sone07-20}}。 ** 当時の所在地表示は[[山口県]][[美祢郡]][[大嶺町|大嶺村]]大嶺奥分であった。 * [[1906年]]（明治39年）[[12月1日]] - 山陽鉄道の[[鉄道国有法|国有化]]により[[官設鉄道]]の駅となる{{R|sone07-20}}。 * [[1909年]]（明治42年）[[10月12日]] - 大嶺線所属駅となる{{R|sone07-20}}。 * [[1924年]]（[[大正]]13年）[[3月23日]] - 美禰線所属駅となる{{R|sone07-20}}。 * [[1939年]]（[[昭和]]14年）5月1日 - 大嶺村が町制施行して[[大嶺町]]となり、所在地表示が山口県美祢郡大嶺町大嶺奥分になる。 * [[1954年]]（昭和29年）[[3月31日]] - [[美祢市]]（第1次）成立に伴い、所在地表示が山口県美祢市大嶺町大嶺奥分になる。 * [[1956年]]（昭和31年） - 町名の改称に伴い、所在地表示が山口県美祢市大嶺町奥分になる。 * [[1963年]]（昭和38年）[[10月1日]] - 美祢線所属駅となる{{R|sone07-21}}。 * [[1984年]]（昭和59年）[[1月1日]] - 貨物取扱を廃止。 * [[1985年]]（昭和60年）[[2月1日]] - [[無人駅]]となる<ref>{{Cite news |title=通報 |newspaper=[[鉄道公報]] |publisher=[[日本国有鉄道]]総裁室文書課 |date=1985-02-01 |page=2 }}</ref>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR西日本の駅となる{{R|sone07-21}}。 * [[1997年]]（[[平成]]9年）4月1日 - 廃止{{R|sone07-21}}。 === 駅名の由来 === * 駅のある地域の村の名前（[[美祢郡]][[大嶺町|大嶺村]]）から。 == 駅構造 == [[File:Oomine-02.jpg|thumb|250px|right|構内（1993年11月6日）]] [[File:Oomine-03.jpg|thumb|250px|right|大嶺駅ホームの[[マイルストーン]]。中央に「12」「15」の数字が見える。（1993年11月6日）]] * プラットホームは線路の西側に一つあるだけだった。駅舎はプラットホームの西側にあったが、廃止前には既に無人化されていた。 * かつては多くの側線があったが、炭鉱閉山後は次々と撤去され、廃止前には線路は一本のみになった。 * プラットホームの礎石に「12M15CH」（「12[[マイル]]15[[チェーン (単位)|チェーン]]」と読む。[[メートル]]に換算すると19,613.88&nbsp;m）の文字が刻まれていた。この距離は厚狭 - 大嶺間の距離（廃止当時の営業キロは19.7&nbsp;km）とほぼ等しく、山陽鉄道の路線として開業した際に記念に刻まれたことやイギリスの技術を用いて日本の鉄道が建設されたことを示している。この礎石の示す地点は、廃止当時の停車場中心とも一致していた。 * なお、隣接する南大嶺駅にも同じく礎石が残っていたが、大嶺支線廃止に伴うホームの埋め立てによって現在は見ることができない。また、南大嶺駅の礎石の地点は、現在の停車場中心と若干離れていた。 == 利用状況 == {| class="wikitable" border="1" cellspacing="0" cellpadding="2" style="font-size:90%; text-align:center; width:30%;" |- !年度 !乗車人員 !貨物発送トン |- |1907||20,696||69,673 |- |1912||19,131||57,030 |- |1916||19,474||73,600 |- |1921||33,957||92,793 |- |1926||62,713||194,308 |- |1931||68,299||208,898 |- |1936||74,509||260,877 |- |1941||158,840||445,028 |- |1945||144,518||144,766 |- |1950||228,289||320,100 |- |1955||246,326||583,248 |- |1960||190,674||837,420 |- |1965||198,406||892,820 |- |1970||96,491||338,814 |- |1992||5,600||- |- |} *『みねぶんか』第6号、美祢市郷土文化研究会、1975年、9頁、山口県統計年鑑1994年 == 駅跡 == * 駅跡地と周辺の線路跡は[[山口県道38号美祢油谷線]]の[[バイパス道路|バイパス]]に転用されている。[[大嶺郵便局]]付近には「大嶺駅跡」という石碑が建立されている。 * 大嶺支線の廃止後は、[[船木鉄道|船鉄バス]]の路線に代替され、その後[[美祢市]]コミュニティバス「[[あんもないと号]]」（船鉄バス受託）へ移行した。駅廃止後も「あんもないと号」および[[ブルーライン交通]]のバス停留所の名称は「大嶺駅」のままであったが、[[2018年]]（平成30年）[[4月1日]]に「大嶺」に改称された。 == 周辺 == * 駅の前を[[厚狭川]]支流の[[麦川川]]が流れており、駅があった当時は橋を渡った先に[[山口県道38号美祢油谷線]]（旧道）が通っていた。 * 駅と山口県道38号美祢油谷線（旧道）の間は[[山口県道236号大嶺停車場線]]が結んでいたが、廃止のちょうど2年前の[[1995年]][[3月31日]]山口県告示第262号で廃止され、美祢市道に降格した。これは[[大嶺支線]]廃止に伴うものではなく、その時同時に廃止された駅に至る県道が他にも存在していたことから（山口県道236号大嶺停車場線を含め19路線）、県による県道路線の合理化が要因とされている。 == 隣の駅 == ; 西日本旅客鉄道（JR西日本） : 美祢線大嶺支線 :: [[南大嶺駅]] -
