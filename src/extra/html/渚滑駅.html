[[鉄道省]]、名寄東線の中湧別 - 興部間開通<ref name="teisha1">停車場変遷大事典 国鉄・JR編I 1998年JTB発行、P251。</ref>に伴い開業<ref name="teisha2">停車場変遷大事典 国鉄・JR編II 1998年JTB発行、P911。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="teisha2"/>。 ** [[10月5日]] - 名寄東線が名寄線に編入され、同線の駅となる<ref name="teisha1"/>。 * [[1923年]]（大正12年）[[11月5日]] - 名寄線が名寄本線に改称され、同線の駅となる<ref name="teisha1"/>。また、渚滑線の当駅 - 北見滝ノ上駅間開通に伴い、同線の分岐駅となる<ref name="teisha1"/>。渚滑機関庫設置<ref name="hyakunenshi">北海道鉄道百年史 下巻 1981年 日本国有鉄道北海道総局発行、巻末年表。</ref>。 * [[1932年]]（昭和7年）[[10月1日]] - 渚滑機関庫が遠軽機関庫渚滑分庫となる<ref name="hyakunenshi"/>。 * [[1949年]]（昭和24年）[[6月1日]] -[[日本国有鉄道]]に移管。 * [[1960年]]（昭和35年）[[5月1日]] - 渚滑機関支区廃止<ref name="hyakunenshi"/>。 * [[1982年]]（昭和57年）[[11月15日]] - 貨物の取り扱いを廃止<ref name="teisha2"/>。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止<ref name="teisha2"/>。 * [[1985年]]（昭和60年）[[4月1日]] - 渚滑線廃止<ref name="teisha1"/>。 * [[1986年]]（昭和61年）[[11月1日]] - 交換設備の運用を取りやめ、同時に無人駅化。 * [[1987年]]（昭和62年）4月1日 - [[国鉄分割民営化]]により、JR北海道に継承<ref name="teisha2"/>。 * [[1989年]]（平成元年）5月1日 - 名寄本線の全線廃止<ref name="teisha1"/>に伴い、廃駅となる<ref name="teisha2"/>。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]と線路を有する[[地上駅]]であった。ホームは、線路の北側（遠軽方面に向かって左手側）に存在した。 国鉄時代末期に無人化されるまでは、[[単式ホーム]]・[[島式ホーム]]を複合した計2面3線のホームと線路を有する、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=210}}</ref>。互いのホームは、駅舎側ホーム西側と島式ホーム西側を結んだ[[跨線橋]]で連絡した<ref name="bunken1983"/>。1983年（昭和58年）時点では、駅舎側単式ホーム（北側）が下りの1番線、島式ホーム駅舎側が上りの2番線、外側が渚滑線発着の上下共用3番線となっていた<ref name="bunken1983"/>。3番線の外側（山側）に旧貨物[[停車場#側線|側線]]を5線有し、そこから車庫線などの行き止まりの側線も数線有した。そのほか、1番線の遠軽方から分岐し駅舎東側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の北側に位置し、ホーム中央部分に接していた<ref name="bunken1983"/>。[[改札口]]は西側1箇所にあった。 == 駅名の由来 == 当駅が所在していた地名、「渚滑町」より。地名は、[[アイヌ語]]の「ショ・コツ」（滝壺）に由来する<ref name="bunken1983"/>。その滝壺から[[砂金]]が産出し、[[鴻之舞鉱山|鴻之舞金山]]の起源となった<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は198人<ref name="bunken1983"/>。 == 駅弁 == * [[1935年]]（昭和10年）から[[1978年]]（昭和53年）まで、駅前の食堂の村上待合所が[[ホタテガイ|帆立]]めしを[[駅弁]]として販売していた。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref> * [[国道273号]]（渚滑国道）<ref name="bunken1980M"/> * 紋別市役所渚滑出張所 *[[紋別警察署]]渚滑駐在所 * 渚滑郵便局 *[[北見信用金庫]]渚滑出張所 * 紋別市立渚滑中学校 * 紋別市立渚滑小学校 * [[渚滑川]]<ref name="bunken1980M"/> * オムサロ原生花園 - 駅から北西に約2.8km<ref name="bunken1983"/>。 * バス乗り場 **流氷もんべつ号「渚滑」停留所：旭川・札幌方面 **[[北紋バス]]「渚滑4丁目」停留所：興部・雄武方面、滝上方面、紋別市内方面 == 駅跡地の再開発 == [[File:20100811渚滑駅跡パークゴルフ場.jpg|right|250px|thumb|「紋別市渚滑ふれあいパークゴルフ場」などとして利用されている渚滑駅跡地。パークゴルフ場休憩所には旧駅にちなみ「渚滑駅」の看板が掲げられている(2010年)]] 名寄本線が廃止となった後、当駅の旧駅本屋はバスの待合所として活用された。この他、旧駅構内は渚滑パークゴルフ協会が管理する[[パークゴルフ]]コースとなり、[[1991年]][[12月]]からは市民有志によって冬季間、旧機関庫を「旧渚滑駅機関庫跡室内パークゴルフ場」として利用するようになった。機関庫内の室内パークゴルフ場は現在も地元住民の同好会が管理し、毎年12月から翌年4月まで開放されている。 当駅の跡地は渚滑地区の中心部にあることから跡地利用が課題となっていたが、紋別市は[[1997年]]度から再開発に着手した<ref>「97年度紋別市補正予算案　渚滑駅跡地再開発に着手　一般会計は196億4千万円　赤井市政初の編成」北海道新聞朝刊北見版、1997年7月17日付</ref>。駅本屋は同年解体され、1999年までに市食品加工センター（うまいっしょ工房）、渚滑高齢者ふれあいセンター、バス待合所が入る紋別市渚滑総合交流促進施設とオホーツク農業共済組合紋別家畜診療所が建設されたほか、パークゴルフコースが「渚滑ふれあいパークゴルフ場」として再整備された<ref>「紋別・渚滑駅跡地　パークゴルフ場完成　地区整備計画が一段落」北海道新聞朝刊遠紋版、1999年7月14日付</ref>。 紋別市は[[2004年]]度、渚滑ふれあいパークゴルフ場などの周辺再整備を行い、この際、市が紋別市運動公園（紋別市南が丘町7丁目）で[[静態保存]]していた[[国鉄9600形蒸気機関車|9600形蒸気機関車]]（69644）が移設された。同機は国鉄から[[1975年]]に無償貸与されたもので、老朽化が進んだため、所管する紋別市教育委員会は機関車を解体して車体の一部のみを保存することにしていたが、全国のSLファンから完全保存を求める声があがり、市教委が方針を転換して補修の上、渚滑駅跡地に移して保存した<ref>「紋別市運動公園のＳＬ→旧渚滑駅跡地に移転　車体の部分補修も　市立博物館『長期の原形保存へ努力』」北海道新聞朝刊遠紋版、2004年3月30日付</ref>。 渚滑線跡は、現在も一部が築堤として残存している。1985年の同線廃止後、レールなどの設備はいったんすべて撤去されたが、駅から数百メートル西方の渚滑小学校地内の校門と運動場の間を横切っていたことから、地元の渚滑小・中学校PTAが「全国で唯一鉄道が走っていた学校」を後世に伝えようと国鉄からレールや踏切遮断機、信号機、駅名標などを譲り受け、校地内の渚滑線跡地に[[1989年]]、線路（約30メートル）を復元して保存している<ref>「線路の輝き再び　渚滑小の校庭に復元　14年たち児童ら補修」北海道新聞朝刊遠紋版、2003年6月24日付</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[富丘駅]] -
