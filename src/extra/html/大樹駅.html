大樹駅」の[[乗車券]]が[[乗車券#縁起物の乗車券|縁起切符]]として発売され、ブームを呼んだことがあった。 == 駅構造 == [[File:JNR Taiki sta 002.jpg|right|thumb|駅ホーム及び線路跡地]] 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）を複合した2面2線のホームと線路を有する[[地上駅]]で、[[列車交換]]が可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム南側と島式ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）140ページより。</ref>。駅舎側ホーム（東側）が下りの1番線、島式ホーム（西側）が上りの2番線となっていた<ref name="bunken1983"/>。島式ホームの外側1線を[[停車場#側線|側線]]として有し、そのほか1番線の広尾方から分岐し駅舎南側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線と、2番線の帯広方から分岐した短い行き止まりの側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の東側に位置しホーム中央部分に接していた<ref name="bunken1983"/>。1976年（昭和51年）に、大樹町が国鉄利用債を利用して改築した鉄筋造り<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）190-191ページより。</ref>の駅舎であった。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「タイキ・ウシ」（[[蚤]]が多いところ）に由来する<ref name="bunken1983"/>。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は204人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道236号]]（広尾国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）12ページより。</ref> * [[大樹町]]役場 * 柏林公園 - 大樹町役場近く。敷地内に[[国鉄9600形蒸気機関車|9600形蒸気機関車]]'''59611号機'''が[[静態保存]]・展示されている<ref name="bunken2011S">書籍『蒸気機関車完全名鑑 ビジュアル改訂版』（[[廣済堂あかつき|廣済堂ベストムック]]、[[2011年]][[1月]]発行）46ページより。</ref>。当線でも活躍した車両であった。 * 大樹郵便局 * 大樹駅前簡易郵便局　‐　2014年1月14日に移転するが、局名はそのままである。 * 広尾警察署大樹駐在所 * 帯広土木現業所大樹出張所 * [[北海道大樹高等学校]] * 大樹中学校 * 大樹小学校 * 大樹町国民健康保険病院 * 大樹町農業協同組合（JA大樹町） * 大樹漁業協同組合 * [[雪印乳業]]株式会社（現・[[雪印メグミルク]]株式会社）大樹工場 * 高野山寺 * 歴舟川 * [[道の駅]][[道の駅コスモール大樹|コスモール大樹]] * [[十勝バス]]大樹案内所、「大樹コスモール前」停留所 == 歴史 == * [[1930年]]（昭和5年）[[10月10日]] - '''[[日本国有鉄道|国有鉄道]]（[[鉄道省]]）広尾線'''中札内駅 - 当駅間の延伸開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。池田機関庫帯広分庫大樹駐泊所設置。 * [[1932年]]（昭和7年）[[11月5日]] - 当駅 - 広尾駅間の延伸開通（広尾線全通）に伴い、中間駅となる。 * [[1976年]]（昭和51年）[[12月3日]] - 駅舎改築。 * [[1982年]]（昭和57年）[[9月10日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[2月2日]] - 広尾線の全線廃止に伴い、廃駅となる。 == 駅跡 == 廃駅後は大樹町により、「大樹町交通公園」として整備された。1999年（平成11年）時点では、駅舎は[[十勝バス]]の待合所に再利用された<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）40-41ページより。</ref>。またホームと[[軌条|レール]]が保存され、ホーム側には[[ブルートレイン]]色に塗られた[[国鉄60系客車#オハ62形|オハ62形客車]]'''オハ62 76'''と[[国鉄60系客車#郵便・荷物車|スユニ60形客車]]'''スユニ60 301'''の2両の旧型[[客車]]と2両の[[貨車]]が[[静態保存]]・展示されていた<ref name="bunken1999H"/>。客車は、[[ライダーハウス]]としても使われた。その後、老朽化に伴い[[2000年]]夏頃に撤去されてしまった。 駅舎は2002年（平成14年）時点でも'''十勝バス大樹案内所'''であり、係員を配置して乗車券窓口と待合所が利用されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）190-191ページより。</ref>。この時点では旧駅舎内に窓口カウンター、改札柵が残存し、旧ホーム側は[[連動装置#論理機構の仕組みと方法|転轍てこ]]の残存を始めとした廃線当時のままの状態、ホームは[[駅名標]]の枠のみが残存していた<ref name="bunken2011H"/>。またレールは撤去され小公園になっていた<ref name="bunken2011H"/>。 その後十勝バス大樹案内所が[[道の駅コスモール大樹]]へ移転した。2005年（平成17年）10月からは北海道衛星株式会社が3代目の利用者となり、本社が開設された<ref>“[http://web.archive.org/web/20200309104929/http://photodb.hokkaido-np.co.jp/detail/0030031973 「北海道衛星」が大樹に本社開設]” , フォト北海道（写真データベース）, ([[北海道新聞社]]), (2005年10月2日), [http://photodb.hokkaido-np.co.jp/detail/0030031973 オリジナル]の2020年3月9日時点によるアーカイブ。 2020年3月9日閲覧。</ref>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）88-89ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。 また、1999年（平成11年）時点では当駅跡の北約1.5kmの地点にある歴舟川に橋台が残存し、大樹駅方面への線路跡も低い築堤や遊歩道として残存・再利用されていた<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）40-41ページより。</ref>。2010年（平成22年）時点でも歴舟川の橋台は残存しているが、藪に埋もれている<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）88-89ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 広尾線 :: [[十勝東和駅]] -
