|統計年度 = |乗換 = |備考 = * 戦後に雄琴駅から改称 }} '''雄琴温泉駅'''（おごとおんせんえき）は、かつて[[滋賀県]][[大津市]]雄琴{{Sfn|江若鉄道の思い出|p=38}}にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 関西の奥座敷と称される温泉街、[[雄琴温泉]]の最寄り駅だった。 == 歴史 == 当駅は1923年（大正12年）4月、江若鉄道の[[叡山駅 (江若鉄道)|叡山駅]]から雄琴駅までの区間が開通したのに合わせて'''雄琴駅'''（おごとえき）として開業した{{Sfn|江若鉄道の思い出|p=38}}{{Sfn|今尾|2008|pp=31&ndash;32}}。同年12月には雄琴から先、[[堅田駅 (江若鉄道)|堅田駅]]までの区間も開通する{{Sfn|田中|宇田|西藤|1998|p=394}}。当時の雄琴（[[滋賀郡]][[雄琴村]]）は農業と漁業を営む村であったが、翌24年ごろから[[雄琴温泉]]の開発が始まると、江若鉄道の開通も手伝って保養地へと変貌{{Sfn|新修大津市史第7巻|pp=341&ndash;342}}。戦前に発行された江若鉄道の観光案内図では雄琴が温泉地として紹介されていて、当地が早くから観光地として整備されていたことが窺える{{Sfn|江若鉄道の思い出|p=40}}。 駅名が雄琴温泉駅に改称されたのは戦後{{Sfn|江若鉄道の思い出|p=38}}。このころの雄琴は大津市への合併（1951年）を機に行われた新しい源泉の掘削事業が成功し、温泉旅館が新規開業するなど温泉街として発展を遂げていた{{Sfn|新修大津市史第7巻|pp=341&ndash;342}}{{Sfn|江若鉄道の思い出|p=41}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。江若鉄道の廃線後に開業した[[湖西線]]にも「雄琴駅」が設けられた（2008年に[[おごと温泉駅]]に改称{{Sfn|今尾|2008|pp=31&ndash;32}}）が、その位置は温泉街の中にあった当駅よりも山側に離れている{{Sfn|レイル|p=80}}。 駅は[[雄琴温泉]]の温泉街の中にあった{{Sfn|レイル|p=80}}。跡地には[[大津市役所]]雄琴市民センター{{Sfn|江若鉄道の思い出|p=38}}やコンビニエンスストア{{Sfn|寺田|2010|pp=24&ndash;25}}が建てられている。当駅付近の線路跡については並走する[[国道161号]]の拡幅に利用されたが、駅の北方、衣川一丁目付近で国道から西へ分かれて山側に伸びる{{Sfn|江若鉄道の思い出|p=41}}{{Sfn|新・鉄道廃線跡を歩く|p=32}}。その先、天神川の手前には未舗装のまま[[築堤]]が残されていて、江若鉄道が営業していた当時の状態を強く留めている{{Sfn|江若鉄道の思い出|p=41}}{{Sfn|寺田|2010|pp=24&ndash;25}}{{Sfn|吉田|1998|p=187}}。 なお、江若鉄道廃線後に開通した[[湖西線]]は雄琴の温泉街を通らず、トンネルを利用し山側を抜けている{{Sfn|寺田|2010|pp=24&ndash;25}}{{Sfn|吉田|1998|p=186}}。これは地形や地質の問題に加えて、温泉街の旅館が騒音や景観問題の点から路線の新設に難色を示したことによる{{Sfn|吉田|1998|p=186}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線
