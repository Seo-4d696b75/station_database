'''新大滝駅'''（しんおおたきえき）は、[[北海道]]（[[胆振支庁]]）[[有珠郡]][[大滝村 (北海道)|大滝村]]字本町（現・[[伊達市 (北海道)|伊達市]]大滝区本町）にあった[[日本国有鉄道]]（国鉄）[[胆振線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''シオ'''。胆振線の[[廃線]]に伴い[[1986年]]（昭和61年）[[11月1日]]に廃駅となった。 [[1980年]]（昭和55年）9月まで運行されていた[[急行列車|急行]]「[[胆振線#急行「いぶり」|いぶり]]」の停車駅であった。 == 歴史 == * [[1940年]]（昭和15年）[[12月15日]] - '''[[胆振縦貫鉄道]]'''[[伊達紋別駅]] - 当駅間開通に伴い'''徳舜瞥駅'''（とくしゅんべつえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1941年]]（昭和16年） ** - 当駅貯鉱槽 - 徳舜瞥（鉄鉱石）鉱山間に第一及び第二索道設置<ref>日鉄鉱業40年史 1979年11月発行 P81。</ref>。 **[[10月12日]] - 当駅 - 西喜茂別駅（後の[[喜茂別駅]]）間延伸開通に伴い中間駅となる。 * [[1944年]]（昭和19年）[[7月1日]] - 胆振縦貫鉄道が[[戦時買収私鉄|戦時買収]]により国有化。線路名を'''胆振線'''に改称、それに伴い同線の駅となる。 * [[1952年]]（昭和27年）[[11月15日]] - '''新大滝駅'''に改称。 * [[1957年]]（昭和32年）6月 - 当駅 - 徳舜瞥硫黄鉱山間に索道設置及び当駅に貯鉱槽新設<ref>日鉄鉱業40年史 P272。</ref>。 * [[1964年]]（昭和39年）3月31日 - 徳舜瞥（鉄鉱石）鉱山閉山<ref>日鉄鉱業40年史 P289。</ref> * [[1971年]]（昭和46年）3月31日 - 徳舜瞥硫黄鉱山閉山<ref>日鉄鉱業40年史 P293。</ref>。 * [[1980年]]（昭和55年）[[5月15日]] - 貨物営業廃止。[[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 胆振線の廃線に伴い廃止となる。 駅開業当時、役場がある徳舜瞥村（旧名）の中心地は[[長流川]]東岸の高台に位置する上野地区で、駅から歩いて1時間近くを要した。その後は本町地区に村の中心が移り、[[1950年]]（昭和25年）の役場庁舎移転と[[大滝村]]への改称につながった。また、上野地区の[[日鉄鉱業]][[徳舜瞥鉱山]]から産出される鉱石を[[索道]]で当駅まで運び、[[貨車]]に積み替え発送した。一日当たりの平均取扱量は100～500トン台で推移していた。[[1971年]]（昭和46年）の閉山後は貨物取扱量が激減した。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名に「新」を冠する<ref group="注" name="ekimei">「新」を冠したのは、[[奥羽本線]][[大滝駅]]や音が同じ木原線[[大多喜駅]]（現・[[いすみ鉄道いすみ線]]）との重複を避けるためと思われる。</ref>。地名（村名）は、村の名所である[[三階滝 (北海道)|三階滝]]に由来した。 == 駅構造 == 廃止時点で、[[単式ホーム]]2面2線を有する[[地上駅]]であった。ホームが千鳥式に配置された[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）92ページより。</ref>。互いのホームは駅舎側ホーム南側と対向ホーム北側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側ホーム（東側）が上りの1番線、対向ホーム（西側）が下りの2番線となっていた<ref name="bunken1983"/>。そのほか2番線の倶知安方から2番線ホーム外側（乗降不可）に分岐し末端部で2線になる行き止まりの[[停車場#側線|側線]]、その側線途中から分岐する行き止まりの側線を1線有した<ref name="bunken1983"/>。[[国鉄9600形蒸気機関車|9600形蒸気機関車]]が入線していた当時は、[[転車台]]を備えていた。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっており、駅舎は構内の南東側に位置し、ホームとは通路及び構内踏切で連絡した<ref name="bunken1983"/>。駅舎の左手側に別棟でトイレを有した<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]]12月発行）280ページより。</ref>。 かつては徳舜瞥鉱山からの[[鉄鉱石]]（[[褐鉄鉱]]）や[[硫黄]]鉱の積み出し駅であった<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（[[JTBパブリッシング]]、[[2001年]]8月発行）70-71ページより。</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は59人<ref name="bunken1983"/>。 == 駅跡 == 2001年（平成13年）時点では旧構内の一部が「平成ふるさとの道公園」として整備され、ホームのあった位置には旧大滝村を象徴するオブジェが設置されていた<ref name="bunken2001H"/>。また伊達紋別方には徳舜瞥鉱山から[[硫化鉄]]を搬送するために敷設された<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）199ページより。</ref>引込線の高架橋が残存していた<ref name="bunken2001H"/>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）153-155ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。優徳方の廃線跡は遊歩道となっていた。近くの駅構内跡には[[雇用促進住宅]]が建っている。 1993年（平成5年）に当駅跡から伊達紋別方へ約0.7kmの地点から優徳駅の先までの線路跡が約5.5kmに渡り[[自転車道|サイクリングロード]]の「平成ふるさとの道」として整備され<ref>{{Cite news |title=生まれ変わった旧胆振線 |newspaper=[[交通新聞]] |publisher=交通新聞社 |date=1996-10-19 |page=2 }}</ref>、2001年（平成13年）時点<ref name="bunken2001H"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H"/>。途中の小川に架かる一部の橋は胆振線の施設が再利用されている<ref name="bunken2011H"/>。 == 駅周辺 ==
