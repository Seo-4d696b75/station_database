'''東京極駅'''（ひがしきょうごくえき）は、[[北海道]]（[[後志支庁]]）[[虻田郡]][[京極町]]字三崎にかつて存在した、[[日本国有鉄道]]（国鉄）[[胆振線]]の[[鉄道駅|駅]]（[[廃駅]]）である。胆振線の[[廃線]]に伴い、1986年（昭和61年）11月1日に廃駅となった。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]と線路を有する[[地上駅]]であった。ホームは、線路の西側（倶知安方面に向かって左手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）92ページより。</ref>。 [[無人駅]]（簡易委託駅）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の西側に位置し、ホームから少し離れていた<ref name="bunken1983"/>。ホーム中央部分の出入口は階段となっていた<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当駅が、所在する地（京極町）の「東」に位置したことから。 == * [[1944年]]（昭和19年）[[7月1日]] - 胆振縦貫鉄道が[[戦時買収私鉄|戦時買収]]により国有化された際、廃止となる。 * [[1962年]]（昭和37年）[[12月17日]] - [[請願駅]]<ref name="bunken1983"/>として再開業。旅客のみ取り扱い。 * [[1963年]]（昭和38年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。
