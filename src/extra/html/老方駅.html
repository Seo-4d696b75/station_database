横荘西線（後に国鉄矢島線を経て現・[[由利高原鉄道]][[由利高原鉄道鳥海山ろく線|鳥海山ろく線]]）前郷駅間約23kmは未開業に終わり{{sfn |無明舎出版|1999| pp=58-59,61}}、結果的に同鉄道最後の開業区間となった。当駅 - 前郷駅間は当駅 - 本荘駅間のバス路線を開通させることにて「横手本荘連絡」を行った{{sfn |若林宣|2004| pp=11-12,18-19,25}}。羽後大森駅 - 二井山駅間5.4kmの建設費が19万円強であったのに対し、二井山駅 - 当駅間12.1kmは140万円であったという{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 二井山駅 - 当駅間は奥羽山脈の分水嶺の1つを通過し、人里稀な区間でもあり、特に冬季は[[雪]]害による運転休止が多く、戦後の燃料費高騰の影響もあり早くも不採算区間とされ、会社経営の建て直しを図るための営業休止を経て{{sfn |若林宣|2004| pp=11-12,18-19,25}}、最終的に復活せず廃止となった{{sfn |若林宣|2004| pp=11-12,18-19,25}}。休止時点のダイヤでは1日2往復の混合列車が運行されていた{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 また、当区間の廃止日に、工事期限延長の手続きが繰り返し行われていた当駅 - 前郷駅間の敷設免許の申請廃止手続きが取られた{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 * [[1930年]]（昭和5年）[[10月5日]] - '''横荘鉄道'''二井山駅 - 当駅間延伸開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref>{{sfn |今尾恵介|2010| p=222}}<ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2010年]]8月発行）25-28,30-31ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="bunken2008SH-1-R"/>。 * [[1944年]]（昭和19年）6月1日 - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/>{{sfn |今尾恵介|2010| p=222}}<ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1946年]]（昭和21年）夏 - [[日本国有鉄道|国鉄]]から借入の[[国鉄C12形蒸気機関車|C12形蒸気機関車]]C12 229号機牽引の当駅発試運転列車が浮蓋駅発車後に過速暴走、八沢木駅手前で停車する{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 * [[1947年]]（昭和22年）7月23日 - 豪雨による路盤および橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/>{{sfn |若林宣|2004| pp=16-17}}。 * [[1948年]]（昭和23年）11月8日 - 運休区間（舘合駅 - 当駅間）が復旧、当駅も営業再開となる<ref name="bunkenKW03"/>{{sfn |若林宣|2004| pp=16-17}}。 * [[1949年]]（昭和24年）12月22日 - 二井山駅 - 当駅間営業休止に伴い[[休止駅]]となる<ref name="bunkenKW03"/>。 * [[1952年]]（昭和27年）2月15日 - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通横荘線'''の駅となる（営業休止中）<ref name="bunken2008M-02"/>{{sfn |今尾恵介|2010| p=222}}<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1953年]]（昭和28年）[[8月5日]] - 二井山駅 - 当駅間部分廃線に伴い廃止となる<ref name="bunken2008M-02"/>{{sfn |今尾恵介|2010| p=222}}<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]であった{{sfn |若林宣|2004| pp=11-12,18-19,25}}。そのほか[[停車場#側線|側線]]として、北側（駅舎の反対側）本線横手方から北に分岐し末端部に車庫を有する行き止りの車庫線を1線と、南側本線終端部方から南に分岐し駅舎西側に至る行き止りの側線を1線有していた{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の南側に位置し、ホーム東側とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した{{sfn |若林宣|2004| pp=11-12,18-19,25}}。 == 駅周辺 == 旧・[[下郷村 (秋田県)|下郷村]]の中心地に位置した。後に合併により旧・由利郡東由利村（さらに[[町村制|町制]]施行して[[東由利町]]）となり、その役場が置かれた。 * [[秋田県道48号横手東由利線]] - 同線の二井山地区 - 老方地区の大部分は横荘線の線路跡がそのまま転用されていた<ref name="bunken2008SH-1"/>（当駅跡附近は別ルートであった{{sfn |無明舎出版|1999| pp=58-59,61}}）。 * [[国道107号]]（本荘街道） * 由利本荘市役所東由利総合支所 - 旧・東由利町役場。当初、当駅跡に建築されていたが後に移転した{{sfn |今尾恵介|2010| pp=202-203}}。 * 老方郵便局 * 石沢川 == 駅跡 == 1996年（平成8年）時点では、駅跡に東由利町役場が建築されていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]]9月発行）35ページより。</ref>。1999年（平成11年）時点でも同様であった{{sfn |無明舎出版|1999| pp=58-59,61}}。その後庁舎は移転し、2007年（平成19年）5月時点では取り壊されて<ref name="bunkenKW03"/>更地になっており、「役場跡」の石碑が建立されていた<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。駅跡を示すものは無かった<ref name="bunken2008SH-1"/>。2010年（平成22年）10月時点では石碑のほか、老人介護施設が建築されていた{{sfn |今尾恵介|2010| p=203}}。 また、二井山駅跡附近から老方の集落入口附近までの線路跡は県道として再利用され、1996年（平成8年）時点では[[盛土|築堤]]や[[切通し]]がほぼ当時の姿を保っていた<ref name="bunken1996H"/>。1999年（平成11年）時点でも同様であった{{sfn |無明舎出版|1999| pp=61}}。この県道は2007年（平成19年）5月時点では[[秋田県道48号横手東由利線]]であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。道床をそのまま舗装したような道路であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった{{sfn |今尾恵介|2010| p=222}}。 この県道は老方の集落入口附近から線路跡から離れ、1999年（平成11年）時点では線路跡再利用部分との境界点となっている橋の隣に横荘線の橋台が片方のみ残存していた{{sfn |無明舎出版|1999| pp=58-59,61}}。老方駅跡附近の線路跡は[[区画整理]]により残っていなかった{{sfn |無明舎出版|1999| pp=58-59,61}}。2007年（平成19年）5月時点も同様であったが<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、県道と別れた附近にまだ一部の線路跡が未舗装の[[農道]]として再利用されていた<ref name="bunkenKW03"/>。2010年（平成22年）10月時点では石沢川の護岸の出っ張りが横荘線の橋台を取り込んだものであると確認できた{{sfn |今尾恵介|2010| p=203}}。 == 隣の駅 == ; 羽後交通 : 横荘線 :: [[浮蓋駅]] -
