[[鉄道省|鉄道院]]湧別軽便線の社名淵（後の開盛） - 下湧別（後の湧別）間延伸開通に伴い、開業<ref name=":022">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=198}}</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1922年]]（大正11年）[[9月2日]] - 湧別軽便線が湧別線に改称され、同線の駅となる。 * [[1932年]]（昭和7年）[[10月1日]] - 遠軽駅 - 下湧別駅間が名寄本線に編入され、同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1959年]]（昭和34年）11月25日 - 駅裏の池内工業専用線使用開始<ref name="choshi">上湧別町史　昭和54年11月発行。</ref>。 * [[1978年]]（昭和53年）[[12月1日]] - 貨物の取り扱いを廃止。 * [[1980年]]（昭和55年）[[10月1日]] - 急行「天都」の廃止に伴い、停車は普通列車のみとなる。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 無人駅化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 === 駅名の由来 === 所在地名より。[[湧別川]]の上流に位置しているために「上」を冠する<ref name=":022" /><ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、2004年2月発行）188ページより。</ref>。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の西側（遠軽方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、1983年7月発行）212ページより。</ref>。かつては千鳥状に設置された2面2線の単式ホームを有する、[[列車交換]]可能な[[交換駅]]であった。1983年（昭和58年）時点では、使われなくなった対向ホーム側の1線は、交換設備運用廃止後も遠軽方の[[分岐器|転轍機]]は維持され[[停車場#側線|側線]]として残っていた<ref name="bunken1983"/>。ホーム前後の線路は、転轍機の名残で湾曲していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の西側に位置し、ホーム北側に接していた<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は221人<ref name="bunken1983"/>。 == 駅周辺 == 旧上湧別町の中心地にあった<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、1980年3月発行）18ページより。</ref>。 * [[北海道道404号上湧別停車場線]]）<ref name="bunken1980M"/> * [[国道242号]]（置戸国道）<ref name="bunken1980M"/> * 湧別町役場（旧・上湧別町役場） *[[遠軽警察署]]上湧別駐在所 *上湧別郵便局 * [[ローソン]]上湧別店 * [[セイコーマート]]たかだ上湧別店 * [[湧別川]]<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「上湧別」停留所 * [[湧別町営バス]]（旧上湧別町営バス）「上湧別屯田市街地」停留所 == 駅跡 == 2001年（平成13年）時点で駅跡地は整地され、駅があった痕跡は残っておらず、駅跡の一部には[[公営住宅|町営住宅]]が建築されていた<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（[[JTBパブリッシング]]、2001年8月発行）36ページより。</ref>。2011年（平成23年）時点でも同様で、更地になっていたが、附近の道路には上湧別駅前通りの名が残り、駅前にあった倉庫群も残存して駅前の雰囲気は残っていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、2011年9月発行）129ページより。</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[北湧駅]] -
