'''丸子鉄道'''大屋駅（後の電鉄大屋駅） - 丸子町駅間開通に伴い開業<ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）42ページより。</ref><ref name="bunkenKW05-R">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）27-28ページより。</ref>。旅客のみ取扱い<ref name="bunken2008SH-2-R">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月21日]] - [[陸上交通事業調整法|交通統合]]に伴い'''上田丸子電鉄丸子線'''の駅になる<ref name="bunkenKW05-R"/><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）210ページより。</ref>。 * [[1969年]]（昭和44年）[[4月20日]] - 丸子線の廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW05-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML073">書籍『RM LIBRARY 73 上田丸子電鉄（上）』（著：宮田道一、諸河久、[[ネコ・パブリッシング]]、[[2005年]][[9月]]発行）11,15,29ページより。</ref>。ホームは線路の東側（丸子町方面に向かって左手側）に存在した<ref name="bunkenRML073"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML073"/>。 [[無人駅]]となっていた<ref name="bunken2008SH-2">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）64-67ページより。</ref>。ホームは[[屋根#形状による分類|片流れ]]屋根の[[待合所]]を有していた<ref name="bunkenRML073"/>。 === 大屋鉄橋（千曲川橋梁） === 電鉄大屋駅 - 当駅間には千曲川が流れており、「大屋鉄橋」<ref name="bunken1996H"/><ref name="bunkenRML073-2">書籍『RM LIBRARY 73 上田丸子電鉄（上）』（著：宮田道一、諸河久、[[ネコ・パブリッシング]]、[[2005年]][[9月]]発行）18,19,31ページより。</ref>或いは「千曲川橋梁」<ref name="bunkenKW05"/>と呼ばれる鉄橋が架けられていた。この鉄橋は丸子方の2連が曲弦ワーレントラス（丸型アーチ）、大屋方の4連が並行弦ワーレントラス（台形アーチ）<ref name="bunkenKW05"/><ref name="bunken2008SH-2"/><ref name="bunken1996H"/>という構造となっていた。 建設当時の同鉄橋は、並行弦ワーレントラス4連と、丸子方は平凡なプレートガーダー2連という構造になっていた<ref name="bunkenRML073-2"/>。しかし1928年（昭和3年）[[8月]]の大洪水によって、[[ガーダー橋]]部分が流失し<ref name="bunkenRML073-2"/>、代替として曲弦ワーレントラスが遠く[[九州]]から運ばれて来たという<ref name="bunkenRML073-2"/>。 == 駅周辺 == 塩川村民にとって丸子町市街地及び上田市街地へ行くのに重宝がられたが、大字石井は依田川を越せば隣接する[[依田村]]（現・上田市）の北部地区に接する地域であるため、同村民にも重宝がられた。その依田村は1955年（昭和30年）4月1日に[[長瀬村 (長野県)|長瀬村]]とともに丸子町に編入されたが、当駅のある[[塩川村]]が丸子町に編入されたのは1956年（昭和31年）9月30日であった。 当駅の丸子町方には、大屋と丸子町を結ぶ幹線道路（丸子線廃止時点では長野県道上田茅野線、現・国道152号）を横切るために、大きな[[踏切]]があった<ref name="bunkenRML073"/>。1964年（昭和39年）にそれまでの腕木式から改良され、廃止まで東京などの都会でよく見かける昇降式の踏切が設置されていた。この型の踏切は、上田丸子電鉄の踏切としては唯一であった。 * [[国道152号]] * [[長野県道186号上田塩川線]] * [[大石橋]] - 当線廃止後、大屋鉄橋（千曲川橋梁）を再利用して作られた道路橋<ref name="bunkenKW05"/>。2001年（平成13年）に架け替えられた。 * 依田川 == 駅跡 == 2010年（平成22年）10月時点では、石井交差点の角付近に位置が特定できた<ref name="bunken2010H-3">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）72ページより。</ref>。周囲は駅前の雰囲気が残っていた<ref name="bunkenKW05"/>。 また、1996年（平成8年）時点では、丸子町駅附近から電鉄大屋駅附近までの線路跡は丸子[[市町村道|町道]]（後に上田市道）となっていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]][[9月]]発行）88-89ページより。</ref>。舗装道路となって整備されているため、2007年（平成19年）8月時点では遺構は無かった<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点でも道路は健在であった<ref name="bunkenKW05"/><ref name="bunken2010H-3-1">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）71ページより。</ref>。 大屋鉄橋（千曲川橋梁）は、1996年（平成8年）時点では、丸子線時代そのままの姿で残っており、「大石橋」と名を変え、道路橋として再利用されていた<ref name="bunken1996H"/>。自動車専用道であった<ref name="bunkenKW05"/><ref name="bunken2008SH-2"/><ref name="bunken2010H-3"/>が、幅員も[[単線]]鉄道時代のままであったため、橋の出入口に[[信号機]]が設置され、[[片側交互通行]]となっていた<ref name="bunken1996H"/>。電車が走っていてもおかしくない風情を残していた<ref name="bunken1996H"/>。 その後2001年（平成13年）夏に台風13号による洪水で流出<ref name="bunkenKW05"/><ref name="bunken2008SH-2"/>などの被害を受けて撤去され<ref name="bunkenKW05"/><ref name="bunken2008SH-2"/><ref name="bunken2010H-3"/>、新しい橋に架け替えられた<ref name="bunkenKW05"/><ref name="bunken2008SH-2"/><ref name="bunken2010H-3"/>。 曲弦ワーレントラス橋のうちの1つは、丸子地区の総合体育施設内に移設されて<ref name="bunken2010H-3"/>「りんどう橋」と名付けられ<ref name="bunkenKW05"/>、施設内を流れる[[内村川]]に架橋されている<ref name="bunken2010H-3"/>。 == 隣の駅 == ; 上田丸子電鉄 : 丸子線 :: [[大屋駅|電鉄大屋駅]] -
