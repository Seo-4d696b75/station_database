[[筑豊本線]]の'''新平恒駅'''（[[鉄道駅#貨物駅|貨物駅]]）として開業。同線上に平恒駅（初代）が存在したため、この駅名とされた。 * [[1929年]]（昭和4年）[[12月7日]] - 筑豊本線から[[飯塚駅]] - [[上山田駅]]間が分離し、上山田線となる。それに伴って、当駅も同線の駅となる。 * [[1939年]]（昭和14年）[[5月29日]] - 平恒駅 (初代) の廃止に伴い、当駅を'''平恒駅'''に改称。 * [[1948年]]（昭和23年）[[5月8日]] - 駅種を[[一般駅]]に変更<ref>[{{NDLDC|2962922/1}} 「運輸省告示第139号」『官報』1948年5月6日]（国立国会図書館デジタルコレクション）</ref>。 * [[1968年]]（昭和43年） ** [[2月1日]] - 駅種を[[旅客駅]]に変更（貨物扱い廃止）。 ** 4月 - [[日本の鉄道駅#直営駅|直営駅]]から[[日本の鉄道駅#業務委託駅|業務委託駅]]となる。 * [[1974年]]（昭和49年）[[3月5日]] - [[無人駅]]化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、日本国有鉄道からJR九州に運営を継承。 * [[1988年]]（昭和63年）[[9月1日]] - 上山田線の全線廃止に伴い、廃駅となる。 == 廃止時の駅構造 == * 1面1線の[[単式ホーム]]を有する、列車交換不能駅、かつ[[地上駅]]であった。 * 無人駅であった。 * 木造の[[駅舎]]があった。 ** 無人化の後には、[[ライオンズクラブ]]の集会所となった。 == 現在 == ホーム跡と待合所が残っており、待合所のそばに当時の駅名標に似せた駅名標が設置されている。 == 駅周辺 == * 駅近くに[[住石マテリアルズ|住友石炭鉱業]]忠隈炭鉱があった。駅の北東には、「[[筑豊富士]]」と呼ばれる同炭鉱の[[ボタ山]]が現存する。 * 穂波町立穂波東中学校 * 穂波町立平恒小学校 * 平恒郵便局 * 旧[[麻生炭鉱]]飯塚鉱業所 * 旧麻生炭鉱中野倶楽部 == 隣の駅== ; 九州旅客鉄道（JR九州） : 上山田線 :: [[飯塚駅]] -
