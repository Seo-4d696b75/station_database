[[鉄道省]]名寄線の下川 - 上興部間延伸開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1978年]]（[[昭和]]53年）[[12月1日]] - 貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 交換設備を撤去し、同時に無人駅化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（[[平成]]元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南側（遠軽方面に向かって右手側）に存在した。 国鉄時代末期に無人化されるまでは、[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム中央部分と対向側ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）208ページより。</ref>。駅舎側（南側）ホームが上りの1番線、対向側ホームが下りの2番線となっていた<ref name="bunken1983"/>。そのほか1983年（昭和58年）時点では、上り線名寄方から駅舎側に分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の南側に位置し、ホーム中央部分に接していた<ref name="bunken1983"/>。ホームの長さは50mであった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地区名より<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）185ページより。</ref>。地区名は、[[1903年]]（[[明治]]36年）に開削された仮定県道（現在の[[国道239号]]）が[[名寄川]]を渡る最上流の橋の名前に由来する。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は90人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道239号]]（下川国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * 一ノ橋郵便局 * 郷土資料展示保存施設「札天山収蔵館」 - 旧一の橋小学校。 * 下川町立知的障害者更生施設「山びこ学園」 * [[名寄川]]<ref name="bunken1980M"/> * 然別川<ref name="bunken1980M"/> * 札天山 - 駅の南。標高596m<ref name="bunken1980M"/>。 * [[名士バス]]「一の橋」停留所 - [[駅名標]]が移設されている。 == 駅跡 == 2000年（平成12年）時点では、集落のコミュニティセンターが建てられていた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）39-40ページより。</ref>。2010年（平成22年）時点では、駅の開業時に植樹された[[マツ|松]]が「夫婦松」と名付けられて「一ノ橋駅開業記念樹」として存在していた<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）33-34ページより。</ref>。2011年（平成23年）時点では、駅跡の近くに上名寄駅と同様のレプリカの[[駅名標]]が設置されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）115ページより。</ref>。コミュニティセンターには、ゲートボール場や駐車場が併設されている。2012年11月には、コミュニティセンターに隣接して一の橋住民センターがオープンし、施設内へ郵便局が移転・入居した。<ref>[http://www.nayoro-np.com/news/2012-11-19.html 名寄新聞 平成24年11月20日記事]</ref> また、2011年（平成23年）時点では駅跡から遠軽方に向かってすぐの場所に、名寄川に架かっていた橋梁の橋台が残存していた<ref name="bunken2011H"/>。さらに上興部駅との間にある天北峠には国道のすぐ脇を通っていた切通しの線路跡が残存し<ref name="bunken2000H"/><ref name="bunken2010H"/>、保線小屋と思われる建物も倒壊しそうな状況ながら残存していた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[幸成駅]] -
