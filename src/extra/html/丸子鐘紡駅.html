丸子線の廃線に伴い廃止となる<ref name="bunken2010H-3-R"/><ref name="bunken2008M-06"/><ref name="bunkenKW05-R"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML073"/>。駅舎側（西側）が上り線（丸子町方面）、外側（東側）が下り線（上田東方面）となっていた<ref name="bunkenRML073"/>。そのほか鐘淵紡績丸子工場への0.8kmの[[専用鉄道|専用線]]を分岐した<ref name="bunkenRML073"/>。この引き込み線は上り線の南側、ホームを過ぎた部分から分岐した<ref name="bunkenRML073"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-2">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）64-67ページより。</ref>。駅舎は構内の西側に位置し、ホーム南側の[[階段]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML073"/>。ホームは上屋を有した<ref name="bunkenRML073"/>。[[駅名標]]には、1959年（昭和34年）時点では仮名表記では「まりこかねぼう」となっていたが、[[ローマ字]]表記では「MARUKO」となっており、記載に相違があった<ref name="bunkenRML073"/>。 === 鐘紡丸子工場引き込み線 === 当駅から工場までの延長は0.8kmで、当駅を出ると依田川側（西側、右手側）方面に大きくカーブし、さらに左側にカーブして工場に至った<ref name="bunkenRML073"/>。途中、工場入口附近から工場内のホーム手前部分までは機回し用に[[複線]]になっていた<ref name="bunkenRML073"/>。工場のホームは1面1線で、上屋を有した<ref name="bunkenRML073"/>。 工場を出た貨物列車は、当駅にて旅客列車の交換が行われる際には、当駅のホーム手前部分で待機していた<ref name="bunkenKW05">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）29,34ページより。</ref>。 == 駅周辺 == 鐘淵紡績（現・[[クラシエホールディングス]]）が上田市大字小牧小字城下（この地は戦後、[[日本専売公社]]（現・[[日本たばこ産業]]）上田工場となるが2006年（平成18年）に閉鎖。以降は更地）に続いて、丸子町大字下丸子の依田川岸に丸子工場を建築した。この工場招致は丸子鉄道によって行われたものであった<ref name="bunkenRML073"/>。引込み線を起点に同工場で作られた繊維製品が東京・大阪方面に輸送された。当線の廃止後はトラック輸送に完全移行した。この丸子工場は1977年（昭和52年）に系列会社のカネボウ絹糸京美人株式会社として独立し、同社本社工場となったがその後解散、閉鎖された。その後はカネボウベルプラザというショッピングセンターとなっている。 * [[国道152号]] * 長野県道丸子東部線 - 現・[[長野県道81号丸子東部インター線]]。 * 中丸子簡易郵便局 * 依田川 == 駅跡 == 2007年（平成19年）8月時点では、遺構は無くなっていた<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点では、専用線の分岐地点は駅の雰囲気が残っていた<ref name="bunkenKW05"/>。駅跡は民家になっていた<ref name="bunken2010H-3">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）71-72ページより。</ref>。 また、1996年（平成8年）時点では、丸子町駅附近から電鉄大屋駅附近までの線路跡は丸子[[市町村道|町道]]（後に上田市道）となっていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]][[9月]]発行）88ページより。</ref>。舗装道路となって整備されているため、2007年（平成19年）8月時点では遺構は無かった<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点でも道路は健在であった<ref name="bunkenKW05"/><ref name="bunken2010H-3"/>。 == 隣の駅 == ; 上田丸子電鉄 : 丸子線 :: [[上長瀬駅]] -
