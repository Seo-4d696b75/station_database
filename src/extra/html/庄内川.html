'''庄内川'''（しょうないがわ）は、[[岐阜県]]南東部および[[愛知県]]北西部を流れ[[伊勢湾]]に注ぐ[[河川]]。[[一級水系]]庄内川の本流である。[[岐阜県]]内では、「'''土岐川'''」と呼ばれている。岐阜・愛知県境の諏訪大橋から下流の玉野渓谷区間では「玉野川」と呼ばれることもある<ref name="kokokei">{{Cite web |url=http://www.rurubu.com/sight/detail.aspx?BookID=A3301900 |title=古虎渓　多治見市の観光スポット|publisher=るるぶ.com|accessdate=2016-08-24}}</ref><ref name="tamanogawa">{{Cite web |url=http://www.city.kasugai.lg.jp/machi/keikan/hyakkei/azone/tamanogawakeikoku.html |title=玉野川渓谷|publisher=春日井市|accessdate=2016-08-24}}</ref>。 == 地理 == === 流域の概要 === 水源は、[[岐阜県]][[恵那市]]の夕立山<ref name="shonaigawa">{{Cite web |url=http://www.cbr.mlit.go.jp/shonai/keikaku/seibi/daijin/pdf/kasenseibi_plan.pdf |title=庄内川水系河川整備計画 |publisher=国土交通省中部地方整備局 |accessdate=2016-08-22}}</ref>。 瑞浪、土岐、多治見の盆地を流れ、愛知・岐阜県境の玉野渓谷を抜けて、[[春日井市]]高蔵寺で[[濃尾平野]]に出る。[[名古屋市]][[港区 (名古屋市)|港区]]で[[伊勢湾]]に注ぐ<ref name="shonaigawa" />。 途中、瑞浪市で[[小里川]]、土岐市で妻木川、多治見市で笠原川、名古屋市[[西区 (名古屋市)|西区]]で[[矢田川 (愛知県)|矢田川]]を合流する。 庄内川は都市河川にもかかわらず、河口域には[[藤前干潟]]等の豊かな河川環境も残されている<ref name="shonaigawa" />。 下流域の名古屋市旧市街地を[[洪水]]から守るために、「[[洗堰緑地|洗堰]]」と呼ばれる越流堤や小田井遊水地（[[庄内緑地|庄内緑地公園]]）などの[[遊水地]]を旧市街地の反対側に整備している。[[新川 (庄内川水系)|新川]]も、そうした庄内川氾濫対策の一つとして整備されたものである。 === 地形 === 庄内川は、夕立山の水源から北西に流れ、その後南西に流れを転じ、屏風山断層に並行して標高200～300メートルの丘陵地を刻むように流れる。瑞浪、土岐、多治見の盆地（東濃盆地）の間は、峡谷を刻んでいる。高蔵寺から下流は瀬戸層群の段丘地形に囲まれ、名古屋市北部から下流は低平地が広がっている。また、庄内川下流域は濃尾平野の[[海抜ゼロメートル地帯]]となっている<ref name="shonaigawa" />。 ==歴史== ===呼称の由来=== 庄内川は昔、土岐川、玉野川、勝川、枇杷島川、番場川、一色川などと、その沿川の地名で呼ばれており、一貫した呼び名はなかった。江戸時代に農業集落である稲内庄、山田庄（現在の名古屋市[[北区 (名古屋市)|北区]]・[[西区 (名古屋市)|西区]]付近）、一色庄などの庄の内を流れる川ということで、庄内川と呼ばれるようになったと考えられる<ref>{{Cite web|title=庄内川について|url=http://www.cbr.mlit.go.jp/kawatomizu/kasen/shonaigawa/shonaigawa.html|website=www.cbr.mlit.go.jp|accessdate=2019-09-06}}</ref><ref>{{Cite web|title=日本の川 - 中部 - 庄内川 - 国土交通省水管理・国土保全局|url=https://www.mlit.go.jp/river/toukei_chousa/kasen/jiten/nihon_kawa/0508_shonai/0508_shonai_02.html|website=www.mlit.go.jp|accessdate=2019-09-06}}</ref>。明治になり、愛知県内では各地の異なった名称を統一し、庄内川と呼ばれるようになったと考えられる<ref name="yobina">{{Cite web |url=http://www.cbr.mlit.go.jp/shonai/tanto/kasen/aramashi/tokigawa/index.html |title=土岐川と庄内川|publisher=国土交通省中部地方整備局庄内川河川事務所|accessdate=2016-08-24}}</ref>。岐阜県内では、今でも土岐川と呼ばれている。 ===水害の歴史=== 庄内川流域では、昔から洪水により浸水被害を被ってきた<ref name="origawa">{{Cite web |url=http://www.cbr.mlit.go.jp/shonai/origawa/familia/saigaishi/index.html |title=流域の災害史 |publisher=国土交通省中部地方整備局庄内川河川事務所|accessdate=2016-08-24}}</ref>。 *1779年（[[安永 (元号)|安永]]8年）8月 - 新川開削と洗堰築造のきっかけとなる大洪水が発生。 *1839年（[[天保]]10年）8月 - 高蔵寺（春日井市）地内の堤防決壊。 *1896年（[[明治]]29年）9月 - 高蔵寺、勝川（春日井市）、瀬古（名古屋市[[守山区]]）地内で破堤。赤痢が大流行。 *1906年（明治37年）7月 - 志段味（名古屋市守山区）、大留（春日井市）、桜佐（春日井市）、勝川、瀬古地内で破堤。 *1911年（明治44年）9月 - 高蔵寺地内で破堤。 *1934年（[[昭和]]9年）9月 - [[室戸台風]]。 *1957年（昭和32年）9月 - 秋雨前線の活動により多治見市中心部のほとんどの家屋が浸水。被災家屋22,428戸（愛知県）、4,540戸（岐阜県）。 *1959年（昭和34年）9月 - [[伊勢湾台風]]。庄内川・新川の13ヶ所が破堤。被災家屋140,569戸（愛知県）、6,227世帯（岐阜県）。 *1971年（昭和46年）9月 - [[昭和46年台風第29号|台風29号]]により、庄内川上流圏域で河川が氾濫。床上浸水19棟、床下浸水753棟。<ref name="aichi">{{Cite web |url=http://www.city.nagoya.jp/ryokuseidoboku/cmsfiles/contents/0000022/22264/syounaizyoulyuuseibikeikaku2.pdf |title=庄内川水系庄内川上流圏域河川整備計画 |publisher=愛知県、名古屋市 |accessdate=2016-08-24}}</ref> *1972年（昭和47年）6月 - 梅雨前線の活動により、上流域での被害甚大。死者6名。被災家屋832棟（愛知県）、1,515棟（岐阜県）。 *1975年（昭和50年）7月 - 梅雨前線の活動により、被災家屋10,315棟（愛知県）、107棟（岐阜県）。 *1976年（昭和51年）9月 - [[昭和51年台風第17号|台風17号]]により、床上浸水1,327棟。被災家屋8,713棟（愛知県）。 *1983年（昭和58年）9月 - [[昭和58年台風第10号|台風10号]]と秋雨前線の活動により、出水。被災家屋7,871棟（愛知県）、164棟（岐阜県）。 *1988年（昭和63年）9月 - 熱帯低気圧と秋雨前線の活動により、上流域で浸水被害、洗堰からも越流。被災家屋1,896棟（愛知県）、94棟（岐阜県）。 *1989年（[[平成]]元年）9月 - [[平成元年台風第22号|台風22号]]により、上流域で浸水被害。被災家屋84棟（愛知県）、571棟（岐阜県）。 *1991年（平成3年）9月 - [[平成3年台風第18号|台風18号]]と秋雨前線の活動により、[[内津川]]などが破堤。[[春日井駅 (JR東海)|JR春日井駅]]等が冠水した。<ref name="aichi" />床上浸水1,722棟。被災家屋6,440棟（愛知県）、16棟（岐阜県）。 *1999年（平成11年）6月 - 梅雨前線の活動により、上流部で床上浸水31棟。被災家屋1棟（愛知県）、120棟（岐阜県）。 *2000年（平成12年）9月 - [[平成12年台風第14号|台風14号]]により、新川が決壊。名古屋市、[[北名古屋市]]、清須市等で浸水被害（[[東海豪雨]]）。被災家屋34,041棟（愛知県）、8棟（岐阜県） *2011年（平成23年）9月 - [[平成23年台風第15号|台風15号]]により、志段味地内で越水。支川では内津川、[[八田川 (愛知県)|八田川]]、[[地蔵川 (春日井市)|地蔵川]]等で越水。地蔵川沿川では400棟以上が浸水。<ref name="aichi" /> ===治水事業の歴史=== [[File:Origawa Dam.jpg|thumb|right|180px|小里川ダム]] 先述の経緯から、治水事業も頻繁に行われてきた<ref name="shonaigawa" />。 *1614年（[[慶長]]19年）- [[徳川家康]]の名古屋築城に伴う治水事業で、現在の堤防位置に大半の堤が完成する。 *1784年（[[天明]]4年）- 「天明の治水」により、新川洗堰を築造、分派し、庄内川とほぼ並行して伊勢湾に至る新川を開削。 *1917年度（[[大正]]7年度） - 愛知県による改修事業着手。 *1932年度（[[昭和]]7年度） - 岐阜県による改修事業着手。 *1932年度（昭和7年度） - 矢田川の河道付替完成。 *1936年度（昭和11年度） - 多治見市脇之島地区の河道付替完成。 *1937年度（昭和12年度） - 直轄砂防事業着手。 *1942年度（昭和17年度） - 直轄改修事業着手。 *1950年度（昭和25年度） - 愛知県による改修事業着手。 *1958年度（昭和33年度） - [[枇杷島町|枇杷島]]「中島」の撤去完成。疎通能力が増強される。 *1963年度（昭和38年度） - 昭和34年9月の伊勢湾台風を受けた、伊勢湾等高潮対策事業で高潮堤完成。 *1969年（昭和44年）3月 - [[一級水系]]に指定。同4月、大臣直轄区域指定。 *1969年度（昭和44年度） - 庄内川水系工事実施基本計画。 *1975年度（昭和50年度） - 庄内川水系工事実施基本計画改定（施行）。 *1989年度（[[平成]]元年度） - [[庄内緑地|小田井遊水地]]概成。 *1999年度（平成11年度） - 同年6月の洪水を受け、土岐川河川災害復旧等関連緊急事業着手。 *2000年度（平成12年度） - 東海豪雨を受け、庄内川・新川河川激甚災害対策特別緊急事業着手。 *2003年度（平成15年度） - 土岐川河川災害復旧等関連緊急事業完成。 *2004年（平成16年）3月 - [[小里川ダム]]竣工。<ref name="origawa2">{{Cite web |url=http://www.cbr.mlit.go.jp/shonai/origawa/know/kojishi/index.html |title=写真で見る小里川ダム工事史 |publisher=国土交通省中部地方整備局庄内川河川事務所 |accessdate=2016-08-30}}</ref> *2004年度（平成16年度） - 庄内川河川激甚災害対策特別緊急事業完成。 *2005年（平成17年）11月 - 庄内川水系河川基本整備方針の策定。 *2008年（平成20年）3月 - 庄内川水系河川基本整備計画の策定。<ref name="seibikeikaku">{{Cite web |url=http://www.mlit.go.jp/river/basic_info/jigyo_keikaku/gaiyou/seibi/ |title=河川整備基本方針・河川整備計画 |publisher=国土交通省 |accessdate=2016-08-30}}</ref> == 河川環境 == === 水質 ===
