くりでん』66頁。</ref>。[[国鉄バス]]も発着していたことから、窓口は（自）沢辺駅（[[自動車駅]]）としての営業もあった。[[1968年]]には仙台盛岡急行線が新設され、当駅には特急バスも発着するようになったが、[[ジェイアールバス東北|JRバス]]による運行は[[1999年]]に廃止、[[東日本急行]]・[[ミヤコーバス]]も当線の廃止を機に乗り入れを取りやめ、バスターミナル機能は金成総合支所前（東日本急行は金成庁舎前）へと移された。[[東北新幹線]]の[[くりこま高原駅]]と連絡するバス「栗夢号しゃとる」は2006年9月30日をもって廃止された。 現在は遺構として交換駅の名残となる3列分の線路とホーム、駅前広場が残る。 == 駅構造 == 2面3線の[[プラットホーム|ホーム]]を持つ[[地上駅]]。線路は東西に走り、木造の駅舎を南側に設置、出入口はこの一箇所であって、北側へは駅の西にある構内踏切を渡って通行した。駅舎の前に駅前広場がある。 開業時に建てられ、廃止まで使われた駅舎は、[[若柳駅]]と同じ設計図で作られたもので、外観も同じであった<ref>『さよなら! くりでん』64頁。</ref>。廃止後しばらくは駅舎がその姿をとどめていたが、のちに解体された。解体後の部材は、保存活動の拠点となる若柳駅駅舎の復元に活用された<ref>[http://rail.hobidas.com/blog/natori/archives/2010/04/post_1241.html 編集長敬白：くりはら田園鉄道は今…。]</ref>。 == 駅周辺 == 駅の周辺は沢辺の市街地である。西から南東に流れる[[三迫川]]の北岸に近い。西約1キロメートルに[[達田橋]]があり[[国道4号]]を渡す。この橋から駅までが沢辺の町の中心となる商店街である。駅の北西約500メートルの町はずれに栗原市役所金成総合支所があり、2005年の合併まで[[金成町]]役場であった。水田地帯を隔てて北に約1キロメートルに金成の市街地の端がかかり、金成の中心部までは駅から約1.5キロメートル離れている。 == 歴史 == * [[1921年]]（[[大正]]10年）[[12月20日]] - 開業。 * [[1987年]]（[[昭和]]62年）[[3月31日]] - 貨物取扱廃止<ref>寺田裕一『私鉄の廃線跡を歩くI 北海道・東北編』、JTBパブリッシング、2007年、168p。ISBN 978-4-533-06847-8</ref>。 * [[2007年]]（[[平成]]19年）[[4月1日]] - 廃止。 == 隣の駅 == ; くりはら田園鉄道 : くりはら田園鉄道線 :: [[大岡駅 (宮城県)|大岡駅]] - '''沢辺駅''' - [[津久毛駅]] == 脚注 == <references/> == 参考文献 == *MGブックス企画・編集『サヨナラ！ くりでん 「くりはら田園鉄道」公式メモリアル・ブック』、エムジー・コーポレーション、2007年。 == 外部リンク == * [http://www.jr-asahi.com/kuriden/eki/sawabe.htm くりでんとと〜ぶHP - 沢辺駅] {{くりはら田園鉄道線}} {{DEFAULTSORT:さわへ}} [[Category:宮城県の廃駅]] [[Category:栗原市の鉄道駅|廃さわへ]] [[Category:栗原市の歴史|さわへえき]] [[Category:日本の鉄道駅 さ|わへ]] [[Category:くりはら田園鉄道の鉄道駅]] [[Category:1921年開業の鉄道駅]] [[Category:2007年廃止の鉄道駅]] 0jpno3hze3d7xcz694q8a58p9e5mas2
