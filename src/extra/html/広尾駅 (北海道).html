eki.jpg|right|thumb|240px|1977年の広尾駅と周囲約500m範囲。右が帯広方面。{{国土航空写真}}]] '''広尾駅'''（ひろおえき）は、[[北海道]]（[[十勝支庁]]）[[広尾郡]][[広尾町]]丸山通り北2丁目にかつて存在した、[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ヒオ'''。広尾線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 == 概要 == 帯広駅から[[十勝平野]]を南下し、当時はこの地方唯一の[[港湾|港]]であった[[十勝港]]を結ぶために敷設された<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]][[10月]]発行）40-41ページより。</ref>長大[[盲腸線]]の[[終着駅|終端駅]]であった。さらに計画では、当駅より[[襟裳岬]]経由で現在[[日高本線]]の終着駅となっている[[様似駅]]まで延伸される予定であった。 一時期は[[最長片道切符]]の起終点だった。[[宮脇俊三]]の著書『[[最長片道切符の旅]]』では、始発駅として登場した。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]で、広尾線の[[終着駅|終端駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）140ページより。</ref>。ホームは、線路の南東側（帯広方面に向かって右手側）に存在した。旅客列車の発着に使用する駅舎側の1番線（上下本線）の北側に、ホームを有さない2-4番線を有した<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]][[4月]]発行）32-33ページより。</ref>。4番線からは北側に水槽及び[[転車台]]を有し（転車台は1983年（昭和58年）時点で現存していた<ref name="bunken1983"/>）、終端部分が車庫2線となっていた[[停車場#側線|側線]]が分岐していた。1-4番線は南西側で収束し、その延長上は入換線となっていた。さらにその途中から分岐する積卸線と、木材線を1線ずつ有した<ref name="bunken2010HE"/>。また、1番線のホーム端部分から駅舎側に戻る形で分岐し駅舎南東側の[[切欠きホーム|切欠き]]部分の貨物ホームへの貨物側線を1本有していた<ref name="bunken2010HE"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の南側に位置しホーム中央部分に接していた<ref name="bunken1983"/><ref name="bunken2010HE"/>。駅舎は1977年（昭和52年）に<ref name="bunken1980"/>開業時からの駅舎から改築された、船底形の鉄骨ヘーベンライト仕上げでモダンな構造<ref name="bunken1980"/>の建物であった。[[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「ピロロ」（陰になったところ）、「ピ・オロ」（石があるところ）、「ピルイ・ペツ」（[[砥石]]の川）、「ピラ・オロ」（崖のところ）など、由来には諸説ある。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は234人<ref name="bunken1983"/>。 == 駅周辺 == 十勝が仙台藩領となり築城された際の土塁や碑などがあり、現在[[広尾小学校]]が建っている。 * 広緑寺 * 広尾スキー場 - 駅より南西に約6km<ref name="bunken1983"/>。 * 大丸山森林公園 * 大丸山 - 駅より北西に約1km<ref name="bunken1983"/>。[[ツツジ]]の名所でもある大丸山展望所が存在する<ref name="bunken1980"/>。 * フンベの滝 - 駅より南に約3.7km<ref name="bunken1983"/>。 == バス路線 == * [[帯広駅バスターミナル]]より十勝バス広尾線で約2時間20分、「広尾」下車 * [[様似駅]]よりジェイ・アール北海道バス[[日勝線]]で約2時間、終点「広尾駅」下車 * [[札幌駅バスターミナル|JR札幌駅]]よりジェイ・アール北海道バス[[日勝線|高速ひろおサンタ号]]で約4時間20分、「広尾駅」下車 == 歴史 == * [[1932年]]（[[昭和]]7年）[[11月5日]] - '''[[日本国有鉄道|国有鉄道]]（[[鉄道省]]）広尾線'''大樹駅 - 当駅間の延伸開通（広尾線全通）に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1977年]]（昭和52年）[[11月15日]] - 駅舎改築。 * [[1982年]]（昭和57年）[[9月10日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[2月2日]] - 広尾線の全線廃止に伴い、廃駅となる。 *[[2018年]]（[[平成]]30年）6月 - 旧駅舎解体。 == 駅跡 == 廃駅後は広尾町が駅舎の払い下げを受け、1987年（昭和62年）7月に「広尾町鉄道記念館」としてリニューアルされた<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]][[11月]]発行）68ページより。</ref>。旧駅舎内には展示スペースが設置され、当時使用していた[[閉塞 (鉄道)|閉塞器]]、通標、[[保線]]用具、備品、[[時刻表]]、[[駅スタンプ]]、写真パネル、沿線模型などが保存・展示されている<ref name="bunken1998S"/>。駅スタンプは、[[さよなら運転|さよなら列車]]運転記念のスタンプが10種類設置されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）192-193ページより。</ref>。2011年（平成23年）時点で改札柵も残存し、旧ホーム側は[[連動装置#論理機構の仕組みと方法|転轍てこ]]や駅の銘板も残存している<ref name="bunken2011H"/>。但しレールは撤去され、駐車場になっていた<ref name="bunken2011H"/>。 この駅舎は、[[十勝バス]]と[[ジェイ・アール北海道バス]]が乗り入れるバスターミナルにもなっている。出札窓口は十勝バス'''広尾案内所'''として活用されており、廃止転換バスである同社広尾線の硬券乗車券の購入も可能である。ジェイ・アール北海道バスの停留所は、日勝線は「広尾」、都市間バスは「広尾駅」を名乗っている。以前はJR北海道[[日高本線]]との連絡運輸も行っていたことから、いわゆる[[自動車駅]]として存続しているが、ジェイ・アール北海道バスの乗車券は取り扱っていない。国鉄時代に廃止され、旧駅舎がバス待合所として活用されている例は、2009年（平成21年）現在で北海道内は当駅と[[北檜山駅]]の2例のみである。 駅構内跡地は「鉄道記念公園」として整備され、一角には[[国鉄C11形蒸気機関車|C11形蒸気機関車]]'''C11 176号機'''の動輪が保存・展示されている<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）40-41ページより。</ref>。またパークゴルフの施設が作られている。2010年（平成22年）時点でも同様であった<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）89ページより。</ref>。 廃止直後は駅舎裏に、[[国鉄キハ20系気動車#キハ22形|キハ22形]][[気動車]]'''キハ22 134'''や旧形[[客車]]・[[緩急車]]・[[貨車]]などが保存されていたが、状態悪化に伴い解体・撤去されている。 その後、2018年（平成30年）6月に、老朽化した旧駅舎が解体された。 <gallery> 画像:広尾駅II.JPG|改札口がそのまま残っている（2007年8月撮影） 画像:広尾駅IV.JPG|出札窓口は十勝バス広尾案内所となっている（2007年8月撮影） 画像:広尾 鉄道記念公園.JPG|駅舎裏手の構内は駐車場と鉄道記念公園になっており、パークゴルフ場も併設している（2007年8月撮影）
