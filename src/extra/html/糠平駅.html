士幌線の[[清水谷駅|清水谷]]駅 - 当駅間開通により、開業。[[一般駅]]。（清水谷10.3km-幌加9.9km）<ref name="Teisha">停車場変遷大事典 国鉄・JR編 Ⅱ JTB 1998年10月発行</ref> ** 帯広営林区署が糠平貯木場設置。 * [[1938年]]（昭和13年） - 貯木場まで専用線350m敷設。 * [[1939年]]（昭和14年）[[11月18日]] - 当駅 - [[十勝三股駅|十勝三股]]駅間が開業し、中間駅となる<ref name="Kushi100">JR釧路支社「鉄道百年の歩み」平成13年12月発行。</ref>。 * [[1955年]]（昭和30年）[[8月1日]] - 糠平ダム建設のため移転。改キロ（清水谷10.9km-幌加11.6km）<ref name="Teisha"/>。 * [[1970年]]（昭和45年）[[9月10日]] - [[車扱貨物]]の取り扱いを廃止<ref name="Teisha"/><ref name="Kushi100"/>。 * [[1974年]]（昭和49年）[[10月1日]] - 貨物の取り扱いを全面廃止<ref name="Teisha"/><ref name="Kushi100"/>。 * [[1978年]]（昭和53年）[[12月25日]] - 当駅 - 十勝三股駅間の列車運転を休止<ref name="Teisha"/>。上士幌タクシーが受託する[[マイクロバス]]による[[バス代行|代行輸送]]となる。休止後も同区間は名目上鉄道路線として存続しており、国鉄の[[時刻表]]にも引き続き掲載されていたが、列車が発着することはなく、事実上は廃線となっていた。 * [[1984年]]（昭和59年）[[2月1日]] - 荷物の取り扱いを廃止<ref name="Teisha"/><ref name="Kushi100"/>。 * [[1987年]]（昭和62年）[[3月23日]] - 士幌線の全線廃止により、廃駅<ref name="Teisha"/>となる。同線が、バス転換される<ref name="Kushi100"/>。 == 駅跡・周辺 == 現在は、上士幌町鉄道資料館となっている。 * [[国道273号]] * [[ぬかびら温泉|ぬかびら源泉郷]] * [[十勝バス]]糠平案内所 ** 士幌線帯広方面代替「糠平営業所前」停留所 ** 旭川方面[[ノースライナー (北海道)|ノースライナーみくに号]]（士幌線十勝三股方面代替）「糠平温泉」停留所 糠平以北では、路線廃止後の上士幌タクシーによる代替バスですら糠平地区以外はほぼ無人化という沿線の極端な過疎化によって2003年9月いっぱいで廃止され、同年10月より帯広 - 糠平 - 旭川間の[[高速バス|都市間バス]]であるノースライナーみくに号が十勝三股停留所を新設、幌加温泉入口停留所と糠平方面の乗降も可能として代替する。 == 隣の駅 == ; 日本国有鉄道 : 士幌線 :: [[黒石平駅]] - <[[電力所前仮乗降場]]> - <[[糠平ダム仮乗降場]]> - '''糠平駅''' - [[幌加駅]] 列車運転休止・バス代行輸送化の際、糠平 - 幌加間に糠平スキー場入口バス停を新設したが時刻表には掲載なし。 == 脚注 == {{脚注ヘルプ}} {{Reflist}} {{士幌線}} {{DEFAULTSORT:ぬかひら}} [[Category:北海道の廃駅]] [[Category:日本の鉄道駅 ぬ|かひら]] [[Category:日本国有鉄道の廃駅]] [[Category:1937年開業の鉄道駅]] [[Category:1987年廃止の鉄道駅]] [[Category:上士幌町の交通|廃ぬかひらえき]] [[Category:上士幌町の建築物|ぬかひらえき]] [[Category:上士幌町の歴史|ぬかひらえき]] lfd1ehkx4mv15hheo1wnxadtc6twqrz
