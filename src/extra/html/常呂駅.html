[[鉄道省]]湧網東線の卯原内駅 - 当駅間延伸開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1940年]]（昭和15年） - 常呂町日吉の奥村鉱業常呂鉱山（通称・奥村鉄山）操業開始。当駅へ専用軌道を敷設（時期不明）し、鉄鉱石運搬。 * [[1945年]]（昭和20年） - 終戦による常呂鉱山操業停止に伴い、奥村鉄山専用軌道運用停止。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1952年]]（昭和27年）[[12月6日]] - 当駅 - 下佐呂間駅（後の浜佐呂間駅）間の延伸開通に伴い、中間駅となる。 * [[1953年]]（昭和28年）[[10月22日]] - 中湧別駅 - 網走駅間全通により路線名を湧網線に改称、それに伴い同線の駅となる。 * [[1982年]]（昭和57年）[[3月27日]] - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[3月20日]] - 湧網線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム東側と対向側ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=161}}</ref>。駅舎側ホーム（南側）が上り線、対向側ホーム（北側）が下り線となっていた（番線表示なし）<ref name="bunken1983"/>。島式ホームの外側の1線は[[停車場#本線|副本線]]として、[[停車場#側線|側線]]扱いで残っていた<ref name="bunken1983"/>。そのほか、上り線の中湧別方から駅舎側に分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を2線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の南側に位置し単式ホーム中央部分に接していた<ref name="bunken1983"/>。[[木構造 (建築)|木造]][[平屋]]の駅舎であった<ref name="bunken2012E">{{Cite book|和書|author=松尾定行 |title=消えた駅舎 消える駅舎 |date=2012-4 |publisher=[[東京堂出版]] |isbn=978-4490207811 |page=190}}</ref>。[[流氷]]の時期には除雪したホームの一部分に、見学のための「流氷観覧席」を設けていた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「ト・コロ・ペッ」（[[湖]]を持つ川）に由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=170}}</ref>。常呂川周辺に沼が多かったことから付いた<ref name="bunken2004N"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は130人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=19}}</ref> * [[北海道道1087号網走常呂自転車道線]] -
