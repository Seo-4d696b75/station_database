'''叡山駅'''（えいざんえき）は、かつて[[滋賀県]][[大津市]][[下阪本]]{{Sfn|江若鉄道の思い出|p=28}}にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 == 歴史 == 叡山駅は1921年（大正10年）、江若鉄道の開業に合わせて開設された駅である{{Sfn|江若鉄道の思い出|p=28}}。当時開通したのは[[三井寺下駅|三井寺]]から叡山までの6.0キロメートルのみで{{Sfn|田中|宇田|西藤|1998|p=394}}、当駅が路線の終点だった{{Sfn|江若鉄道の思い出|p=28}}。路線開業の翌日からは[[比叡山]][[延暦寺]]で開祖[[最澄]]の1100年大遠忌が開催されていて、叡山までの開業はこの大遠忌へ向かう参詣客輸送に間に合わせるためであった{{Sfn|江若鉄道の思い出|p=28}}{{Sfn|レイル|p=72}}{{Refnest|延暦寺は江若鉄道の設立にあたり最高額の3500株を負担し{{Sfn|田中|宇田|西藤|1998|p=395}}、同社の筆頭株主であった{{Sfn|寺田|2010|p=9}}。出資は大遠忌の参詣客輸送のため、鉄道敷設に協力したものとみられている{{Sfn|田中|宇田|西藤|1998|p=395}}。}}。路線は2年後の1923年（大正12年）に叡山から先、[[雄琴温泉駅|雄琴]]まで延伸する{{Sfn|田中|宇田|西藤|1998|p=394}}{{Sfn|寺田|2010|p=9}}。 線内では[[日吉駅 (滋賀県)|日吉駅]]と並び比叡山参詣の玄関口を担う駅であった{{Sfn|江若鉄道の思い出|p=34}}。江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]されている{{Sfn|今尾|2008|pp=31&ndash;32}}。 * 叡山駅は旅客と貨物の取り扱いを行う[[一般駅]]で{{Sfn|寺田|2010|p=10}}、構内には[[プラットホーム|ホーム]]と[[停車場#側線|側線]]が設けられていた{{Sfn|竹内|1967}}{{Sfn|寺田|2010|p=15}}。[[列車交換]]が可能な停車場であり{{Sfn|寺田|2010|p=10}}、ホームは2本の線路の間に置かれ、両側に線路が接する[[島式ホーム]]が1面のみ{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=78}}。ホームには木が生えていた{{Sfn|レイル|p=80}}。 駅があった場所は下阪本4丁目交差点の南に当たる{{Sfn|江若鉄道の思い出|p=28}}。江若鉄道の廃線後に開通した[[湖西線]]と[[国道161号]][[西大津バイパス]]が通っていて、駅の痕跡は残されていない{{Sfn|江若鉄道の思い出|p=28}}{{Sfn|寺田|2010|p=24}}。 なお、湖西線にも当駅と同名の「叡山駅」が設けられた（1994年に[[比叡山坂本駅]]に改称{{Sfn|今尾|2008|pp=31&ndash;32}}）が、その位置は当駅ではなく隣の[[日吉駅 (滋賀県)|日吉駅]]に一致する{{Sfn|江若鉄道の思い出|p=35}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[滋賀駅]] - '''叡山駅''' - [[日吉駅
