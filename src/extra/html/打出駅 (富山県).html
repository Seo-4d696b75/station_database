'''越中電気軌道'''四方駅 - 打出浜駅間延伸開通に伴い'''打出駅'''（初代、新富山駅起点8.1km地点）、'''打出浜駅'''（うちいではまえき、新富山駅起点8.5km地点）の2駅が開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref>。 * [[1927年]]（昭和2年）[[2月13日]] - 鉄道会社名を'''越中鉄道'''に改称。それに伴い同鉄道の駅となる<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1929年]]（昭和4年）[[7月2日]] - 打出浜駅 - 堀岡駅間延伸開通に伴い打出浜駅が中間駅となる<ref name="bunken2010H-3-R"/>。 * [[1932年]]（昭和7年）- [[1934年]]（昭和9年）頃 - 打出駅（初代）を'''富山飛行場前駅'''（とやまひこうじょうまえ）に改称<ref name="bunken2008M-6"/>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]に伴い'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1946年]]（昭和21年）以前 - 富山飛行場前駅廃止<ref name="bunken2008M-6"/>。また打出浜駅を新富山方に0.1km移設の上'''打出駅'''（2代目）に改称<ref name="bunken2008M-6"/><ref group="注" name="ekigyoumu">累計キロは新富山駅起点8.4kmとなるが、『RM LIBRARY 107 富山地鉄笹津・射水線』、『新 鉄道廃線跡を歩く3 北陸・信州・東海編』、『新 消えた轍 7 北陸』の各書籍では累計キロは新富山駅起点8.1kmとする。</ref>。 * 時期不詳 - 無人化<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）40-41ページより。</ref>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/>。ホームは線路の北側（新港東口方面に向かって右手側）に存在した<ref name="bunkenRML107"/>。 [[無人駅]]となっていた<ref name="bunkenRML107"/>。[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は撤去されたがホームに[[待合所]]を有した<ref name="bunkenRML107"/>。 == 駅周辺 == * [[国道415号]] * 貴船神社 == 駅跡 == 線路跡は四方駅跡南方附近から終点・新港東口駅附近まで[[サイクリングロード]]となっている<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（JTBパブリッシング、[[1997年]]5月発行）108-109ページより。</ref>。1997年（平成9年）時点では、線路跡地はそのサイクリングロードであった<ref name="bunken1997H"/>。2006年（平成18年）時点<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）66-67ページより。</ref>、2010年（平成22年）時点でも同様であった<ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）53ページより。</ref><ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』50-51ページより。</ref>。 == 隣の駅 == ; 富山地方鉄道 : 射水線 :: [[四方駅]] -
