[[鉄道省]]雨龍線幌加内 - 当駅間延伸開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 ** [[10月10日]] - 線路名を幌加内線に改称、それに伴い同線の駅となる。 * [[1932年]]（昭和7年）[[10月25日]] - 当駅 - 朱鞠内駅間延伸開通に伴い中間駅となる。 * [[1941年]]（昭和16年）[[10月10日]] - 線路名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1982年]]（昭和57年）[[3月30日]] - 貨物・[[チッキ|荷物]]取扱い廃止。無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1995年]]（平成7年）[[9月4日]] - 深名線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の東側（名寄方面に向かって右手側）に存在した。[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）76-77ページより。</ref>。かつては[[列車交換]]可能な[[交換駅]]であった。1983年（昭和58年）時点では使われなくなった対向ホーム側の1線は交換設備運用廃止後も分岐器が深川方、名寄方の両方向とも維持された形で[[停車場#側線|側線]]として残っていた（但しホームは撤去されていた）<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）205ページより。</ref>。またその側線の名寄方から外側の深川方に伸びる行き止まりの短い側線を1線有した<ref name="bunken1983"/>。そのほか本線の深川方から分岐し駅舎南側への側線を1線有していた<ref name="bunken1993"/>。その後線路は側線を含め1993年（平成5年）までには撤去されたが、ホーム前後の線路は分岐器の名残で湾曲していた<ref name="bunken1993"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の東側に位置しホーム北側に接していた<ref name="bunken1993"/>。 [[貨物列車]]が運行されていた時代は、木材、[[デンプン|澱粉]]、[[雑穀]]の搬出が主体であった<ref name="bunken1983"/>。名残であった広い構内には夏場は[[ルピナス属|ルピナス]]の花で覆われていたとのことである<ref name="bunken1993"/>。 == 駅名の由来 == 所在地名より。[[アイヌ語]]の「ソウ<small>シ</small>ナイ<ref>[[ラテン文字化|アイヌ語ラテン翻字]]: so-us-nay</ref>」〔滝・ある・川〕に字をあてたものである。由来は駅西側の霧立峠へ上る道のかたわらで、滝になって落ちている現在のソーウンナイ川に由来すると推測される<ref name="bunken1993"/><ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/5160P.pdf|title=アイヌ語地名リスト ゴ～シブ　P51-60P|accessdate=2017-10-20|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は20人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は2人<ref name="bunken1993"/>。 == 駅周辺 == * [[国道239号]]（観月国道/霧立国道） * [[国道275号]]（空知国道/美深国道） * [[雨竜川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> * ソーウンナイ川<ref name="bunken1980M"/> * 相雲内岳 -
