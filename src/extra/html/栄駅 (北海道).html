'''[[北海道鉄道 (2代)|北海道鉱業鉄道]]金山線'''生鼈駅（後の旭岡駅） - 当駅間延伸開通に伴い'''似湾駅'''（にわんえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 ** [[11月11日]] - 当駅 - 辺富内駅（後の富内駅）間延伸開通に伴い中間駅となる。 * [[1924年]]（大正13年）[[3月3日]] - 鉄道会社名を'''北海道鉄道'''（2代目）に改称、それに伴い同鉄道の駅となる。 * [[1943年]]（昭和18年）[[8月1日]] - 北海道鉄道が[[戦時買収私鉄|戦時買収]]により国有化。線路名を'''富内線'''に改称、それに伴い同線の駅となる。同時に'''栄駅'''に改称。 * [[1977年]]（昭和52年）[[2月1日]] - 貨物・[[チッキ|荷物]]取扱い廃止。 * 時期不詳<ref group="注" name="ekigyoumu">1981年（昭和56年）4月時点では既に無人駅となっている（書籍『コロタン文庫36 国鉄駅名全百科 56.4訂補版』（[[小学館]]、[[1981年]][[10月]]発行）54ページより）。</ref> -無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の廃線に伴い廃止となる。 == 駅構造 == 昭和30年代の栄駅には保線区があり、多くの鉄道員が配置されていた。駅前には、周辺の森林から切り出された丸太を集積する土場があり、[[日本通運]]の担当者が丸太を[[貨車]]に積み込む作業が行われていた<ref>{{Cite book |和書 |author=辻喜美男 　「昔の栄地区を思い出して」|year=2014 |title=穂別高齢者の語り聞き史（昭和編）大地を踏みしめて 上　|page=p24 |publisher=穂別高齢者の語りを聞く会 }}</ref>。 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。ホームは線路の北側（日高町方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）106ページより。</ref>。かつては島式ホーム1面2線を有する[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎側の1線は、交換設備運用廃止後も鵡川方の[[分岐器|転轍機]]及びホーム端までの線路が[[停車場#側線|側線]]として残っていた（但し1983年（昭和58年）時点では転轍機の先、ホームに至る間の部分に[[車止め]]が設置されていた<ref name="bunken1983"/>）。 [[無人駅]]（簡易委託駅）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の北側に位置した<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名より。 旧駅名の似湾（にわん）は、[[アイヌ語]]の「ニ・アン・ペツ」（木の生えた川）に由来する<ref name="bunken1983"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は36人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道74号穂別鵡川線]] * [[北海道道59号平取厚真線]] * 似湾郵便局（集配局） * むかわ町立仁和小中学校 * [[鵡川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）11ページより。</ref> * 似湾川<ref name="bunken1980M"/> 昭和20年代の駅周辺には、駅前の道路沿いに20軒以上の民家、商店街（旅館、菓子店、薬局、パチンコ屋、鍛冶、劇場、診療所等）があった<ref>{{Cite book |和書 |author=布施武光 「86年間思いつくまま」|year=2014 |title=穂別高齢者の語り聞き史（昭和編）大地を踏みしめて 上|page=p39 |publisher=穂別高齢者の語りを聞く会 }}</ref>。 == 駅跡 == 木材加工場の敷地となっている。2011年（平成23年）時点では旧駅構内には雑草が生い茂っている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）84ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 富内線 :: [[旭岡駅]] -
