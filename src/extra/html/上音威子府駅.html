'''上音威子府駅'''（かみおといねっぷえき）は、[[北海道]]（[[上川支庁]]）[[中川郡 (天塩国)|中川郡]][[音威子府村]]字上音威子府にあった[[北海道旅客鉄道]]（JR北海道）[[天北線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''カオ'''。同線の[[廃線]]に伴い[[1989年]]（平成元年）[[5月1日]]に廃駅となった。 [[1987年]]（昭和62年）から[[臨時駅]]となっており、廃止直前は、下り（[[稚内駅|稚内]]行）の一番列車のみが停車し、上り（音威子府方面）は全列車通過していた。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の西側（南稚内方面に向かって左手側）に存在した。[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。かつては千鳥状にずれた単式ホーム2面2線を有する[[列車交換]]可能な[[交換駅]]であった。交換設備運用廃止後は線路は撤去されたが、上り方面ホームは残存していた。太平洋戦争以前には、周辺の[[森林]]（[[北海道大学]]演習林）から出荷される木材を積み込むためのヤードを有していた。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の西側に位置しホームに接していた。木造板壁<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）188ページより。</ref>で、柱を露出させてその間を縦板で埋めるという遊びの入ったデザインの<ref name="bunken2000E">書籍『駅舎再発見』（著：杉崎行恭、[[JTBパブリッシング]]、[[2000年]]12月発行）148ページより。</ref>駅舎であった。 == 駅名の由来 == 当駅の所在する地名から。地名は「音威子府」の語源（[[音威子府村]]参照）ともなった音威子府川の上流に位置するところから付けられている。 == 駅周辺 == 上音威子府集落があり、上音威子府小中学校が存在したが、離農により集落が消滅し、現在は無人地帯となっている。 * [[国道275号]]（頓別国道） - 駅から約0.1km<ref>『国鉄全線各駅停車1』200ページより。</ref>。 * 天北川 * [[宗谷バス]]「上音威子府」停留所（[[天北線 (宗谷バス)|天北宗谷岬線]]、都市間バス） == 歴史 == * [[1914年]]（大正3年）[[11月7日]] - [[鉄道省|鉄道院]]宗谷線音威子府駅 - 小頓別駅間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1919年]]（大正8年）[[10月20日]] - 線路名を宗谷本線に改称、それに伴い同線の駅となる。 * [[1930年]]（昭和5年）[[4月1日]] - 音威子府駅 - 稚内駅間を宗谷本線から分割し線路名を北見線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1961年]]（昭和36年）[[4月1日]] - 線路名を天北線に改称、それに伴い同線の駅となる。 * [[1973年]]（昭和48年）[[9月17日]] - 貨物・[[チッキ|荷物]]取扱い廃止。交換設備を廃止。同時に無人化。 * [[1987年]]（昭和62年） ** [[4月1日]] - [[国鉄分割民営化]]により、北海道旅客鉄道（JR北海道）の駅となる。 ** 11月10日 - [[臨時駅]]（4月1日から11月30日まで営業）に種別変更。 * [[1989年]]（平成元年）[[5月1日]] - 天北線の廃線に伴い廃止となる。 == 利用状況 |1958 |56 |- |1964 |55 |- |1971 |8 |} == 駅跡 == [[ファイル:Kamiotoineppu station 2009.JPG|thumb|right|250px|上音威子府駅ホーム跡及び線路跡（2009年）。写真上が小頓別方面]] 1997年（平成9年）時点では駅舎、ホーム、[[鉄道信号機|信号機器設備]]が残存していた<ref name="bunken199712H">書籍『鉄道廃線跡を歩くIV』（[[JTBパブリッシング]]、[[1997年]]12月発行）23-24ページより。</ref>。駅舎は廃屋一歩手前の状況ながらも[[農業機械|農機材]]置き場に再利用され<ref name="bunken2000E"/>、2004年（平成16年）9月頃までは残存していたが、倒壊の危険があるため取り壊された。駅舎があった頃は、当時の椅子などもそのまま残存していた。 2009年（平成21年）秋時点で、国道から駅前広場までの数十メートルの道（未舗装）、駅前広場（未舗装）、石積みの[[プラットホーム]]、音威子府方面及び小頓別方面への線路跡が確認できた。ホームや線路跡には草木が生い茂り、貨物扱いなどを行った構内跡も原野化している。ホームは2010年（平成22年）時点でも残存しており<ref>書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）15ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）242ページより。</ref>。 2014年、開業100年を記念して、音威子府村民有志による「天北線上音威子府機関支区鉄道倶楽部（現：nociw＊）」が駅の修復に乗り出した。村の「住民提案型まちづくり補助金制度」による助成を受け、業者によってプラットホームの修復と、国鉄仕様を模したレプリカの駅名標設置が行われた。2015年8月4日、駅名標の除幕式を行った<ref>[https://response.jp/article/2015/07/28/256635.html
