[[雄琴温泉駅|雄琴駅]]間の開通に合わせて使用が開始された{{Sfn|江若鉄道の思い出|p=34}}。ただしこの時駅は完成しておらず、仮停車場としての開業だった{{Sfn|江若鉄道の思い出|p=34}}。駅としての正式な開業日は同年8月1日である{{Sfn|江若鉄道の思い出|p=34}}。 駅のあった[[坂本 (大津市)|坂本]]地区は[[比叡山]]麓に位置し[[延暦寺]]や[[日吉大社]]を抱える観光地であったことから、当駅開業に前後して他の輸送機関も当地区に進出している{{Sfn|江若鉄道の思い出|p=34}}。1922年（大正11年）には比叡辻に新唐崎港が開港し汽船が乗り入れ{{Sfn|江若鉄道の思い出|p=34}}、1927年（昭和2年）には[[琵琶湖鉄道汽船]]（のちに合併し[[京阪電気鉄道]]）が坂本駅（現在の[[坂本比叡山口駅]]）を開設{{Sfn|江若鉄道の思い出|p=35}}{{Sfn|新修大津市史第7巻|p=395}}。同年には[[比叡山鉄道]]が坂本と比叡山上を結ぶ[[比叡山鉄道線|ケーブルカー路線]]を開通させている{{Sfn|江若鉄道の思い出|p=35}}{{Sfn|新修大津市史第7巻|p=395}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1923年]]（大正12年） ** [[4月1日]] - 叡山から雄琴までの延伸開業と同時に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2955325/9}} 「地方鉄道運輸開始」『官報』1923年4月6日]（国立国会図書館デジタル化資料）</ref>。仮停車場として使用を開始する{{Sfn|江若鉄道の思い出|p=34}}。 ** [[8月1日]] - 正式に駅として開業する{{Sfn|江若鉄道の思い出|p=34}}{{Sfn|寺田|2010|p=10}}。当時の所在地は[[滋賀郡]][[坂本村 (滋賀県)|坂本村]]坂本 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=日吉駅線路配置図|left=[[叡山駅 (江若鉄道)|叡山]]}} {{鉄道配線図|0|0|0|0|0|0|BV|0|0}} {{鉄道配線図|0|0|0|quai|quai|quai|0|0|0}} {{鉄道配線図|voie|voie|voie|voie|voie|voie|voie|voie|voie}} {{鉄道配線図|0|0|courbehg|voie|voie|voie|courbehd|0|0}} {{鉄道配線図|0|0|0|quai|quai|quai|0|0|0}} {{駅配線図終了|right=[[雄琴温泉駅|雄琴温泉]]|source={{Sfn|竹内|1967}}|note=本線に接続していない線路は撤去済みを示す}} 日吉駅は旅客と貨物の両方を扱うことができた[[一般駅]]{{Sfn|寺田|2010|p=10}}。ただし[[プラットホーム|ホーム]]は1面のみ、線路も[[単線]]の[[棒線駅]]であり、規模の小さい駅だった{{Sfn|江若鉄道の思い出|p=34}}{{Sfn|レイル|p=79}}。かつては[[列車交換]]が可能だったが、線路が撤去された結果列車交換ができない棒線駅となっている{{Sfn|寺田|2010|p=10}}{{Sfn|レイル|p=81}}。 駅にはその規模に合わず大きな木造の[[駅舎]]があり{{Sfn|レイル|p=81}}{{Sfn|寺田|2010|p=15}}、委託駅員が配置されていた{{Sfn|レイル|p=80}}。 == 利用状況 == 開業から数年間の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1923年||40,545人||35,472人||-||159トン||<ref>[{{NDLDC|973989/387}} 『滋賀県統計全書』大正12年版]（国立国会図書館デジタル化資料）</ref> |- |1924年||82,647人||61,903人||36トン||47トン||<ref>[{{NDLDC|1909568/388}} 『滋賀県統計全書』大正13年版]（国立国会図書館デジタル化資料）</ref> |- |1931年||49,506人||48,147人||1,109トン||1,227トン||<ref>[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料）</ref> |- |1934年||46,937人||47,720人||357トン||341トン||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||51,871人||51,685人||209トン||287トン||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||50,951人||50,202人||184トン||221トン||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅前の道を西に行くと[[京阪石山坂本線]]の終点、[[坂本比叡山口駅]]に行き着く{{Sfn|レイル|p=80}}。駅があった場所は江若鉄道廃線後に開業した[[湖西線]]の[[比叡山坂本駅]]に重なる{{Sfn|江若鉄道の思い出|p=35}}。 [[滋賀駅]]付近から当駅までの江若鉄道の線路跡は湖西線に利用されている{{Sfn|吉田|1998|p=186}}。いっぽう当駅から北へ伸びていた線路の跡は、山側を抜ける湖西線とは違い[[琵琶湖]]側に向かっているために道路として残されていて、{{読み仮名|苗鹿|のうか}}3丁目交差点付近で[[国道161号]]に出合う{{Sfn|レイル|p=80}}{{Sfn|吉田|1998|p=186}}{{Sfn|寺田|2010|p=24}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[叡山駅 (江若鉄道)|叡山駅]] -
