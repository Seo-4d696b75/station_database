'''三河海老駅'''（みかわえびえき）は、かつて[[愛知県]][[新城市]]海老字千原田に存在した[[豊橋鉄道]][[豊橋鉄道田口線|田口線]]の[[鉄道駅|駅]]である。 [[1929年]]（[[昭和]]4年）に田口鉄道により開業。[[1956年]]（昭和31年）に豊橋鉄道の駅となるが、[[1968年]]（昭和43年）に路線の廃線に伴って[[廃駅|廃止]]された。[[南設楽郡]][[海老町]]（廃止時は[[鳳来町]]、現・新城市）にあった駅の一つである。 == 歴史 == 当駅のあった海老は、[[豊橋市|豊橋]]と[[長野県]]の[[飯田市|飯田]]を結ぶ「伊那街道」の[[宿場|宿場町]]として発展した地域である。[[明治]]に入ってもなお街道の賑わいとともにあり、次第に人口も増加して[[1894年]]（明治27年）には町制をしいて海老町となった<ref>[[#kadokawa|『角川日本地名大事典』23]]、pp169,246-247</ref><ref name="horai-p814">[[#horai|『鳳来町誌』歴史編]]、pp814-816</ref>。 海老町に鉄道が伸びるのには時間を要した。<br />[[1894年]]に、官営鉄道が通じていた[[豊橋駅]]を起点とする東参鉄道が海老村（当時）までの路線敷設を申請したが、先願の[[豊川鉄道]]に敷設権を取られ、同じ南設楽郡内の[[大海駅]]までの[[1900年]]（明治33年）の延伸に止まった。現在のJR[[飯田線]]南部にあたる区間である。<br />[[大正]]に入ってからは[[1923年]]（大正12年）に、豊川鉄道の姉妹会社[[鳳来寺鉄道]]が、大海駅から[[宇連川]]に沿って路線を伸ばした。海老町はこれらの鉄道網からは外れていたが、豊川鉄道大海駅延伸に伴い駅と町の間で[[乗合馬車]]が営業を開始し<ref>『鳳来町誌』歴史編、p817</ref>、[[1919年]]（大正8年）には大海から海老を経て田口（現・[[設楽町]]田口）へ至る[[路線バス]]が開通する<ref>[[#horai-ko|『鳳来町誌』交通史編]]、p242</ref>など、交通機関は確保されていた。 当駅を開設した田口鉄道は、上記の豊川鉄道・鳳来寺鉄道の系列会社である。当駅は、同鉄道の第一期線にあたる鳳来寺口駅（後の[[本長篠駅]]） - 当駅間の開通に伴って、[[1929年]]（昭和4年）[[5月22日]]に開業した<ref name="chizucho">[[#chizucho|『日本鉄道旅行地図帳』7号]]、pp37-38</ref><ref>[[#horai-ta|『鳳来町誌』田口鉄道史編]]、pp58-59,64</ref>。よって、開通時は路線の[[終着駅]]であった。海老以北、第二期線の当駅 - [[清崎駅]]間は[[1930年]]（昭和5年）に開通、その後第三期線が[[1932年]]（昭和7年）に田口の[[三河田口駅]]まで開通して、鳳来寺口駅 - 三河田口駅間が全通している<ref>『鳳来町誌』田口鉄道史編、pp58-59</ref>。鉄道の開通は、町に人口の減少という影響を与えた。これは、鉄道開通により中心地としての役割が薄れて人々が都会へと出ていくことが多くなったためであった<ref name="horai-p814" />。 戦後の[[1956年]]（昭和31年）[[10月1日]]、田口鉄道は[[豊橋鉄道]]に[[合併 (企業)|合併]]された。これに伴い、当駅は豊橋鉄道田口線の駅となった<ref>『鳳来町誌』田口鉄道史編、p152</ref>。 それから8年後の[[1964年]]（昭和39年）、豊橋鉄道は[[赤字]]の拡大に伴って田口線の廃止を決定する<ref>『鳳来町誌』田口鉄道史編、pp161-162</ref>。田口線は[[1968年]]（昭和43年）の廃止までに数度[[台風]]の被害に遭い、最終的な運転区間は本長篠駅 - 当駅間となり当駅が再度終点になっていた<ref>『鳳来町誌』田口鉄道史編、p186</ref>。[[9月1日]]付で路線は[[廃線]]、バスへの転換が実施され<ref>『鳳来町誌』田口鉄道史編、pp172-173</ref>、駅もあわせて廃止された<ref name="chizucho"/>。 [[2012年]]（平成24年）現在、本長篠駅と設楽町田口を結ぶのは[[豊鉄バス]][[豊鉄バス新城営業所#田口新城線|田口新城線]]で、[[新城市民病院]]から本長篠駅前経由で田口まで運行されている<ref>「{{PDFlink|[http://www.toyotetsu.jp/routemap/img/shinshiro.pdf 豊鉄バス路線系統図 新城営業所管内]}}」（豊鉄バス公式サイト）、2015年10月22日閲覧</ref>。海老地内では、バスは[[愛知県道32号長篠東栄線]]（伊那街道）を走行しており、駅のあった字千原田の北東にあたる字南貝津に「海老」[[バス停留場|バス停]]が設置されている<ref>[http://www.mapion.co.jp/phonebook/M12001/23221/L0232079/ Mapion電話帳 海老バス停]、2012年7月20日閲覧</ref>。 == 構造 == 相対式2面2線と本長篠方折返しの片面1線で構成された<ref>{{Cite journal|和書|author = 白井良和|title = 奥三河に咲いたローカル線 田口線の回想|date = 1986-03|publisher = 鉄道図書刊行会 |journal = 鉄道ピクトリアル|number = 461|page = 59}}</ref>、列車の[[列車交換|行き違い]]が可能な[[交換駅]]であった<ref>『鳳来町誌』田口鉄道史編、p64</ref>。構内には[[車両基地|車庫]]や[[変電所]]があるなど、同線の中枢として機能していた<ref>[[#timeslip|『タイムスリップ飯田線』]]、p168</ref>。 [[日本の鉄道駅#有人駅|有人駅]]であり、1956年（昭和31年）の時点で[[駅長]]を含めて7人配置されていた<ref>[[#zuroku|『図録』]]、p8</ref>。 == 利用状況 == 1957年度の乗車人員は12万7千人（1日平均348人）で、そのうち4割にあたる5万6千人が[[定期乗車券]]での利用客であった。これは、田口線全11駅では[[本長篠駅]]・[[鳳来寺駅]]・[[三河田口駅]]に次いで4番目に多い<ref name="tokei">[[#tokei|『愛知県統計年鑑』昭和34年版]]、p386</ref>。 一方同年度における貨物取扱量は、発送が2,783トン、到着が506トンであった。田口線の貨物取扱駅は7駅あったが、取扱量はその中では少なく上から6番目である<ref name="tokei" />。 == 駅の跡地 == 駅の跡地は整地され[[駐車場]]となっているほか、本長篠側に新城市海老構造改善センターがある<ref>[[#haisen|『私鉄の廃線跡を歩く』2]]、p55</ref>。<br /> 駅舎は廃止後も残っていたが、[[1994年]]（平成6年）に解体された<ref>[[#wadachi|『消えた轍』3]]、p126</ref>。<br /> [[2016年]]3月末までは、跡地に本長篠駅前 - 海老車庫前間の系統のみ発着した「海老車庫前」という名称のバス停もあったが<ref>[[#sin-haisen|『新鉄道廃線跡を歩く』3]]、p131,133</ref><ref>[http://www.mapion.co.jp/phonebook/M12001/23221/L0232088/ Mapion電話帳 海老車庫前バス停]、2012年7月20日閲覧</ref>、前述の新城市民病院 - 田口間の系統は海老車庫前は経由しなかった。 == 隣の駅 == ; 豊橋鉄道 : 田口線 :: [[三河大石駅]] - '''三河海老駅''' - [[滝上駅]] == 脚注 == {{脚注ヘルプ}} {{reflist}} == 参考文献 == * {{Cite book|和書|author=愛知県|title=愛知県統計年鑑|volume=第8回
