'''上興部駅'''（かみおこっぺえき）は、[[北海道]]（[[網走支庁]]）[[紋別郡]][[西興部村]]字上興部にあった[[北海道旅客鉄道]]（JR北海道）[[名寄本線]]の[[鉄道駅|駅]]（[[廃駅]]）である。名寄本線の[[廃線]]に伴い[[1989年]]（平成元年）[[5月1日]]に廃駅となった。 == 歴史 == * [[1920年]]（大正9年）[[10月25日]] - [[鉄道省]]名寄線下川駅 - 当駅間延伸開通に伴い終着駅として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1921年]]（大正10年）[[10月5日]] - 当駅 - 興部間延伸開通に伴い途中駅となる。名寄機関庫上興部分庫設置。 * [[1923年]]（大正12年）[[11月5日]] - 線路名を名寄本線に改称、それに伴い同線の駅となる。 * [[1929年]]（昭和4年）[[11月14日]] - 名寄機関庫上興部分庫廃止。 * [[1935年]]（昭和10年）4月 - 北海道庁直営の上興部石灰鉱業所操業開始に伴い、専用線運用開始<ref>北海道農材工業社史「北海道の土地改良と農材十五年の歩み」昭和41年6月発行より。道庁は昭和16年に北海道農材工業の前身である北海道興農公社へ経営を委託した。昭和26年版-45年版全国専用線一覧にて 総延長1.3～1.5km。</ref>。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1978年]]（昭和53年）[[12月1日]] - 貨物取扱を専用線発着車扱貨物に限定。 * [[1982年]]（昭和57年）[[11月15日]] - 貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の廃線に伴い廃止となる。廃止当日まで、ここを終着とする名寄発の列車が運転されていた。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]複合型2面3線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）208ページより。</ref>。互いのホームは駅舎側ホーム中央部分と島式ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した。駅舎側単式ホーム（南側）が上りの1番線、[[待合所]]が設置された<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]][[12月]]発行）72ページより。</ref>島式ホーム駅舎側が下り及び当駅折り返し用の2番線、島式ホームの外側が下り貨物列車専用の副本線である3番線となっていた<ref name="bunken1983"/>。そのほか1983年（昭和58年）時点では、1番線の名寄方から分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、3番線の外側に貨物用であった[[停車場#側線|側線]]を2線、及びそこから分岐する行き止まりの側線を多数有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南側に位置し上り線ホーム中央部分に接していた<ref name="bunken1983"/>。 [[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅の所在する地名より。地名は、興部川の上流に位置したため<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）185-186ページより。</ref>、「興部」に「上」を冠する。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は87人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道239号]]（天北国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * 上興部警察官駐在所 * 北海道農材工業株式会社上興部石灰砿業所 * [[道の駅にしおこっぺ花夢]]（駅廃止後に建設） * 知的障害者更生施設 清流の里（駅廃止後に建設） * 興部川<ref name="bunken1980M"/> * 札骨川<ref name="bunken1980M"/> * [[名士バス]]「上興部」停留所 == 駅跡 == [[File:JR Kami-Okoppe sta 001.jpg|thumb|250px|上興部鉄道資料館（2014年9月）]] 旧駅構内は1994年（平成6年）5月1日から西興部村により「上興部鉄道記念館」として整備されている<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]][[11月]]発行）50ページより。</ref>。開業当時からの木造駅舎を現役当時のままに保存すると共に自由に見学出来る<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）39-40ページより。</ref>鉄道資料館とし、舎内には備品、[[蒸気機関車]]の動輪、[[軌道自転車]]、駅銘板、制服などが保存・展示され<ref name="bunken1998S"/>、専用鉄道の歴史も伝えられている<ref name="bunken2000H"/>。[[軌条|レール]]とホームも保存され（但し2面3線から2面2線になっている）、旧2番線上に横付けする形で[[国鉄キハ56系気動車#キハ27形|キハ27形]][[気動車]]キハ27 109と[[国鉄DD14形ディーゼル機関車|DD14形]][[雪かき車|除雪車]]DD14 302号機の[[ロータリー車#鉄道車両|ロータリー式除雪機構部]]が連結された状態で[[静態保存]]・展示されている<ref name="bunken1998S"/>。また[[鉄道信号機#機械式信号機|腕木式信号機]]<ref name="bunken1998S"/>、[[踏切警報機]]<ref name="bunken2000H"/>も設置されている。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）34ページより。</ref>、2011年（平成23年）時点でも同様で、ホームには本物の[[駅名標]]も保存されていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）116ページより。</ref>。 <gallery> File:JR Kami-Okoppe sta 003.jpg|駅舎とホーム跡 File:JR Kami-Okoppe sta 002.jpg|駅跡地から外方向 </gallery> == 脚注 ==
