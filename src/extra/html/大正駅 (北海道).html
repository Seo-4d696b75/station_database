'''大正駅'''（たいしょうえき）は、[[北海道]]（[[十勝支庁]]）[[帯広市]]大正本町にあった[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''イヨ'''。広尾線の[[廃線]]に伴い[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）138ページより。</ref>。互いのホームは駅舎側ホーム中央部分と島式ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側単式ホーム（西側）が上りの1番線、島式ホーム（東側）が下りの2番線となっていた<ref name="bunken1983"/>。島式ホームの外側1線は、[[停車場#側線|側線]]として残っており、そのほか1番線の帯広方から分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、島式ホーム外側の側線からさらに分岐し広尾方も接続する側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南西側に位置しホーム中央部分に接していた<ref name="bunken1983"/>。[[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 「大正駅 - 幸福駅」の[[乗車券]]を「あやしげな語呂合わせ」とも評された<ref name="bunken1983"/>「たいそう幸福」という[[乗車券#縁起物の乗車券|縁起切符]]として発売し、売れ行きも良かった<ref name="bunken1983"/>。駅舎のホーム側にはこの切符の宣伝文が書かれたパネルが掲示されていた<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]][[12月]]発行）230ページより。</ref>。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名より。 == 利用状況 == 1981年度（昭和56年度）の1日乗降客数は321人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道419号大正停車場線]] * name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）186ページより。</ref>、旧木造駅舎に似せて建設された公衆トイレが広場内に設置されている。また、0系は[[池田町 (北海道)|池田町]]の民宿・ワインの国に移設された。 == 隣の駅 == ; 日本国有鉄道 : 広尾線 :: [[愛国駅]] -
