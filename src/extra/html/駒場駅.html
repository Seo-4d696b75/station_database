国有鉄道の駅として開業。[[一般駅]]<ref name="Teisha">停車場変遷大事典 国鉄・JR編 Ⅱ JTB 1998年10月発行。</ref>。 * [[1970年]]（昭和45年）[[9月10日]] - 貨物・荷物の取り扱いを廃止（特別扱小荷物（新聞、雑誌）のみ取り扱い）<ref name="Teisha"/>。同時に無人駅化<ref name="Kushi100">JR釧路支社「鉄道百年の歩み」平成13年12月発行。</ref>。 * [[1974年]]（昭和49年）[[10月1日]] - 特別扱小荷物の取り扱いを廃止<ref name="Teisha"/>。 * [[1976年]]（昭和51年）[[9月11日]] - 駅舎を改築<ref name="Kushi100"/>。 * [[1987年]]（昭和62年）[[3月23日]] - 士幌線の全線廃止に伴い、廃駅となる<ref name="Teisha"/>。 == 周辺 == * [[北海道道337号上士幌士幌音更線]] * [[北海道道133号音更新得線]] * [[独立行政法人]][[家畜改良センター]]十勝牧場 * [[北海道音更高等学校]] * [[北海道拓殖バス]]「駒場ターミナル」停留所 == 駅跡 == 鉄道設備を撤去後、バスの待合室が設置され小公園「ききょうの里」として整備されている。[[2014年]]（平成26年）10月に駅跡を示す案内板が設置されており（[[木野駅 (北海道)|木野駅]]・[[音更駅]]・[[武儀駅]]跡と同仕様）、[[2018年]]（平成30年）7月においても同様であった。 == 脚注 == <references /> == 隣の駅 == ; 日本国有鉄道 : 士幌線 :: [[音更駅]] -
