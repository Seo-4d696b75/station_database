[[比良駅 (江若鉄道)|比良]]間に新設された駅である{{Sfn|江若鉄道の思い出|p=66}}。青柳浜水泳場の最寄り駅として、水泳客の便宜を図って開設された{{Sfn|江若鉄道の思い出|p=66}}{{Sfn|レイル|p=82}}。江若鉄道の沿線には水泳場が点在し、夏になると多くの水泳客が押し寄せた{{Sfn|江若鉄道の思い出|p=68}}{{Sfn|レイル|p=7}}。当駅のような水泳客向けの駅<ref>当駅のほかには[[近江舞子南口駅]]や[[白鬚浜駅]]がある。</ref>の設置は、江若鉄道が夏場の水泳客輸送に力を入れていたことの表れでもある{{Sfn|江若鉄道の思い出|p=68}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1949年]]（昭和24年）[[7月1日]] - 開業{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=青柳ヶ浜駅線路配置図|left=[[近江木戸駅|近江木戸]]}} {{鉄道配線図|voie|voie|voie|voie|voie}} {{鉄道配線図|0|quai|quai|quai|0}} {{駅配線図終了|right=[[比良駅 (江若鉄道)|比良]]|source={{Sfn|竹内|1967}}}} 青柳ヶ浜駅は旅客の乗降のみを扱う[[旅客駅]]{{Sfn|寺田|2010|p=10}}。水泳客のための駅だったことから、夏期のみ列車が停車する[[臨時駅]]であった{{Sfn|江若鉄道の思い出|p=66}}。[[プラットホーム|ホーム]]は線路の片側（[[近江今津駅 (江若鉄道)|近江今津]]方面に向かって右側）に1面のみの[[片面ホーム]]{{Sfn|江若鉄道の思い出|p=66}}{{Sfn|レイル|p=80}}。コンクリート製のホームに[[枕木]]を積んで作ったホームを接いでホーム長を延ばしていた{{Sfn|江若鉄道の思い出|p=66}}。 駅出入口付近には水色の小屋があり、そこで駅業務が行われていた{{Sfn|江若鉄道の思い出|p=67}}。 == 駅周辺 == 駅があった場所は[[大谷川 (大津市)|大谷川]]のやや北側に相当する{{Sfn|江若鉄道の思い出|p=66}}。江若鉄道の廃線後に開業した[[湖西線]]が江若鉄道の線路跡を利用し建設された中、当駅付近は線路跡と湖西線の位置がずれていて、湖西線の線路の東側に江若鉄道の路盤が残されているのが確認できる{{Sfn|吉田|1998|p=190}}。ただし駅の痕跡は残されていない{{Sfn|寺田|2010|p=25}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[近江木戸駅]] -
