'''清崎駅'''（きよさきえき）は、[[愛知県]][[北設楽郡]][[設楽町]]にかつて存在した[[豊橋鉄道]][[豊橋鉄道田口線|田口線]]（旧・田口鉄道）の[[鉄道駅|駅]]。北設楽郡[[田口町]]（1956年以降設楽町）にあった駅の一つである。 == 歴史 == 清崎駅は[[1930年]]（[[昭和]]5年）[[12月10日]]、田口鉄道により開設された<ref name="chizucho">[[#chizucho|『日本鉄道旅行地図帳』7号]]、pp37-38</ref><ref name="horai-ta64">[[#horai-ta|『鳳来町誌』田口鉄道史編]]、p64</ref>。 田口鉄道は、現在のJR[[飯田線]][[本長篠駅]]と設楽町を結んだ鉄道である。[[三河海老駅]]を終点としていた同鉄道の第一期線に続いて第二期線が開通したとき、この清崎駅が新たな[[終着駅]]となった。[[三河田口駅]]までの第三期線が開通し、田口鉄道が全通したのは2年後の[[1932年]]（昭和7年）である<ref>『鳳来町誌』田口鉄道史編、pp58-59</ref>。 駅の置かれた清崎は、[[豊橋市|豊橋]]と[[長野県]]の[[飯田市|飯田]]を結んだ伊那街道沿いの地域である。街道の[[宿場]]として栄えた旧萩平村（[[愛知県立田口高等学校]]周辺）を含む地域であるが、駅は南部の東田内地区に立地した<ref>[[#kadokawa|『角川日本地名大辞典』23]]、p490,1906-1907 および[[#shitara|『設楽町誌』村落誌]]、p364</ref>。 [[1956年]]（昭和31年）[[10月1日]]、田口鉄道が豊橋鉄道に[[合併 (企業)|合併]]されたため豊橋鉄道田口線の駅となる。それから8年後の[[1964年]]（昭和39年）、豊橋鉄道は[[赤字]]の拡大を理由に田口線の廃止を決定、沿線自治体などとの協議を開始した<ref>『鳳来町誌』田口鉄道史編、pp161-162</ref>。その最中の[[1965年]][[9月17日]]から翌[[9月18日|18日]]にかけて襲った[[台風]]（[[昭和40年台風第24号|台風24号]]）の被害で、清崎・三河田口間が不通、[[バス (交通機関)|バス]]代行となってしまう<ref>『鳳来町誌』田口鉄道史編、p166,186</ref>（同区間は[[1966年]]（昭和41年）[[10月1日]]より休止<ref name="chizucho"/>）。同区間も含めて田口線は[[1968年]]（昭和43年）[[9月1日]]付で全線廃止される<ref name="chizucho"/>が、その直前の[[8月29日]]に再度台風（[[昭和43年台風第10号|台風10号]]）で被災し、今度は[[田峯駅]]から清崎駅までの区間が線路破壊により不通となり、最終営業日は三河海老駅から先の区間は運転していなかった<ref>『鳳来町誌』田口鉄道史編、p186</ref>。 廃線となった田口線はバスに転換された。2012年現在、田口線があったルートに沿って走るバスは[[豊鉄バス]][[豊鉄バス新城営業所#田口新城線|田口新城線]]である。設楽町清崎には、駅名と同じ「清崎」という名の[[バス停留所|バス停]]がある<ref>[http://www.mapion.co.jp/phonebook/M12001/23561/L0260642/ Mapion電話帳 滝上バス停]、2012年9月17日閲覧</ref>。駅の跡地は[[国道257号]]清崎交差点の南方で、線路跡は国道に取り込まれている<ref>[[#new|『新鉄道廃線跡を歩く』3]]、p133</ref>。[[駅名標]]が廃止後も残っていたが、[[1997年]]（平成9年）に撤去された<ref>[[#wadachi|『消えた轍』3]]、p126</ref>。 == 構造 == 島式1面2線ホームとその両側に貨物側線各1線を持つ<ref>{{Cite journal|和書|author = 白井良和|title = 奥三河に咲いたローカル線 田口線の回想|date = 1986-03|publisher = 鉄道図書刊行会 |journal = 鉄道ピクトリアル|number = 461|page = 59}}</ref>[[日本の鉄道駅#有人駅|有人駅]]で、1956年（昭和31年）の時点で[[駅長]]を含めて4人配置されていた<ref>[[#zuroku|『図録』]]、p8</ref>。 == 利用状況 == 1957年度の乗車人員は6万1千人（1日平均168人）で、そのうち5割にあたる3万1千人が[[定期乗車券]]での利用客であった。この乗車人員は田口線全11駅の中で[[三河大草駅]]・[[長原前駅]]・[[三河大石駅]]・[[滝上駅]]に次い5番目に少ない<ref name="tokei">[[#tokei|『愛知県統計年鑑』昭和34年版]]、p386</ref>。 同年度における貨物取扱量は、発送が2,190トン、到着が1,524トンであった。田口線の貨物取扱駅は7駅あったが、取扱量はその中では少なく上から5番目である<ref name="tokei" />。 == 隣の駅 == ; 豊橋鉄道 : 田口線 :: [[長原前駅]] - '''清崎駅''' - （貨）[[大久賀多駅]] - （臨）[[鮎淵駅]] - [[三河田口駅]] == 参考文献 == *
