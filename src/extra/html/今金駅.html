'''[[日本国有鉄道|国有鉄道]]瀬棚線'''の花石駅 - 当駅間延伸開通に伴い、新設<ref name=":022">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=25}}</ref><ref name="haruka">『道南鉄道100年史 遥』 [[北海道旅客鉄道函館支社]] 2003年2月発行。</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 *[[1932年]]（昭和7年）[[11月1日]] - 当駅 - [[瀬棚駅]]間延伸開通（瀬棚線の全線開通）に伴い、中間駅となる。 *[[1983年]]（昭和58年）[[9月1日]] - 貨物の取り扱いを廃止<ref name="haruka" />。 *[[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止<ref name="haruka" />。 *[[1987年]]（昭和62年）[[3月16日]] - 瀬棚線の全線廃止に伴い、廃駅となる<ref name="haruka" />。 === 駅名の由来 === {{See also|今金町#町名の由来}} 所在地名より。のちに町名となった。地名は初期の入植者今村藤次郎と金森石郎の名の頭文字を採って命名された<ref name=":022" /><ref>{{Cite book|和書|title=駅名の起源|editor=札幌鉄道局編|year=1939|publisher=北彊民族研究会|id={{NDLJP|1029473}}|page=25}}</ref><ref>[http://www.town.imakane.lg.jp/gyousei/gyouseiinfo/gaiyou/post_2.html 今金町の概要] - 今金町、2015年5月21日閲覧。</ref>。なお町名・地名は「いま'''か'''ね」、駅名は「いま'''が'''ね」と読みに相違がある。 「いま金になる」という駅名の縁起をかついで、1983年（昭和58年）頃には月間平均100枚以上の[[入場券]]が売れたという<ref name="bunken1983" />。 == 駅構造 == 廃止時点で、[[相対式ホーム]]2面2線を有する[[地上駅]]で、[[列車交換]]が可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム西側と対向側ホーム西側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）65ページより。</ref>。線路北側の駅舎側ホームが上り1番線、対向側ホームが下り2番線となっていた。そのほか対向側ホームの外側に[[停車場#本線|副本線]]様の[[停車場#側線|側線]]を1線有し（乗降不可）、この側線から国縫方に短い側線を1線分岐した。また、本線国縫方の1、2番線分岐部分手前から旧貨物側線を分岐した<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の北側（瀬棚方面に向かって右側）に位置し、ホーム中央部に接していた<ref name="bunken1983"/>。開業当初からの駅舎は改築され、鉄筋コンクリート平屋建ての、待合室や事務室が広くゆったりと取られ、売店も有した建物となっていた<ref name="bunken1983"/>。ホームは他の瀬棚線内の駅に比し、比較的長い有効長であった<ref name="bunken1983"/>。 == 利用状況 == * 1981年度の1日乗降客数は610人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道230号]]（国縫道路） * [[北海道道686号鈴岡今金停車場線]] *[[せたな警察署]]今金交番 * 今金町役場 * 今金郵便局 * [[北海道銀行]]今金支店 * [[渡島信用金庫]]今金支店 * 今金町国保病院 * 八雲保健所今金支所 * [[後志利別川]] *[[函館バス]]「今金」停留所 == 駅跡 == 2000年（平成12年）時点で駅舎跡に[[函館バス]]の待合所が建築され、旧構内に「[[デ・モーレン・いまかね]]」という名の風車型の建物が建っている。また、レールと今金町内に所在した6駅の[[駅名標]]（レプリカ<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）76-77ページより。</ref>）が設置されている。近辺の線路跡は、「オランダ通り」という遊歩道になっている<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）76-78ページより。</ref>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）163ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。 == その他 == 当駅を発着駅とする区間列車が下り1本、上り3本設定されていた（1987年（昭和62年）廃止時点<ref name="bunken2010HE">書籍『廃線終着駅を訪ねる 国鉄・JR編』（著：[[三宅俊彦]]、JTBパブリッシング、2010年4月発行）11ページより。</ref>）。 == 隣の駅 == ; 日本国有鉄道 : 瀬棚線 :: [[種川駅]] -　'''今金駅''' -
