開業。当時は貨物のみの取り扱い<ref name="Takayama"/>。 * [[1920年]]（大正9年）[[5月10日]] - 旅客扱いを開始する<ref name="Takayama"/>。 * [[1923年]]（大正12年）[[5月21日]] - 当駅から（貨）稲築駅までの貨物支線が開通<ref name="Takayama"/>。 * [[1966年]]（[[昭和]]41年）[[3月10日]] - 当駅 - 嘉穂信号場（ - 下山田駅）間が開通。 * [[1968年]]（昭和43年）[[11月1日]] - 当駅-（貨）稲築間の貨物線を廃止<ref name="Takayama"/>。 * [[1986年]]（昭和61年）[[4月1日]] - 漆生線の全線廃止に伴い、廃駅となる<ref name="Takayama"/>。 == 廃止時の駅構造 == <!--駅構造ご存知の方加筆願います--> * 島式ホーム1面2線と複数の側線を有していたが、末期には石炭輸送も行われていなかったため放置されていた。 * 旅客用ホームのうち、駅舎側の1線は下鴨生方面にはつながっていなかった。 * [[委託駅]]で、木造の[[駅舎]]があった。 * 駅舎から[[プラットホーム|ホーム]]へは、通路と構内踏切で連絡していた。 == 駅周辺 == * 旧三井鉱山漆生炭鉱 * [[国道211号]] == 隣の駅 == ; 日本国有鉄道 : 漆生線 :: [[鴨生駅]] -
