[[日本の鉄道駅#業務委託駅|業務委託]]化<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。 * [[1975年]]（昭和50年）[[4月1日]] - 笹津線の廃線に伴い廃止となる<ref name="bunken2008M-06"/><ref name="bunkenKW07"/><ref name="bunken2008SH-3-R"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/>。ホームは線路の西側（地鉄笹津方面に向かって右手側）に存在した<ref name="bunkenRML107"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた<ref name="bunkenRML107"/><ref name="bunkenKW07"/>。駅舎は構内の西側に位置し、ホーム中央部分に接していた<ref name="bunkenToyama2006H">書籍『鉄道の記憶』（編著：草卓人、[[桂書房]]、[[2006年]][[2月]]発行）301ページより。</ref>。[[屋根#形状による分類|片流れ]]屋根の建物であった<ref name="bunkenToyama2006H"/>。また列車接近時には[[駅自動放送|テープによる案内放送]]がされていた<ref name="bunkenRML107"/>。 == 駅周辺 == 田園風景を抜け、大沢野町の家並みの中に位置した<ref name="bunkenRML107"/>。 * [[国道41号]]（越中東街道） * 旧飛騨街道 * 教順寺<ref name="bunkenKW07"/> == 駅跡 == 1998年（平成10年）時点では、駅時代の名残である[[自転車]]置場と広場となっていた<ref name="bunken1998H">書籍『鉄道廃線跡を歩くV』（[[JTBパブリッシング]]、[[1998年]][[6月]]発行）81ページより。</ref>。2008年（平成20年）時点でも同様に道路脇の空地となっていた<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、[[2008年]][[7月]]発行）80-81ページより。</ref>。 また、当駅跡附近の線路跡は、1998年（平成10年）時点では当線廃線後に南富山駅 - 日本繊維前駅の間を東西に走る形で開通した国道359号の取付け道路附近から当駅跡南側付近までは2車線の舗装道路となっていた<ref name="bunken1998H"/>。2008年（平成20年）時点でも同様であった<ref name="bunkenToyama2008H"/>。当駅跡南側から先の線路跡は1998年（平成10年）時点では幅の狭い生活道路になっており<ref name="bunken1998H"/>、さらにその先の大沢野町八木山駅方は[[盛土|築堤]]が残存し、一部には[[バラスト軌道|バラスト]]も残存していた<ref name="bunken1998H"/>。 == 隣の駅 == ; 富山地方鉄道 : 笹津線 :: [[大沢野北口駅]] -
