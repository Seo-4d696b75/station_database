'''朱鞠内駅'''（しゅまりないえき）は、[[北海道]]（[[空知支庁]]）[[雨竜郡]][[幌加内町]]字朱鞠内にあった[[北海道旅客鉄道]]（JR北海道）[[深名線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''マリ'''。深名線の[[廃線]]に伴い[[1995年]]（平成7年）[[9月4日]]に廃駅となった。 == 歴史 == * [[雨竜第一ダム]]建設に伴う水没地域の立木伐採運搬のため、[[王子製紙 (初代)|王子製紙]]<ref group="注">ダム建設を行った雨竜電力㈱は王子製紙の傍系企業。</ref>より依頼を受けた川上林業株式会社がブトカマベツ川<ref group="注">母子里から現在の朱鞠内湖の中央を通って雨竜第一ダムの出口へ流れ、三股で[[雨竜川]]へ合流していた。</ref>沿いに軽便軌道を敷設、ガソリン機関車にて当駅裏土場へ林材輸送<ref name="shinchoshi">新幌加内町史 2008年3月発行、P211。</ref><ref 廃止時点で、[[相対式ホーム]]2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは駅舎側ホーム中央部分と対向側ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）76ページより。</ref>。駅舎側ホーム（西側）、対向側ホーム（東側）共に上下線共用となっていた。対向側ホームの外側1線が[[停車場#側線|側線]]となっており、そのほか駅舎側本線の名寄方から分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1993"/>。1983年（昭和58年）時点では構内外側にさらに3線の側線を有し<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）206ページより。</ref>、最も外側の側線からは名寄方に分岐して[[転車台]]を有し、その先で更に3線に分岐し車庫に至る行き止まりの側線を有した<ref name="bunken1983"/>。その後1993年（平成5年）までには撤去された<ref name="bunken1993"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の西側に位置しホーム中央部分に接していた<ref name="bunken1993"/>。駅舎は1964年（昭和39年）の大火の後に建て替えられた耐火ブロック造の平屋の建物であった<ref name="choshi">幌加内町史 1971年9月発行、P723。</ref>。1983年（昭和58年）時点では駅舎内に[[キク]]の[[造花]]や木の根の大株が飾られていた<ref name="bunken1983"/>。また、[[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。焼失した旧駅舎は一部二階建ての木造で、面積は旧駅舎の方が1.5倍程広かった<ref name="choshi"/>。 また、旧駅舎も開設当初は北母子里駅より若干大きい程度の平屋であったが、1956年（昭和31年）まで（時期不詳）に増改築されている<ref>幌加内村史 1958年11月発行、P525。</ref>。[[深名線#名羽線|名羽線]]が開業すれば、当駅は[[乗換駅]]になる予定だった。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は48人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は8人<ref name="bunken1993"/>。 == 駅周辺 == 当駅ができる以前は湖畔駅近くの三股が朱鞠内の中心市街であったが、当駅がかなり離れた位置に設置されたため、駅の建設と共に多くが三股から移転して新市街が形成された。また駅構内の南北両端に接する敷地にはそれぞれ木工所が開設されて、特に北側の三津橋木工所は専用線が廃止された後も長く操業を続けていた。 駅前広場には「歓迎 道立朱鞠内湖公園」と記載された鉄製の[[アーチ]]が設置されていた<ref name="bunken1983"/>。 * [[国道275号]]（美深国道） * [[北海道道528号蕗の台朱鞠内停車場線]] * 朱鞠内小学校 * [[朱鞠内湖]]（[[雨竜第一ダム]]） * 朱鞠内郵便局 * [[雨竜川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> * 朱鞠内川<ref name="bunken1980M"/>。 * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「朱鞠内」バス停 == その他 == * 北海道内最大の[[人造湖]]である[[朱鞠内湖]]を擁す雨竜第一・第二ダム建設の際には、当線の延伸工事も重なったため当駅は大変に活況を呈し<ref name="bunken1993"/>、水没地域の伐採林材や建設資材の運搬のために臨時貨物列車が頻繁に増発された<ref name="shinchoshi"/>。ダム建設完了とともに1949年（昭和24年）には[[宇津内駅]]での林材貨物積込も無くなって潮が引くように貨物便数も減り<ref name="shinchoshi"/>、翌1950年（昭和25年）には機関支区が廃止となった<ref name="shinchoshi"/>。 * 当駅はもともと[[夜間滞泊]]が設定されていたが、廃止直前には設定はなく、[[深川駅]]まで戻って[[旭川運転所]]まで回送していた。 == 駅跡 == 廃駅後しばらくは駅舎がバス[[待合所]]として再利用され、相対式ホームも残っていた。2000年（平成12年）初頭時点では駅舎は現役当時のままと言える状態であったが<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）35-36ページより。</ref>、同年夏に跡地の再利用として全て解体され、跡地は「朱鞠内コミュニティー公園」となった。駅舎跡地にはバス待合所が新築され<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）42-43ページより。</ref>、ロータリー内に[[駅名標]]と[[軌条|レール]]の一部が移設されている。 また、2000年（平成12年）時点では駅跡の湖畔方にあった最初の[[踏切]]附近から左側に分岐する[[未成線]]の[[深名線#名羽線|名羽線]]の道床が残存しており<ref name="bunken2000H"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[共栄駅]] - '''朱鞠内駅''' - [[湖畔駅]] == 脚注・出典 == {{脚注ヘルプ}} === 脚注 === {{Reflist|group="注"}} === 出典 ===
