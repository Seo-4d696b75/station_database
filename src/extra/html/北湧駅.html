[[日本国有鉄道]]名寄本線の'''厚生病院前仮乗降場'''が開業。 * [[1966年]]（昭和41年）[[10月1日]] - '''北湧仮乗降場'''（[[鉄道管理局|局]]設定）として開業。同時に厚生病院前仮乗降場廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。同時に駅に昇格。'''北湧駅'''となる。 * [[1989年]]（[[平成]]元年）[[5月1日]] - 名寄本線の廃線に伴い廃止となる。 == 駅構造 == {{節スタブ}} 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。[[仮乗降場]]に出自を持つ[[無人駅]]となっていた。 厚生病院前仮乗降場は中湧別駅と当駅との間に位置した。 == 駅名の由来 == かつて当駅附近に6年制の学校である「北湧校」が存在したことに由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=188}}</ref>。 == 駅周辺 == * [[国道242号]]（置戸国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref> * [[湧別川]]<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「出合橋」停留所 == 駅跡 == 駅跡地には[[かみゆうべつチューリップ公園]]が開設されている<ref name="bunken2001H">{{Cite book|和書|editor=[[宮脇俊三]] |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |pages=36-37}}</ref><ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=[[北海道新聞社]] |isbn=978-4894536128 |page=128}}</ref>。また、「ふるさと館JRY（屯田歴史博物館）」西隣の道路にもなっている。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[中湧別駅]] -
