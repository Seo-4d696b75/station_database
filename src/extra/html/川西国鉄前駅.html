[[川西能勢口駅|川西能勢口]]間廃線に伴い、[[1981年]]（[[昭和]]56年）に廃止された<ref name="sone14"/>。 == 歴史 == [[鉄道院|院線]]池田駅（後の[[川西池田駅]]）と貨物の連絡を図るために設置された駅であるが、当駅が存在していた時期の大半において川西池田駅の駅舎は現在より西にあり、徒歩連絡が容易になったのは川西池田駅が移転された[[1980年]]（昭和55年）から1年余りの間に過ぎない。 当初「池田駅前」として開業したが、[[1951年]]（昭和26年）に[[日本国有鉄道|国鉄]]池田駅が「川西池田駅」へ改称されたため、後の[[1965年]]（昭和40年）に当駅名を「川西国鉄前」に改称している。 廃止後、駅跡は駅前再開発により整地され、往時を偲べるものは残されていない。 === 年表 === * [[1917年]]（[[大正]]6年）[[9月2日]] - 妙見線が能勢口駅（現・[[川西能勢口駅]]）より延伸した際に'''池田駅前駅'''として開業<ref name="sone14"/>。 * [[1948年]]（昭和23年） - 貨物取扱を廃止<ref>柳谷政人「ぷらっと沿線紀行67　能勢電鉄　カーブにまっすぐ勝負」『朝日新聞』大阪本社版　2008年9月27日付夕刊、第1面および第3面。</ref>。 * [[1965年]]（昭和40年）[[4月1日]] - '''川西国鉄前駅'''に改称<ref name="sone14"/>。 * [[1981年]]（昭和56年）[[12月20日]] - 当駅 - 川西能勢口間廃線に伴い廃止<ref name="sone14"/>。 == 駅構造 == [[単式ホーム]]1面1線のみを持つ[[地上駅]]。当初は木造のホーム上屋があった<ref>複数の写真で確認できる（『朝日新聞』「ぷらっと沿線紀行67」、『のせでん 昔ギャラリー』2頁など）</ref>が、廃止直前には[[駅名標]]とホームがあるだけであった。 貨物の取り扱いがあった頃には貨物線が国鉄池田駅（後の川西池田駅）構内まで延びていたが、貨物の取扱いのなくなった戦後は放置されていた<ref>能勢電鉄株式会社編　『能勢電鉄　80年史』、1991年、210頁。</ref>。この貨物線は[[国道176号]]を跨いでおり、阪急電鉄から貸与または譲渡された車両を搬出入する際に国道176号上の踏切が活用されたこともある<ref>能勢電鉄株式会社・鉄道事業部編　『のせでん 昔ギャラリー』、2009年、32頁および47頁の画像を参照</ref>。 == 隣の駅 == ; 能勢電鉄 : 妙見線 :: '''川西国鉄前駅''' -
