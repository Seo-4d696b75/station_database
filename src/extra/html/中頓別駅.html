[[鉄道省|鉄道院]]宗谷線の小頓別駅 - 当駅間延伸開通に伴い、開業<ref name="sone">{{Cite book|和書|author=[[曽根悟]]（監修）|title=週刊 歴史でめぐる鉄道全路線 国鉄・JR|editor=朝日新聞出版分冊百科編集部（編集）|publisher=[[朝日新聞出版]]|series=週刊朝日百科|volume=20号・宗谷本線／留萌本線|pages=14-17|date=2009-11-2}}</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1918年]]（大正7年）[[8月25日]] - 宗谷線の当駅 - 浜頓別駅間延伸開通に伴い、中間駅となる<ref name="sone"/>。 * [[1919年]]（大正8年）[[10月20日]] - 路線名を宗谷本線に改称、それに伴い同線の駅となる<ref name="sone"/>。 * [[1930年]]（昭和5年）[[4月1日]] - 音威子府駅 - 稚内駅間を宗谷本線から削除し路線名を北見線に改称、それに伴い同線の駅となる<ref name="sone"/>。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1961年]]（昭和36年）4月1日 - 路線名を天北線に改称、それに伴い同線の駅となる<ref name="sone"/>。 * [[1975年]]（昭和50年）12月 - 駅舎改築。 * [[1984年]]（昭和59年）[[2月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（[[昭和]]62年）4月1日 - [[国鉄分割民営化]]により、北海道旅客鉄道（JR北海道）の駅となる。 * [[1989年]]（[[平成]]元年）[[5月1日]] - 天北線の全線廃止に伴い、廃駅となる<ref name="sone"/>。 * [[1990年]]（平成2年）11月 - 駅跡地に、天北線メモリアルパークが完成<ref name="hokkaido-np-1990-11-30">“公園全面完成 駅周辺が一変 中頓別”. [[北海道新聞]] (北海道新聞社). (1990年11月30日)</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]2面2線を有する[[地上駅]]であった。ホームが千鳥式に配置された、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）188ページより。</ref>。互いのホームは、駅舎側ホーム北側と対向ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側ホーム（西側）が下り線、対向ホーム（東側）が上り線となっていた<ref name="bunken1983"/>。貨物用の[[停車場#側線|側線]]などは撤去されていたが、下り線の浜頓別方から駅舎側に分岐た元貨物側線の留置線が、保線車両（[[モーターカー]]）用に残されていた。[[1983年]]（昭和58年）時点では駅舎の対向側の単式ホームは島式ホームであり（両側とも乗降可能）<ref name="bunken1983"/>、上り線から分岐しその島式ホーム外側への側線を1線と、そこから南稚内方に分岐する行き止まりの側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の西側に位置し両ホームとは通路及び構内踏切で連絡した。1975年（昭和50年）に中頓別町が国鉄利用債を購入して鉄筋コンクリート製に改築した駅舎で<ref name="bunken1983"/>、駅舎に当時の[[ローカル線]]では珍しい[[水洗式便所|水洗式のトイレ]]を有していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名はアイヌ語の「トウ・ウン・ベツ」（沼から出る川）に由来し、川の中流域にあることから「中」を冠している<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=84 }}</ref>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は263人<ref name="bunken1983"/>。 {| class="wikitable" style="margin: 1em 0.2em;
