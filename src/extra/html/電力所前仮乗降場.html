士幌線の仮乗降場として、黒石平駅と糠平駅の間に開業。上り列車のみ停車。同時に、[[黒石平駅]]は下り列車のみの営業となる。 * [[1987年]]（昭和62年）[[3月23日]] - 士幌線の全線廃止に伴い、営業を停止、廃止となる。 == 周辺 == * [[国道273号]] * [[電源開発]]糠平発電所 * [[十勝バス]]「黒石平」停留所 == 隣の駅 == ; 日本国有鉄道 : 士幌線 :: [[黒石平駅]] -
