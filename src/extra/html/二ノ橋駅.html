[[運輸省]]名寄本線の'''二ノ橋仮乗降場'''（[[鉄道管理局|局]]設定）として開業<ref>『下川町史』では1948年とある。</ref> 。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1950年]]（昭和25年）[[1月15日]] - 駅に昇格、'''二ノ橋駅'''となる<ref>[{{NDLDC|2963439/4}} 「日本国有鉄道公示第2号」『官報』1950年1月9日]（国立国会図書館デジタルコレクション）</ref>。一般駅。 * [[1962年]]（昭和37年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1978年]]（昭和53年）[[12月1日]] - 貨物および荷物の取扱い廃止。同時に無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化<ref>[[日本国有鉄道]][[公示]]S53.11.28公157</ref> 。 * [[1986年]]（昭和61年）[[11月1日]] - 簡易委託廃止、完全無人化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）208ページより。</ref>。そのほか、遠軽方から南側に分岐した行き止まりの保線用[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。 [[無人駅]]となっており、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、[[中名寄駅]]や[[渚滑線]][[中渚滑駅]]と同型の[[プレハブ工法|プレハブ]]駅舎となっていた。駅舎は構内の北側に位置し、ホームに接していた<ref name="bunken1983"/>。ホームの長さは50mであった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在する地区名より<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）185ページより。</ref>。地区名は、[[1903年]]（明治36年）に開削された仮定県道（現在の[[国道239号]]）が[[名寄川]]を渡る上流から2番目の橋の名前に由来する。 == 駅周辺 == 畑になっている。 * [[国道239号]]（下川国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * [[名寄川]]<ref name="bunken1980M"/> * [[名士バス]]「二の橋」停留所 == 駅跡 == 2000年（平成12年）時点では跡形も無く、道床跡を利用して用水路の工事が進められていた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）39ページより。</ref>。2010年（平成22年）時点でも跡形は無く<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）33ページより。</ref>、2011年（平成23年）時点でも同様で、農耕地に取り込まれ遺構は何も無かった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）114ページより。</ref>。畑になっている。 また、2000年（平成12年）時点では駅跡附近の国道の「二の橋跨線橋」から線路跡が望め<ref name="bunken2000H"/>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[下川駅]] -
