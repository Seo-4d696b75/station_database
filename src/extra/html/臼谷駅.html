[[運輸省]]羽幌線の'''臼谷[[仮乗降場]]'''（[[鉄道管理局|局]]設定）として開業。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1950年]]（昭和25年）[[1月15日]] - 駅に昇格、'''臼谷駅'''となる<ref name="kanpo1950">[{{NDLDC|2963439/4}} 「日本国有鉄道公示第2号」『官報』1950年1月9日]（国立国会図書館デジタルコレクション）</ref>。旅客・[[チッキ|荷物]]を取扱い<ref name="kanpo1950"/>。 * [[1964年]]（昭和39年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1972年]]（昭和47年）[[2月8日]] - [[チッキ|荷物]]取扱い廃止。同時に無人化。 * [[1987年]]（昭和62年）[[3月30日]] - 羽幌線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の東側（幌延方面に向かって右手側）に存在した。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の東側に位置し、ホームに接していた。海水浴シーズンには駅員が派遣された<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）199ページより。</ref>。 == 駅名の由来 == 当駅が所在する地名より。地名は、[[アイヌ語]]の「ウシ・ヤ→ウシヤ」（入江・岸）に由来する。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は約60人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道232号]]（天売国道/[[日本海オロロンライン]]） * 臼谷海水浴場 -
