明知鉄道に転換<ref name="sone26"/>。同時に'''明智駅'''に改称<ref name="sone26"/>。 == 駅構造 == [[駅舎]]に接して1面1線の[[単式ホーム]]がある[[地上駅]]。構内に[[車両基地]]があり、車両の[[夜間滞泊]]もある。また、多くの[[停車場#側線|側線]]を有する。 駅舎の隣には2012年、喫茶店や地元産品の販売コーナー、トイレを備えた交流施設「明智駅前プラザ」が開設された<ref>出典 : [http://www.city.ena.lg.jp/news/201203/20120326_4/ 恵那市ホームページ]</ref>。 == 駅周辺 == *[[日本大正村]]<ref name="sone26"/> *明智地域振興事務所（旧恵那郡明智町役場） *恵那市立明智小学校 *[[恵那市立明智中学校]] *[[岐阜県立恵那南高等学校]]明智校舎 *[[国道363号]] === バス === * [[東濃鉄道]]バス **明智線（[[瑞浪駅]]方面） 概ね1時間に1便運行されている。 * [[恵那市自主運行バス]] == 乗客・貨物の推移 == {| class="wikitable" border="1" cellspacing="0" cellpadding="2" style="font-size:90%; text-align:center; width:60%;" |- !年度 !乗車人員 !降車人員 !貨物発送トン !貨物到着トン |- |1934||36,106||34,777||901||3,073 |- |1935||44,217||40,813||988||4,048 |- ||||||||| |- |1955||155,905||141,862||7,486||28,950 |- |1956||173,909||154,763||12,110||3,486 |- |1957||189,203||173,994||12,413||26,016 |- |1958||193,585||167,311||10,164||21,930 |- |1959||195,331||182,768||13,212||26,126 |- |1960||196,065||183,253||13,561||27,268 |- |1961||190,389||180,309||13,581||17,795 |- |1962||195,140||185,394||12,899||16,561 |- |1963||229,337||216,337||16,863||15,914 |- |} 出典 -
