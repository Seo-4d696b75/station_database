和歌山港駅間の和歌山港線は、和歌山県が[[鉄道事業者|第三種鉄道事業]]（保有）を行っている区間で、南海電気鉄道は第二種鉄道事業者（運送）となる。この境界は当駅廃止後もそのままとなっているため、当駅廃止後は'''県社分界点'''（けんしゃぶんかいてん）となった<ref name = "乗りものニュース2" /><ref name = "乗りものニュース3" />。 == 歴史 == * [[1956年]]（昭和31年）[[5月6日]] - 和歌山港線とともに開業<ref name = "乗りものニュース2">{{Cite web | url = https://trafficnews.jp/post/79744/2 | title = 意外? 日本一短い鉄道は「和歌山県」だった 南海和歌山港線の不思議 2ページ | author = 草町義和 | date = 2018-03-05 | publisher = 乗りものニュース | accessdate = 2018-03-07}}</ref>。 * [[2005年]]（平成17年）[[11月27日]] -
