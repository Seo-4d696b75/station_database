41</ref>、この地方の[[特産品]]である[[九谷焼]]の積み出しの他、近辺の湯谷、徳山、和気に点在していた[[瓦]]工場の瓦と燃料の[[石炭]]の入出荷、鍋谷にあった服部鉱山からの陶石の積み出しにも利用され、構内には[[重油]]タンクも有った<ref name="nomiden">[https://www.city.nomi.ishikawa.jp/www/contents/1001000001291/simple/nomiden.pdf 能美市『能美電ものがたり 運行時の各駅と現在』]（PDF）</ref>。晩年は[[列車交換]]機能（島式ホーム北側の線路）、貨物側線も無くなり[[プラットホーム#単式ホーム|棒線]]化された<ref name="rm"/>。なお、工事施行申請時は駅名を「石子湯谷」としていたが、開業3日前の届出で「湯谷石子」と改めていた<ref name="rm"/>。 == 歴史 == * [[1925年]]（大正14年）[[3月21日]] - 能美電気鉄道の駅として開業。 * [[1926年]]（大正15年）[[10月2日]]（届出）- [[列車交換]]機能を設置<ref name="rm"/>。 * [[1939年]]（昭和14年）[[8月1日]] - 金沢電気軌道の駅となる。 * [[1968年]]（昭和43年）[[9月1日]]～[[9月6日]] - [[閉塞 (鉄道)|閉塞]]の取り扱いと列車交換機能及び貨物側線を廃止、[[プラットホーム#単式ホーム|単式ホーム]]1面1線となる<ref name="rm"/>。 * [[1980年]]（昭和55年）[[9月14日]] - 能美線廃止に伴い廃駅。 == 隣の駅 == ; 北陸鉄道 : 能美線 :: [[徳久駅]] - '''湯谷石子駅''' - [[加賀佐野駅]] == 現状 == 鉄道路線廃止後は[[公園]]として再整備されており、駅跡を示す[[石碑]]が設置されている<ref name="nomiden"/>。 == 関連項目 == * [[日本の鉄道駅一覧]] * [[北陸鉄道]] * [[廃駅]] == 脚注 == {{脚注ヘルプ}} {{Reflist}} {{北陸鉄道能美線}} {{DEFAULTSORT:ゆのたにいしこ}} [[category:日本の鉄道駅 ゆ|のたにいしこ]] [[Category:石川県の廃駅]] [[category:北陸鉄道の廃駅]] [[Category:1925年開業の鉄道駅]] [[Category:1980年廃止の鉄道駅]] [[Category:能美市の鉄道駅|廃ゆのたにいしこ]] [[Category:能美市の歴史|ゆのたにいしこえき]] 7zn0mb3krbr9w8249qx5u0g2825lksy
