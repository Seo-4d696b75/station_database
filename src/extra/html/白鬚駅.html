|廃止年月日= [[1969年]]（昭和44年）[[11月1日]] |乗車人員 = |乗降人員 = |統計年度 = |乗換 = |備考 = }} '''白鬚駅'''（しらひげえき）は、かつて[[滋賀県]][[高島郡 (滋賀県)|高島郡]][[高島町 (滋賀県)|高島町]]大字鵜川（現在の[[高島市]]鵜川{{Sfn|江若鉄道の思い出|p=84}}）に所在した[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。全国の白鬚社の総本社である[[白鬚神社]]の南隣にあった{{Sfn|江若鉄道の思い出|p=84}}{{Sfn|寺田|2010|p=15}}。 駅は[[琵琶湖]]の西岸を走る江若鉄道の線路が最も湖岸に接近する区間にあり、湖上に建つ白鬚神社の大鳥居を脇に見ながら湖のすぐそばを列車が走り抜けるという、沿線有数の風光明媚な光景を見ることができた{{Sfn|江若鉄道の思い出|pp=86&ndash;87}}。 == 歴史 == 白鬚駅は1927年（昭和2年）、江若鉄道の[[北小松駅 (江若鉄道)|北小松駅]] - [[高島町駅 (滋賀県)|大溝駅]]間が開通したのに合わせて開業した{{Sfn|江若鉄道の思い出|p=84}}。鉄道の建設に際し、当駅前後の区間は琵琶湖に面した白鬚神社の前を通る県道（のちに[[国道161号]]となる）の、さらに湖岸寄りを埋め立てて線路を敷設{{Sfn|江若鉄道の思い出|p=85}}。そのため当駅前後の区間は、江若鉄道の中で最も琵琶湖に接近する区間となった{{Sfn|江若鉄道の思い出|p=86}}。 琵琶湖に面する白鬚神社はその立地からかつては船による参拝が一般的であったが、当駅の開業以後は鉄道を利用しての参拝に取って代わり{{Sfn|江若鉄道の思い出|pp=85&ndash;86}}、当駅は神社参拝の玄関口としての役割を担った{{Sfn|江若鉄道の思い出|p=84}}{{Sfn|ありし日の江若鉄道|p=19}}。9月に神社で行われる「なるこ参り」の際には一年で最も多くの参拝者が訪れ、駅周辺は出店などで大いに賑わった{{Sfn|江若鉄道の思い出|p=86}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1927年]]（昭和2年）[[12月15日]] - 北小松駅から大溝駅までの区間の開通に伴い、[[滋賀郡]][[小松村 (滋賀県)|小松村]]に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956758/11}} 「地方鉄道運輸開始」『官報』1927年12月24日]（国立国会図書館デジタル化資料）</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == 白鬚駅は旅客と貨物の両方を取り扱うことができた[[一般駅]]{{Sfn|寺田|2010|p=10}}。[[プラットホーム|ホーム]]は線路の片側（[[近江今津駅 (江若鉄道)|近江今津]]方面に向かって左側）に1面のみであった（[[単式ホーム]]）{{Sfn|レイル|p=80}}。当初は[[列車交換]]が可能であったが、線路が撤去された結果列車交換ができない棒線駅となっている{{Sfn|寺田|2010|p=10}}{{Sfn|レイル|p=80}}。普段は[[無人駅]]であったが、なるこ参りの際には駅員が配置され、臨時の乗降口を設けて乗客をさばいていた{{Sfn|江若鉄道の思い出|p=89}}。 駅舎があり、その入り口には[[唐破風]]の屋根が取り付けられていた{{Sfn|江若鉄道の思い出|p=84}}。白鬚神社のすぐ近くにあることから、神社建築をモチーフにしたものとされる{{Sfn|江若鉄道の思い出|p=84}}。 == 利用状況 == 初期の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == [[File:Shirahige-jinja (Takashima) otorii.JPG|thumb|琵琶湖上に建つ白鬚神社の大鳥居]] 琵琶湖に向かって山がせり出してきている地にあり、[[白鬚神社]]があるほかには江若鉄道の線路と国道が通じているのみであった{{Sfn|レイル|p=85}}。 駅のすぐ横には琵琶湖が広がり{{Sfn|レイル|p=84}}、駅北側の湖上には白鬚神社の大鳥居が造立されている{{Sfn|江若鉄道の思い出|pp=86&ndash;87}}。「近江の[[厳島神社|厳島]]」とも称される大鳥居と琵琶湖を背景に列車が湖のすぐそばを駆け抜けていく様子は、江若鉄道を代表する光景として親しまれた{{Sfn|江若鉄道の思い出|pp=86&ndash;87}}{{Sfn|レイル|p=84}}。 江若鉄道の線路跡は[[国道161号]]の拡幅用の土地や歩道に転用され、鉄道の遺構は残っていない{{Sfn|寺田|2010|p=26}}{{Sfn|吉田|1998|p=193}}。白鬚神社の前を通る国道が湖と山に挟まれた土地にもかかわらず充分広い道幅を持っているのは、線路跡が国道の拡幅に使われたためである{{Sfn|吉田|1998|p=193}}。また、江若鉄道の廃線後に開通した[[湖西線]]は白鬚神社裏手の山をトンネル（[[高島トンネル]]を参照）にて抜けることとなったため、かつてのように湖上の鳥居を眺めながら列車が駆け抜けるという光景は見られなくなっている{{Sfn|江若鉄道の思い出|p=88}}。
