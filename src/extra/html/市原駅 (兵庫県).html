播州鉄道の駅として開業。[[鉄道駅#一般駅|一般駅]]。 ** 当時の所在地表示は[[兵庫県]][[多可郡]][[日野村 (兵庫県)|日野村]]市原であった。 * [[1923年]]（大正12年）[[12月21日]] - 播丹鉄道に譲渡。 * [[1943年]]（[[昭和]]18年）[[6月1日]] - 播丹鉄道が国有化、[[鉄道省]]鍛冶屋線の駅となる。 * [[1952年]]（昭和27年）[[4月1日]] - [[西脇市]]成立に伴い、所在地表示が兵庫県西脇市市原町になる。 * [[1959年]]（昭和34年）[[8月20日]] - 貨物の取扱いを廃止（[[鉄道駅#旅客駅|旅客駅]]となる）。 * [[1973年]]（昭和48年）[[10月1日]] - [[チッキ|荷物]]の取扱いを廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR西日本が継承。 * [[1990年]]（[[平成]]2年）4月1日 - 鍛冶屋線廃線に伴い廃止、駅舎解体。 * （時期不明） - 旧駅舎を復元した鍛冶屋線市原駅資料館が竣工。 == 隣の駅 == ; 西日本旅客鉄道 : 鍛冶屋線 :: [[西脇駅]] - '''市原駅''' - [[羽安駅]] == 脚注 == {{脚注ヘルプ}} {{Reflist}} == 関連項目 == * [[日本の鉄道駅一覧]] * [[廃駅]] == 外部リンク == * [http://www.city.nishiwaki.lg.jp/kankotokusan/nishiwakikitaharimanokanko/kankou_shigen/1358384173303.html　鍛冶屋線市原駅記念館] {{鍛冶屋線}} {{Eki-stub}} {{DEFAULTSORT:いちはら}} [[Category:兵庫県の廃駅]] [[Category:西脇市の鉄道駅|廃いちはら]] [[Category:日本の鉄道駅 い|ちはら]] [[Category:西日本旅客鉄道の廃駅]] [[Category:日本国有鉄道の鉄道駅]] [[Category:播丹鉄道]] [[Category:1921年開業の鉄道駅]] [[Category:1990年廃止の鉄道駅]] [[Category:西脇市の歴史|いちはらえき]] n8ighpdj2fwzkqasahtx8lpe8iq084r
