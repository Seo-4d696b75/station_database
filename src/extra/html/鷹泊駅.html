'''鷹泊駅'''（たかどまりえき）は、[[北海道]]（[[空知支庁]]）[[深川市]]多度志町字鷹泊にあった[[北海道旅客鉄道]]（JR北海道）[[深名線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''タマ'''。深名線の[[廃線]]に伴い[[1995年]]（平成7年）[[9月4日]]に廃駅となった。 == 歴史 == * [[1926年]]（大正15年）[[11月10日]] - [[鉄道省]]雨龍線多度志駅 - 当駅間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1929年]]（昭和4年）[[11月8日]] - 当駅 - 幌加内駅間延伸開通に伴い中間駅となる。 * [[1931年]]（昭和6年）[[10月10日]] - 線路名を幌加内線に改称、それに伴い同線の駅となる。 * [[1941年]]（昭和16年）10月10日 - 線路名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1951年]]（昭和26年）6月 - 鷹泊ダム建設や河川改修、灌漑施設建設等の雨竜川綜合開発事業に伴い、石狩川治水事業所専用線を設置<ref>多度志町史 1965年9月発行、P251,668等。</ref>。 * [[1982年]]（昭和57年）[[3月29日]] - 貨物取扱い廃止。 * [[1984年]]（昭和59年） ** [[2月1日]] - [[チッキ|荷物]]取扱い廃止。 ** 11月10日 - 旅客無人化。運転要員は1986年まで引き続き配置。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1995年]]（平成7年）[[9月4日]] - 深名線の廃線に伴い廃止となる。 === 駅名の由来 === 当駅の所在する地名より。地名は、市街の少し下流の対岸にある巨岩（現在の鷹泊岩）に鷹がよくとまっていたことからとされるが<ref name=":2">{{Cite book|和書|page=35|editor=札幌鉄道局|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|author=|author3=|author2=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref><ref name="Yamada2018">{{Cite book|title=北海道の地名|date=2018-11-30|year=|publisher=草風館|author=[[山田秀三]]|edition=2|series=アイヌ語地名の研究　別巻|location=[[浦安市]]|isbn=978-4-88323-114-0|page=82}}</ref>、国鉄北海道総局発行の『駅名の起源』は1954年（昭和29年）版から[[アイヌ語]]の「チカㇷ゚オッナイ（chikap-ot-nay：鳥・多くいる）」に由来するという説を掲載しており<ref name="Yamada2018" /><ref name=":03">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|page=113}}</ref>、不明である<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/7180P.pdf|title=アイヌ語地名リスト セッ～ツキガ　P71|accessdate=2018-06-10|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の西側（名寄方面に向かって左手側）に存在した<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）74-75ページより。</ref>。そのほか本線の名寄方から分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の旧貨物ホームへの側線を1線有していた<ref name="bunken1993"/>。かつては[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型2面2線を有する[[列車交換]]可能な[[交換駅]]であり、ここを終着とする深川からの区間列車も運転されていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）204ページより。</ref>。互いのホームは駅舎側ホーム中央部分と対向ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側（西側）が下りの1番線、対向側（東側）が上りの2番線となっていた<ref name="bunken1983"/>。また島式ホームの外側1線が[[停車場#側線|側線]]として残っていた<ref name="bunken1983"/>。この側線からは名寄方に伸びる行き止まりの短い側線を1線有した<ref name="bunken1983"/>。交換設備運用廃止後は線路は側線を含め1993年（平成5年）までには撤去されたが、ホーム前後の線路は[[分岐器]]の名残で湾曲していた<ref name="bunken1993"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の西側に位置しホーム中央部分に接していた<ref name="bunken1993"/>。昔ながらの古びた木造で、入口の上には「たかどまり」と記載された大きな駅銘板が設置されていた<ref name="bunken1993"/>。有人駅時代には待合室に[[スイセン属|水仙]]と[[バラ]]の[[造花]]が飾られていた<ref name="bunken1983"/>。 駅前広場には30m近い[[トドマツ|大椴松]]が数本そびえていた。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は86人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は50人<ref name="bunken1993"/>。 == 駅周辺 == 鷹泊ダムの完成後は[[田|水田]]地帯に位置した<ref name="bunken1993"/>。 * [[国道275号]]（空知国道） * [[北海道道693号鷹泊鷹泊停車場線]] * 鷹泊郵便局 * 鷹泊神社 * リフレッシュプラザ鷹泊 - 深名線廃線後に廃校となった旧・深川市立鷹泊小学校の敷地跡に1997年（平成9年）、深川市により建築された公共施設。館内に「旧深名線資料室」が開設され、[[駅名標]]、[[閉塞 (鉄道)|閉塞器]]、[[鉄道信号機#機械式信号機|腕木式信号機]]、[[軌条|レール]]、備品、[[写真]]パネルなどが保存・展示されている<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]]11月発行）41ページより。</ref>。 * 鷹泊自然公園 - 駅から北西に約2km<ref name="bunken1983"/>。 * 鷹泊ダム - 駅から北西に約6km<ref name="bunken1983"/>。 * [[雨竜川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> * 幌加内峠 - 駅の北北東<ref name="bunken1980M"/>。 * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「鷹泊」停留所 == 駅跡 == [[File:Shinmeisen Takadomari.jpg|thumb|駅跡　(2011年8月9日)]] 2000年（平成12年）時点では、駅舎、ホーム、別棟の[[トイレ]]が残存していた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（JTBパブリッシング、[[2000年]]1月発行）34-35ページより。</ref>。駅舎は窓や扉の開口部は板で打ち付けられて塞がれていた<ref name="bunken2000H"/>。また駅前広場にあった大椴松の木はなくなっていた<ref name="bunken2000H"/>。2010年（平成22年）時点でも同様で、駅舎は倉庫として再利用されている模様であった<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）40-41ページより。</ref>。2011年（平成23年）時点でも同様であったが、トイレの劣化は激しく、傾いていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）175-176ページより。</ref>。 また、当駅と沼牛駅との間にあった幌加内峠を抜ける幌加内トンネルは、2000年（平成12年）時点では草木に阻まれ調査不能状態であったが<ref name="bunken2000H"/>、2010年（平成22年）時点では国道275号線の工事が進行中で立ち入ることができない状態であった<ref name="bunken2010H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[下幌成駅]] - '''鷹泊駅''' - [[沼牛駅]] == 脚注 == {{脚注ヘルプ}} {{reflist}} == 関連項目 == *
