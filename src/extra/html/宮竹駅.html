能美電気鉄道により開業。 * [[1931年]]（昭和6年）[[12月18日]]（届出） - [[列車交換]]設備を新設<ref name="rm"/>。 * [[1939年]]（昭和14年）[[8月1日]] - 金沢電気軌道が能美電気鉄道を吸収合併。 * [[1941年]]（昭和16年）8月1日 - 北陸合同電気（現在の北陸電力）が金沢電気軌道を合併。 * [[1942年]]（昭和17年）[[3月26日]] - 事業譲渡により北陸鉄道となる。 * [[1968年]]（昭和43年）[[9月1日]]～[[9月6日]] - [[閉塞 (鉄道)|閉塞]]の取り扱いと列車交換機能及び貨物側線を廃止、[[プラットホーム#単式ホーム|単式ホーム]]1面1線となる<ref name="rm"/>。[[無人駅]]化<ref name="rm"/>。 * [[1980年]]（昭和55年）[[9月14日]] - 能美線[[廃線|廃止]]に伴い廃駅。 == 駅周辺 == * 宮竹[[郵便局]] * 宮竹[[保育園]] * 宮竹[[小学校]] * [[北陸先端科学技術大学院大学]] * 東京プロセスサービス * 石川ハイテク交流センター * 正林寺 * 宮本酒造店 == 隣の駅 == ; 北陸鉄道 : 能美線 :: [[灯台笹駅]] -
