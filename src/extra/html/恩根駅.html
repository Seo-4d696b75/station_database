[[日本国有鉄道]]相生線の'''恩根仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1956年]]（昭和31年）[[9月20日]] - 駅に昇格。'''恩根駅'''となる。旅客のみ取扱い。 * [[1985年]]（昭和60年）[[4月1日]] - 相生線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の西側（北見相生方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）159ページより。</ref>。[[無人駅]]であった。 == 駅名の由来 == 当駅の所在する地名より。 地名は、和人入植時、この地を流れる網走川の支流、オンネキキン川の[[アイヌ語]]名である「オンネキキン（onne-''kikin''）」（大きい・キキン川）に、「恩根木檎」と当てたものが後略されたものである<ref name=":0">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/3140P.pdf|title=アイヌ語地名リスト オニシベ～キタ　P31-40P|accessdate=2018-12-02|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 なお「オンネキキン」は現在の本岐地区の名称の由来となった「ポンキキン（pon-''kikin''）」（小さい・キキン川）<ref name=":03">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/121130P.pdf|title=アイヌ語地名リスト ホロナ～モオ　P121-130|accessdate=2018-12-02|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>と対になる名称である<ref name=":0" />。詳細は[[本岐駅#ポンキキン川]]についてを参照。 == 駅周辺 == * [[国道240号]] * [[津別町営バス|津別まちバス（旧・津別町営バス）]]「恩根入口」停留所 == 駅跡 == 畑となっている。 == 隣の駅 == ; 日本国有鉄道 : 相生線 :: [[津別駅]] -
