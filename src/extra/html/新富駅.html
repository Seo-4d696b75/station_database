駅設置工事竣工<ref name="shinchoshi"/>。 ** [[10月7日]] - 上り2番列車から30秒停車が実現する（非公式開業？）<ref name="shinchoshi"/>。 * [[1955年]]（昭和30年）[[9月2日]] - [[日本国有鉄道]]深名線の'''新富仮乗降場'''（[[鉄道管理局|局]]設定）として開業<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年10月 JTB編集発行。</ref>。 * [[1956年]]（昭和31年）[[9月20日]] - 駅に昇格。'''新富駅'''となる<ref name=":0222">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=114}}</ref><ref name="hensen"/>。旅客のみ取扱い<ref name="hensen"/>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承<ref name="hensen"/>。 * [[1990年]]（平成2年）[[9月1日]] - 利用者減少により廃止となる<ref name="hensen"/>。 ** 同年月日に、[[宗谷本線]][[琴平駅 (北海道)|琴平駅]]、[[石北本線]][[伊奈牛駅]]が[[廃駅]]となった。 === 駅名の由来 === もともと地名は「豊富」であった。しかし宗谷本線[[豊富駅]]と紛らわしいため「新富」とした<ref name=":0222" />。その後住民から地区名も新富に変更するよう要望があり、1960年（昭和35年）1月1日から地区名も新富となった<ref>新幌加内町史 2008年（平成20年）3月発行、P199-200。</ref>。 なお、1973年（昭和48年）に国鉄北海道総局が発行した『北海道　駅名の起源』では、字名改正が先であったとしている<ref name=":0222" />。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]で、[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。ホームは石組み土盛りの造りで、線路の東側（名寄方面に向かって右手側）に設置されていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）205ページより。</ref>。また、[[無人駅]]であったが、ホーム中央部分の出入口附近に[[待合所]]機能のみの木造駅舎を有した<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）178,183ページより。</ref>。 == 駅周辺 == 駅前に民家は無く、周囲は休耕[[田]]が目立っていた<ref name="bunken1983"/>。 * [[国道275号]]（空知国道） * [[北海道大学]][[演習林]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> * [[雨竜川]]<ref name="bunken1980M"/> * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「新富」停留所 == 駅跡 == 廃駅と同時に撤去され、2011年（平成23年）時点では駅関連施設はすべて撤去されている<ref name="bunken2011H"/>。 == その他 == 駅の廃止に当たってJR北海道から廃止を打診された幌加内町が、利用していた3戸の住民に確認せず廃止に同意をしたため、列車で病院に通っていた老夫婦が孤立するという問題が起きた経緯がある<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（小学館、[[1993年]]6月発行）76ページより。</ref>。 なお、2015年の国勢調査によると幌加内町字新富は消滅集落となった。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[政和駅]] -
