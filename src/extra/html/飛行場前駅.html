'''飛行場前駅'''（ひこうじょうまええき）は、[[北海道]]（[[宗谷支庁]]）[[枝幸郡]][[浜頓別町]]字共和にあった[[北海道旅客鉄道]]（JR北海道）[[天北線]]の[[鉄道駅|駅]]（[[廃駅]]）である。天北線の[[廃線]]に伴い[[1989年]]（平成元年）[[5月1日]]に廃駅となった。 == 歴史 == * [[1955年]]（昭和30年）[[12月2日]] - [[日本国有鉄道]]北見線の'''飛行場前仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1961年]]（昭和36年）[[4月1日]] - 線路名を天北線に改称、それに伴い同線の仮乗降場となる。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、北海道旅客鉄道（JR北海道）の駅となると共に駅に昇格。'''飛行場前駅'''となる。 * [[1989年]]（平成元年）[[5月1日]] - 天北線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]だった。ホームは線路の西側、南稚内方面に向かって左手側に存在した<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=27}}</ref>。ホームは木製のデッキ式で<ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |pages=249-250}}</ref>、音威子府方（南側）に[[斜路|スロープ]]を有し<ref name="bunken1981">{{Cite book|和書|author=高浜博隆 |others=[[鉄道友の会]]東京支部 |title=国鉄駅名全百科 |edition=56.4訂補版 |date=1981-10 |publisher=[[小学館]] |series=コロタン文庫 |isbn= |volume=36 |pages=pp.12-13,24,26}}</ref>踏切際の道路に連絡していた。 [[仮乗降場]]に出自を持つ[[無人駅]]であり、駅舎は存在せず、ホーム出入口から少し離れた位置に[[マンサード屋根|マンサード]]<ref name="bunken2001H">{{Cite book|和書|editor=[[宮脇俊三]] |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |pages=40-42}}</ref>の[[待合所]]を有していた<ref>{{Cite web |accessdate=2012-11-4}}「飛行場前乗降場」1988年9月5日撮影の画像あり。</ref>。待合所入口には「飛行場前停畄場」と表記されていた<ref name="bunken1981" />。 仮乗降場時代の一時期、[[駅名標]]には駅名の読みとして「ひこうじょまえ」と記載されていた<ref name="bunken1981" />が、その後訂正されている<ref name="bunken2011P" />。 == 駅名の由来 == 駅名は、かつて付近に[[大日本帝国陸軍]]の[[浅茅野飛行場|浅茅野第一飛行場]]があったことに由来する。浅茅野飛行場は、[[太平洋戦争]]中の[[1944年]]（昭和19年）に建設され、その1年後の[[1945年]]（昭和20年）には戦争は終結したためほとんど使われないままにその役目を終え、兵舎や将校官舎といった建物は1951年（昭和26年）に解体撤去された。 当駅が[[鉄道管理局|局]]設定の[[仮乗降場]]として開業した[[1955年]]（昭和30年）には、周囲に特段の構築物などの無い原野となっていた。テレビや書籍でも「飛行場がないところなのに飛行場前駅と名乗る駅」として紹介されることがあった<ref name="bunken1981" />。 == 駅周辺 == 周辺は、整備されていない笹原と原野、牧場、湿地等である。 *
