[[鉄道省]]渚滑線の渚滑駅 - 北見滝ノ上駅間開通に伴い、開業。[[日本の鉄道駅#一般駅|一般駅]]。 *[[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 *[[1960年]]（昭和35年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。 *[[1978年]]（昭和53年）[[12月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。同時に無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 *[[1985年]]（昭和60年）[[4月1日]] - 渚滑線の全線廃止に伴い、廃駅となる。 === 駅名の由来 === 当駅が所在した地名より。[[渚滑川]]にある滝の下に位置することに由来する<ref name=":02">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=202}}</ref><ref>{{Cite book|和書|title=駅名の起源|editor=札幌鉄道局編|year=1939|publisher=北彊民族研究会|id={{NDLJP|1029473}}|page=94}}</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南側（北見滝ノ上方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）215ページより。</ref>。そのほかに旧貨物線線である、北見滝ノ上方から駅舎側に分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分への[[保線]]用の[[停車場#側線|側線]]を1線有した<ref name="bunken1983"/>。かつては[[相対式ホーム]]2面2線を有する、[[列車交換]]可能な[[交換駅]]であった。交換設備運用廃止後は線路は撤去されたが、ホーム前後の線路は転轍機の名残で湾曲していた。 [[無人駅]]（簡易委託駅）となっており、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、下渚滑駅や中渚滑駅と全く同じ形状の駅舎となっていた。駅舎は構内の東側に位置し、ホームに接していた。 == 駅周辺 == * [[国道273号]]（渚滑国道） * 滝上町立滝下小学校 * [[渚滑川]] * [[北紋バス]]「滝下」停留所（渚滑線代替：旧駅前、[[高速バス|都市間バス]]：滝下小学校前） == 駅跡 == 1997年（平成9年）11月時点では駅舎とホームの一部が残存し、駅舎は宿泊所にも利用されていた模様であった<ref name="bunken1998H">書籍『鉄道廃線跡を歩くV』（[[JTBパブリッシング]]、[[1998年]][[6月]]発行）23-24ページより。</ref>。また駅前にあったモミの木や農業倉庫も残存していた<ref name="bunken1998H"/>。2011年（平成23年）時点でも同様で駅舎は倉庫として利用されている模様であった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）170ページより。</ref>。ホームの一部及び線路跡も残存している<ref name="bunken2011H"/>。駅舎は個人所有となっており、立ち入ることはできない。2018年（平成30年）時点では少なくとも駅舎は解体されている。 == 隣の駅 == ; 日本国有鉄道 : 渚滑線 :: [[上渚滑駅]] -
