* 1929年に雄松駅から改称 }} '''近江舞子駅'''（おうみまいこえき）は、かつて[[滋賀県]][[滋賀郡]][[志賀町 (滋賀県)|志賀町]]大字南小松（現在の[[大津市]]南小松{{Sfn|江若鉄道の思い出|p=76}}）にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 [[琵琶湖]]の代表的な水泳場である近江舞子水泳場のうち、中浜（雄松崎）の最寄り駅として多くの水泳客で賑わう駅であった。 == 歴史 == 当駅は1926年（大正15年）、江若鉄道の[[近江木戸駅]] - 雄松駅間の開通に合わせて'''雄松駅'''（おまつえき）として滋賀郡[[小松村 (滋賀県)|小松村]]に開業した{{Sfn|江若鉄道の思い出|p=76}}<ref name=kampo19260814>[{{NDLDC|2956343/4}} 「地方鉄道運輸開始」『官報』1926年8月14日]（国立国会図書館デジタル化資料）</ref>。翌27年に路線はさらに延伸、[[北小松駅 (江若鉄道)|北小松駅]]までの区間が開通する{{Sfn|寺田|2010|p=10}}{{Sfn|田中|宇田|西藤|1998|p=394}}。 江若鉄道では当初、沿線の各町村には原則として1駅を各町村役場の近くに設置する計画であったが、その計画に従うと[[木戸村 (滋賀県)|木戸村]]の近江木戸駅から小松村の北小松駅まで8キロメートルにわたり駅がない状態となる{{Sfn|田中|宇田|西藤|1998|pp=395&ndash;396}}。そのため木戸村と小松村、それに[[葛川村]]の一部の地区の住民はこの区間に駅を設置するよう1919年（大正8年）に江若鉄道に要望、当駅はその請願に応えて開設が決まった{{Sfn|田中|宇田|西藤|1998|pp=395&ndash;396}}。 雄松駅の名は、[[琵琶湖]]の景勝地「雄松崎」に由来する{{Sfn|江若鉄道の思い出|p=76}}。琵琶湖の中でも遠浅の地形で水泳や観光に適していた雄松崎は{{Sfn|江若鉄道の思い出|p=76}}、昭和に入ると観光地として開発が進み、白砂の砂浜が広がるその風景は兵庫の[[舞子 (神戸市)|舞子]]の浜になぞらえて「近江舞子」と称されるに至った{{Sfn|江若鉄道の思い出|p=77}}<ref>{{Cite web |url=http://www.rekihaku.otsu.shiga.jp/db/jiten/data/231.html |title=近江舞子（雄松崎） |accessdate=2017-10-16 |work=大津の歴史事典 |publisher=大津市歴史博物館}}</ref>。当駅が近江舞子駅に改称したのも昭和のはじめ、1929年（昭和4年）のことである{{Sfn|江若鉄道の思い出|p=76}}。当地には江若鉄道による鉄道路線のほか[[琵琶湖汽船|太湖汽船]]による観光船も就航し{{Sfn|江若鉄道の思い出|p=77}}、キャンプ村やホテルといった宿泊施設の相次ぐ開設も奏功して夏場には水泳客が多く訪れる琵琶湖の代表的な水泳場となった{{Sfn|江若鉄道の思い出|p=78}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1926年]]（大正15年）[[8月11日]] - 近江木戸駅から当駅までの延伸開業に伴い、'''雄松駅'''として滋賀郡小松村に開業{{Sfn|寺田|2010|p=10}}<ref name=kampo19260814/>。 * [[1927年]]（昭和2年）[[4月1日]] - 当駅から北小松駅まで延伸{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956537/6}} 「地方鉄道運輸開始」『官報』1927年4月6日]（国立国会図書館デジタル化資料）</ref>。 * [[1929年]]（昭和4年）[[8月10日]] - '''近江舞子駅'''に改称{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == 近江舞子駅は[[列車交換]]が可能な交換駅{{Sfn|寺田|2010|p=10}}。[[プラットホーム|ホーム]]は1面で、2本の線路の間に配される[[島式ホーム]]である{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=80}}。旅客と貨物の双方を扱うことができた[[一般駅]]であり{{Sfn|寺田|2010|p=10}}、貨物線と[[停車場#側線|側線]]を1本ずつ有した{{Sfn|寺田|2010|p=15}}。 (江若鉄道)|北小松駅]]にかけての沿線には別荘や保養所が建ち並んだ{{Sfn|レイル|p=82}}。 戦前は駅周辺に「[[千本桜]]」と称される[[桜並木]]が広がっていた{{Sfn|ありし日の江若鉄道|p=17}}。これは1936年（昭和11年）、当時の江若鉄道社長の要請に応じて植物学者[[笹部新太郎]]が育てていた山桜500本を植えたものであったが、戦時中の工場疎開により一部が伐採され、残りも手入れが行き届かず失われた{{Sfn|ありし日の江若鉄道|p=17}}{{Sfn|江若鉄道の思い出|p=81}}。 駅のあった場所は江若鉄道の廃線後に開通した[[湖西線]]の[[近江舞子駅]]の南方にあたり、当駅から北小松駅にかけての線路跡は[[滋賀県道307号北小松大物線|県道307号]]に転用されている{{Sfn|江若鉄道の思い出|p=78}}。この区間は廃線後早いうちから生活道路として利用されていた{{Sfn|寺田|2010|p=25}}{{Sfn|吉田|1998|p=192}}。 == 隣の駅 == ;
