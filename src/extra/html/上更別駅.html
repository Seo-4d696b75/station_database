eki.jpg|right|thumb|1977年の上更別駅と周囲約500m範囲。右下が広尾方面。{{国土航空写真}}]] '''上更別駅'''（かみさらべつえき）は、[[北海道]]（[[十勝支庁]]）[[河西郡]][[更別村]]字上更別にあった[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''サヘ'''。広尾線の[[廃線]]に伴い[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）139ページより。</ref>。互いのホームは駅舎側ホーム中央部分と島式ホーム北側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側単式ホーム（東側）が下りの1番線、島式ホーム（東側）が下りの2番線となっていた<ref name="bunken1983"/>。島式ホームの外側1線は、帯広方から分岐した行き止まりの[[停車場#側線|側線]]となっており、そのほか1番線の広尾方から分岐し駅舎南側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。 [[無人駅]]扱いの、運転取扱い要員のみが配置されていた駅であった。駅舎は構内の北東側（広尾方面に向かって左側）に位置し1番線ホーム中央部に接していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅の所在する地名より。地名はアイヌ語の「サラ・ベツ」（[[葦]]や[[茅]]が生える川）に由来し<ref>[http://www.sarabetsu.jp/cms/index.php?ID=160 沿革] - 更別村、2015年5月21日閲覧。</ref>、川の上流に位置することから「上」を冠している<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=72 }}</ref>。 == 利用状況 == 1981年度（昭和56年度）の1日乗降客数は20人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道210号尾田豊頃停車場線]] * [[国道236号]]（広尾国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）13ページより。</ref> * [[十勝スピードウェイ]] * 更別村立上更別小学校 * 更別湿原 - 駅から北東に約0.6km<ref name="bunken1983"/>。北海道[[天然記念物]]の[[カバノキ属|ヤチカンバ]]が生育している<ref name="bunken1983"/>。 * [[十勝バス]]「上更別」停留所 == 歴史 == * [[1930年]]（昭和5年）[[10月10日]] - '''[[日本国有鉄道|国有鉄道]]広尾線'''中札内駅 - 大樹駅間延伸開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1974年]]（昭和49年）[[12月15日]] - 貨物・[[チッキ|荷物]]（発送手荷物を除く）取扱い廃止。出札・改札業務を停止し旅客業務について無人化。列車交換設備を有し、運転要員は営業最終日まで継続配置。 * [[1984年]]（昭和59年）[[2月1日]] - 手荷物発送取扱い廃止。 * [[1987年]]（昭和62年）[[2月2日]] - 広尾線の廃線に伴い廃止となる。 == 駅跡 == 1999年（平成11年）時点では「上更別駅」と記載された記念碑が建立されていた<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）38-40ページより。</ref>。この記念碑は更別駅と同様のもので、表側下段の碑文は同文となっているが、表側上段の駅舎をスケッチした陶板と裏面の[[駅名標]]は当駅のものになっている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）188-189ページより。</ref>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）87ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。駅跡自体は空地になっており、線路跡も確認できる<ref name="bunken2011H"/>。 == 隣の駅 == ; 日本国有鉄道 : 広尾線 :: [[更別駅]] -
