1943年に大溝駅から改称 }} '''高島町駅'''（たかしまちょうえき）は、かつて[[滋賀県]][[高島郡 (滋賀県)|高島郡]][[高島町 (滋賀県)|高島町]]大字勝野（現在の[[高島市]]勝野{{Sfn|江若鉄道の思い出|p=94}}）にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 == 歴史 == 当駅は1927年（昭和2年）、江若鉄道が[[北小松駅 (江若鉄道)|北小松駅]]から延伸し、[[高島郡 (滋賀県)|高島郡]]内に初めて鉄道が開通した際に開業した{{Sfn|江若鉄道の思い出|p=94}}。開通に際しては、当時の滋賀県知事[[今村正美]]も臨席しての大規模な祝賀会が挙行されている{{Sfn|江若鉄道の思い出|p=94}}。この時開通したのは当駅までで、路線は2年後の1929年（昭和4年）にさらに延伸、[[安曇川駅 (江若鉄道)|安曇駅]]までの区間が開業した{{Sfn|寺田|2010|p=10}}{{Sfn|田中|宇田|西藤|1998|p=394}}。 開業当時の町名は高島郡[[大溝町]]であり、駅名も'''大溝駅'''（おおみぞえき）と称していたが{{Sfn|江若鉄道の思い出|p=94}}、1943年（昭和18年）に大溝町が周辺の村と合併して[[高島町 (滋賀県)|高島町]]となったのに伴い駅名も高島町駅に改称されている{{Sfn|江若鉄道の思い出|p=95}}。合併を経たあとも駅周辺は町の中心地であり、商業や観光の拠点として賑わいを見せた{{Sfn|江若鉄道の思い出|p=96}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1927年]]（昭和2年）[[12月15日]] - 江若鉄道の北小松駅 - 当駅間の開業に伴い、'''大溝駅'''として高島郡大溝町に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956758/11}} 「地方鉄道運輸開始」『官報』1927年12月24日]（国立国会図書館デジタル化資料）</ref>。 * [[1929年]]（昭和4年）[[6月1日]] - 当駅から安曇川駅までの区間が延伸開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957195/6}} 「地方鉄道運輸開始」『官報』1929年6月6日]（国立国会図書館デジタル化資料）</ref>。 * [[1943年]]（昭和18年）4月ごろ - 高島町の発足により'''高島町駅'''に改称{{Sfn|今尾|2008|pp=31&ndash;32}}{{Sfn|寺田|2010|p=15}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=高島町線路配置図|left=[[白鬚浜駅|白鬚浜]]}} 周辺はかつての[[大溝藩]]大溝城の城下町であり、駅は大溝城の[[堀|水堀]]跡に立地していた{{Sfn|江若鉄道の思い出|p=95}}{{Sfn|江若鉄道の思い出|p=96}}。1956年（昭和31年）には駅前に[[高島市民病院|高島診療所]]が開所し、病院利用者を当て込んだ商店なども駅前に出店している{{Sfn|江若鉄道の思い出|p=96}}。 駅のあった場所は、江若鉄道の廃線後に開業した[[湖西線]][[近江高島駅]]の駅前広場に相当する{{Sfn|レイル|p=85}}{{Sfn|寺田|2010|p=26}}。当駅より先、[[近江今津駅 (江若鉄道)|近江今津]]方面の線路跡については、近江高島駅の北側でカーブを描く湖西線の線路の外側に[[築堤]]や橋台が一部残されているのを確認することができるが、それ以降はおおむね湖西線に取り込まれている{{Sfn|寺田|2010|p=26}}{{Sfn|吉田|1998|pp=193&ndash;195}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[白鬚駅]] -
