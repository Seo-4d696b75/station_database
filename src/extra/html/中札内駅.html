'''中札内駅'''（なかさつないえき）は、[[北海道]]（[[十勝支庁]]）[[河西郡]][[中札内村]]字札内にかつて設置されていた、[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ナサ'''。広尾線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 == 駅構造 == 廃止時点で、2面2線の[[相対式ホーム]]を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム中央部分と対向側ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）139ページより。</ref>。駅舎側ホーム（西側）が上りの1番線、対向側ホーム（東側）が下りの2番線となっていた<ref name="bunken1983"/>。対向側ホームの外側1線を[[停車場#側線|側線]]として有し、その他、1番線の帯広方から分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の北西側（広尾方面に向かって右側）に位置し1番線ホーム中央部分に接していた<ref name="bunken1983"/>。改修されていたものの、土壁と板張りの駅舎であった<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名はアイヌ語の「サツナイ」（乾いた川）に由来し、札内川の中流に位置することから「中」を冠している<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=72 }}</ref>。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は232人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道321号中札内停車場線]] * [[国道236号]]（広尾国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）13ページより。</ref> * [[道の駅なかさつない]] *
