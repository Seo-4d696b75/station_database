播州鉄道の駅として開業。[[鉄道駅#一般駅|一般駅]]。 ** 当時の所在地表示は[[兵庫県]][[多可郡]][[西脇町 (兵庫県)|津万村]]西脇であった。 * [[1917年]]（大正6年）[[11月1日]] - 津万村が町制施行・改称して[[西脇町 (兵庫県)|西脇町]]となり、所在地表示が兵庫県多可郡西脇町西脇になる。 * [[1923年]]（大正12年）[[12月21日]] - 播丹鉄道に譲渡。 * [[1943年]]（[[昭和]]18年）[[6月1日]] - 播丹鉄道が国有化、[[鉄道省]]鍛冶屋線の駅となる。 * [[1952年]]（昭和27年）[[4月1日]] - [[西脇市]]成立に伴い、所在地表示が兵庫県西脇市西脇になる。 * [[1953年]]（昭和28年）2月 - 2代目の駅舎に改築。 * [[1974年]]（昭和45年）[[10月1日]] - 貨物の取扱いを廃止（[[鉄道駅#旅客駅|旅客駅]]となる）。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取扱いを廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR西日本が継承。 * [[1990年]]（[[平成]]2年）4月1日 - 鍛冶屋線廃線に伴い廃止。隣駅の野村駅が[[西脇市駅]]に改称。 == 隣の駅 == ; 西日本旅客鉄道 : 鍛冶屋線 :: [[西脇市駅|野村駅]] - '''西脇駅''' - [[市原駅 (兵庫県)|市原駅]] == 関連項目 == * [[日本の鉄道駅一覧]] * [[廃駅]] {{鍛冶屋線}} {{Eki-stub}} {{DEFAULTSORT:にしわき}} [[Category:兵庫県の廃駅]] [[Category:西脇市の鉄道駅|廃にしわき]] [[Category:日本の鉄道駅 に|しわき]] [[Category:西日本旅客鉄道の廃駅]] [[Category:日本国有鉄道の鉄道駅]] [[Category:播丹鉄道]] [[Category:1913年開業の鉄道駅]] [[Category:1990年廃止の鉄道駅]] [[Category:西脇市の歴史|にしわきえき]] exrhrtdf3usgu4m00s81l9l762n6hr1
