[[鹿島参宮鉄道]]の駅として開業。 * [[1965年]]（昭和40年）[[6月1日]] - 鹿島参宮鉄道と[[常総筑波鉄道]]が合併して関東鉄道が発足。同社鉾田線の駅となる。 * [[1979年]]（昭和54年）[[4月1日]] ‐ 鉾田線が鹿島鉄道に分社化。 * [[2007年]]（[[平成]]19年）4月1日 - 廃止。 * [[2010年]]（平成22年）[[8月30日]] - 駅跡地にかしてつバス「東田中駅」停留所が開業。 === 現況 === バス専用道化工事によって、ホーム等は全て撤去された。 2010年8月30日にはかしてつバス[[バス専用道路|専用道]]開通に伴い、駅跡地に停留所が開業している。 == その他 == 毎年9月に石岡市で行われる[[常陸國總社宮大祭]]開催時は臨時便として、石岡駅 - 小川駅間の増便と石岡駅 - 当停留所の区間便も運行される。当停留所止まりの便は通常の乗降口とは異なり、石岡方の交差点で一般道路へ右折後、石岡運動公園の前が降車口となる。当停留場始発の便は通常の乗降口と同じである。 == 隣の駅 == ;鹿島鉄道線 :鹿島鉄道 ::[[石岡南台駅]] - '''東田中駅''' - [[玉里駅]] == 脚注 == {{脚注ヘルプ}} {{Reflist}} == 関連項目 == *[[日本の鉄道駅一覧]] *[[東田中駅 (愛知県)]] {{鹿島鉄道線}} {{デフォルトソート:ひかしたなか}} [[Category:日本の鉄道駅 ひ|かしたなか]] [[Category:茨城県の廃駅]] [[Category:関東地方のバス停|ひかしたなかえき]] [[Category:鹿島鉄道の鉄道駅]] [[Category:鹿島参宮鉄道|ひかしたなかえき]] [[Category:1964年開業の鉄道駅]] [[Category:2007年廃止の鉄道駅]] [[Category:石岡市の交通|廃ひかしたなかえき]] [[Category:石岡市の歴史|ひかしたなかえき]] 3t61n4yh6mk6izsqxiwk542ckfqajg7
