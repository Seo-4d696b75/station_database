'''今井田駅'''（いまいだえき）は、[[広島市]][[安佐北区]][[可部町]]今井田に存在した[[西日本旅客鉄道]]（JR西日本）[[可部線]]の[[鉄道駅|駅]]（[[廃駅]]）。 可部線非電化区間（[[可部駅|可部]] - [[三段峡駅|三段峡]]間）の廃線に伴い、[[2003年]]（[[平成]]15年）12月1日に廃止された。 == 駅構造 == 1面1線の[[単式ホーム]]のみを持つ[[地上駅]]。[[無人駅]]で駅舎はなく、線路の南側に設置されたホームの上に[[待合所]]があるだけであった。 == 利用状況 == 以下の情報は、広島市統計書及び広島市勢要覧に基づいたデータである。 {| class="wikitable" |- !rowspan|年度 !rowspan|1日平均<br />乗車人員||年度毎<br |} == 駅周辺 == 駅の北側を[[広島県道267号宇津可部線]]が通っている。駅の南側を[[太田川]]が流れており、駅のすぐ東側にある筒瀬橋（赤色の吊橋）か下流にある安佐北大橋（[[広島県道269号今井田緑井線]]になっている）で対岸の広島市安佐北区[[安佐町]]筒瀬に渡ることができる（筒瀬橋は重量制限がある上、標識には無いが事実上の車幅制限があり3ナンバー車は注意を要する。その上、貨物車も実質的に[[軽自動車]]（軽トラ）程度しか通れない）。 * [[広島交通]] 今井田バス停 == 歴史 == * [[1956年]]（[[昭和]]31年）11月19日 - [[日本国有鉄道|国鉄]]可部線の'''今井田[[仮乗降場]]'''として開業。 * 1956年（昭和31年）12月20日 - '''今井田駅'''に昇格。[[鉄道駅#旅客駅|旅客駅]]。 ** 当時の所在地表示は広島県[[安佐郡]]可部町今井田であった。 * [[1972年]]（昭和47年）4月1日 - 可部町の広島市編入に伴い、所在地表示が広島市可部町今井田に変更。 * [[1980年]]（昭和55年）4月1日 - 安佐北区成立に伴い、所在地表示が広島市安佐北区可部町今井田に変更される。 * [[1987年]]（昭和62年）4月1日 - [[国鉄分割民営化]]によりJR西日本が継承。 * [[2003年]]（[[平成]]15年）12月1日 - 廃止。
