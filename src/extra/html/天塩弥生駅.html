[[鉄道省]]名雨線名寄 - 当駅間開通に伴い'''初茶志内駅'''（はっちゃしないえき）として開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1941年]]（昭和16年）[[10月10日]] - 当駅 - [[朱鞠内駅|朱鞠内]]間が延伸開業し全通。路線名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1951年]]（昭和26年）[[7月20日]] - '''天塩弥生駅'''に改称<ref>[{{NDLDC|2963904/9}} 「日本国有鉄道公示第176号」『官報』1951年7月14日]（国立国会図書館デジタルコレクション）</ref>。 * [[1960年]]（昭和35年）[[9月15日]] - 貨物・[[チッキ|荷物]]取扱い廃止。 * [[1982年]]（昭和57年）[[3月29日]] - 無人化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1995年]]（平成7年）[[9月4日]] - 深名線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]（片面使用）1面1線を有する[[地上駅]]であった。ホームは線路の北側（名寄方面に向かって左手側）に存在した。[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]][[6月]]発行）77ページより。</ref>。かつては[[島式ホーム]]1面2線を有する[[列車交換]]可能な[[交換駅]]であり、ここを始発・終着とする名寄からの区間列車も運転されていた。そのほか駅舎側の線から駅舎側に分岐し駅舎東側の貨物ホームへの貨物[[停車場#側線|側線]]を1線有していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）206ページより。</ref>。使われなくなった駅舎側の1線は、交換設備運用廃止後は撤去された。貨物側線は1983年（昭和58年）4月時点では、分岐器が深川方、名寄方の両方向とも維持されていた形で残存していたが<ref name="bunken1983"/>、1993年（平成5年）までに撤去された<ref name="bunken1993"/>。ホーム前後の線路は分岐器の名残で湾曲していた<ref name="bunken1993"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が改築されて残っていた。駅舎は構内の北側に位置しホーム西側とを結ぶ通路で連絡した<ref name="bunken1993"/>。駅舎は無人化後縮小され、待合室部分のみの間口の狭い建物となっていた（1993年（平成5年）時点ではこの形状<ref name="bunken1993"/>）。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名に[[旧国名#駅名|旧国名]]の「天塩」を冠する。 旧駅名の初茶志内（はっちゃしない）は、[[アイヌ語]]の「ハツクシ・ウシ・ナイ」（[[ブドウ|葡萄]]、[[クワ|子桑]]の多い川）に由来する<ref name="bunken1993"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は22人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は4人<ref name="bunken1993"/>。 == 駅周辺 == 名寄市内の山間に位置した<ref name="bunken1983"/>。 * [[北海道道798号西風連名寄線]] * 弥生共同墓地 * 初茶志内山 - 駅の南。標高201m。 * [[ジェイ・アール北海道バス]][[深名線 (ジェイ・アール北海道バス)|深名線]]「天塩弥生」停留所 == 駅跡 == [[File:Shinmeisen Teshioyayoi.jpg|thumb|駅跡　(2011年8月8日)]] 廃駅後しばらく駅舎やホームが残っていたが、1998年（平成10年）に全て解体され、跡地は整地された。2000年（平成12年）時点では跡地が残るのみで<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）36ページより。</ref>、2010年（平成22年）時点では民家の裏の荒廃した空き地となっていた<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）43ページより。</ref>。2011年（平成23年）時点では駅附近の線路跡が確認出来、[[バラスト軌道|バラスト]]が残存していた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）180-181ページより。</ref>。[[駅名標]]のレプリカが名寄市の「[[名寄市北国博物館]]」に[[キマロキ編成]]とともに保存されている<ref name="bunken2011H"/>。 2015年、ある夫妻が天塩弥生駅跡地と周辺の土地を名寄市から購入し、昭和40年代までに建設された道内の駅舎をイメージした民宿を建築した<ref>「昭和の駅舎を再現する民宿 元鉄道マンのこだわりに応える」[https://web.archive.org/web/20151208042612/http://kitacraft.sakura.ne.jp/sblo_files/kitacraft/image/image.jpeg 北海道住宅新聞2015年9月25日]</ref><ref>「天塩弥生駅復活へ 名寄の富岡さん夫妻・レストラン来春オープン」[http://www.nayoro-np.com/news/2015-11-22.html 名寄新聞2015年11月22日]</ref>。2016年3月から「天塩弥生駅」の屋号で営業しており、昼の時間帯はレストランとして営業している<ref>「【天塩弥生駅】旅人宿&田舎食堂」https://www.facebook.com/teya841/</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[北母子里駅]] -
