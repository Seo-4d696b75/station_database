[[日本国有鉄道]]（国鉄）湧網線の'''興生沢仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年）[[3月20日]] - 湧網線の全線廃止に伴い、営業を停止、廃止となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の南側（網走方面に向かって右手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=177}}</ref>。 廃止時まで[[仮乗降場]]であり、[[無人駅]]となっていた。駅舎は無かったが、ホーム中央部分に高床式の[[待合所]]を有していた<ref name="bunken2011P"/>。ホームは中湧別方に[[斜路|スロープ]]を有し<ref name="bunken2011P"/>、駅施設外に連絡していた。 == 駅名の由来 == 当地に入植した実習生が当地附近の[[沢]]を興して皆で生きよう、という意味で付けた名に由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=169}}</ref>。 == 駅周辺 == * [[北海道道103号留辺蘂浜佐呂間線]] * 佐呂間別川<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=19}}</ref> * [[佐呂間町ふれあいバス]]（[[フリー乗降制|フリー乗降]]） == 駅跡 == [[2011年]]（平成23年）時点では駅の設置場所は確認できたが、更地となり、何も残っていない<ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |page=103}}</ref>。 == 隣の駅 == ; 日本国有鉄道 : 湧網線 :: [[佐呂間駅]] -
