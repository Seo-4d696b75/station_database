終着駅として開業<ref>[{{NDLDC|2961659/9}} 「運輸通信省告示第78号」『官報』1944年3月20日] - [[国立国会図書館]]デジタルコレクション</ref>。 ** 当時の所在地表示は[[兵庫県]][[多紀郡]][[福住村 (兵庫県)|福住村]]福住であった。 * [[1955年]]（昭和30年）[[4月15日]] - [[多紀町|多紀村]]成立に伴い、所在地表示が兵庫県多紀郡多紀村福住になる。 * [[1958年]]（昭和33年）[[11月1日]] - 営業範囲を「一般運輸営業」から、「一般運輸営業。ただし、代金引換は行わない。」へ改める<ref>1958年（昭和33年）10月29日日本国有鉄道公示第384号「篠山線篠山外四停車場の営業範囲を改正する件」</ref>。 * [[1960年]]（昭和35年） ** [[1月1日]] - 多紀村が町制施行して[[多紀町]]となり、所在地表示が兵庫県多紀郡多紀町福住になる。 ** [[12月15日]] - 営業範囲を「旅客、手荷物、小荷物および車扱貨物。ただし、配達および代金引換の取扱はしない。」へ改める<ref>1960年（昭和35年）12月13日日本国有鉄道公示第623号「福知山線藍本停車場等の営業範囲を改正する件」</ref>。 * [[1972年]]（昭和47年）[[3月1日]] - 篠山線全線廃止に伴い廃駅となる<ref>1972年（昭和47年）2月1日日本国有鉄道公示第547号「運輸営業の廃止の件」</ref>。 == 駅構造 == 1面1線の[[単式ホーム]]に駅舎を持つ駅であった。 == 駅周辺 == 福住地区の中心部に位置している。かつての駅前通りには福住バス停があり、丹波篠山市街方面への篠山線代替バスが[[神姫グリーンバス]]、園部方面の元[[西日本JRバス]]の路線が[[京阪京都交通]]により運転されており、交通の連絡点という役割は変わっていない。 * 地主稲荷神社 * 丹波篠山市役所多紀支所 * 福住郵便局 <gallery widths=180 heights=120> Fukusumi bus stop 20180119.jpg|福住バス停 Fukusumi Station, name plate 2018.jpg|2018年撮影の福住駅跡碑 </gallery> == 隣の駅 == ; 日本国有鉄道 : [[篠山線]] :: [[村雲駅]] -
