41</ref>、読みは異なるが[[日本国有鉄道|国鉄]][[岩内線]]の[[岩内駅]]と同名となるのを避けるために旧国名を冠していた。そのため、「ばんばのばあちゃん」として知られた近隣のマッサージ店へ通う利用者のために、国鉄線からの連絡切符が販売され、その連絡切符が能美線各駅の中で当駅が最も多く販売された駅だった時期もあった<ref name="nomiden">[https://www.city.nomi.ishikawa.jp/www/contents/1001000001291/simple/nomiden.pdf 能美市『能美電ものがたり 運行時の各駅と現在』]（PDF）</ref>。 [[有人駅]]時代は駅員が2名常駐し、貨物駅が併設されていた頃は[[米]]、[[麦]]などの農産物、[[藁]]工品、[[肥料]]、[[織物]]製品などの積み降ろしで賑わっていた<ref name="nomiden"/><ref name="rm"/>。 == 駅構造 == 能美線の駅の中では大きな駅で、全盛期は貨物[[停車場#側線|側線]]を有していた。晩年は[[片面ホーム]]の1面1線の[[地上駅]]であり、[[木造]]の[[駅舎]]を設けていた<ref name="nomiden"/><ref name="rm"/>。 == 歴史 == * [[1925年]]（大正14年）[[6月5日]] - 能美電気鉄道により、'''岩内駅'''として開業。当初は省線とは非連帯駅<ref name="rm"/>。 * [[1938年]]（昭和13年）[[10月27日]]（届出）- ホーム擁壁を木造から[[コンクリート]]に変更、ホーム長を9m144cmから15mに延伸<ref name="rm"/>。 * [[1939年]]（昭和14年）[[8月1日]] - 金沢電気軌道が能美電気鉄道を吸収合併。 * [[1941年]]（昭和16年）8月1日 - 北陸合同電気（現在の北陸電力）が金沢電気軌道を合併。 * [[1942年]]（昭和17年）[[3月26日]] - 事業譲渡により北陸鉄道となる。 * [[1946年]]（昭和21年）以降 - '''加賀岩内駅'''に改称<ref>[[日本鉄道旅行地図帳]]（[[新潮社]]）6号「北信越」p.28（ ISBN 9784107900241 ）</ref>。この頃から国鉄との連帯運輸を開始<ref name="rm"/>。 * [[1967年]]（昭和42年）[[1月14日]] - 貨物側線を廃止<ref name="rm"/>。[[無人駅]]化<ref name="nomiden"/>。 * [[1980年]]（昭和55年）[[9月14日]] - 能美線[[廃線|廃止]]に伴い廃駅となる。 == 駅周辺 == * 岩内[[体育館]] * 岩内[[保育園]] == 隣の駅 == ; 北陸鉄道 : 能美線 :: [[三ツ口駅]] - '''加賀岩内駅''' - [[火釜駅]] == 現状 == 鉄道廃止後、駅跡は[[公園]]として再利用されており、駅跡には石碑が設置されている<ref name="nomiden"/>。ただし、火釜方は水田の区画整理により旧線跡がわからない。 == 関連項目 == * [[日本の鉄道駅一覧]] * [[北陸鉄道]] == 脚注 == {{脚注ヘルプ}} {{Reflist}} {{北陸鉄道能美線}} {{DEFAULTSORT:かかいわうち}} [[category:日本の鉄道駅 か|かいわうち]] [[Category:石川県の廃駅]] [[category:北陸鉄道の廃駅]] [[Category:1925年開業の鉄道駅]] [[Category:1980年廃止の鉄道駅]] [[Category:能美市の鉄道駅|廃かかいわうち]] [[Category:能美市の歴史|かかいわうちえき]] nihysl2f56380ygaons2f5u4yybh2sv
