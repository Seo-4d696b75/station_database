'''豊牛駅'''（とようしえき）は、[[北海道]]（[[宗谷支庁]]）[[枝幸郡]][[浜頓別町]]字豊寒別にかつて設置されていた、[[日本国有鉄道]]（国鉄）[[興浜北線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''ヨウ'''。興浜北線の[[廃線]]に伴い、[[1985年]]（昭和60年）[[7月1日]]に廃駅となった。 == 歴史 == * [[1936年]]（昭和11年）[[7月10日]] - [[鉄道省]]興浜北線の浜頓別駅 - 北見枝幸駅間開通に伴い、開業<ref name=":022">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=190}}</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1944年]]（昭和19年）[[11月1日]] - 興浜北線の[[不要不急線]]指定に伴い、営業休止<ref name=":022" />。 * [[1945年]]（昭和20年）[[12月5日]] - 営業再開<ref name=":022" />。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1961年]]（昭和36年）[[4月1日]] - [[日本の鉄道駅#業務委託駅|業務委託]]化。 * [[1973年]]（昭和48年）[[9月17日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止し、同時に無人駅化<ref>北海道鉄道百年史 下巻 日本国有鉄道北海道総局</ref>。駅舎改築<ref name="bunken1983"/>。 * [[1985年]]（昭和60年）[[7月1日]] - 興浜北線の全線廃止に伴い、廃駅となる。 === 駅名の由来 === 当地は「豊寒別」と「プイタウシ」の中間にあり、「牛が豊かになるようにとの地元の要望<ref name=":022" />」から「豊寒別」から「豊」、「プイタウシ」から「ウシ」すなわち「牛」をとり、命名した<ref name=":022" />。 なお、「豊寒別」は[[アイヌ語]]の「トイカムペッ（toy-kamu-pet）」（土・かぶさる・川）に由来するとされ<ref name=":022" /><ref name=":3">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/8190P.pdf|title=アイヌ語地名リスト ツキサ～トヨコ　P81-90P|accessdate=2018-10-31|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>、「プイタウシ」はアイヌ語で「[[エゾノリュウキンカ]]の根を掘る所」の意とされている<ref name=":022" />。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の東側（北見枝幸方面に向かって左手側）に存在した<ref name="bunken1983">{{Cite |page=226}}</ref>の[[大量生産|マスプロ設計]]の小型駅舎となっていた<ref name="bunken2010E">{{Cite book|和書|author=杉崎行恭 |title=絶滅危惧駅舎 |date=2010-6 |publisher=二見書房 |series=二見文庫 |isbn=978-4576100906 |page=150}}</ref>。1983年（昭和58年）時点では、屋根の配色は青であった<ref name="bunken1983"/>。駅舎内にトイレを有していた<ref name="bunken2010E"/>。駅舎は構内の北東側に位置し、ホームから少し離れていた。 == 利用状況 == {| class="wikitable" style="margin: 1em 0.2em;
