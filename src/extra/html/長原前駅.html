1957年度 |乗換 = |備考 = 無人駅 |備考全幅 = 特記のないデータは廃止時 }} [[ファイル:Nagaramae_station_map.jpg|thumb|1977年の長原前駅跡とその周囲。赤で囲った場所に駅があった。赤い線は田口線の廃線跡。{{国土航空写真}}]] '''長原前駅'''（ながらまええき<ref name="chizucho"/>）は、[[愛知県]][[北設楽郡]][[設楽町]]清崎にかつて存在した[[豊橋鉄道]][[豊橋鉄道田口線|田口線]]（旧・田口鉄道）の[[鉄道駅|駅]]。北設楽郡[[田口町]]（1956年以降設楽町）にあった駅の一つである。 == 歴史 == 長原前駅は[[1950年]]（[[昭和]]25年）[[10月1日]]、田口鉄道により開設された<ref name="chizucho">[[#chizucho|『日本鉄道旅行地図帳』7号]]、pp37-38</ref><ref name="horai-ta64">[[#horai-ta|『鳳来町誌』田口鉄道史編]]、p64</ref>。 田口鉄道は、現在のJR[[飯田線]][[本長篠駅]]と[[北設楽郡]][[設楽町]]を結んだ鉄道である。駅の開業と路線の開通の時期には大きな開きがあり、駅をはさむ[[三河海老駅]]から[[清崎駅]]までの区間が同鉄道の第二期線として開通したのは駅設置より20年も前の[[1930年]]（昭和5年）のことである<ref>『鳳来町誌』田口鉄道史編、pp58-59</ref>。 駅が置かれたのは[[豊川]]（寒狭川）左岸の設楽町清崎であるが、駅名にある「長原」はその対岸の田峯にある地名（設楽町田峯字長原）である。駅近くの清崎字釜渕には町立清嶺中学校（1949年創立、2001年廃校<!--跡地は「グループホーム設楽の家-->）があった<ref>[[#shitara|『設楽町誌』教育・文化編]]、p80,130</ref>。 [[1956年]]（昭和31年）[[10月1日]]、田口鉄道が豊橋鉄道に[[合併 (企業)|合併]]されたため豊橋鉄道田口線の駅となる。それから8年後の[[1964年]]（昭和39年）、豊橋鉄道は[[赤字]]の拡大を理由に田口線の廃止を決定<ref>『鳳来町誌』田口鉄道史編、pp161-162</ref>。路線の廃止は[[1968年]]（昭和43年）[[9月1日]]付<ref name="chizucho"/>であったが、その直前の[[8月29日]]に[[台風]]（[[昭和43年台風第10号|台風10号]]）で被災して田峯・清崎間が線路破壊により不通となり、三河海老駅から先の区間は運転していなかった<ref>『鳳来町誌』田口鉄道史編、p186</ref>。 == 構造 == 片面ホーム1面1線で列車の[[列車交換|行き違い]]が不可能な駅であった<ref>{{Cite journal|和書|author = 白井良和|title = 奥三河に咲いたローカル線 田口線の回想|date = 1986-03|publisher = 鉄道図書刊行会 |journal = 鉄道ピクトリアル|number = 461|page = 59}}</ref><ref name="horai-ta64"/>。[[駅員]]は配置されていない（[[無人駅]]）<ref>[[#zuroku|『図録』]]、p8</ref>。 == 利用状況 == 1957年度の乗車人員は1万9千人（1日平均53人）で、そのうち約4割にあたる7千人が[[定期乗車券]]での利用客であった。この乗車人員は田口線の全11駅中で[[三河大草駅]]に次いで少ない。一方、貨物の取り扱いはない<ref>[[#tokei|『愛知県統計年鑑』昭和34年版]]、p386</ref>。 == 隣の駅 == ; 豊橋鉄道 : 田口線 :: [[田峯駅]] - '''長原前駅'''
