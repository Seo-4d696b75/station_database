'''[[日本国有鉄道|国有鉄道]]富内線'''振内駅 - 日高町駅間の延伸開通に伴い、開業。旅客のみ取り扱い。 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の東側（日高町方面に向かって右手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）108ページより。</ref>。 開業時からの[[無人駅]]で駅舎は無かったが、ホーム北側出入口附近にホームに屋根が掛けられた形の[[待合所]]を有していた<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当初この辺りの南側を三菜頃(サンナコロ)、北側を岡春部(オカシュンベ)と言い、その二集落の間に所在したため其々の頭文字から三岡、[[旧国名#駅名|旧国名]]の「日高」を冠した。なお、三菜頃は後に更に南側の岩内と統合し三岩になり、岡春部は富岡になった。旧国名が冠されたのは、[[小海線]][[三岡駅]]と重複するためと考えられるという<ref name="bunken1983"/> == 駅周辺 == 山の中の駅であった<ref name="bunken1983"/>。 * [[国道237号]]（日高国道） * [[北海道道847号三岩日高線]] * [[沙流川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）11ページより。</ref> * 岡春部川<ref name="bunken1980M"/> * 轟淵 -
