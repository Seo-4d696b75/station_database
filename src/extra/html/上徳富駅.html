[[石狩沼田駅]]間が[[廃線|廃止]]になるとともに廃駅となった。 == 駅構造 == 駅舎は札幌に向かって左手の東側に設置され、駅裏側利用の島状の単式ホーム1面1線と、駅舎とホーム間に貨物線1本を有していた。両端の分岐は貨物線側が開いた片開分岐の[[1線スルー]]状で、石狩追分に良く似ていたが、駅舎横札幌方に貨物ホームがあった<ref>昭和7年 線路一覧略図 札幌鉄道局発行。</ref>。 == 歴史 == * [[1931年]]（昭和6年）[[10月10日]] - [[札沼線|札沼北線]]石狩沼田 - 中徳富（初代、現在の[[新十津川駅|新十津川]]）間開通に伴い開業。[[一般駅]]。 * [[1943年]]（昭和18年）[[10月1日]] - [[石狩月形駅|石狩月形]] - [[石狩追分駅|石狩追分]]間の営業休止（[[不要不急線]]）により休止。 * [[1953年]]（昭和28年）[[11月3日]] - [[浦臼駅|浦臼]] - [[雨竜駅|雨竜]]間復活に伴い営業再開。 * [[1964年]]（昭和39年）[[4月1日]] - [[業務委託駅|業務委託化]]。 * [[1972年]]（昭和47年）[[6月19日]] - 札沼線新十津川 - 石狩沼田間廃止に伴い廃駅となる。 === 駅名の由来 === {{Main2|「徳富」の由来|下徳富駅#駅名の由来}} 地域名の「徳富」に、石狩川を基準にして中流にあることから「上」を冠した<ref>{{Cite book|和書|author=|editor=札幌鉄道局編|title=駅名の起源|date=|year=1939|accessdate=|publisher=北彊民族研究会|page=33|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|id={{NDLJP|1029473}}}}</ref>。 == 駅跡 == [[駅舎]]は農機具倉庫として利用されていたが、[[2008年]]（平成20年）7月に取り壊された。 民間に払下げられた駅長・助役用の官舎が残っている。 駅名標は、郷土資料館に展示されている。 * [[国道275号]] * [[北海道中央バス]]「大和市街」停留所 ** [[ジェイ・アール北海道バス]][[石狩線]]廃止により引き継がれた路線。 == 隣の駅 == ; 日本国有鉄道 : 札沼線 :: [[石狩橋本駅]] -
