江若鉄道の和邇 - 近江木戸間の延伸に伴い、'''比良口駅'''として滋賀郡[[木戸村 (滋賀県)|木戸村]]に開業{{Sfn|今尾|2008|pp=31–32}}[{{NDLDC|2956241/9}} 「地方鉄道運輸開始」『官報』1926年4月15日]（国立国会図書館デジタル化資料）。 * [[1941年]]（昭和16年） - '''蓬萊駅'''に改称{{Sfn|今尾|2008|pp=31–32}}。 * [[1965年]]（昭和40年） - 沿線にサンケイバレイが開業{{Sfn|江若鉄道の思い出|p=59}}。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31–32}}。 == 駅構造 == {{駅配線図開始|title=蓬萊駅線路配置図|left=[[和邇駅 (江若鉄道)|和邇]]}} {{鉄道配線図|0|0|0|quai|quai|0|0|0|0|0|0}} {{鉄道配線図|0|0|courbebg|voie|voie|butoird|quai|quai|quai|BV|0}} {{鉄道配線図|voie|voie|voie|voie|voie|voie|voie|voie|voie|voie|voie}} {{鉄道配線図|0|0|0|0|0|courbehg|voie|voie|voie|courbehd|0}} {{鉄道配線図|0|0|0|0|0|0|quai|quai|quai|0|0}} {{駅配線図終了|right=[[近江木戸駅|近江木戸]]|source={{Sfn|竹内|1967}}|note=本線に接続していない線路は撤去済みを示す}} 蓬萊駅は旅客と貨物の取り扱いを行うことができた[[一般駅]]{{Sfn|寺田|2010|p=10}}。[[プラットホーム|ホーム]]は線路の片側に1面のみ配される（[[単式ホーム]]）{{Sfn|竹内|1967}}{{Sfn|レイル|p=79}}。当初は[[列車交換]]が可能な駅だったが、[[停車場#側線|側線]]が撤去された結果列車交換ができない棒線駅になった{{Sfn|竹内|1967}}{{Sfn|寺田|2010|p=10}}{{Sfn|レイル|p=83}}。 サンケイバレイの最寄り駅であったことから、駅構内には宣伝用の小旗が多く掲げられていた{{Sfn|江若鉄道の思い出|p=58}}。駅は高台にあり、改札口越しには[[琵琶湖]]を望むことができた{{Sfn|レイル|p=83}}。 駅名は「蓬'''萊'''草かんむりに來（「来」の旧字）」と表記するのが正しいが、ホームや駅舎に掲げられていた[[駅名標]]では「蓬'''莱'''」という表記も用いられた{{Sfn|レイル|p=83}}。 == 利用状況 == 初期の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1931年||14,541人||14,702人||195トン||339トン||[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料） |- |1934年||14,850人||15,384人||368トン||110トン||[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料） |- |1935年||14,627人||15,042人||84トン||129トン||[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料） |- |1936年||16,100人||16,372人||53トン||131トン||[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料） |} == 駅周辺 == 駅の周りは木に覆われていた{{Sfn|レイル|p=82}}。比良山地の山々が迫り、山上に向かって伸びるサンケイバレイの[[カーレーター]]は車窓からも見ることができた{{Sfn|レイル|p=82}}{{Sfn|江若鉄道の思い出|p=60}}。なおカーレーターについては、当駅廃止後の1975年（昭和50年）に廃止されている{{Sfn|レイル|p=82}}。 駅のあった場所は、江若鉄道の廃線後に開業した[[湖西線]]の[[蓬萊駅]]に重なる{{Sfn|江若鉄道の思い出|p=58}}。当駅付近の線路跡には湖西線が通っていて、駅の痕跡は残されていない{{Sfn|寺田|2010|p=25}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[和邇駅 (江若鉄道)|和邇駅]] -
