'''雄勝鉄道'''湯沢駅 - 西馬音内駅間開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年8月発行）30,32-33,36ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月16日]] - [[陸上交通事業調整法|交通統合]]に伴い'''横荘鉄道'''の駅となる<ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1944年]]（昭和19年）[[6月1日]] - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''雄勝線'''に制定。それに伴い'''羽後鉄道雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">『私鉄の廃線跡を歩くI』106-109ページより。</ref>。 * [[1947年]]（昭和22年） ** [[7月23日]] - 豪雨による路盤及び橋脚損壊により雄勝線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、ネコ・パブリッシング、2003年11月発行）17ページより。</ref>。 ** [[8月14日]] - 羽後山田駅 - 梺駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 * [[1952年]]（昭和27年）[[2月15日]] - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1973年]]（昭和48年）[[4月1日]] - 雄勝線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML052">『RM LIBRARY 52』4,15,24-25ページより。</ref>。ホームは線路の北側（梺方面に向かって右手側）に存在した<ref name="bunkenRML052"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML052"/>。 [[無人駅]]となっていた<ref name="bunken2008SH-1-R"/><ref name="bunkenRML052"/>が駅舎を有し、構内の北側に位置しホーム中央部分に接していた<ref name="bunkenRML052"/>。あぐりこ稲荷に由来する寺社風の建物であった<ref name="bunkenRML052"/>。ホーム端は梺方、湯沢方ともに[[斜路|スロープ]]を有し<ref name="bunken2008SH-1"/><ref name="bunkenRML052"/>、[[枕木]]による柵も設けられていた<ref name="bunken2008SH-1"/><ref name="bunkenRML052"/>。 駅舎前には[[鳥居]]が建てられていた<ref name="bunken2008SH-1"/>。 == 駅周辺 == 駅名の由来ともなっている<ref name="bunkenRML052"/>、近在の信仰の厚い元稲田稲荷神社<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/><ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）64ページより。</ref>（別名・あぐりこ稲荷<ref name="bunkenRML052"/><ref name="bunkenTouhoku1999"/>）の下車駅で、祭り（初午）の日には大変多くの乗降客があった<ref name="bunkenRML052"/>。周囲は水田であった<ref name="bunkenRML052"/>。 * [[国道398号]]柳田バイパス - 現在の状況。 * 元稲田稲荷神社（あぐりこ稲荷） - 駅北側<ref name="bunkenTouhoku1999"/>の駅前<ref name="bunkenRML052"/>。 * 羽後交通バスあぐりこ[[バス停留所]]<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/> == 駅跡 == 1995年（平成7年）時点では、水田の中に消滅していた<ref name="bunken1995H">書籍『鉄道廃線跡を歩く』（JTBパブリッシング、[[1995年]]11月発行）37ページより。</ref>。2007年（平成19年）5月時点<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1">『新 鉄道廃線跡を歩く1』201ページより。</ref>。 また、当駅跡附近の線路跡は1999年（平成11年）時点では林の南側の[[農道]]に転用されていた<ref name="bunkenTouhoku1999"/>。 == 隣の駅 == ; 羽後交通 : 雄勝線 :: [[羽後三輪駅]] -
