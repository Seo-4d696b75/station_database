[[日本国有鉄道]]深名線の'''下政和仮乗降場'''（[[鉄道管理局|局]]設定）として開業<ref name="hensen"/>。 * [[1961年]]（昭和36年）[[12月1日]] - 200m深川方に移転して'''政和温泉仮乗降場'''（局設定）に改称<ref name="hensen"/><ref>幌加内町史 1971年9月発行、P727。</ref><ref group="注" name="chizu">一方、国土地理院 2万5千分の1地形図では、[http://mapps.gsi.go.jp/contentsImageDisplay.do?specificationId=1300568&isDetail=false 1956年（昭和31年）測量図]・[http://mapps.gsi.go.jp/contentsImageDisplay.do?specificationId=1300569&isDetail=false 1968年（昭和43年）測量図]・[http://mapps.gsi.go.jp/contentsImageDisplay.do?specificationId=1300570&isDetail=false 1983年（昭和58年）測量図]の当駅の推移を見ると、下政和仮乗降場が<u>そのままの位置で</u>政和温泉仮乗降場となり、その後（1968年から1983年の間）に移転した形になっている。</ref>。 * 不明（1966年以降） - 待合室<ref name="bunken2011P"/>設置<ref group="注">幌加内町史 P891、1966年（昭和41年）7月11日に待合室設置を鉄道局へ請願。</ref>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承<ref name="hensen"/>。同時に[[臨時駅]]へ昇格<ref name="hensen"/>（毎年12月1日 - 4月20日の間は全列車通過）。'''政和温泉駅'''となる<ref name="hensen"/>。 * [[1990年]]（平成2年）[[3月10日]] - 冬季休業中のまま廃止<ref name="hensen"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の東側（名寄方面に向かって右手側）に存在した<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]]12月発行）101ページより。</ref>。[[分岐器]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。 [[仮乗降場]]に出自を持つ[[無人駅]]となっており、駅舎はないがホーム中央部分に[[待合所]]を有していた<ref name="bunken2011P"/>。 [[政和温泉]]の泉源がいちど枯渇したために、温泉旅館は1981年（昭和56年）5月に休止した後、1986年（昭和61年）に建物を解体して廃業した<ref>新幌加内町史 2008年10月発行、P1026。</ref>。駅はその後もしばらく営業した後廃止となった<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（小学館、[[1993年]]6月発行）76ページより。</ref>。なお、温泉は場所を変えて町有の観光施設「せいわ温泉ルオント」として1992年（平成4年）に復活している<ref name="choshi1">新幌加内町史 P1024-1027。</ref>。 == 駅名の由来 == 当駅の所在地近辺の温泉名より。この地の入植者が鉱泉を発見し、1918年（大正7年）に温泉免許を取得して温泉宿を開いたが、経営が振るわず、冷泉だったため燃料代に窮した事や主人が亡くなったため一旦廃業となった<ref name="choshi1"/>。1960年（昭和35年）になって町が温泉の成分を分析をしたところ有効成分が認められ、また交通面でも下政和仮乗降場が近くにあることから、町内有志が旧温泉宿の後継者と共に1961年（昭和36年）10月に㈱政和温泉を立ち上げ、町議会にも働きかけて翌1962年（昭和37年）6月に温泉旅館を新築し開業した<ref name="choshi1"/>。またこれに合わせて仮乗降場を温泉旅館の近くへ移転させ、駅名も改めた<ref name="hensen"/><ref group="注" name="chizu"/>。 == 駅周辺 == * [[国道275号]]（空知国道） * [[北海道道938号伊文政和線]] * [[道の駅森と湖の里ほろかない]] ** せいわ温泉ルオント - 当駅の廃駅後の開業。 * 政和温泉 - [[種村直樹]]の著書『鈍行列車の旅』（日本交通公社、1979年）に、1976年（昭和51年）3月に種村自身が訪問した際の様子が当駅とともに紹介されている<ref group="注">逝去後の2015年（平成27年）に刊行された『鈍行最終気まぐれ列車』（実業之日本社）pp.49 - 51にこの内容が再録されている。</ref>。旅館は駅前にある一軒家で、利用客向けに[[ジンギスカン (料理)|ジンギスカン]]が販売されており、腰にタオルを巻いただけの姿だけでも食べることができた<ref>前記種村直樹の文による。</ref>。駅の休止する冬期は温泉も営業を休止していた。旅館は1981年（昭和56年）に泉源の枯渇で休止した後、1986年（昭和61年）に廃業している。 * [[雨竜川]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）15ページより。</ref> == 駅跡 == 2011年（平成23年）時点では駅関連施設はすべて撤去されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）178ページより。</ref>。 また、2000年（平成12年）時点では、駅跡の雨煙別方の雨竜川に架橋されていた鉄骨[[桁橋|ガーダー橋]]である「第三雨竜川橋梁」が残存し<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）35-36ページより。</ref>、2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）41-43ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。この橋は[[公益社団法人]][[土木学会]]により[[土木学会選奨土木遺産]]に選定されている<ref name="bunken2010H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 深名線 :: [[上幌加内駅]] -
