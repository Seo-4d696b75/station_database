'''[[日本国有鉄道|国有鉄道]]富内線'''振内駅 - 当駅間延伸開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1982年]]（昭和57年）[[11月15日]] - 貨物取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。　 * [[1986年]]（昭和61年）[[11月1日]] - 富内線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、富内線の[[終着駅|終端駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）108ページより。</ref>。駅舎側（東側）、外側（西側）共に上下共用（番線表示なし）となっていた。両線はホーム端で収束していた。そのほか旅客の本線それぞれの外側に副本線を各1線ずつ有し、駅舎側の副本線の駅舎傍には貨物用の[[単式ホーム]]を有し<ref name="bunken1983"/>途中で分岐し、本線に合流するほか、行き止まりの[[停車場#側線|側線]]ともなっていた。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の東側に位置し、ホーム北側とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎は鉄骨ブロック建て平屋の、明るく広い近代的な建物であった<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]]10月発行）56ページより。</ref>。 駅前には左手側に公園が設置され、樹齢1,500年の[[イチイ]]の巨木、池、日高町特産の銘石による[[庭石]]が配置されていた<ref name="bunken1980"/>。 予定では当駅から延伸し、[[占冠駅]]を経由して[[根室本線]][[金山駅 (北海道)|金山駅]]附近に接続することになっていたが、延伸部分の新線建設は凍結中で<ref name="bunken1980"/>、そのまま廃線となった。 貨物列車は木材の搬出が主体であった<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当駅の所在する地名より。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は94人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道237号]]（日高国道） * [[国道274号]]（穂別国道/石勝樹海ロード） * [[北海道道847号三岩日高線]] * [[道の駅樹海ロード日高]] * 日高町役場日高総合支所（旧・[[日高町 (北海道)|日高町]]役場） * [[門別警察署]]日高駐在所 * 日高郵便局 * [[北海道日高高等学校]] * 日高町立日高中学校 * 日高町立日高小学校 * 日高神社 * [[苫小牧信用金庫]]日高代理店 * びらとり農業協同組合（JAびらとり）日高支所 * [[沙流川温泉]] * [[国立日高青少年自然の家]] * [[日高山岳ビラパーク]] -
