当駅間の開通に伴い、[[日本の鉄道駅#一般駅|一般駅]]として開業。 * [[1946年]]（昭和21年）[[12月15日]]：当駅 - [[渡島大沢駅]]間の開通に伴い、中間駅となる。 * [[1949年]]（昭和24年）[[6月1日]]：日本国有鉄道法施行に伴い、[[日本国有鉄道]]（国鉄）に継承。 * [[1953年]]（昭和28年）[[11月8日]]：福山線が'''[[松前線]]'''に改称される。 * [[1963年]]（昭和38年）[[12月1日]]：[[準急列車|準急]]「'''松前'''」運行開始。 * [[1980年]]（昭和55年）[[10月1日]]：急行「松前」廃止により、[[優等列車]]が停車しなくなる。 * [[1982年]]（昭和57年）[[11月15日]]：貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]]：[[チッキ|荷物]]の取り扱いを廃止。 * [[1986年]]（昭和61年）11月1日：出札・改札業務を停止し、旅客業務については無人化され、[[日本の鉄道駅#簡易委託駅|簡易委託駅]]となる。閉塞扱いの運転要員が出務する事があったが、原則として千軒駅 - 松前駅間を一閉塞として運行。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]に伴い、日本国有鉄道から[[北海道旅客鉄道]]（JR北海道）に継承。 * [[1988年]]（昭和63年）[[2月1日]]：松前線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。その他に、上下線に副本線（[[停車場#側線|側線]]扱い）を有していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）62ページより。</ref>。 最終営業日まで、[[無人駅]]扱いの運転取り扱い要員のみが配置されていた駅であった。[[乗車券]]類の販売は、[[日本の鉄道駅#簡易委託駅|簡易委託]]化されていた。駅舎はホームの南側に位置し、ホームとは[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。 == 駅名の由来 == {{節スタブ}} 当駅が所在した地名に、[[旧国名#駅名|旧国名]]の「[[渡島国|渡島]]」を付した。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は224人<ref name="bunken1983"/>。 == 駅周辺 == * [[青函トンネル]][[吉岡定点]] -
