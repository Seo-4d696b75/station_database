[[鉄道省|鉄道院]]宗谷線、音威子府駅 - 当駅間の開通に伴い開業<ref name="sone">{{Cite book|和書|author=[[曽根悟]]（監修）|title=週刊 歴史でめぐる鉄道全路線 国鉄・JR|editor=朝日新聞出版分冊百科編集部（編集）|publisher=[[朝日新聞出版]]|series=週刊朝日百科|volume=20号・宗谷本線／留萌本線|pages=14-17|date=2009-11-2}}</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1916年]]（大正5年）[[10月1日]] - 宗谷線の当駅 - 中頓別駅間延伸開通に伴い、中間駅となる<ref name="sone"/>。 * [[1919年]]（大正8年）[[10月20日]] - 路線名を宗谷本線に改称し、それに伴い同線の駅となる<ref name="sone"/>。 * [[1929年]]（昭和4年）[[12月1日]] - [[歌登町営軌道|歌登村営軌道]]当駅 - 上幌別（歌登）間開業 * [[1930年]]（昭和5年）[[4月1日]] - 音威子府駅 - 稚内駅間を宗谷本線から削除し路線名を北見線に改称、それに伴い同線の駅となる<ref name="sone"/>。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1961年]]（昭和36年）4月1日 - 路線名を天北線に改称、それに伴い同線の駅となる<ref name="sone"/>。 * [[1970年]]（昭和45年）[[11月1日]] - 歌登町営軌道廃止。 * [[1982年]]（昭和57年）6月1日 - 貨物の取り扱いを廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）4月1日 - [[国鉄分割民営化]]により、北海道旅客鉄道（JR北海道）の駅となる<ref name="sone"/>。 * [[1989年]]（平成元年）[[5月1日]] - 天北線の全線廃止に伴い、廃駅となる<ref name="sone"/>。 === 駅名の由来 === 当駅が所在した地名より。地名の「頓別」はアイヌ語の「トウ・ウン・ベツ」（沼から出る川）に由来し、付近一帯を頓別村と称していたが、植民の際に付近の地名と区別をするため「小」を冠した<ref>{{Cite book|和書|title=駅名の起源 |editor=札幌鉄道局編 |year=1939 |publisher=北彊民族研究会 |id={{NDLJP|1029473}} |page=84 }}</ref>。 == 駅構造 == 廃止時点で、2面2線の[[単式ホーム]]を有する[[地上駅]]であった。ホームが千鳥式に配置された、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）188ページより。</ref>。互いのホームは、駅舎側ホーム西側と対向ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側ホーム（南側）が上り線、対向ホーム（北側）が下り線となっていた<ref name="bunken1983"/>。そのほか上り線の音威子府方から分岐し上り線ホームはす向かいの貨物ホームへの貨物[[停車場#側線|側線]]を1線、本線の音威子府方から分岐し下り線の外側への貨物側線を2線有していた<ref name="bunken1983"/>。かつては豊富な林産資源を背景に多くの側線を有したが、末期にはその多くは撤去されていた。また1983年（昭和58年）時点で、[[分岐器|転轍機]]が撤去された[[専用鉄道|専用線]]が残存していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の南側に位置し両ホームとは通路及び構内踏切で連絡した。 1970年（昭和45年）までは、北海道開拓のため建設された[[殖民軌道]]の一つであった[[歌登町営軌道]]が接続していたが、路線の一部区間が重複する[[美幸線]][[未成線|未成区間]]建設促進のため廃止された。 [[急行列車|急行]]「[[宗谷 (列車)#宗谷本線・天北線優等列車沿革|天北]]」の停車は[[歌登町|歌登]]・[[枝幸町|枝幸]]方面へ向かう乗客の便宜を図ったもので、駅前からは枝幸発着の[[宗谷バス]]が連絡していた。 == 利用状況 == * 1981年度（昭和56年度）の1日当たりの乗降客数は67人<ref name="bunken1983"/>。 {| class="wikitable" style="margin: 1em 0.2em; text-align: center; font-size: 85%;" |- style="background: #ddd;" !colspan="2"|乗車人員推移<ref name="choshi">中頓別町史 平成9年5月発行 P397/8,855</ref> |- !年度 !1日平均人数 |- |1921 |63 |- |1935 |80 |- |1953 |133 |} 歌登町営軌道 小頓別駅 {| class="wikitable" style="margin: 1em 0.2em; text-align: center; font-size: 85%;" |- style="background: #ddd;" !colspan="2"|乗車人員推移<ref name="choshi"/> |- !年度 !1日平均人数 |- |1921 |… |- |1935 |23 |- |1953 |18 |} == 駅周辺 == 小市街を形成している。[[旧丹波屋旅館]]（2000年、[[登録有形文化財]]に登録）がかつての繁栄の痕跡として残る。 * [[北海道道400号小頓別停車場線]] * [[北海道道12号枝幸音威子府線]] * [[国道275号]]（頓別国道） * 小頓別郵便局 * 天北峠 -
