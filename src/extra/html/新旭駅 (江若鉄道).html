[[近江今津駅 (江若鉄道)|近江今津]]間の開通に合わせて開業した{{Sfn|江若鉄道の思い出|p=108}}{{Sfn|ありし日の江若鉄道|p=24}}。この区間は当初1930年（昭和5年）8月末の開通を見込んでいたものの、安曇駅と当駅との間にある安曇川橋梁の架設に時間を要したことなどから開通が翌年にずれ込むこととなった{{Sfn|江若鉄道の思い出|p=108}}{{Sfn|ありし日の江若鉄道|p=24}}。開業当初の駅名は当時の村名（高島郡[[新儀村]]）をとって'''新儀駅'''（しんぎえき）としたが、村が1955年（昭和30年）に合併して[[新旭町]]となると、駅名もそれに従って改称されている{{Sfn|江若鉄道の思い出|p=109}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1931年]]（昭和6年）[[1月1日]] - 安曇駅から近江今津駅までの延伸開業に伴い、'''新儀駅'''として高島郡新儀村に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957723/4}} 「地方鉄道運輸開始」『官報』1931年3月9日]（国立国会図書館デジタル化資料）</ref>。 * 時期不詳 - '''新旭駅'''に改称<ref>{{Harvnb|今尾|2008|p=32}}では1952年から1962年の間とする。</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=新旭駅線路配置図|left=[[安曇川駅 (江若鉄道)|安曇川]]}} {{鉄道配線図|0|0|0|0|courbebg|voie|voie|voie|courbebd|0|0|0|0|0}} {{鉄道配線図|0|0|0|Mvoie|0|quai|quai|quai|0|Dvoie|0|0|0|0}} {{鉄道配線図|0|0|Mbifd|voie|voie|voie|voie|voie|voie|voie|Dbifg|0|0|0}} {{鉄道配線図|voie|bifhd|voie|voie|voie|voie|voie|voie|voie|voie|voie|bifhg|bifbg|voie}} {{鉄道配線図|0|0|0|0|0|quai|quai|quai|quai|quai|0|Mvoie|0|0}} {{鉄道配線図|0|0|0|0|0|0|0|quai|butoirg|voie|courbehd|0|0|0}} {{鉄道配線図|0|0|0|0|0|BV|0|quai|quai|quai|0|0|0|0}} {{駅配線図終了|right=[[饗庭駅|饗庭]]|source={{Sfn|竹内|1967}}}} 新旭駅は2面の[[プラットホーム|ホーム]]が線路を挟んで向かい合う[[相対式ホーム]]という構造であった{{Sfn|竹内|1967}}{{Sfn|寺田|2010|p=15}}。[[列車交換]]が可能な交換駅である{{Sfn|寺田|2010|p=10}}。旅客と貨物の双方を取り扱うことができた[[一般駅]]であり{{Sfn|寺田|2010|p=10}}、駅には[[停車場#側線|側線]]や[[貨物上屋]]が設けられていた{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=85}}。 駅員が配置されていた有人駅である{{Sfn|レイル|p=85}}。 == 利用状況 == 開業から数年間の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1931年||30,132人||29,870人||1,098トン||1,023トン||<ref>[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料）</ref> |- |1934年||33,371人||33,613人||1,257トン||1,860トン||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||35,413人||35,432人||876トン||1,780トン||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||39,609人||39,519人||1,062トン||1,879トン||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅の所在地である北畑やその周辺は新旭町の中心部で、町役場や中学校、銀行など町の公共施設や商店が集積していた{{Sfn|江若鉄道の思い出|p=109}}。ただ駅の周辺に人家は見られなかった{{Sfn|レイル|p=85}}。 駅があった場所は、江若鉄道の廃線後に開業した[[湖西線]]の[[新旭駅]]から500メートルほど南方にあたり、2015年現在は[[大津第一交通]]高島営業所が付近に立地している{{Sfn|江若鉄道の思い出|p=109}}{{Sfn|寺田|2010|p=26}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[安曇川駅 (江若鉄道)|安曇川駅]] -
