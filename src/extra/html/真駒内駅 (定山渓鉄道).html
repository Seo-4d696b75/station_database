}} '''真駒内駅'''（まこまないえき）は[[北海道]][[札幌市]][[南区 (札幌市)|南区]]にあった[[定山渓鉄道線]]の[[鉄道駅|駅]]である。同線の[[廃線]]により[[1969年]]（昭和44年）に[[廃駅]]となった。 == 概要 == [[定山渓鉄道線|定山渓鉄道]]の真駒内駅は、現在の[[札幌市営地下鉄南北線|地下鉄南北線]][[自衛隊前駅]]と[[真駒内駅]]の中間地点に設置されていた。 設置当時、周辺は官営の[[真駒内種畜場]]であり、種畜場の建物に最も近い位置に当初は停留所として設置された<ref name="kubo p82">[[#kubo|久保 『定山渓鉄道』p.82]]</ref>。種畜場の建物は500 mほど離れた場所にあり、一面の牧草地の中にホームがあるだけの状態であった<ref name="sapporo p239">[[#sapporo|さっぽろ文庫 11『札幌の駅』p.239]]</ref>。 また、当時の真駒内の集落は[[豊平川]]岸の真駒内下町（現在の真駒内本町に相当）にあり、現在の地下鉄真駒内駅周辺に相当する地区は開けていなかった。駅はこの町に近くなるように置かれたが、その下町との間には真駒内種畜場が広がっており、設置時に駅周辺に人家は一つもなかった。 戦前までは、駅の裏手には小高い丘が広がっており、多くの[[サクラ|桜]]が自生していたことから「桜山」と呼ばれていた。駅開設後は花見の名所として知られ、秋の紅葉と併せて多くの行楽客で賑わった<ref name="sapporo p239"/><ref name="kubo p83"/><ref>[[#kubo|久保 『定山渓鉄道』p.87-88]]</ref>。 戦後は、進駐軍が牧場を接収し、キャンプ・クロフォードを設営し<ref>岡田利夫 『戦中戦後20年 北海道木材・林業の変遷』 152頁 北海道林材新聞刊 昭和63年6月10日刊</ref>、当駅からキャンプ・クロフォードへの専用の引き込み線が設置された<ref name="kubo p39">[[#kubo|久保 『定山渓鉄道』p.39]]</ref><ref name="kubo p83"/>。 == 歴史 == * [[1920年]]（[[大正]]9年）4月1日 - [[定山渓鉄道線]]の停留所として、[[豊平駅]] - [[石切山駅]]間<ref group="注釈">真駒内駅の設置当初、[[澄川駅 (定山渓鉄道)|澄川駅]]、[[慈恵学園停留所]]、[[緑ヶ丘停留所]]は未開業であった。</ref>に設置される<ref name="kubo p82"/>。設置当初は客扱いは4月から11月までの期間のみであった<ref name="kubo p82"/>。 * [[1929年]]（[[昭和]]4年） - 電化に伴い、行き違い可能な停車場となる<ref name="kubo p82"/>。 * [[1946年]]（昭和21年） - 当駅からキャンプ・クロフォードへの専用線が設置される<ref name="kubo p39"/>。 * [[1969年]]（昭和44年）11月1日 - 定山渓鉄道線の全線廃止に伴い廃駅。 == 駅構造 == 停留所としての設置当初は枕木を積んだ板張りのホームのみだったが<ref name="kubo p82"/>、電化に伴う輸送力増強のため、行き違い可能な島式ホーム、側線、信号が設置され停車場となり、駅舎と駅員住宅が建てられた<ref name="kubo p82"/>。また、荷物積み下ろし専用ホームも設けられ、真駒内種畜場の家畜の運搬に利用された<ref name="kubo p83">[[#kubo|久保 『定山渓鉄道』p.83]]</ref>。 <!--さらに貨物線を加えて4本の線路を持った。--> 昭和21年に設置されたキャンプ・クロフォードへの専用線は、[[日本国有鉄道|国鉄]]が運行管理し定山渓鉄道が線路保守を行う方法がとられ、国鉄から直通の進駐軍専用列車が運行されていた<ref name="kubo p83"/>。 == 駅跡 == 駅跡は広い敷地を活用し[[札幌市営地下鉄南北線]]の南車両基地への分岐箇所となっている。[[平岸通]]を挟んだ向側にはかつて[[日本通運|日通]]の荷物取扱所であった建物が現存している<ref name="kubo p85">[[#kubo|久保 『定山渓鉄道』p.85]]</ref>。 == 隣の駅 == ; 定山渓鉄道（廃止） : 定山渓鉄道線 :: [[慈恵学園停留所]] -
