駅に昇格し、同時に改称。'''北遠軽駅'''となる。旅客のみ取り扱い。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、国鉄からJR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の東側（遠軽方面に向かって左手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）213ページより。</ref>。 [[無人駅]]となっており、駅舎は無いがホーム中央部分に高床式の[[待合所]]を有していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在する地（遠軽町）の北方に位置していたために<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）188ページより。</ref>、「北」を冠した。 仮乗降場時代の「学田」は、この地に最初に入植してきたキリスト教団体が、私立大学建設を目的とする北海道同志教育会を設立し、その開拓農場を運営するに当たって入植者に農業の傍ら道徳や教育を重視するなど同教育会の主旨の理解と義務を課したことから、この開拓地を学田と名付けたことによる<ref>遠軽町史 1977年（昭和52年）10月発行、P159-207。</ref>。なお仮乗降場から駅に昇格した折は、[[富良野線]]に一足先に[[学田駅]]が設置されていたことから駅名変更に至ったと思われるが詳細不明。 == 駅周辺 == * [[国道242号]]（置戸国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * [[湧別川]]<ref name="bunken1980M"/> * 社名渕川<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「学田3丁目」停留所 == 駅跡 == 2001年（平成13年）時点では何も残っていない<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（[[JTBパブリッシング]]、[[2001年]][[8月]]発行）36ページより。</ref>。2011年（平成23年）時点でも同様で、雑草が生える更地となっていた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）129-130ページより。</ref>。駅跡の遠軽方には、線路跡の築堤と防雪林が残存していた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[開盛駅]] -
