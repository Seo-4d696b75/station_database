当駅 - 舘合駅間延伸開通に伴い中間駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1944年]]（昭和19年）6月1日 - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1947年]]（昭和22年） ** 7月23日 - 豪雨による路盤および橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）16-17ページより。</ref>。 ** 7月29日 - 横手駅 - 舘合駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1952年]]（昭和27年）2月15日 - 横荘線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、当駅 - 舘合駅間部分廃線前までは[[列車交換]]可能な交換駅であった<ref name="bunkenRML061">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）6,10-11,19,25,30-31ページより。</ref>。駅舎側（西側）が下り線（老方方面）、外側（東側）が上り線（横手方面）となっていた<ref name="bunkenRML061"/>。そのほか[[停車場#側線|側線]]として、下り線から西に分岐し駅舎との間を通過する[[機回し線]]を1線と、機回し線横手方から西に分岐し駅舎南側に至る行き止りの側線を1線有していた<ref name="bunkenRML061"/>。浅舞駅とほぼ同じ配線となっていた<ref name="bunkenRML061"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の西側に位置し、ホーム北側の[[階段]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML061"/><ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]]9月発行）35ページより。</ref>。ホームは[[待合所]]を有した<ref name="bunkenRML061"/>。待合所の隣、出入口側のホーム上に[[鉄道信号機#機械式信号機|腕木式信号機]]が設置されていた<ref name="bunkenRML061"/>。 列車交換の通標は浅舞駅 - 当駅間は「□」、当駅 - 羽後大森駅間は「△」であった<ref name="bunkenRML061"/>。 駅名は、駅舎出入口上部に掲示されていた駅銘板には「沼'''館'''駅」と表記されていたが<ref name="bunken1996H"/>、羽後交通の資料には「沼'''館'''」「沼'''舘'''」の双方が使用されていた<ref name="bunkenRML061"/>。 == 駅周辺 == 旧・雄物川町（さらに以前は沼館町）の中心駅であった<ref name="bunkenRML061"/>。 * [[秋田県道13号湯沢雄物川大曲線]] -
