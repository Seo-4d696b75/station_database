[[相模鉄道]] '''東河原駅'''として開業。当初は砂利を取り扱う[[貨物駅]]。 *[[1939年]]（[[昭和]]14年） **10月1日 - '''昭和産業駅'''に改称。 *[[1940年]]（昭和15年） **4月20日 - 旅客営業開始。 *[[1942年]]（昭和17年） **10月30日 - '''四之宮口駅'''に改称。 * [[1944年]]（昭和19年） **6月1日 - [[鉄道省]]による国有化により[[鉄道省]][[相模線]]の駅となる。同時に'''西寒川駅'''に改称。当駅から四之宮駅間が廃止され終着駅となる。 *[[1946年]]（昭和21年）頃 - 旅客営業が行われなくなる。 *[[1954年]]（昭和29年） **10月1日 - 旅客営業を正式に廃止<ref>1954年（昭和29年）9月29日日本国有鉄道公示第285号「相模線寒川・西寒川の旅客運輸営業廃止」</ref>。 *[[1960年]]（昭和35年） **[[11月15日]] - 旅客営業を再開<ref>1960年（昭和35年）11月12日日本国有鉄道公示第572号「相模線西寒川停車場において旅客の取扱を開始する件」</ref>。 *[[1965年]]（昭和40年） **10月1日 - 営業範囲を「旅客及び同停車場接続専用線発着貨物」から「旅客及び同停車場接続専用線発着車扱貨物」へ改正<ref>1965年（昭和40年）9月29日日本国有鉄道公示第571号「停車場の営業範囲を改正」</ref>。 *[[1970年]]（昭和45年） **7月10日 - 貨物取扱を廃止し、営業範囲を「旅客」へ改正<ref>1970年（昭和45年）7月9日日本国有鉄道公示第276号「停車場の営業範囲の改正」</ref>。 *[[1984年]]（昭和59年） **[[3月31日]] - 廃止<ref name="kouji216" />。 == 駅跡地 == {{See also|一之宮公園}} [[画像:Nishi-Samukawa Stn. Monument.JPG|200px|thumb|right|西寒川駅跡にある記念碑]] 寒川支線の跡地は、本線からの分岐点からのほぼ全てが「一之宮緑道」として整備されている。一部区間には線路も残され、車輪などが展示されている。また、一部のレールは[[西暦]]ではなく[[皇紀]]での刻印がなされている<ref>{{Cite web|title=一般社団法人　寒川町観光協会　思い出の西寒川支線・西寒川駅跡|url=http://www.samukawa-kankou.jp/entry.html?id=229575|accessdate=2018-08-04}}</ref>。 西寒川駅の跡は「八角広場」という小さな[[公園]]になっており、[[線路 (鉄道)|線路]]が一部残されている。当地には、当駅と[[海軍工廠|相模海軍工廠]]跡の[[記念碑]]がある。 [[2002年]]に旧相模海軍工廠で製造されたと思われる化学兵器が発掘され、話題になった。 == 隣の駅 == ; 日本国有鉄道 : 相模支線 :: [[寒川駅]] - '''西寒川駅''' == アクセス == * 寒川駅南口より[[神奈川中央交通]]バス<茅53>茅ケ崎駅行きで「八角広場前」下車。 * [[茅ケ崎駅]]（北口）より神奈川中央交通バス<茅53>寒川駅南口行きで「八角広場前」下車。 == 脚注 == {{脚注ヘルプ}} {{Reflist}} == 関連項目 == * [[廃駅]] * [[日本の鉄道路線一覧]] == 外部リンク == [http://www.samukawa-kankou.jp/entry.html?id=229575 一般社団法人　寒川町観光協会ホームページ] {{相模線|mode=1}} {{eki-stub}} {{DEFAULTSORT:にしさむかわ}} [[Category:神奈川県の廃駅]] [[Category:日本の鉄道駅 に|しさむかわ]] [[Category:日本国有鉄道の廃駅]] [[Category:相模鉄道の廃駅|しのみやくち]]<!--買収当時の駅名--> [[Category:1923年開業の鉄道駅]] [[Category:1984年廃止の鉄道駅]] [[Category:相模線|廃にしさむかわ]] [[Category:寒川町の交通|廃にしさむかわえき]] sjix0ge9693jb6vhoae4vpyhokrgh7k
