紋別 - [[興部駅]]間の急行「[[天都 (列車)|天都]]」（[[網走市]]の[[天都山]]より命名）も設定されていたほか、[[渚滑線]]の列車の多くが当駅まで乗り入れていた。 また、1943年（昭和18年）から1948年（昭和23年）の短い期間であるが、南西方向の藻鼈川（モベツ川）上流の[[鴻之舞鉱山]]から、[[簡易軌道]]の[[鴻紋軌道]]が本駅裏に駅を設けて接続していて、人や物資を運んでいた。 なお、当駅の[[みどりの窓口]]に設置されていた[[マルス (システム)|マルス]]は通常の専用回線を用いるものではなく、[[マルス (システム)#マルス105|マルス105]]のK型端末〈通称・簡易マルス〉と呼ばれていた[[鉄道電話]]回線を利用するものであった。 == 歴史 == * [[1921年]]（大正10年）[[3月25日]] - [[鉄道省]]名寄東線の中湧別 - 興部間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1968年]]（昭和43年）[[10月1日]] - コンテナ基地設置<ref>{{Cite book|和書|editor=日本国有鉄道旭川鉄道管理局 |title=旭川・鉄道八十八年の歩み |date=1987-3 |publisher= |isbn= |page= }}</ref>。 * [[1971年]]（昭和46年） - 駅舎改築。PC造駅舎となる。 * [[1984年]]（昭和59年）[[2月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、日本国有鉄道から北海道旅客鉄道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]を複合した計2面3線のホームと線路を有する[[地上駅]]であり、[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=211}}</ref>。互いのホームは、駅舎側ホーム北側と島式ホーム北側を結んだ[[跨線橋]]で連絡した<ref name="bunken1983"/>。駅舎側単式ホーム（東側）が下りの1番線、上屋を有した<ref name="bunken2011P">{{Cite book|和書|author=工藤裕之 |title=北海道廃止ローカル線写真集 追憶の鉄路 |date=2011-12 |publisher=[[北海道新聞社]] |isbn=978-4894536197 |page=p.124,130,134}}</ref>島式ホーム駅舎側が上りの2番線、外側が1983年（昭和58年）時点では[[停車場#側線|側線]]扱いの3番線となっていた<ref name="bunken1983"/>。3番線の外側（西側）に旧貨物側線を2線有し、そこから行き止まりの側線も南北に各1線有した<ref name="bunken1983"/>。そのほか、1番線の遠軽方から分岐し駅舎南側のホーム[[切欠きホーム|切欠き]]部分の貨物ホームへの貨物側線を2線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の東側に位置し単式ホーム中央部分に接していた<ref name="bunken1983"/>。ホームの有効長は120mあった<ref name="bunken1983"/>。[[士別駅]]と同規格のコンクリート製の駅舎であった。 [[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「モ・ペッ」（静かな川）に由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=187}}</ref>。地名起源となった川は、元紋別駅附近を流れる藻鼈川である<ref name="bunken2004N"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は800人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M-2">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=35}}</ref> * [[北海道道713号中渚滑紋別停車場線]]（旧・紋別停車場線）<ref name="bunken1980M-2"/> * 紋別市役所 * [[紋別警察署]] * 紋別警察署中央交番 * [[紋別郵便局]] * 紋別市立紋別中学校<ref name="bunken1980M-2"/> * 紋別市立紋別小学校<ref name="bunken1980M-2"/> * [[北海道労働金庫]]紋別支店 * [[北見信用金庫]]紋別支店<small>（旧：[[紋別信用金庫]]本店）</small> * [[遠軽信用金庫]]紋別支店 * [[北洋銀行]]紋別支店 * [[北海道銀行]]紋別支店 * オホーツクはまなす農業協同組合（JAオホーツクはまなす）本所 * 紋別漁業協同組合 * 紋別公園 -
