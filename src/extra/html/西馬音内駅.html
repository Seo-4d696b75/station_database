'''雄勝鉄道'''湯沢駅 - 当駅間開通に伴い開業<ref name="bunken2008M-02">書籍『日本鉄道旅行地図帳 全線全駅全廃線 2 東北』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]6月発行）43ページより。</ref><ref name="bunken2010H-1-R">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）222ページより。</ref><ref name="bunkenKW03">書籍『新 消えた轍 3 東北』（著：寺田裕一、[[ネコ・パブリッシング]]、2010年8月発行）30,32-33,36ページより。</ref><ref name="bunken2008SH-1-R">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、2007年9月発行）165ページより。</ref>。 * [[1935年]]（昭和10年）[[2月13日]] - 当駅 - 梺駅間延伸開通（雄勝鉄道全通）に伴い中間駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1943年]]（昭和18年）[[10月16日]] - [[陸上交通事業調整法|交通統合]]に伴い'''横荘鉄道'''の駅となる<ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/>。 * [[1944年]]（昭和19年）[[6月1日]] - 鉄道会社名を'''羽後鉄道'''に改称。路線名を'''雄勝線'''に制定。それに伴い'''羽後鉄道雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">『私鉄の廃線跡を歩くI』106-109ページより。</ref>。 * [[1947年]]（昭和22年） ** [[7月23日]] - 豪雨による路盤および橋脚損壊により雄勝線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2">書籍『RM LIBRARY 52 羽後交通雄勝線』（著：若林宣、ネコ・パブリッシング、2003年11月発行）17ページより。</ref>。 ** [[8月14日]] - 羽後山田駅 - 梺駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML052-2"/>。 * [[1951年]]（昭和26年）[[7月18日]] - 信号機増設および第二種連動装置設置を行い交換駅となる<ref name="bunkenKW03"/><ref name="bunkenRML052">『RM LIBRARY 52』4,15,19-21,23,27,31,34,38,46ページより。</ref>（日付は認可日<ref name="bunkenRML052"/>）。 * [[1952年]]（昭和27年）[[2月15日]] - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通雄勝線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1967年]]（昭和42年）[[12月1日]] - 当駅 - 梺駅間部分廃線に伴い[[終着駅]]となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1970年]]（昭和45年）[[10月1日]] - 車扱貨物営業が休止となる<ref name="bunkenRML052"/>。 * [[1972年]]（昭和47年）[[9月14日]] - 廃止に向け駅設備を縮小<ref name="bunkenRML052"/>、駅舎およびホーム改築<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1973年]]（昭和48年）[[4月1日]] - 雄勝線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。そのほか[[停車場#側線|側線]]として、発着線の南側に1線と、その南側に末端部分に車庫の建屋を有する車庫線を1線有していた<ref name="bunkenRML052"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の北側に位置していた。廃止時点の構内設備は、路線廃止に向けて、廃止に先駆けて構内設備の縮小化（解体）がされた結果のもので、ホーム、駅舎、車庫のいずれも仮設のもので<ref name="bunkenRML052"/>、ホームは板敷、駅舎は[[プレハブ工法|プレハブ]]造りの簡易な建物となっていた<ref name="bunkenRML052"/>。 構内設備の縮小化前までは、[[島式ホーム]]1面2線を有し、当駅 - 梺駅間部分廃線までは[[列車交換]]可能な交換駅であった<ref name="bunkenRML052"/>。外側（南側）が下り本線（梺方面）、駅舎側（北側）が上り本線（湯沢方面）となっていた<ref name="bunkenRML052"/>。そのほか側線として、本線湯沢方から北側に分岐し駅舎東側の貨物積卸場および駅舎との間を通過し上り本線に合流する[[貨物線]]を1線と、下り本線湯沢方から南側に分岐しさらに2線に分岐、構内西側の末端部分に車庫（車輌基地）を有す行き止まりの車庫1番線、車庫2番線を有した<ref name="bunkenRML052"/>。 駅舎は構内の北西側に位置し、ホーム西側の[[斜路|スロープ]]とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML052"/>。2階建ての特徴ある建物であった<ref name="bunkenRML052"/>。ホームは[[切妻造|切妻]][[屋根#形状による分類|屋根]]の上屋を有した<ref name="bunkenRML052"/>。この上屋は開業時には作られていなかった<ref name="bunkenRML052"/>。 構内には車輌基地である「西馬音内電車区」が併設されていた<ref name="bunkenRML052"/>。 1971年（昭和46年）7月20日の内燃化以降、雄勝線の廃止に向けて駅の縮小化が始まり、1972年（昭和47年）9月14日には設備を約50m湯沢方に移動させプレハブ駅舎と板張りホームを設置<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、旧駅舎は取り壊され跡地には[[ボウリング]]場が建築された<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。島式ホーム、車庫も取り壊された<ref name="bunkenRML052"/>。 閉塞取扱廃止前までの列車交換の通標は、羽後三輪駅 - 当駅間は「□」、当駅 - 梺駅間は「△」であった<ref name="bunkenRML052"/>。また、1965年（昭和40年）12月8日<ref name="bunken2008SH-1-R"/>までの列車交換の通標は、羽後三輪駅 - 当駅間は「□」、当駅 - 梺駅間は「◇」であった<ref name="bunkenRML052"/>。 雄勝鉄道時代は、本社も置かれていた<ref name="bunkenRML052"/>。 == 駅周辺 == 当線最大の駅で<ref name="bunkenTouhoku1999"/>、中間駅では最も乗降客が多かった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。雄勝郡羽後町（さらに以前は旧・雄勝郡西馬音内町）の中心駅であった。 * [[秋田県道276号下開清水線]] - 現在の状況。 * [[秋田県道57号十文字羽後鳥海線]] - 現在の状況。 * [[国道398号]]柳田バイパス - 現在の状況。 * 羽後町役場<ref name="bunkenRML052"/> * 西馬音内郵便局 * 川原田簡易郵便局 * [[羽後町立羽後病院]] * [[羽後町立羽後中学校]] * 羽後町立西馬音内小学校 * 羽後町歴史民俗資料館 * 西馬音内川<ref name="bunkenRML052"/> == 駅跡 == 1995年（平成7年）時点では、[[こまち農業協同組合|農協]]巨福会館になっていた<ref name="bunken1995H">書籍『鉄道廃線跡を歩く』（[[JTBパブリッシング]]、[[1995年]]11月発行）37ページより。</ref>。また、建物の片隅に駅跡の記念碑が建立され<ref name="bunken1995H"/>、駅前で焼き餅を販売していた「三浦屋」の店先には当地で解体された車輌の[[マスター・コントローラー]]が保存されていた<ref name="bunken1995H"/>。1999年（平成11年）時点でも同様で、その他にAコープ西馬音内にもなっており、このAコープとJA西馬音内の建物の間に先の記念碑があった<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）64ページより。</ref>。この記念碑には前面に「雄勝鉄道・西馬音内駅跡」と、横に開業および廃止年月日が記載されていた<ref name="bunkenTouhoku1999"/>。また巨福会館の裏に線路跡が道路に転用されており<ref name="bunkenTouhoku1999"/>、ホーム跡地には食堂が建築されていた<ref name="bunkenTouhoku1999"/>。2007年（平成19年）5月時点でも同様で<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）時点では記念碑が新しくなっていた<ref name="bunken2010H-1">『新 鉄道廃線跡を歩く1』200-201ページより。</ref>。 また、当駅跡附近の線路跡は1999年（平成11年）時点では西馬音内小学校附近までの部分が道路に転用されていた<ref name="bunkenTouhoku1999"/>。 そのほか、1995年（平成7年）時点では、西馬音内の南にある浅井という集落の農家の片隅にロータリー式[[雪掻車]]'''ユキ3形'''の廃車体が置かれていた<ref name="bunken1995H"/>。2007年（平成19年）5月時点も同様で<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>、2010年（平成22年）時点でも崩れかけながらも残存していた<ref name="bunken2010H-1"/>。 == 隣の駅 == ; 羽後交通 : 雄勝線 :: [[あぐりこ駅]] -
