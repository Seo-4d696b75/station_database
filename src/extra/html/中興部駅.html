[[鉄道省]]名寄線の上興部 - 興部間の延伸開通に伴い開業<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2954862/2 『官報』 1921年09月27日　鉄道省告示第129号]（国立国会図書館デジタルコレクション）</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 線路名を名寄本線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1968年]]（昭和43年）[[12月24日]] - 駅舎改築。 * [[1974年]]（昭和49年）[[10月1日]] - 貨物取扱い廃止。 * [[1978年]]（昭和53年）[[12月1日]] - [[チッキ|荷物]]取扱い廃止。同時に出札・改札業務を停止し、旅客業務について無人化。但し、閉塞扱いの運転要員は継続配置（JR化後に入場券のみ発売）。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[相対式ホーム]]2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは、駅舎側ホーム中央部分と対向側ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）209ページより。</ref>。駅舎側（南西側）ホームが上り線、対向側ホームが下り線となっていた<ref name="bunken1983"/>。そのほか、下り線の遠軽方から対向側ホーム外側に分岐した行き止まりの[[停車場#側線|側線]]を1線有していた<ref name="bunken1983"/>。 [[無人駅]]扱いの運転取扱い要員のみが配置されていた駅であった。駅舎は構内の南西側に位置し、上り線ホーム中央部分に接していた。 JR化後は、入場券のみ発売を行っていた。 == 駅名の由来 == 当駅が所在する地名より。地名は、興部川の中流に位置したため<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）186ページより。</ref>、「興部」に「中」を冠する。 == 駅周辺 == * [[国道239号]]（天北国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * 興部川<ref name="bunken1980M"/> * [[名士バス]]「中興部」停留所 == 駅跡 == 2000年（平成12年）時点で木造駅舎が完全な形で残存し、ホームと植え込み、構内の信号施設も残存していた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）40ページより。</ref>。2010年（平成22年）時点でも同様で<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）34ページより。</ref>、2011年（平成23年）時点では別棟のトイレの残存も確認出来た<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）117-118ページより。</ref>。ホーム側は[[軌条|レール]]は撤去されたがホーム上は現役当時の状態であった<ref name="bunken2011H"/>。現在は個人所有となっており<ref name="bunken2011H"/>、駅舎内は非公開となっている<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]][[12月]]発行）143ページより。</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[六興駅]] -
