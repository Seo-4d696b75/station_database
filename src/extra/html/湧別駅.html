[[鉄道省|鉄道院]]湧別軽便線社名淵（後の[[開盛駅|開成]]） - 当駅間延伸開通に伴い'''下湧別駅'''（しもゆうべつえき）として開業。[[一般駅]]。 * [[1922年]]（大正11年）[[9月2日]] - 線路名を湧別線に改称、それに伴い同線の駅となる。 * [[1930年]]（[[昭和]]5年）[[2月12日]] - [[湧別軌道]]当駅 - 丁寧間が開業。 * [[1932年]]（昭和7年）[[10月1日]] - [[遠軽駅]] - 当駅間が名寄本線に編入され同線の駅となる。 * [[1939年]]（昭和14年） ** [[8月1日]] - 湧別軌道の軌道運輸営業廃止が許可。 ** [[9月1日]] - 湧別軌道の廃止が[[官報]]掲載（廃止公示）。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1954年]]（昭和29年）[[11月10日]] - '''湧別駅'''に改称。 * [[1978年]]（昭和53年）[[12月1日]] - [[貨物]]取扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止。 * [[1986年]]（昭和61年）[[11月1日]] - 無人（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の廃線に伴い廃止となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の北西側（湧別方面に向かって左手側）に存在した。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）212ページより。</ref>。かつては貨物用の[[停車場#側線|側線]]を2線有しており、駅舎側から1、2、3番線となっていた（1975年（昭和50年）3月時点ではこの配線）<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）40ページより。</ref>。そのほか1番線の延長上から遠軽方に戻る形で駅舎側に分岐し駅舎北側の貨物ホームに至る側線を1線有していた<ref name="bunken2010HE"/>（以上の側線は1981年（昭和56年）3月時点では撤去済みであった<ref name="bunken2010HE"/>）。 [[無人駅]]（簡易委託駅）となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の北西側に位置し、ホームとは通路で連絡した<ref name="bunken1983"/>。開業当初からの<ref name="bunken1980">書籍『終着駅 国鉄全132』（[[雄鶏社]]、[[1980年]]10月発行）26-27ページより。</ref>、民家の様な木造平屋建ての建物であった<ref name="bunken2010HE"/>。ホームの有効長は旅客ホームが50 m、貨物ホームが30 mであった<ref name="bunken2010HE"/>。 == 駅名の由来 == 当駅の所在する地名より。地名は、[[アイヌ語]]の「ユペ・オッ」（カレイサメが多い）に由来する<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]]2月発行）186ページより。</ref>。別説として「イペ・オッ・イ」（魚の豊富である所）に由来するという説もある<ref name="bunken2004N"/>。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は55人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道656号湧別停車場サロマ湖線]] * [[北海道道204号湧別上湧別線]] * 湧別町役場湧別総合支所（旧・湧別町役場） * 湧別郵便局 * 湧別町運動公園 * [[オホーツク海]] - ホームからも望めた<ref name="bunken1980"/>。 * [[サロマ湖]] * [[湧別川]] == バス路線 == * [[北海道北見バス]]、[[北紋バス]]、[[湧別町営バス]]「湧別」停留所 - 北海道道204号沿いに待合室付きの停留所が整備される。 == 駅跡 == 駅跡地には遠軽地区広域組合消防署湧別出張所及び湧別町文化センターさざ波が建設されており<ref name="bunken2001H">書籍『鉄道廃線跡を歩くVIII』（JTBパブリッシング、[[2001年]]8月発行）36-37P</ref>、消防署前の歩道には「湧別駅の跡」の記念碑が建立されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）128P</ref>。 <gallery> ファイル:Yubetsu.Fire-Station.JPG|遠軽地区広域組合消防署湧別出張所 ファイル:Yubetsu-CultureCenter-Sazanami.JPG|湧別町文化センターさざ波 </gallery> == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線（支線） :: [[四号線駅]] -
