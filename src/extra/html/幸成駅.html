[[日本国有鉄道]]（国鉄）名寄本線の'''幸成仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。同時に駅に昇格、'''幸成駅'''となる。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の北側（遠軽方面に向かって左手側）に存在した<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]][[12月]]発行）132ページより。</ref>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。 [[仮乗降場]]に出自を持つ[[無人駅]]で、駅舎は無かったが[[プレハブ工法|プレハブ]]の物置を[[待合所]]として利用していた<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）114ページより。</ref><ref name="bunken2011P"/>。ホームは名寄方に[[斜路|スロープ]]を有し<ref name="bunken2011P"/>、駅施設外に連絡していた。 == 駅名の由来 == {{節スタブ}} 当駅が所在した地の近辺の地名、「幸成」より<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）185ページより。</ref><ref group="注" name="chimei">下川町一の橋の西、二の橋の東に「幸成」の地名あり（書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより）。</ref>。 == 駅周辺 == * [[国道239号]]（下川国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）18ページより。</ref> * [[名寄川]]<ref name="bunken1980M"/> * モサンル川<ref name="bunken1980M"/> * [[名士バス]]「幸成」停留所 == 駅跡 == 2000年（平成12年）時点では、鉄道関連の遺構は何も残されていなかった<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]][[1月]]発行）39ページより。</ref>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）33ページより。</ref>、2011年（平成23年）時点でも同様で、[[クマザサ|熊笹]]に覆われた原野と化していた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[二ノ橋駅]] -
