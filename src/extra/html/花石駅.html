当駅 - 今金駅間の延伸開通に伴い、中間駅となる。 *[[1979年]]（昭和54年）[[3月10日]] - 貨物取扱い廃止<ref name="haruka" />。 *[[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止<ref name="haruka" />。 *[[1987年]]（昭和62年）[[3月16日]] - 瀬棚線の全線廃止に伴い、廃駅となる<ref name="haruka" />。 === 駅名の由来 === 所在地名より。この地で[[メノウ]]が多く産出し、その美しさを花にたとえての名称である<ref name=":022" /><ref>{{Cite book|和書|title=駅名の起源|editor=札幌鉄道局編|year=1939|publisher=北彊民族研究会|id={{NDLJP|1029473}}|page=25}}</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型の2面2線を有する[[地上駅]]で、[[列車交換]]が可能な[[交換駅]]であった。互いのホームは、単式ホーム東側と島式ホーム東側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた。線路南側の駅舎側単式ホームが下り1番線、島式ホーム駅舎側が上り2番線となっていた。また、島式ホームの外側の1線は1983年（昭和58年）時点で[[停車場#側線|側線]]扱いで残っていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）64ページより。</ref>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は構内の南東側（瀬棚方面に向かって左側）に位置し単式ホームに接していた<ref name="bunken1983"/><ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）10-11ページより。</ref>。駅舎は古い木造駅舎で白いペンキ塗りの建物で<ref name="bunken1983"/>、丹羽駅とほぼ同型の建物であった。ホームは[[アスファルト]]舗装されていた<ref name="bunken1983"/>。 == 利用状況 == * 1981年度の1日乗降客数は42人<ref name="bunken1983"/>。 == 駅周辺 == * [[国道230号]] * 花石郵便局 * [[後志利別川]] == 駅跡 == 2000年（平成12年）時点で駅跡近辺は、道路工事で出た土砂の置き場となっていた<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）76-78ページより。</ref>。2011年（平成23年）時点では、近辺の築堤以外は何も残っていない<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）75ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 瀬棚線 :: [[美利河駅]] -　'''花石駅''' -
