日置郡日置村大字日置に'''毘沙門駅'''（びしゃもんえき）として開業{{Sfn|日吉町郷土誌編さん委員会|1988|page=240}}。 * [[1934年]]（昭和9年）[[6月1日]] - '''上日置駅'''に改称{{Sfn|日吉町郷土誌編さん委員会|1988|page=240}}。 * [[1945年]]（昭和20年）[[3月18日]] - 日置村に対して[[アメリカ軍]]の空襲が行われる{{Sfn|日吉町郷土誌編さん委員会|1988|page=136}}。上日置駅に停車中であった通学通勤時間中の列車に対しても[[機銃掃射]]が行われ、3名（学生2名、車掌）が死亡したほか、多数の負傷者が出た{{Sfn|日吉町郷土誌編さん委員会|1988|page=138}}。列車の機関士は銃弾が足を貫通する重傷を負いながら上日置駅から大田トンネルに列車を退避させた{{Sfn|日吉町郷土誌編さん委員会|1988|page=138}}。 * [[1971年]]（昭和46年）[[4月1日]] - 無人化される{{Sfn|日吉町郷土誌編さん委員会|1988|page=242}}。同時にスイッチバックの使用が停止される。 * [[1984年]]（昭和59年）[[3月17日]] - 同線廃線と共に廃止{{Sfn|日吉町郷土誌編さん委員会|1988|page=242}}。 == 駅構造 == 日置駅側から伊集院駅側へ向かって上り勾配が続く途中にある駅で、[[プラットホーム]]は対向式に2面設けられていた。しかし北側のプラットホームに面した線は、伊集院側からしか入線できないスイッチバック構造になっていた。従ってこのホームを使用する列車は、上りの場合一旦駅を通り過ぎてからバックして入線することになる。 == 廃止後 == プラットホームと[[給水塔]]が現地に残されている。長年、スイッチバック側の線路跡やプラットホームなどは藪に覆われて確認することができない状態であったが、[[2014年]][[7月11日]]から[[7月13日|13日]]にかけて[[南さつま市]]で行われた南薩鉄道100年記念展に合わせて整備が行われた結果、廃線から30年ぶりに姿を現した。 == ギャラリー == <gallery> File:Ruin of Kamihioki station.jpg|上日置駅跡に残る給水塔とプラットホーム、スイッチバック引き上げ線はこれよりも右手の藪の中になる File:Kamihiokista01.jpg|スイッチバック線路 File:Kamihiokista02.JPG|スイッチバック線路側プラットホーム File:太田トンネル 上日置駅側坑門 .jpg|上日置駅側　太田トンネル坑門 </gallery> == 隣の駅 == ; 鹿児島交通（南薩鉄道） : 枕崎線 :: [[伊集院駅]] -
