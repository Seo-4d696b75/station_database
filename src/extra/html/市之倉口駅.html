笠原鉄道開業に際し開設。 * [[1944年]]（昭和19年）[[3月1日]] - 笠原鉄道が[[東濃鉄道駄知線|駄知鉄道]]等と合併し、東濃鉄道笠原線となる。 * [[1950年]]（昭和25年）頃 - 交換設備が設置される。 * [[1971年]]（昭和46年）[[6月12日]] - 乗客数の減少により、笠原線の旅客鉄道営業休止。以降同線は貨物鉄道営業のみとなる。当駅は[[東京窯業]]の規模拡大に伴う耐火煉瓦などの運搬があり、取扱量も多かったという。 * [[1978年]]（昭和53年） ** [[10月31日]] - 東京窯業関連の貨物輸送がトラック輸送に切り替えられたことにより、これをもって笠原線の貨物営業を終了する。 ** [[11月1日]] - 笠原線廃止により廃駅。 == 駅構造 == * 交換施設があった。また、貨物用[[プラットホーム|ホーム]]も存在したという。 * 隣接する東京窯業多治見工場への専用線があり、廃止まで使用されていたという。 == 現在の状況 == * [[1979年]]（昭和54年）、多治見市は笠原線の線路跡を[[自転車専用道路]]・[[歩行者専用道路]]として整備することを決定し、平和町 - 滝呂町の整備を行い[[1998年]]（平成10年）に完成する。駅跡地はこの整備で無くなっているが、東京窯業多治見工場付近にわずかに痕跡があるという。 == 隣の駅 == ;東濃鉄道 :笠原線 :: [[本多治見駅]] - '''市之倉口駅''' - [[下滝呂駅]] == 関連項目 == * [[日本の鉄道駅一覧]] * [[廃駅]] {{東濃鉄道笠原線}} {{Eki-stub}} {{DEFAULTSORT:いちのくらくち}} [[Category:日本の鉄道駅 い|ちのくらくち]] [[Category:岐阜県の廃駅]] [[Category:東濃鉄道（2代）の鉄道駅]] [[Category:1928年開業の鉄道駅]] [[Category:1978年廃止の鉄道駅]] [[Category:多治見市の鉄道駅|廃いちのくらくち]] [[Category:多治見市の歴史|いちのくらくちえき]] d5uvj7q279bu4kfzrkpfxxpjdwm2mt4
