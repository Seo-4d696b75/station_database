当駅間が開業し、開設。 * [[1915年]]（大正4年）[[8月26日]] 当駅 - 岡本新間が開業。 * [[1918年]]（大正7年）[[3月19日]] 武岡鉄道に社名変更。 * [[1924年]]（大正13年）[[3月7日]] 今立鉄道（未開業）と合併し、南越鉄道となる。 * [[1941年]]（昭和16年）[[7月2日]] 福武電気鉄道に合併、同社の南越線の駅となる。 * [[1945年]]（昭和20年）[[8月1日]] 鯖浦電気鉄道が合併して福井鉄道となり、同社南越線の駅となる。 * [[1971年]]（昭和46年）[[9月1日]] 当駅 - 戸ノ口間が[[廃線|廃止]]され、当駅が[[終着駅]]となる。同時に[[列車交換#交換施設|交換設備]]の撤去や[[無人駅|無人駅化]]が行われた。<ref name="haisen3_p166">[[寺田裕一]] 「私鉄の廃線跡を歩くⅢ 北陸・上越・近畿編」、[[JTBパブリッシング]]、2008年5月、p.98、ISBN 978-4-533-07145-4</ref> * [[1981年]]（昭和56年）[[4月1日]] 社武生 - 当駅間の廃止により廃駅となる。 == 駅構造 == [[相対式ホーム]]2面2線と引込線2線を有する駅だった<ref name="shuchaku">昭和の終着駅 北陸・信越篇（[[交通新聞社]] 2017年6月）p.6 - 9</ref>。晩年は[[駅舎|駅舎側]]のホームのみを使用していた<ref name="shuchaku"/>。 小口の貨物取扱いもあり、貨物としては近くの魚市場への魚の輸送や米などの需要があった。 当駅 - [[戸ノ口駅|戸ノ口]]間が廃止されたときは当駅が終点だった。 == 駅跡 == 駅の名をとどめるものは残っていない。跡地にはコミュニティセンターが建っている。 == 隣の駅 == 福井鉄道 : 南越線（廃止） :: [[五分市駅]] - '''粟田部駅''' - [[岡本新駅]] == 脚注 == {{Reflist}} == 参考文献 == *『ありし日の南越線』[[福井鉄道]] 鉄道運輸課 1999年 *『南越線開業100年』[[越前市]] [[武生公会堂記念館]] 2014年7月18日 *『昭和の終着駅 北陸・信越篇』[[交通新聞社]] 2017年6月 {{福井鉄道南越線}} {{Eki-stub}} {{DEFAULTSORT:あわたへ}} [[Category:福井鉄道の廃駅]] [[Category:越前市の鉄道駅|廃あわたへ]] [[Category:越前市の歴史|あわたへえき]] [[Category:日本の鉄道駅 あ|わたへ]] [[Category:1914年開業の鉄道駅]] [[Category:1981年廃止の鉄道駅]] [[Category:福井県の廃駅]] q0l3gzh66q2y8lt9m8rxmwo9eipb8xm
