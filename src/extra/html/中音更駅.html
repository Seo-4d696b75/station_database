鹿追 - 当駅延伸開業に伴い開設。[[日本の鉄道駅#一般駅|一般駅]]<ref>昭和12年度現在鉄道停車場一覧 鉄道省発行 P259。</ref>。 * [[1931年]]（昭和6年）[[11月15日]] - 当駅 - 上士幌間延伸開業。中間駅となる。 * [[1932年]]（昭和7年）[[4月2日]] - 機関車方向転換用のY線（[[デルタ線]]）を瓜幕駅へ移設<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?XSLT_NAME=frame&NO=1&BID=F0000000000000047927&IS_STYLE=default&IS_TYPE=AJAX&LANG=default&ID=&act=&GID=&IMG_FLG=&TYPE= 中音更外一停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1933年]]（昭和8年） ** [[6月3日]] - 鹿追駅の機関車方向転換用のY線（デルタ線）を当駅へ移設<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047927&ID=M0000000000000394539&TYPE=&NO= 中音更及鹿追停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1934年]]（昭和9年）[[1月11日]] - 炭水設備を南新得駅へ移設<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047927&ID=M0000000000000394542&TYPE=&NO= 中音更及南新得停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1942年]]（昭和17年）[[12月16日]] - 構内改装、相対式ホームを単式ホームのみに変更<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047928&ID=M0000000000000394618&TYPE=&NO= 停車場設備変更ノ件] 国立公文書館 デジタルアーカイブ。免許日。</ref>。 * [[1949年]]（昭和24年）[[9月1日]] - 東瓜幕 - 上士幌間廃止により廃駅。 == 駅名の由来 == 当地の地域名に由来する。「音更」の由来については[[士幌線]][[音更駅]]を参照。 == 駅構造 == * 駅舎は上士幌駅へ向かって右側（東側）にあった。 * ホームは初期には千鳥状にずれた相対式ホーム2面2線を有していたが、1942年（昭和17年）に駅裏側が撤去されて駅舎側の単式ホーム1面となった。 * 駅舎横の瓜幕側に貨物ホームと2本の引込線、その他駅裏側に側線1本を有した。 * 駅裏側に機関車の方向転換のためのY線（デルタ線）を有していた。 * 開設当初にダイヤ上の理由により機関車夜間滞泊用に機関庫と給炭台が駅裏の側線上に設置された<ref>[http://www.digital.archives.go.jp/DAS/meta/listPhoto?KEYWORD=&LANG=default&BID=F0000000000000047926&ID=M0000000000000394485&TYPE=&NO= 中音更駅仮設物設置ノ件] 国立公文書館 デジタルアーカイブ。この記述では1930年（昭和）5年7月末日までの仮設となっているが、機関庫は1933年（昭和8年）に、炭水設備は1934年（昭和9年）に南新得駅へ移設されるまで置かれていたと思われる。</ref>。 == 駅周辺 == 周囲は畑作地帯。 * 音更川 * 北中音更小学校 == 隣の駅 == ; 北海道拓殖鉄道 : 北海道拓殖鉄道線 :: [[東瓜幕駅|東瓜幕]] -
