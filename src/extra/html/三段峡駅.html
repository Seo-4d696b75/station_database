Station.jpg|thumb|200px|right|ホームに停車中の列車（1999年5月）]] '''三段峡駅'''（さんだんきょうえき）は、[[広島県]][[山県郡]][[戸河内町]]（現在の[[安芸太田町]]）滝川にあった[[西日本旅客鉄道]]（JR西日本）[[可部線]]の[[鉄道駅|駅]]（[[廃駅]]）である。 同線の[[終着駅]]であったが、可部線の非電化区間の廃線に伴い[[2003年]]（平成15年）[[12月1日]]に廃止された。 == 駅構造 == [[単式ホーム]]1面1線を持つ[[地上駅]]であった。駅北端の線路終端部分には第4種[[車止め]]が設置されていた。また、本線の外側に[[停車場#側線|側線]]を1本有していたが、両端とも本線から分岐するポイントを含む数メートルの線路が残るのみで、中間の大部分が撤去されており機能していなかった。 駅舎は観光地[[三段峡]]への入り口駅らしくコンクリート造り二階建てで斜めになった屋根を持つ大きなものであった。この駅は廃止まで[[日本の鉄道駅#簡易委託駅|簡易委託駅]]であり、委託駅員がピンク色の[[乗車券#乗車券の素材|常備軟券]]を販売する[[出札]]窓口があった。 [[ファイル:sandankyo1.png|三段峡駅配線図]] == 駅周辺 == 駅前には[[三段峡]]へのハイキングコースの入口（三段峡正面口）があり、三段滝へは徒歩2時間ほどの距離である<ref>[http://www.akioota-navi.jp/kanko/sandankyo/index.html 安芸太田町観光協会HP「あきおおたナビ」三段峡の紹介ページ]による。</ref>。また、駅横には[[柴木川]]が流れている。土産物屋・食堂・宿泊施設などの観光客向け施設がある。 かつて駅舎横には[[蒸気機関車]]（[[国鉄C11形蒸気機関車|C11]] 189号機）が[[静態保存]]されていたが、可部線の部分廃止後は広島県安芸郡[[府中町]]にあるキリンビアパーク広島（ダイヤモンドシティ・ソレイユ、現・[[イオンモール広島府中]]）の敷地内に移設された。 <gallery> ファイル:C11 SL Sandankyo.jpg|三段峡駅前に保存されていた頃のC11-189 ファイル:JNR C11 189 20040327.jpg|イオンモール広島府中ソレイユに移設されたC11-189 </gallery> [[ファイル:Sandankyou koryuhiroba2.JPG|thumb|200px|right|駅跡モニュメント]] == 歴史 == この駅は[[1969年]]（昭和44年）、[[日本国有鉄道|国鉄]]可部線が[[加計駅]]から当駅まで延伸したのに伴い開業した。当初は戸河内までの計画だったが、地元の要望などで当駅まで建設された経緯があるといわれる。当駅から先、島根県の[[浜田駅]]まで線路を敷設すべく今福線の名で建設が進められたが、[[1980年]]（昭和55年）に[[日本国有鉄道経営再建促進特別措置法|国鉄再建法]]が施行されたため工事は中止された。今福線旧線は、当駅からスイッチバックして川を渡りトンネルに入る予定だったとされるが定かではない。新線は当駅から真っ直ぐ川を渡り長大トンネルに入る予定で未着工。なお当駅の右隣には本線から分岐する形で空き地が直線状に川まで伸びており延伸の際には直線に改良される予定だったものと思われる。 * 1969年（昭和44年）[[7月27日]] - 国鉄可部線の駅（[[鉄道駅#旅客駅|旅客駅]]）として開業{{R|停車場}}。 * [[1985年]]（昭和60年）[[2月1日]] - 無人駅化<ref>{{Cite news |title=通報 |newspaper=[[鉄道公報]] |publisher=[[日本国有鉄道]]総裁室文書課 |date=1985-02-01 |page=2 }}</ref>（[[加計駅]]管理による乗車券類発売は継続→後に[[簡易委託]]駅化） * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR西日本が継承{{R|停車場}}。 * [[2003年]]（平成15年）[[12月1日]] - 可部線部分廃線に伴い廃止。 * [[2005年]]（平成17年） - 駅舎解体。 1970年代、特に紅葉シーズン時期の休日には、三段峡へ向かう観光客を乗せた四両編成のディーゼルカーが満席状態で駅に到着するほど賑わう反面、平日は閑散した状況となるなど乗客数の落差は大きかった。1日の利用者は平均100人程度で、平日の利用者の半数以上は戸河内へ通う小・中学生であった。通学する児童らの存在は鉄道が地域の重要な交通機関であることを証明したものの、低額な通学定期券利用者が中心ということもあり、管理事務所が「バカらしくて計算したくなくなるほど」悪化する営業係数の改善には貢献しなかった<ref>「終着駅の町 休日以外はガラあき」『中國新聞』昭和46年11月12日.5面</ref>。 駅舎解体後の駅跡地には安芸太田町立の交流施設（三段峡交流広場）が設置されており、広場内には三段峡駅跡の[[モニュメント]]がある。このモニュメントの車止めは、現役時代の当駅の[[車止め]]（第4種車止め）とは異なり、第3種車止めである。なお、広場には[[広電バス]]の三段峡[[バス停留所]]が近接しており、三段峡～[[広島バスセンター]]間を結ぶ[[バス (交通機関)|バス]]路線が設定されている。 == 隣の駅 == ; 西日本旅客鉄道 : 可部線 :: [[戸河内駅]] - '''三段峡駅''' : 今福線（未成線） :: '''三段峡駅''' - 橋山駅 == 脚注 == {{脚注ヘルプ}} {{reflist}} == 関連項目 == * [[可部線#可部駅 - 三段峡駅間の廃止]] {{可部線|mode=1}}
