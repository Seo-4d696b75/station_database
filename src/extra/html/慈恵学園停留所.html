開業<ref name="kubo p78"/>。 * [[1969年]]（昭和44年）[[11月1日]] - 定山渓鉄道線の[[廃線]]に伴い廃止<ref name="kubo p79">[[#kubo|久保 『定山渓鉄道』p.79]]</ref>。 == 駅構造 == 単式ホーム1面1線の停留所であったが、広い待合室を備えた立派な駅舎を持ち、通学時間帯には駅員が配置されていた<ref name="kubo p78"/>。定期券の販売も行われていたが<ref name="kubo p78"/>、学生の増加に伴い混雑が激しくなり、学校への出張販売も行われていた<ref name="sapporo p239">[[#sapporo|さっぽろ文庫 11『札幌の駅』p.239]]</ref>。朝夕には当駅発着の学生専用電車も運行されていた<ref name="kubo p78"/><ref name="sapporo p239"/>。 == 駅周辺 == [[札幌新陽高等学校|札幌慈恵女子高等学校]]は駅から600m程東側の丘の上にあった<ref name="kubo p78"/>。駅開業後、急速に宅地化が進み<ref name="kubo p78"/>、停留所周辺には学生相手の商店等もできた<ref name="sapporo p239"/>。定山渓鉄道線廃止後の[[札幌市営地下鉄南北線]]建設にあたっては、近隣住民から当停留所跡地への駅設置が期待されていたが実現せず、400mほど南側に[[自衛隊前駅]]が設置されることとなった<ref name="kubo p79"/>。 == 駅跡 == 駅舎は札幌市営地下鉄南北線自衛隊前駅より400mほど北側にあった。駅跡には札幌市澄川児童会館と札幌南消防署澄川出張所が建つ<ref name="kubo p79"/>。 == 隣の駅 == ; 定山渓鉄道 : 定山渓鉄道線 :: [[澄川駅 (定山渓鉄道)|澄川駅]] -
