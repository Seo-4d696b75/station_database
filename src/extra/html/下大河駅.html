1945.jpg|thumb|300px|1945年アメリカ軍作成の広島市地図。"Ōkō Station（大河）"が確認できる。]] '''下大河駅'''（しもおおこうえき）は、かつて[[広島県]][[広島市]]旭町（現在の[[南区 (広島市)|南区]][[旭 (広島市)|西旭町]]）に所在していた[[日本国有鉄道]]（国鉄）[[宇品線]]の[[鉄道駅#旅客駅|旅客駅]]である。 この項目では下大河駅の前身ともいえる「'''大河地蔵前停留場'''」（おおこうじぞうまえていりゅうじょう）についても扱う。 == 概要 == [[芸備線|芸備鉄道]]により宇品線唯一の[[交換駅|行き違い駅]]として設置された。その後芸備鉄道の国有化による[[日本国有鉄道|国鉄]]移管を経て、戦後の[[1966年]]、宇品線の一般旅客営業廃止に伴いこの駅も廃止となった。2012年現在、駅施設・線路は完全に撤去されているが、跡地には小公園が作られモニュメントが設置されている。 == 歴史 == 下大河駅は[[1931年]]11月、[[芸備線|芸備鉄道]]により大河停車場として開業され、この時点では前駅の[[上大河駅]]（初代駅。当時の名称は「被服支廠前停留場」）、次駅の[[丹那駅]]とはそれぞれ0.6km、0.5kmの距離があった。下大河駅の前身となったのは同年3月、同じく芸備鉄道によってこの駅より13m南側の位置に開業した「大河地蔵前停留場」<ref>のちの下大河駅南側の旭町踏切（現在の「ポッポ広場」前の交差点）よりもやや南側に所在していた。なお、宇品線の踏切の名称については長船友則『宇品線92年の軌跡』pp.42-43、掲載の国鉄中国支社作成の図面にしたがう。</ref>であるが、大河停留場の開業と同時に廃止された。この駅は開業当初から'''宇品線唯一の[[交換駅|行き違い駅]]'''<ref>戦後の一時期（[[1947年]]〜[[1958年]]）においては[[南段原駅]]にも[[行き違い施設|行き違い設備]]が設けられた。</ref>であり、これによってそれまで1日24往復であった運行が30往復に増発され[[宇品線]]利用者の増加がはかられた。その後[[1937年]]11月、芸備鉄道国有化に伴い前駅が上大河駅と改称されたのに併せ、「下大河駅」と改称され<ref>ただし駅は旭町（当時の町名）に所在し[[大河 (広島市)|大河地区]]とは離れた位置にある。</ref>。戦時中はガソリンカー運転休止で宇品線の一部の駅が営業休止となった際にもこの駅は存続した。 戦後における駅の乗降客の多くは[[比治山女子中学校・高等学校|比治山女子高]]・[[広島大学附属中学校・高等学校|広大附高]]・[[広陵高等学校 (広島県)|広陵高]]<ref>[[1973年]]まで現在の南区[[宇品]]に校地が所在していた（現・[[みゆきプラザ]]）。</ref>など近隣の学校に通学する学生・生徒であり、徒歩あるいは[[バス (交通機関)|バス]]により学校と駅の間を往来していた。しかし乗降人数が次第に減少に向かったため、[[1966年]]12月の[[上大河駅|上大河]]〜[[宇品駅|宇品]]間の旅客運輸営業廃止に伴って次駅の丹那駅とともに廃止された<ref>終点の宇品駅は旅客営業は停止したものの「宇品貨物取扱所」と改称され[[1986年]]の「宇品四者協定線」廃止まで存続した。</ref>。これにより宇品線では旅客線で行き違い駅が消滅することとなった。 これ以降1972年4月の旅客営業全面廃止に至る時期の宇品線では、上大河駅での旅客列車の折り返し運転にさいして、同駅が[[単式ホーム|単式片面ホーム]]であるため、機関車を前後に付け替えることができなくなった。このため旅客列車は客車の前後に機関車を接続する「双頭列車」で運転された。 === 年表 === * [[1931年]]（昭和6年）3月20日 - [[芸備線|芸備鉄道]]により大河地蔵前停留場の開業。 * 1931年11月29日 - 芸備鉄道、大河地蔵前停留所の13m北に'''大河停車場'''を開業。同日、大河地蔵前停留場は廃止。 * [[1937年]]（昭和12年）7月1日 - 芸備鉄道国有化により'''下大河駅'''に改称。 * [[1966年]]（昭和41年）12月20日 - 上大河駅〜[[宇品駅]]間の旅客運輸営業廃止に伴い廃止される。 * [[2001年]]（平成13年）12月 - 駅址に「ポッポ広場」が設置される。 == 駅の所在地・構造 == 広島市旭町（現在の南区西旭町1番）に所在し、2面2線の[[相対式ホーム|相対式]]構造となっていた線路・プラットホームの西側に1階建て駅舎が設置され、駅の南側に[[踏切]]（旭町踏切）があった。また列車の行き違いのため、ホームで駅員による[[閉塞 (鉄道)#タブレット|タブレット]]交換で[[閉塞 (鉄道)|閉塞]]が行われ、駅舎には駅員が宿直するための控所が設けられていた。 == 隣の駅 == 廃止（[[1966年]][[12月20日]]）時点のもの。 ; 日本国有鉄道 : 宇品線 :: [[上大河駅]] - '''下大河駅''' - [[丹那駅]] == 近隣の施設 == 廃止時点のもの。 * [[広島県立広島工業高等学校]] - 戦前の[[広島陸軍被服支廠]]の敷地を継承し現在に至っている。 * [[広島市立大河小学校]] * [[広島大学附属小学校]]・[[広島大学附属中学校・高等学校|中学校・高等学校]] - 戦前の[[広島高等学校 (旧制)|旧制広島高等学校]]校地、戦後初期の広島大学教養部校地（旧皆実キャンパス）を継承し現在に至っている。 * 旭商店街 == 駅址の現状 == [[1966年]]の廃止後、駅舎・ホームなどの施設は早々に撤去された。宇品駅までの貨物列車の運行は[[1986年]]10月の「宇品四者協定線」<ref>[[1972年]]4月の旅客業務全面廃止以降の宇品線の正式名称。</ref>廃止まで続いていたため線路は使用されていたが、運行は早朝の1往復に限られたため日中にこの近辺を列車が通過することはなく、近隣住民により通路や遊び場として使われていた。その後、四者協定線廃止により列車の運行がなくなったのちも線路は[[1990年代]]初頭まで撤去されないまま放置されていた。 現在、この駅の跡地近辺のみならず、[[国道2号|国道2号線]]から[[広島県道86号翠町仁保線|黄金山通り]]までの旧宇品線区間（すなわちかつての霞通り踏切より丹那踏切に至る区間）は、完全に線路が撤去され、廃線跡をやや拡幅してほぼそのままの形で車道として造成されているため、旭町踏切跡は交差点となっている。また駅舎跡地は2012年現在、西旭町集会所および地元住民によって整備された小公園「ポッポ広場」となっており、かつてこの地に駅が所在していたことを示す記念物が設置されている（旧宇品線の駅跡地に記念物が置かれているのはこの駅と丹那駅・下丹那駅・宇品駅の4駅である<ref>南段原駅の付近にも宇品線を記念するモニュメント・公園が設置されているが駅跡地とは異なる位置にある。</ref>）。
