[[鹿島参宮鉄道]]により開業。 * [[1965年]]（[[昭和]]40年）[[6月1日]] - 鹿島参宮鉄道と[[常総筑波鉄道]]が合併して関東鉄道が発足。同社鉾田線の駅となる。 * [[1979年]]（昭和54年）[[4月1日]] ‐ 鉾田線が鹿島鉄道に分社化。 * [[2007年]]（[[平成]]19年）4月1日 - 廃止。 * [[2010年]]（平成22年）[[8月30日]] - 駅跡地にかしてつバス「新高浜駅」停留所が開業。 == 隣の駅 == ; 鹿島鉄道 : 鹿島鉄道線 :: [[玉里駅]] -
