'''更別駅'''（さらべつえき）は、[[北海道]]（[[十勝支庁]]）[[河西郡]][[更別村]]字更別にかつて存在した、[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''サラ'''。広尾線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の北東側（広尾方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）139ページより。</ref>。かつては2面2線の[[相対式ホーム]]を有する、[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎と反対側の1線は、交換設備運用廃止後も[[停車場#側線|側線]]として残っていた（但し、1983年（昭和58年）時点ではプラットホームは撤去されていた<ref name="bunken1983"/>）。 その他、本線広尾方から駅舎側に分岐し駅舎南側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、また上記旧交換線である側線の帯広方から分岐した行き止まりの側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、駅舎は構内の北東側に位置しホーム中央部分に接していた<ref name="bunken1983"/>。間口が狭い小ぢんまりとした駅舎であった。 更別村農協の[[専用鉄道|専用線]]が接続していた。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[アイヌ語]]の「サル・ペツ」（[[葦]]が生えている川）に由来する<ref name="bunken1983"/>。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は178人<ref name="bunken1983"/>。 == 駅跡地・周辺 == 火山灰の畑作地帯で、[[アスパラガス]]や[[トウモロコシ|スイートコーン]]の産地であった<ref name="bunken1983"/>。 * 広尾線の全線廃止に伴い、廃駅となる。 == 駅跡 == 1999年（平成11年）時点では、駅跡地に「更別駅」と記載された記念碑が建立されていた<ref name="bunken1999H">書籍『鉄道廃線跡を歩くVI』（[[JTBパブリッシング]]、[[1999年]][[3月]]発行）38-40ページより。</ref>。この記念碑は[[花崗岩|御影石]]製で、1997年（平成9年）に開村50年に合わせて建立され、表側上段には駅舎をスケッチした陶板がはめ込まれており、下段には碑文として広尾線の意義や歴史が記載されている。裏面には、[[駅名標]]を模した陶板がはめ込まれている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）188ページより。</ref>。2010年（平成22年）時点<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）87ページより。</ref>、2011年（平成23年）時点でも同様であった<ref name="bunken2011H"/>。駅舎などの施設は撤去され、構内の跡は空き地になっている。 駅周辺の農業倉庫なども2010年（平成22年）時点で残存しており、駅の雰囲気を残している<ref name="bunken2010H"/>。 また、2010年（平成22年）時点では当駅跡と上更別駅跡の間にある南4線川にガーダー橋が残存している<ref name="bunken2010H"/>。 == 隣の駅 == ; 日本国有鉄道 : 広尾線 :: [[中札内駅]] - '''更別駅''' - [[上更別駅]] == 脚注 == {{脚注ヘルプ}} {{reflist}} == 関連項目 ==
