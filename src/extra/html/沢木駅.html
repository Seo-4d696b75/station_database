[[鉄道省]]興浜南線、興部駅 - 雄武駅間の開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1944年]]（昭和19年）[[11月1日]] - 興浜南線の[[不要不急線]]指定に伴い、営業を休止。 * [[1945年]]（昭和20年）[[12月5日]] - 営業再開。 * [[1949年]]（昭和24年）[[6月1日]] - [[公共企業体]]である[[日本国有鉄道]]に移管。 * [[1978年]]（昭和53年）[[12月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止し、[[無人駅]]化。 * [[1985年]]（昭和60年）[[7月15日]] - 興浜南線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]と線路を有する[[地上駅]]であった<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=214}}</ref>。ホームは、線路の北西側（雄武方面に向かって左手側）に存在した。このほか、雄武方から駅舎側に分岐する[[停車場#側線|側線]]を1本有していた<ref name="bunken1983"/>。このホームは本来[[島式ホーム]]で、興浜線全通の際には[[列車交換]]可能な配線とする計画があった事が指摘されている<ref name="bunken1983"/>。 [[無人駅]]となっていた。駅舎は構内の南西側に位置し、ホームとは通路で連絡した<ref name="bunken1983"/>。[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、[[渚滑線]][[下渚滑駅]]や[[興浜北線]][[豊牛駅]]などと同型の駅舎となっていた。改築の際に線路未敷設側のホーム上に位置変更している<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在していた地名より。地名は[[アイヌ語]]の「サラキ」（[[葦]]）、「サマケ」（傍らの所）など、由来には諸説ある。 == 駅周辺 == * [[国道238号]] * [[北海道道883号宇津沢木線]] * 雄武町立沢木小学校 * 沢木郵便局 * 日の出岬 - 駅から北に約1.5km<ref name="bunken1983"/>。 * [[北紋バス]]「沢木」停留所 == 駅跡 == [[2000年]]（平成12年）時点で当駅の跡地は小公園になっており、[[駅名標]]や名所案内も残っていた<ref name="bunken2000H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVII |date=1999-12 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533033766 |page=33}}</ref>。また、雄武方の日の出岬に向かう線路跡は道路となっている<ref name="bunken2000H"/>。[[2010年]]（平成22年）時点も同様であった<ref>{{Cite book|和書|editor=今尾恵介 |title=新 鉄道廃線跡を歩く1 北海道・北東北編 |date=2010-3 |publisher=JTBパブリッシング |isbn=978-4533078583 |page=25}}</ref>。[[2016年]]（平成28年）4月時点で駅名標と名所案内は撤去されていた。 == 隣の駅 == ; 日本国有鉄道 : 興浜南線 :: [[興部駅]]- '''沢木駅''' -
