[[鉄道省]]名寄線の上興部 - 興部間の延伸開通（名寄線全通）に伴い、開業<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2954862/2 『官報』 1921年09月27日　鉄道省告示第129号]（国立国会図書館デジタルコレクション）</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1978年]]（昭和53年）[[12月1日]] - 貨物・[[チッキ|荷物]]の取り扱いを廃止。民間委託駅となる<ref>興部町百年史 平成5年3月発行。</ref>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。プラットホームは、線路の南西側（遠軽方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）209ページより。</ref>。そのほか旧貨物線線である、名寄方から駅舎側に分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分への[[停車場#側線|側線]]を1線有した<ref name="bunken1983"/>。かつては2面2線の[[相対式ホーム]]を有する、[[列車交換]]可能な[[交換駅]]であった。交換設備運用廃止後は線路は撤去されたが、ホーム前後の線路は[[分岐器|転轍機]]の名残で湾曲していた<ref name="bunken1983"/>。 [[無人駅]]となっていたが、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎が残っていた。駅舎は構内の南西側に位置し、ホーム中央部分に接していた。開業以来の古い木造駅舎であった<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）40ページより。</ref>。 == 駅名の由来 == 当駅が所在した地名より。[[アイヌ語]]の「ウッ<ref>[[ラテン文字化|アイヌ語ラテン翻字]]: ut</ref>」（肋骨、転じて横の川）に由来している<ref name=":022">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/1120P.pdf|title=アイヌ語地名リスト イチャ～エリ　P11-20|accessdate=2017-10-20|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 「ウッ」、あるいは「ウッナイ<ref>アイヌ語ラテン翻字: ut-nay</ref>」（肋骨・川）に由来する地名・駅名は北海道内各地にある（[[宗谷本線]][[歌内駅]]など）が、本流の両岸から小さい支流が流れ込む様を、背骨とあばら骨にたとえた地名である<ref name="bunken1983"/><ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]]2月発行）186ページより。</ref><ref name=":022" />。 == 駅周辺 == * [[国道239号]]（天北国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）18ページより。</ref> * [[北海道道883号宇津沢木線]] * 興部川<ref name="bunken1980M"/> * 三井山 - 駅の東。標高237m<ref name="bunken1980M"/>。 * [[名士バス]]「宇津小学校前」停留所 == 駅跡 == 2000年（平成12年）時点で空地となっており、駅までの舗装道路のみが残存していた<ref name="bunken2000H"/>。2010年（平成22年）時点でも同様で<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）34ページより。</ref>、2011年（平成23年）時点でも更地であり何も残っていなかった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）118ページより。</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[班渓駅]] -
