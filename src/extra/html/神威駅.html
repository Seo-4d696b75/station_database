[[北海道炭礦鉄道]]空知線開通に伴い、北炭神威砿専用線使用開始<ref name="shishi1"/>。 * [[1896年]]（明治29年）[[10月21日]] - 北海道炭礦鉄道の駅として、砂川駅-歌志内駅間に開駅。一般駅。 * [[1906年]]（明治39年）[[10月1日]] - 北海道炭礦鉄道の鉄道路線[[鉄道国有法|国有化]]により、[[官設鉄道]]に移管。 * [[1907年]]（明治40年） - 北炭空知砿西山坑から当駅西側貯炭場へ索道設置<ref name="shishi1"/><ref group="注">索道設置以前は坑道を利用して当駅貯炭場へ馬搬していた。</ref>。 * 不明 - 新歌志内炭礦から当駅東側貯炭場へ索道設置<ref name="shinshishi">新歌志内市史 1994年3月発行。</ref><ref group="注">1917年（大正6年）12月に奔別炭礦（株）により開砿、翌1918年（大正7年）12月に山下鑛業の経営となり、1928年（昭和3年）12月に住友坂炭礦に買収。住友は索道ごと買収した（『わが社の歩み』住友石炭鉱業発行）ことから1918年-1928年の間に設置されたとみられる。</ref>。 * [[1931年]]（昭和6年） - 北炭西山坑閉山<ref name="shishi1"/>。 * [[1944年]]（昭和19年） - 駅舎改築<ref name="shishi1"/>。 * [[1953年]]（昭和28年）[[7月31日]] - 住友石炭鉱業新歌志内砿閉山<ref name="shinshishi"/>。 * [[1963年]]（昭和38年）9月 - 北炭神威砿閉山<ref name="shinshishi"/>。 * [[1972年]]（昭和47年）[[3月15日]] - 貨物の取り扱いを廃止。 * [[1979年]]（昭和54年）[[6月15日]] - [[チッキ|荷物]]の取り扱いを廃止し、[[無人駅|無人]]（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化<ref name="shinshishi"/>。 * [[1986年]]（昭和61年）11月 - [[乗車券]]の発売を取りやめ、完全無人化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1988年]]（昭和63年）[[4月25日]] - 歌志内線の[[廃線]]に伴い、[[廃駅]]となる。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[無人駅]]であった。有人駅時代の駅舎が残っていた。 == 駅名の由来 == 北海道炭礦鉄道が1893年（明治26年）に申請した、鉱区名の神威（カムイ）から採られた<ref name="shishi2">歌志内市史 P132。</ref>。当会社が鉱区をこの名前にした由来はよくわかっておらず、近くの「カムイヌプリ」（神威岳）もしくはペンケウタシナイ川の支流「カムイシンレルマップ」（ただしシンレルの意味が不明なため全体の意味も不明）から「カムイ」を採ったのではないかとされる<ref name="shishi2"/>。また「かむい」ではなく「かもい」と訛っている理由は不明。ちなみに地区の名前は「かむい」<ref name="shishi2"/>で訛っていない<ref group="注">当時のアイヌ語表記の揺れと思われる。当時は地域名も炭礦名も「かもい」と読んでいた可能性もあるが、残されているのが漢字表記だけなので実態は不明。</ref>。 == 周辺施設 == * [[北海道道114号赤平奈井江線]] * [[北海道道1027号砂川歌志内線]] * [[北海道中央バス]]「神威市街」停留所 == 隣駅 == ; 北海道旅客鉄道 : 歌志内線 :: [[西歌駅]] -
