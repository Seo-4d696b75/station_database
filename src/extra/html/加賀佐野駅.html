39</ref>。ホームは線路の南側に位置し、[[木造]]の[[駅舎]]を設けていた<ref name="nomiden">[https://www.city.nomi.ishikawa.jp/www/contents/1001000001291/simple/nomiden.pdf 能美市『能美電ものがたり 運行時の各駅と現在』]（PDF）</ref>。通勤通学の乗降客のほか、[[九谷焼]]を全国に発送する貨物も取り扱っていた<ref name=":0">{{Cite news|title=「加賀さの」バス停名で復活　旧北鉄能美線の駅名|date=2020-04-01|newspaper=北國新聞|url=https://headlines.yahoo.co.jp/hl?a=20200401-00844286-hokkoku-l17|accessdate=2020-04-06}}</ref>。 佐野の集落に近いのでこの駅名とされたが、駅の所在地は石子であり、駅前を流れる用水を境に区分が変わっていた。なお、工事施行申請時は駅名を「佐野」としていたが、開業3日前の届出で「加賀佐野」と改めていた<ref name="rm"/>。 == 歴史 == [[1945年]]（昭和20年）には、[[北陸鉄道小松線|小松線]]の[[鵜川遊泉寺駅]]から当駅までの鉄道の延長が計画され、着工もされたが、[[未成線|未成]]のまま計画は中止となった<ref name="nomiden"/>。 *[[1925年]]（大正14年）[[3月21日]] - 能美電気鉄道の駅として開業。 *[[1939年]]（昭和14年）[[8月1日]] - 金沢電気軌道の駅となる。 *[[1967年]]（昭和42年）[[1月14日]] - 側線を廃止<ref name="rm"/>。[[無人駅]]化<ref name="nomiden"/>。 *[[1980年]]（昭和55年）[[9月14日]] - 能美線廃止に伴い廃駅。 *[[2020年]]（[[令和]]2年）[[4月1日]] - 地元要望により旧駅から約100[[メートル]]東側に「加賀さの」バス停として名称が復活した<ref name=":0" />。 == 隣の駅 == ; 北陸鉄道 : 能美線 :: [[湯谷石子駅]] - '''加賀佐野駅''' - [[末信牛島駅]] == 現状 == 鉄道路線廃止後は[[サイクリングロード]]及び小公園として再整備されているが、駅跡を示す石碑や遺構は残っていない<ref name="nomiden"/>。 == 参考文献 == {{Reflist}} == 関連項目 == * [[日本の鉄道駅一覧]] * [[北陸鉄道]] * [[佐野駅]] {{北陸鉄道能美線}} {{DEFAULTSORT:かかさの}} [[category:日本の鉄道駅 か|かさの]] [[Category:石川県の廃駅]] [[category:北陸鉄道の廃駅]] [[Category:1925年開業の鉄道駅]] [[Category:1980年廃止の鉄道駅]] [[Category:能美市の鉄道駅|廃かかさの]] [[Category:能美市の歴史|かかさのえき]] ht5dxcsnc2odprfq1zkbtur3opzv872
