'''越中鉄道'''打出浜駅（後の打出駅（2代目）） - 堀岡駅間開通に伴い開業<ref name="bunken2008M-6">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]]10月発行）34ページより。</ref><ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]]4月発行）211-212ページより。</ref>。 * [[1943年]]（昭和18年）[[1月1日]] - [[陸上交通事業調整法|交通統合]]により'''富山地方鉄道射水線'''の駅となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 * [[1971年]]（昭和46年）[[5月1日]] - 交換設備撤去<ref name="bunkenRML107">書籍『RM LIBRARY 107 富山地鉄笹津・射水線』（著：服部重敬、[[ネコ・パブリッシング]]、2008年7月発行）35,40-41ページより。</ref>。 * 時期不詳 - [[日本の鉄道駅#業務委託駅|業務委託]]化<ref name="bunkenRML107"/>。 * [[1980年]]（昭和55年）[[4月1日]] - 射水線の廃線に伴い廃止となる<ref name="bunken2008M-6"/><ref name="bunken2010H-3-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML107"/>。ホームは線路の北側（新港東口方面に向かって右手側、旧上り線）に存在した<ref name="bunkenRML107"/>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった<ref name="bunkenRML107"/>。線路北側の駅舎側ホームが上り線、対向側ホームが下り線となっていた<ref name="bunkenRML107"/>。使われなくなった対向ホーム側の1線は、交換設備運用廃止後は撤去されたが、ホームは残存していた<ref name="bunkenRML107"/>。 [[日本の鉄道駅#業務委託駅|業務委託駅]]となっていた<ref name="bunkenRML107"/>。駅舎は構内の北東側に位置しホーム北東側に接していた<ref name="bunkenRML107"/>。ホームに[[待合所]]を有した<ref name="bunkenRML107"/>。 == 駅周辺 == [[富山の売薬]]の[[行商|行商人]]が多く住む地域であった<ref name="bunkenRML107"/>。線路の南側に並行して[[灌漑]]用[[水路]]が走っていた<ref name="bunkenRML107"/>。 * [[国道415号]] * 海老江郵便局 * 海老江海浜公園 * [[加茂神社 (射水市)|加茂神社]]（[[分霊|分社]]） == 駅跡 == 線路跡は四方駅跡南方附近から終点・新港東口駅附近まで[[サイクリングロード]]となっている<ref name="bunken1997H">書籍『鉄道廃線跡を歩くIII』（[[JTBパブリッシング]]、[[1997年]]5月発行）108-109ページより。</ref>。1997年（平成9年）時点では、線路跡地はそのサイクリングロードであった<ref name="bunken1997H"/>。2006年（平成18年）時点<ref name="bunkenToyama2008H">書籍『富山廃線紀行』（著：草卓人、[[桂書房]]、2008年7月発行）66-67ページより。</ref>、2010年（平成22年）時点でも同様であった<ref name="bunkenKW07">書籍『新 消えた轍 7 北陸』（著：寺田裕一、ネコ・パブリッシング、2010年12月発行）53ページより。</ref><ref name="bunken2010H-3">『新 鉄道廃線跡を歩く3』50-51ページより。</ref>。 駅跡地附近はサイクリングロードから一旦外れ、2006年（平成18年）時点では広々とした旧構内が残り、駅舎側のホームが残存していた<ref name="bunkenToyama2008H"/>。2輌編成の<ref name="bunkenKW07"/>車輌停車位置を示すペンキ書きの表示や[[柵]]も残っていた<ref name="bunkenToyama2008H"/>。また線路跡部分は[[コミュニティバス]]の休憩場所になっていた<ref name="bunkenToyama2008H"/>。2010年（平成22年）時点でも同様であった<ref name="bunkenKW07"/>。 2018年(平成30年)時点[[柵]]は撤去されたものの、車輌停車位置を示すペンキ書きは辛うじて、残っていた == 関連項目 == * [[海老江駅]] -
