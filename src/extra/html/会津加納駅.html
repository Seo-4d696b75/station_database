'''会津加納駅'''（あいづかのうえき）は、[[福島県]][[耶麻郡]][[熱塩加納村]]（現[[喜多方市]]熱塩加納町）にあった[[日本国有鉄道]]（国鉄）[[日中線]]の[[鉄道駅|駅]]（[[廃駅]]）である。日中線の[[廃線|廃止]]にともない、[[1984年]]（[[昭和]]59年）に廃止された。 == 駅構造 == [[単式ホーム]]1面1線の[[地上駅]]であった。また、貨物扱い時代の名残として、側線2本<ref>[[1972年]]（昭和47年）の鉱山廃鉱による貨物扱い減少により、[[1974年]]（昭和49年）頃には駅舎側1本と外側半分の大部分が撤去されていた。</ref>と喜多方側に側線1本を有した。 == 歴史 == * [[1938年]]（昭和13年）[[8月18日]] - 開業。[[一般駅]] * [[1983年]]（昭和58年）[[11月1日]] - 貨物取り扱い廃止。 * [[1984年]]（昭和59年）[[2月1日]] - 荷物取り扱い廃止。 * 1984年（昭和59年）[[4月1日]] - 廃止。 == 現状 == 駅舎は撤去され、構内跡地は[[福島県道335号大平喜多方線|県道335号]]として整備されており、貨物用側線2本があった場所は道路幅がやや広くなっている。駅前通りは更地になっており、喜多方側の貨物用側線があった場所には[[国道121号]]に繋がる道路が新しく出来ている。道路沿いには、再現された駅名標が建っている。熱塩寄りにあった[[跨線橋|陸橋]]は取り壊されて[[福島県道336号熱塩加納会津坂下線|県道336号]]の起点となっている。
