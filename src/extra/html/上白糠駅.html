日本国有鉄道白糠線 白糠駅 - 上茶路駅間の開通に伴い開業。旅客のみ取扱い。 *[[1983年]]（昭和58年）[[10月23日]] - 白糠線の全線廃止に伴い、廃駅となる。 === 駅名の由来 === 白糠川の上流にあることによる<ref name=":2">{{Cite book|和書|title=北海道　駅名の起源|date=1973-3-25|year=|publisher=日本国有鉄道北海道総局|ref=ekimei1973|url=|edition=第1版|ASIN=B000J9RBUY|location=札幌市|author=|accessdate=|author2=|author3=|author4=|author5=|author6=|author7=|author8=|author9=|page=151}}</ref>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南側（[[北進駅|北進]]方面に向かって左手側）に存在した。開業時からの[[無人駅]]で駅舎はないが、ホーム東側の出入口附近に[[待合室]]を有した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）141ページより。</ref>。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は27人<ref name="bunken1983"/>。 == 駅周辺 == 周囲は[[酪農]]地帯である<ref name="bunken1983"/>。 * [[国道392号]] == 駅跡 == [[駅前広場]]が残っていて、外灯も残っている。 [[2001年]]（[[平成]]13年）時点では広い空き地になっており<ref>書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2001年]]1月発行）48ページより。</ref>、[[2010年]]（平成22年）時点でも同様であった<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（JTBパブリッシング、[[2010年]]4月発行）69ページより。</ref>。駅の前後の路盤は[[盛土|築堤]]となって残っており<ref name="bunken2010H" />、近くに「第三大曲橋梁」も残存している<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）147ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 白糠線 :: [[白糠駅]] -
