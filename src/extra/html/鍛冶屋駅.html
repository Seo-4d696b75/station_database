播州鉄道の駅として開業。[[鉄道駅#一般駅|一般駅]]。 *** 当時の所在地表示は[[兵庫県]][[多可郡]][[中町 (兵庫県)|中村]]鍛冶屋であった。 ** [[12月21日]] - 播丹鉄道に譲渡。 * [[1924年]]（大正13年）[[4月1日]] - 中村が町制施行して[[中町 (兵庫県)|中町]]となり、所在地表示が兵庫県多可郡中町鍛冶屋になる。 * [[1943年]]（[[昭和]]18年）[[6月1日]] - 播丹鉄道が国有化、[[鉄道省]]鍛冶屋線の駅となる。 * [[1974年]]（昭和45年）[[10月1日]] - 貨物の取扱いを廃止（[[鉄道駅#旅客駅|旅客駅]]となる）。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]の取扱いを廃止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR西日本が継承。 * [[1990年]]（[[平成]]2年）4月1日 - 鍛冶屋線廃線に伴い廃止。 == ギャラリー == <gallery widths="180" heights="120"> Kaziyasen-kinenkan2.JPG|鍛冶屋線記念館（鍛冶屋駅 旧駅舎）とキハ30 69 JNR Kajiya Station.JPG|国鉄時代の駅舎（1986年12月31日） Kajiya Station-05.jpg|ホーム上に保存されている駅名標。 </gallery> == 隣の駅 == ; 西日本旅客鉄道 : 鍛冶屋線 :: [[中村町駅]] -
