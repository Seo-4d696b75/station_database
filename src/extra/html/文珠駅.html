[[運輸省]]（国鉄）歌志内線の'''文珠[[仮乗降場]]'''（局設置）として、砂川駅 - 神威駅間に開設<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年10月 JTB編集・発行。</ref>。 * [[1947年]]（昭和22年）[[2月20日]] - 一般駅として正式開業<ref name="hensen"/>。[[チッキ|荷物]]・貨物の取り扱いを開始<ref name="hensen"/>。 * [[1960年]]（昭和35年）[[2月10日]] - 民間委託駅化<ref>新歌志内市史 P1083。</ref>。 * [[1961年]]（昭和36年）2月10日 - 荷物・貨物の取り扱いを廃止<ref name="hensen"/>。 * [[1986年]]（昭和61年）11月 - [[乗車券]]の発売を停止。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、国鉄からJR北海道に承継される<ref name="hensen"/>。 * [[1988年]]（昭和63年）[[4月25日]] - 歌志内線の全線廃止に伴い、[[廃駅]]となる<ref name="hensen"/>。 == 駅構造 == 廃止時点で、単式ホーム1面1線を有する[[無人駅]]であった。 == 駅名の由来 == この地の地域名より採られた。近くにあった[[炭鉱]]が三人の共同経営であったため、「三人寄れば[[文殊菩薩|文殊]]（文珠）の知恵」の[[ことわざ]]にちなみ命名されたといわれている<ref>新歌志内市史 P1434。</ref>。この炭鉱名が地域名となった<ref>新歌志内市史 P101-124。</ref><ref group="注">文珠炭砿については[[西歌駅]]を参照のこと。</ref>。 == 周辺施設 == * [[北海道道114号赤平奈井江線]] * [[北海道歌志内高等学校]]（現・[[歌志内市立歌志内中学校]]） * [[北海道中央バス]]「文珠」停留所 == 隣駅 == ; 北海道旅客鉄道 : 歌志内線 :: [[焼山駅]] -
