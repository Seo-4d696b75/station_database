'''安曇川駅'''（あどがわえき）は、かつて[[滋賀県]][[高島郡 (滋賀県)|高島郡]][[安曇川町]]大字西万木（現在の[[高島市]]安曇川町西万木{{Sfn|江若鉄道の思い出|p=104}}）にあった[[江若鉄道]]の[[鉄道駅|駅]]（[[廃駅]]）。 == 歴史 == 当駅は1929年（昭和4年）、江若鉄道の[[高島町駅 (滋賀県)|大溝駅]]から安曇駅までの区間が開通したのに合わせて'''安曇駅'''（あどえき）の名で開業した{{Sfn|江若鉄道の思い出|p=104}}{{Sfn|今尾|2008|pp=31&ndash;32}}。駅名の「安曇」は当時の村名（高島郡[[安曇町|安曇村]]）でもある<ref name=kampo19290606/>。1931年（昭和6年）には安曇から先へ路線が延伸、[[近江今津駅 (江若鉄道)|近江今津駅]]までの区間が開通した{{Sfn|寺田|2010|p=10}}{{Sfn|田中|宇田|西藤|1998|p=394}}。 [[高島郡 (滋賀県)|高島郡]]内における主要な商業地であった安曇にとって鉄道の存在はかねてから希求されていたものであり、当駅まで鉄路が伸びた際には大規模な祝賀会が挙行された{{Sfn|江若鉄道の思い出|p=105}}。住民は安曇駅に到着した一番列車を[[万歳]]で迎えたことが、当時の新聞によって報じられている{{Sfn|江若鉄道の思い出|p=105}}{{Sfn|ありし日の江若鉄道|p=23}}。<!--また安曇は江戸時代の儒学者中江藤樹ゆかりの地でもあり、彼が[[修身]]の教材として取り上げられていた戦前・戦中期、当駅は[[藤樹書院]]跡などの史跡の最寄り駅として知られていた{{Sfn|江若鉄道の思い出|p=105}}。-->安曇村は町制施行と合併を通じて1954年（昭和29年）に[[安曇川町]]となっていて、当駅の名前ものちに安曇川駅へ改称されている。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1929年]]（昭和4年）[[6月1日]] - 江若鉄道の大溝 - 当駅間の延伸開業に伴い、'''安曇駅'''として高島郡安曇村に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref name=kampo19290606>[{{NDLDC|2957195/6}} 「地方鉄道運輸開始」『官報』1929年6月6日]（国立国会図書館デジタル化資料）</ref>。駅前で開通祝賀会を実施、翌日午後には安曇小学校でも祝賀会を開催{{Sfn|江若鉄道の思い出|p=105}}。 * [[1931年]]（昭和6年）[[1月1日]] - 当駅から近江今津駅までの区間が延伸開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957723/4}} 「地方鉄道運輸開始」『官報』1931年3月9日]（国立国会図書館デジタル化資料）</ref>。 * 時期不詳 - '''安曇川駅'''に改称<ref>{{Harvnb|今尾|2008|p=32}}では1952年から1962年の間とする。</ref>。 安曇川駅は[[列車交換]]が可能な交換駅で{{Sfn|寺田|2010|p=10}}、2本の線路の間に挟み込まれるように[[プラットホーム|ホーム]]が1面配されていた（[[島式ホーム]]）{{Sfn|レイル|p=81}}。旅客と貨物の両方を取り扱うことができる[[一般駅]]であり{{Sfn|寺田|2010|p=10}}、駅には[[停車場#側線|側線]]や貨物用のホームも設けられていた{{Sfn|竹内|1967}}{{Sfn|レイル|p=81}}。 また、駅の敷地内には大規模な木材の集積場が設けられていた。これは町の西隣の山間部に位置した[[朽木村]]から切り出された木材を当駅で貨物列車に積み込み、南へ輸送するためのものである{{Sfn|江若鉄道の思い出|p=106}}。朽木の木材はかつて[[安曇川]]、[[琵琶湖]]を利用する舟運によって専ら輸送されていたが、当駅開業後はその輸送ルートの一端を鉄道が担うこととなった{{Sfn|江若鉄道の思い出|p=106}}。 駅前には旅館や食堂などが軒を連ねる商店街が形成されていた{{Sfn|江若鉄道の思い出|p=106}}。駅に隣接してバスの車庫があり、駅前からはバスが発着した{{Sfn|レイル|p=84}}{{Sfn|寺田|2010|p=15}}。 安曇川は「近江聖人」と称された江戸時代の儒学者[[中江藤樹]]ゆかりの地であり、当駅は[[藤樹書院]]跡や藤樹神社など藤樹に関係する施設の最寄り駅であった{{Sfn|江若鉄道の思い出|p=104}}。彼が[[修身]]の教材として採用されていた戦前・戦中期、これらの史跡は高島郡内有数の観光スポットであり、江若鉄道沿線を紹介する当時の観光パンフレットでも大きく取り上げられている{{Sfn|江若鉄道の思い出|p=104}}{{Sfn|江若鉄道の思い出|p=113}}。また駅に設置されていた観光案内板では、藤樹関連の史跡のほかに朽木スキー場や朽木渓谷など朽木村の名所も紹介されていて、当駅は朽木方面への観光の玄関口でもあった{{Sfn|江若鉄道の思い出|p=105}}。 駅のあった場所は、江若鉄道の廃線後に開業した[[湖西線]][[安曇川駅]]のやや北方に当たる{{Sfn|江若鉄道の思い出|p=105}}。駅の痕跡は残っていない{{Sfn|新・鉄道廃線跡を歩く|p=33}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[水尾駅]] - '''安曇川駅''' - [[新旭駅 (江若鉄道)|新旭駅]] :*
