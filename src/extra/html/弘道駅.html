[[日本国有鉄道]]（国鉄）名寄本線の'''弘道仮乗降場'''（[[鉄道管理局|局]]設定）として開業。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、国鉄からJR北海道に承継。同時に駅に昇格し、'''弘道駅'''となる。 * [[1989年]]（平成元年）5月1日 - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の北東側（遠軽方面に向かって左手側）に存在した。 [[仮乗降場]]に出自を持つ[[無人駅]]となっており、駅舎は無いがホーム横に[[待合所]]を有していた<ref name="bunken2011H-2">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=[[北海道新聞社]] |isbn=978-4894536128 |page=133}}</ref>。ホームは木造で、遠軽方に[[斜路|スロープ]]を有し<ref name="bunken2011H-2"/>駅施設外に連絡していた。ホーム は短い簡易型であった。 == 駅名の由来 == 当駅が所在した地名より。地名は、当地の入植者が[[水戸学]]の[[弘道館|弘道精神]]を伝えたことに由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=187}}</ref>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref> * オンネコムケナイ川<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「弘道」停留所 == 駅跡 == 駅施設は全て撤去され、現在、当駅の跡地は空き地になっている。 また、[[2001年]]（平成13年）時点では当駅跡から遠軽方にある共進川に架かっていた「水谷橋」のコンクリート製の橋台が残存し<ref name="bunken2001H">{{Cite book|和書|editor=[[宮脇俊三]] |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |pages=35-36}}</ref>、[[2011年]]（平成23年）時点でも同様であった<ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=北海道新聞社 |isbn=978-4894536128 |pages=125-126}}</ref>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[小向駅]] -
