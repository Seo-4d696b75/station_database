[[鉄道省]]名寄東線の中湧別 - 興部間の開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1923年]]（大正12年）[[11月5日]] - 路線名を名寄本線に改称し、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1974年]]（昭和49年）[[10月1日]] - 貨物の取り扱いを廃止。 * [[1978年]]（昭和53年）[[12月1日]] - [[チッキ|荷物]]の取り扱いを廃止し、同時に無人駅化。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]により、JR北海道に継承。 * [[1989年]]（平成元年）[[5月1日]] - 名寄本線の全線廃止に伴い、廃駅となる。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは、線路の南西側（遠軽方面に向かって右手側）に存在した<ref name="bunken1983">{{Cite book|和書|editor=[[宮脇俊三]] |others=[[原田勝正]] |title=国鉄全線各駅停車1 北海道690駅 |date=1983-7 |publisher=[[小学館]] |isbn=978-4093951012 |page=211}}</ref>。かつては[[相対式ホーム]]2面2線を有する、[[列車交換]]可能な[[交換駅]]であった。1983年（昭和58年）時点では、使われなくなった対向ホーム側の1線は、交換設備運用廃止後も遠軽方の[[分岐器|転轍機]]は維持され[[停車場#側線|側線]]として残っていた<ref name="bunken1983"/>。ホーム前後の線路は、転轍機の名残で湾曲していた<ref name="bunken1983"/>。 [[無人駅]]となっており、[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、[[中名寄駅]]などと同型の[[プレハブ工法|プレハブ]]駅舎となっていた。駅舎は構内の南西側に位置し、ホームに接していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、当地が[[コムケ湖]]とシブノツナイ湖の2つの沼が望める高台となっていることに由来する<ref name="bunken2004N">{{Cite book|和書|author=太田幸夫 |title=北海道の駅878ものがたり 駅名のルーツ探究 |date=2004-2 |publisher=富士コンテム |isbn=978-4893915498 |page=187}}</ref>。 == 駅周辺 == * [[国道238号]]（オホーツク国道）<ref name="bunken1980M">{{Cite book|和書|editor= |title=北海道道路地図 改訂版 |edition= |date=1980-3 |publisher=地勢堂 |page=18}}</ref> * 沼ノ上郵便局 * 紋別市立沼ノ上小学校 * 旧[[紋別空港]] - 駅の北<ref name="bunken1980M"/>。 * [[コムケ湖]] - 駅の北西<ref name="bunken1980M"/>。 * シブノツナイ湖 - 駅から東に約2.3km<ref name="bunken1983"/>。 * シブノツナイ川<ref name="bunken1980M"/> * [[北海道北見バス]]・[[北紋バス]]「沼の上」停留所 == 駅跡 == 2001年（平成13年）時点では雑木林の中にホームが崩れかけた状態で残存し、ホーム上には[[駅名標]]も残存していた<ref name="bunken2001H">{{Cite book|和書|editor=宮脇俊三 |title=鉄道廃線跡を歩くVIII |date=2001-7 |publisher=[[JTBパブリッシング]] |series=[[JTBキャンブックス]] |isbn=978-4533039072 |pages=35-36}}</ref><ref name="bunken2011H">{{Cite book|和書|author=本久公洋 |title=北海道の鉄道廃線跡 |date=2011-9 |publisher=[[北海道新聞社]] |isbn=978-4894536128 |page=126}}</ref>が、[[2003年]]（平成15年）に紋別市が公園として整備した。2011年（平成23年）時点では駅名標はレプリカに変わり、ホームも修復の上[[アスファルト]]舗装されていた<ref name="bunken2011H"/>。 また、2011年（平成23年）時点ではシブノツナイ湖に注ぐ小さな[[沢]]に架橋されている鉄骨[[桁橋|ガーダー橋]]が残存していた<ref name="bunken2011H"/>。 == 隣の駅 == ; 北海道旅客鉄道 : 名寄本線 :: [[弘道駅]] -
