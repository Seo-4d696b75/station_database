'''丸子鉄道'''当駅 - 大屋駅（後の電鉄大屋駅）間延伸開通に伴い開業<ref name="bunken2010H-3-R">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）210ページより。</ref><ref name="bunken2008M-06">書籍『日本鉄道旅行地図帳 全線全駅全廃線 6 北信越』（監修：[[今尾恵介]]、[[新潮社]]、[[2008年]][[10月]]発行）42ページより。</ref><ref name="bunkenKW05-R">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）27-28ページより。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="bunken2008SH-2-R">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）165ページより。</ref>。 * [[1943年]]（昭和18年）[[10月21日]] - [[陸上交通事業調整法|交通統合]]に伴い'''上田丸子電鉄丸子線'''の駅になる<ref name="bunken2010H-3-R"/><ref name="bunkenKW05-R"/>。 * [[1949年]]（昭和24年） - 駅舎改築<ref name="bunken2008SH-2">書籍『私鉄の廃線跡を歩くII 関東・信州・東海編』（著：寺田裕一、JTBパブリッシング、2008年1月発行）64-67ページより。</ref><ref name="bunkenKW05">書籍『新 消えた轍 5 上信越』（著：寺田裕一、[[ネコ・パブリッシング]]、[[2011年]][[2月]]発行）28-29,34ページより。</ref>（増築<ref name="bunkenRML073">書籍『RM LIBRARY 73 上田丸子電鉄（上）』（著：宮田道一、諸河久、[[ネコ・パブリッシング]]、[[2005年]][[9月]]発行）26-29ページより。</ref>）。 * [[1969年]]（昭和44年）[[4月20日]] - 丸子線の廃線に伴い廃止となる<ref name="bunken2010H-3-R"/><ref name="bunken2008M-06"/><ref name="bunkenKW05-R"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML073"/>。ホームは線路の南側（丸子町方面に向かって右手側）に存在した<ref name="bunkenRML073"/>。そのほか発着線がホームを外れた東側に[[機回し線|機回し]]の出来る[[停車場#側線|側線]]を1線、その側線の丸子町方から南側に分岐し駅舎東側附近に伸びた行き止りの側線を1線、さらにその側線から丸子町方に分岐しさらに2線に分かれる行き止りの側線を有した<ref name="bunkenRML073"/>。貨車の扱い及び電車の留置を行っていた<ref name="bunkenRML073"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-2"/>。駅舎は構内の南側に位置しホーム中央部分に接していた<ref name="bunkenRML073"/>。堂々とした佇まいの建物であった<ref name="bunkenRML073"/>。ホームは[[木構造 (建築)|木造]]の上屋を2棟有し、うち1棟はホームを過ぎて線路の末端部分も覆っていた<ref name="bunkenRML073"/>。ホームの西側は利用客の自転車置き場になっていた<ref name="bunkenRML073"/>。 == 駅周辺 == 開通時は丸子鉄道で唯一の長野県上田市の駅であった。[[日本国有鉄道|国鉄]][[上田駅]]とは離れた、市街地の入口<ref name="bunkenKW05"/>に当たる、東に位置した独自のターミナル駅であった<ref name="bunkenRML073"/>。発着する列車本数が上田駅よりも多かったために当駅周辺は賑わっていた。[[ほていや百貨店]]や上田デパートといった大型店舗や、[[映画館]]などが建てられていた<ref name="bunkenRML073"/>。 * [[上田交通|上田丸子電鉄]][[上田交通真田傍陽線|真田傍陽線]][[川原柳駅]] - 駅から北に約0.5km<ref name="bunkenKW05"/>。 * [[国道18号]] 当駅が開業するまでは、願行寺という[[上田城]]初代城主[[真田氏]]ゆかりの寺の土地であった。駅の開業と同時に実施された上田市の都市計画事業により、土地を横断する形で都市計画道路が建設され、寺社地が墓地と寺の土地に分割され、道路沿いに通りが建設された（大門町）。駅の開業により上田市街地東部の開発が始まり、廃止直前頃までにそれが進んだ。 == 駅跡 == 2007年（平成19年）8月時点では、上田交通経営の上田東駐車場という有料[[駐車場]]になっていた<ref name="bunken2008SH-2"/>。2010年（平成22年）10月時点では[[ドラッグストア]][[クスリのアオキ]]上田中央店店舗と同店舗駐車場が建設されており<ref name="bunkenKW05"/><ref name="bunken2010H-3">書籍『新 鉄道廃線跡を歩く3 北陸・信州・東海編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）71ページより。</ref>、一部は引き続き上田交通経営の上田東駐車場であった<ref name="bunken2010H-3"/>。 また、2010年（平成22年）10月時点では、当駅附近の線路跡は細い道路に転用されていた<ref name="bunkenKW05"/>。但し遺構は残っていなかった<ref name="bunkenKW05"/>。 2017年（平成29年）1月時点では、引き続きドラッグストアクスリのアオキ上田中央店店舗と同店舗駐車場、上田交通経営の上田東駐車場の敷地となっている。 == 隣の駅 == ; 上田丸子電鉄 : 丸子線 :: '''上田東駅''' -
