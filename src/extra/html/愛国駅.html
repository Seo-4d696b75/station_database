'''愛国駅'''（あいこくえき）は、[[北海道]]（[[十勝支庁]]）[[帯広市]]愛国町にあった[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''アコ'''。広尾線の[[廃線]]に伴い[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 同線には[[幸福駅]]があり、「'''愛'''の'''国'''から'''幸福'''へ」の[[キャッチコピー|キャッチフレーズ]]とともにブームを巻き起こした。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の西側（広尾方面に向かって右手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）138ページより。</ref>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった。使われなくなった駅舎と反対側の1線は、交換設備運用廃止後も[[停車場#側線|側線]]として残っていた（但し1983年（昭和58年）時点ではプラットホームは撤去されていた<ref name="bunken1983"/>）。そのほか本線帯広方から駅舎側に分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1線、また上記旧交換線である側線の帯広方から分岐し2線に分かれる行き止まりの側線を有していた<ref name="bunken1983"/>。 本来は[[無人駅]]であるが、大正駅の助役が当駅の助役を兼務しており<ref name="bunken1983"/>、実質的に有人駅であった。開業時の駅舎は改築され、「しゃれた」駅舎を有していた<ref name="bunken1983"/>。駅舎は構内の南西側に位置しホーム中央部分に接していた。ホームには「愛国から幸福ゆき」の乗車券をかたどった[[駅名標]]や<ref name="bunken1983"/>庭園<ref name="bunken1981">書籍『コロタン文庫36 国鉄駅名全百科 56.4訂補版』（[[小学館]]、[[1981年]][[10月]]発行）39ページより。</ref>が作られていた。[[駅スタンプ#わたしの旅|「わたしの旅スタンプ」]]が設置されていた<ref name="bunken1983"/>。 また、駅横には[[国鉄ヨ3500形貨車|ヨ3500形]][[車掌車]]の車体を再利用した売店も存在した<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]][[12月]]発行）232ページより。</ref>。 == 駅名の由来 == 当駅の所在する地名より。[[愛国#帯広市愛国|愛国の地名]]は「愛国青年団」という名の開拓団があったことによる。 == 利用状況 == 1981年度（昭和56年度）の1日乗降客数は742人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道595号愛国停車場線]] * [[北海道道962号愛国停車場古舞線]] * [[国道236号]]（広尾国道）<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]][[3月]]発行）13ページより。</ref> * [[帯広広尾自動車道]][[帯広川西インターチェンジ]] * 愛国簡易郵便局 * 愛の国郷土博物館 * [[札内川]]<ref name="bunken1980M"/> * [[十勝バス]]「愛国」停留所 == 歴史 == * [[1929年]]（昭和4年）[[11月2日]] - '''[[日本国有鉄道|国有鉄道]]（[[鉄道省]]）広尾線'''帯広駅 - 中札内駅間開通に伴い開業。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1945年]]（昭和20年）[[7月16日]] - 砂利線使用開始<ref>『釧路鉄道管理局史』1972年10月発行、年代表。</ref>。 * [[1974年]]（昭和49年）[[12月15日]] - 貨物・[[チッキ|荷物]]取扱い廃止。無人化。 * [[1979年]]（昭和54年）[[8月1日]] - 駅舎改築。記念乗車券4万5千枚販売。 * [[1987年]]（昭和62年）[[2月2日]] - 広尾線の廃線に伴い廃止となる。 == 駅跡 == 帯広市により「愛国交通記念館」として整備されている<ref name="bunken1998S">書籍『全国保存鉄道III 東日本編』（監修：白川淳、[[JTBパブリッシング]]、[[1998年]][[11月]]発行）66ページより。</ref>。旧駅舎内には当時使用していた[[閉塞 (鉄道)|閉塞器]]、通標、備品、[[乗車券]]、駅スタンプ、写真パネルなどが保存・展示されている<ref name="bunken1998S"/>。また駅構内は営業当時そのままの状態で保存されており、線路上に[[国鉄9600形蒸気機関車|9600形蒸気機関車]]'''19671号機'''が[[静態保存]]・展示されている<ref name="bunken2011S">書籍『蒸気機関車完全名鑑 ビジュアル改訂版』（[[廣済堂あかつき|廣済堂ベストムック]]、[[2011年]][[1月]]発行）46ページより。</ref>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]][[4月]]発行）86-87ページより。</ref>。「愛国から幸福ゆき」の乗車券をモチーフにした[[石碑]]、同切符の販売1,000万枚を記念した石碑も設置されている<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]][[9月]]発行）185-186ページより。</ref>。尚、駅横にあったヨ3500形車掌車を再利用した売店は2011年（平成23年）時点では廃棄された状態となっている<ref name="bunken2011H"/>。 また、記念館を含む周辺は公園として整備され、ハート型の噴水、遊具、バーベキュー小屋などが設置されている。 == 「愛の国から幸福へ」ブーム == {{main|幸福駅#「愛の国から幸福へ」ブーム}}'' [[1973年]]（昭和48年）[[3月]]、[[日本放送協会|NHK]]の紀行番組で幸福駅と共に紹介され、一大ブームとなり、
