鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1946年]]（昭和21年）夏 - [[日本国有鉄道|国鉄]]から借入の[[国鉄C12形蒸気機関車|C12形蒸気機関車]]C12 229号機牽引の老方駅発試運転列車が浮蓋駅発車後に過速暴走、当駅にて対向列車到着直後に高速で通過する<ref name="bunkenRML061">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）6,11,25ページより。</ref>。 * [[1947年]]（昭和22年）7月23日 - 豪雨による路盤および橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）16-17ページより。</ref>。 * [[1948年]]（昭和23年）11月8日 - 運休区間（舘合駅 - 老方駅間）が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1949年]]（昭和24年）12月22日 - 当駅 - 老方駅間営業休止に伴い当駅が一時的な[[終着駅]]となる<ref name="bunkenKW03"/>。 * [[1952年]]（昭和27年）2月15日 - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1953年]]（昭和28年）8月5日 - 当駅 - 老方駅間部分廃線に伴い終着駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1965年]]（昭和40年） ** 7月9日 - 水害による雄物川橋梁橋脚洗掘により舘合駅 - 当駅間運休、当駅も営業休止となる<ref name="bunkenRML061-2 19-20P">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）19-20ページより。</ref><ref name="bunkenKW03"/>。 ** 10月21日 - 舘合駅 - 当駅間が正式に営業休止となり、当駅も[[休止駅]]となる<ref name="bunkenKW03"/>。 * [[1966年]]（昭和41年）6月15日 - 舘合駅 - 当駅間部分廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[島式ホーム]]1面2線を有する[[地上駅]]で、当駅 - 老方駅間部分廃線前までは[[列車交換]]可能な交換駅であった<ref name="bunkenRML061-6-11P">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）6,11ページより。</ref>。そのほか[[停車場#側線|側線]]として、西側（駅舎側）本線横手方から西に分岐し駅舎北側に至る行き止りの側線を1線有していた<ref name="bunkenRML061"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっていた<ref name="bunken2008SH-1-R"/>。駅舎は構内の西側に位置し、ホーム南側とを結ぶ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunkenRML061"/>。 列車交換の通票は羽後大森駅 - 当駅間は「□」であった<ref name="bunkenRML061"/>。 == 駅周辺 == * [[秋田県道164号二井山大森線]] - 現在の状況。 * [[湯殿山神社]]（[[分霊|分社]]） == 駅跡 == 2001年（平成13年）7月には、駅跡の道路の脇に、雄物川町（当時）により「横荘線 二井山駅跡」と記載され、沿革も記された白い[[石碑|記念碑]]が建立された<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2007年（平成19年）5月時点でも同様であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。また、駅名を引き継いだ羽後交通のバス停が附近にあった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 また、当駅跡附近から老方の集落入口附近までの線路跡は県道として再利用され、1996年（平成8年）時点では[[盛土|築堤]]や[[切通し]]がほぼ当時の姿を保っていた<ref name="bunken1996H">書籍『鉄道廃線跡を歩くII』（[[JTBパブリッシング]]、[[1996年]]9月発行）34-35ページより。</ref>。1999年（平成11年）時点でも同様であった<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）61ページより。</ref>。この県道は2007年（平成19年）5月時点では[[秋田県道48号横手東由利線]]であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。道床をそのまま舗装したような道路であった<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。また、2007年（平成19年）5月時点では県道に転用されなかった部分が雑草の生い茂った切通しや未舗装の小道として残存していた<ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。2010年（平成22年）時点でも同様であった<ref name="bunken2010H-1 202-203P">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）202-203ページより。</ref>。 なお、当駅 - 老方駅間には5か所の[[トンネル]]があり、1996年（平成8年）時点では崩落の危険のある最も当駅に近い「二井山トンネル」（全長190m）は通行禁止となっていたが、坑口が木々に覆われた形で残存していた<ref name="bunken1996H"/>。1999年（平成11年）時点<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）61ページより。</ref>、2010年（平成22年）10月時点でも同様であった<ref name="bunken2010H-1 203P">書籍『新 鉄道廃線跡を歩く1 北海道・北東北編』（[[JTBパブリッシング]]、[[2010年]]4月発行）203ページより。</ref>。「二井山トンネル」は2003年（平成13年）春に老方側坑口から2m地点付近で崩壊し閉塞している<ref>[http://yamaiga.com/tunnel/niiyama.html 二井山隧道]</ref>。 == 隣の駅 == ; 羽後交通 : 横荘線 :: [[八沢木駅]] -
