和邇駅から当駅までの延伸開業に伴い、滋賀郡[[木戸村 (滋賀県)|木戸村]]に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2956241/9}} 「地方鉄道運輸開始」『官報』1926年4月15日]（国立国会図書館デジタル化資料）</ref>。 ** [[8月11日]] - 当駅から雄松駅までの区間が開業{{Sfn|田中|宇田|西藤|1998|p=394}}<ref>[{{NDLDC|2956343/4}} 「地方鉄道運輸開始」『官報』1926年8月14日]（国立国会図書館デジタル化資料）</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=近江木戸駅線路配置図|left=[[蓬莱駅 (江若鉄道)|蓬莱]]}} {{鉄道配線図|0|0|0|0|quai|quai|quai|0|0|0}} {{鉄道配線図|voie|bifbd|bifbd|voie|voie|voie|voie|voie|bifbg|voie}} {{鉄道配線図|0|0|Vcourbehg|courbehg|voie|voie|voie|courbehd|0|0}} {{鉄道配線図|0|quai|Vvoie|quai|quai|quai|quai|0|0|0}} {{鉄道配線図|0|quai|Vbutoirb|quai|0|0|0|BV|0|0}} {{鉄道配線図|0|quai|quai|quai|0|0|0|0|0|0}} {{駅配線図終了|right=[[青柳ヶ浜駅|青柳ヶ浜]]|source={{Sfn|竹内|1967}}}} 近江木戸駅は旅客と貨物の双方を扱うことができた[[一般駅]]{{Sfn|寺田|2010|p=10}}。[[列車交換]]が可能な交換駅であり{{Sfn|寺田|2010|p=10}}、[[プラットホーム|ホーム]]は2本の線路を挟んで向かい合わせに2面配される[[相対式ホーム]]{{Sfn|寺田|2010|p=15}}{{Sfn|レイル|p=80}}。このほかに[[停車場#側線|貨物側線]]を有した{{Sfn|寺田|2010|p=15}}。 ホームの脇には茶色の[[給水塔]]が建っていて、[[蒸気機関車]]の水の補給に使用された{{Sfn|江若鉄道の思い出|p=64}}{{Sfn|レイル|p=83}}。 == 利用状況 == 初期の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1931年||20,498人||20,305人||149トン||212トン||<ref>[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料）</ref> |- |1934年||23,341人||23,074人||216トン||263トン||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||23,737人||23,231人||71トン||217トン||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||28,032人||28,019人||78トン||261トン||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅のあった場所は、江若鉄道の廃線後に開業した[[湖西線]]の[[志賀駅]]におおむね一致する{{Sfn|江若鉄道の思い出|p=64}}。当駅付近の線路跡には湖西線の線路が通っており、駅の痕跡は残っていない{{Sfn|寺田|2010|p=25}}。 当駅と[[蓬莱駅 (江若鉄道)|蓬莱駅]]の間には木戸川、当駅と[[青柳ヶ浜駅]]の間には大谷川という[[天井川]]が流れていて、江若鉄道はこの2河川をいずれも[[トンネル]]にてくぐり抜けていた{{Sfn|江若鉄道の思い出|p=61}}{{Sfn|レイル|p=76}}。線内にあったトンネルはこの2か所のみである{{Sfn|江若鉄道の思い出|p=61}}。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[蓬莱駅 (江若鉄道)|蓬莱駅]] -
