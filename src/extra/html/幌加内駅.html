[[鉄道省]]雨龍線鷹泊 - 当駅間延伸開通に伴い開業<ref> 『北海道鉄道百年史 下巻』1981年3月 日本国有鉄道北海道総局 編集・発行。</ref>。[[日本の鉄道駅#一般駅|一般駅]]<ref name="hensen">『停車場変遷大事典 国鉄・JR編 II』1998年10月 JTB編集発行。</ref>。 * [[1931年]]（昭和6年） ** [[9月15日]] - 当駅 - 添牛内駅間延伸開通に伴い中間駅となる<ref name="hensen"/>。 ** [[10月10日]] - 線路名を幌加内線に改称、それに伴い同線の駅となる。 * [[1941年]]（昭和16年）10月10日 - 線路名を深名線に改称、それに伴い同線の駅となる。 * [[1949年]]（昭和24年）[[6月1日]] - [[日本国有鉄道]]に移管。 * [[1982年]]（昭和57年） ** [[3月29日]] - [[業務委託駅]]化<ref>新幌加内町史 P597-598 。</ref>。 ** [[11月1日]] - 貨物取扱い廃止<ref name="hensen"/>。 * [[1984年]]（昭和59年）[[2月1日]] - [[チッキ|荷物]]取扱い廃止<ref name="hensen"/>。 * [[1987年]]（昭和62年）[[4月1日]] - [[国鉄分割民営化]]によりJR北海道に継承<ref name="hensen"/>。 * [[1995年]]（平成7年）[[9月4日]] - 深名線の廃線に伴い廃止となる<ref name="hensen"/>。 * [[2000年]]（平成12年）[[3月19日]] - 旧駅舎が火災により焼失。 == 駅構造 == 廃止時点で、[[相対式ホーム]]2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは駅舎側ホーム南側と対向側ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]]6月発行）75ページより。</ref>。線路西側の駅舎側ホーム（西側）が下りの1番線、対向側ホーム（東側）が上りの2番線となっていた。そのほか1番線の名寄方から駅舎側に分岐し駅舎北側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の貨物ホームへの貨物側線を1本有していた<ref name="bunken1993"/>。かつては対向側ホームは[[島式ホーム]]で<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（小学館、[[1983年]]7月発行）205ページより。</ref>、1983年（昭和58年）時点では外側の1線は[[停車場#側線|側線]]として残っていた<ref name="bunken1983"/>。この側線からは深川方に伸びる行き止まりの短い側線を1線有した<ref name="bunken1983"/>。その後1993年（平成5年）までには撤去された<ref name="bunken1993"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]で、駅舎は木造平屋建てで構内の西側に位置し1番線ホーム中央部分に接していた<ref name="bunken1993"/>。開業時は北母子里駅舎より若干大きい程度であったが、1956年（昭和31年）まで（時期不詳）に増改築されている<ref>幌加内村史 1958年11月、P516。</ref><ref group="注">63.6㎡→87.6㎡（幌加内町史による）。</ref>。その後も改修されており、外壁は[[防水工事|防水]][[合板|ベニヤ]]張りであった<ref name="bunken1983"/>。1983年（昭和58年）時点では駅舎内に売店を有した<ref name="bunken1983"/>が、1991年（平成3年）に撤退した。 駅前広場には大きな[[庭石]]を配した[[築山]]と、「歓迎 幌加内町」と記載された[[塔]]が設置されていた<ref name="bunken1983"/><ref name="bunken1993"/>。 <gallery widths="200px" perrow="4"> File:JR Horokanai Station 95-02.jpg|当駅に停車中の列車<br />（奥は朱鞠内方面。1995年8月） File:JR Horokanai Station 95-03.jpg|当駅構内を車窓から望む<br />（奥は深川方面。1995年8月） </gallery> == 駅名の由来 == 当駅の所在する地名より。地名は、[[アイヌ語]]の「ホロカ・ナイ」（後に戻る川）に由来する。付近を流れる幌加内川が、本流である[[雨竜川]]が北から南に流れるのに対して逆向きの流路を取っており、遡って山中に行こうとすればかえって遠回りになることから付けられた。 [[士幌線]][[幌加駅]]と同じ語源である。 == 利用状況 == * 1981年度（昭和56年度）の1日乗降客数は151人<ref name="bunken1983"/>。 * 1992年度（平成4年度）の1日乗降客数は42人<ref name="bunken1993"/>。 == 駅周辺 == * [[国道275号]]（空知国道） * [[北海道道48号和寒幌加内線]] * [[北海道道126号小平幌加内線]] * [[北海道道373号幌加内停車場線]] * 幌加内町役場 * [[士別警察署]]幌加内駐在所 * 幌加内郵便局 * 幌加内小学校 * 幌加内中学校 * [[北海道幌加内高等学校]] * [[幌加内交流プラザ]] -
