[[1913年]]（[[大正]]2年）[[10月1日]] |廃止年月日= |乗車人員= |統計年度= |乗換= |備考= 無人駅 }} '''浜厚真駅'''（はまあつまえき）は、[[北海道]]（[[胆振総合振興局]]）[[勇払郡]][[厚真町]]字浜厚真にある[[北海道旅客鉄道]]（JR北海道）[[日高本線]]の[[鉄道駅|駅]]である。[[電報略号 (鉄道)|電報略号]]は'''ハマ'''。 一部の普通列車（上り最終列車）は当駅を通過する。 == 歴史 == [[ファイル:HamaAtsuma eki.jpg|right|thumb|1975年の浜厚真駅と周囲約1km範囲。右が静内方面。千鳥状の相対式ホーム2面2線と駅表貨物ホーム及び駅裏静内方ストックヤードにそれぞれ引込み線を有している。この後、駅裏側線は貨物取扱廃止に伴い撤去された。また、1983年（昭和58年）時点<ref name="bunken1983"/>では、まだ貨物ホーム側側線は残されている。{{国土航空写真}}]] * [[1913年]]（大正2年）[[10月1日]]：'''苫小牧軽便鉄道'''の'''厚真駅'''（あづまえき）として開業<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2952457/7 『官報』 1913年10月06日　鉄道院彙報「軽便鉄道運輸開始」]（国立国会図書館デジタルコレクション）</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1925年]]（大正14年）[[11月15日]]：'''浜厚真駅'''（はまあづまえき）に改称<ref>[https://dl.ndl.go.jp/info:ndljp/pid/2956122/4 『官報』 1925年11月19日　鉄道省彙報「地方鉄道駅名改称」]（国立国会図書館デジタルコレクション）</ref>。 * [[1927年]]（昭和2年）[[8月1日]]：苫小牧軽便鉄道が[[鉄道敷設法|国有化]]により[[日本国有鉄道|国有鉄道]]に移管。線路名が日高線に改称され、同線の駅となる。同時に駅名の読みを「'''はまあつま'''」に改称。 * [[1943年]]（昭和18年）[[11月1日]]：線路名を日高本線に改称、それに伴い同線の駅となる。 * [[1977年]]（昭和52年）[[2月1日]]：貨物・[[チッキ|荷物]]取扱い廃止。同時に出札・改札業務を停止し旅客業務について無人化。但し閉塞扱いの運転要員は継続配置。乗車券は[[日本の鉄道駅#簡易委託駅|簡易委託]]化。 * [[1986年]]（昭和61年）[[11月1日]]：電子閉塞の導入により運転要員無人化。 * [[1987年]]（昭和62年）[[4月1日]]：[[国鉄分割民営化]]によりJR北海道に継承。 * [[1987年]]（昭和62年）：貨車駅舎に改築<ref>書籍『JR・私鉄全線各駅停車1 北海道620駅』（[[小学館]]、[[1993年]][[6月]]発行）100ページより。</ref>。 * 時期不詳<ref group="注" name="ekigyoumu">1990年（平成2年）7月1日時点では完全無人化されている（小冊子『HANDBOOK 1990』「駅は旅の出発点」（発行：北海道旅客鉄道、[[1990年]]発行）より）。</ref>：簡易委託廃止、完全無人化。 === 駅名の由来 === 当駅の所在する地名より。地名のうち「厚真」は、[[アイヌ語]]の「アツ・オマ・プ」の転訛した<ref name="bunken2004N">書籍『北海道の駅878ものがたり 駅名のルーツ探究』（監修：太田幸夫、富士コンテム、[[2004年]][[2月]]発行）99-100ページより。</ref>「アトマプ」（オヒョウ（[[ニレ科]]の落葉高木）のあるところ）に由来する。[[アットゥシ]]の生地となる繊維を[[オヒョウ (植物)|オヒョウ]]より取ったとのことである<ref name="bunken1993"/><ref name="bunken2004N"/>。当駅所在地周辺は「厚真」の海岸寄りにあるため「浜」を冠す<ref name="bunken2004N"/>。 == 駅構造 == [[単式ホーム]]1面1線を有する、[[地上駅]]。ホームは線路の南側（鵡川方面に向かって右手側、旧1番線）に存在する。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっている<ref name="bunken1993">書籍『JR・私鉄全線各駅停車1 北海道630駅』（[[小学館]]、[[1993年]][[6月]]発行）100ページより。</ref>。かつては[[相対式ホーム]]2面2線を有する[[列車交換]]可能な[[交換駅]]であった<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]][[7月]]発行）100ページより。</ref>。互いのホームは少しずれた位置にあり、駅舎側ホーム中央部分と対向ホーム南側を結んだ[[踏切#構内踏切|構内踏切]]で連絡した<ref name="bunken1983"/>。駅舎側（南側）が上りの1番線、対向側（北側）が下りの2番線となっていた<ref name="bunken1983"/>。そのほか1983年（昭和58年）時点4月では1番線の苫小牧方から分岐し駅舎西側の[[プラットホーム#切欠きホーム|ホーム切欠き]]部分の旧貨物ホームへの側線を1線有していた<ref name="bunken1983"/>。交換設備運用廃止後は線路は側線を含め1993年（平成5年）までには撤去された<ref name="bunken1993"/>。 [[苫小牧駅]]管理の[[無人駅]]となっている。駅舎は構内の南側に位置しホームに接している。[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、[[車掌車]]改造のものとなっている。一時期はピンク色に塗装されていた<ref name="bunken2008">書籍『北海道鉄道駅大図鑑』（著：本久公洋、[[北海道新聞社]]、[[2008年]][[8月]]発行）343ページより。</ref>が、2017年時点では波が描かれた白基調に塗装が変わっている（冒頭画像参照）。 <gallery widths="180" heights="120"> ファイル:Hama-atsuma Station, Hidaka Main
