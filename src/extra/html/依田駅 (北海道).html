(十勝国)|中川郡]][[幕別町]]字依田にかつて設置されていた、[[日本国有鉄道]]（国鉄）[[広尾線]]の[[鉄道駅|駅]]（[[廃駅]]）である。広尾線の[[廃線]]に伴い、[[1987年]]（昭和62年）[[2月2日]]に廃駅となった。 一部の普通列車は通過した（1986年（昭和61年）11月1日改定の時刻（廃止時の時刻表）で、朝一番の下り列車、及び最終の上り列車<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）33ページより。</ref>）。 == 駅構造 == 廃止時点で、1面1線の[[単式ホーム]]を有する[[地上駅]]であった。ホームは、線路の東側（広尾方面に向かって左手側）に存在した。また、[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）138ページより。</ref>。 開業時からの[[無人駅]]で、駅舎及び[[待合所]]は存在しなかった。上屋などもなかったが、ベンチのみ設置されていた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅が所在した地名より。地名は、[[明治|明治時代]]の開拓者である晩成社の[[依田勉三]]の[[名字]]に由来する<ref name="bunken1983"/>。 == 利用状況 == 1981年度（昭和56年度）の1日当たりの乗降客数は3人<ref name="bunken1983"/>。 == 駅周辺 == 周囲は[[畑]]作地帯である<ref name="bunken1983"/>。 * [[北海道道151号幕別帯広芽室線]] * [[北海道道238号更別幕別線]]<ref name="bunken1980M">書籍『北海道道路地図 改訂版』（地勢堂、[[1980年]]3月発行）13ページより。</ref> * [[札内川]]<ref name="bunken1980M"/> - 駅の右手側<ref name="bunken1983"/>。 == バス路線 == [[十勝バス]]が運行する[[バス代行|代替バス]]は道路事情の関係で帯広 - 依田 - 北愛国 - 愛国の別系統が設定されたが、路線統合を経て2003年（平成15年）9月をもって廃止されている。現在は一般客も乗車できるスクールバスが「依田近隣センター前」に停車する。 == 歴史 == * [[1957年]]（昭和32年）[[12月25日]] -
