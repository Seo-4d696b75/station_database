鉄道会社名を'''羽後鉄道'''に改称。路線名を'''横荘線'''に制定。それに伴い'''羽後鉄道横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunken2008SH-1">書籍『私鉄の廃線跡を歩くI 北海道・東北編』（著：寺田裕一、JTBパブリッシング、[[2007年]]9月発行）82-85ページより。</ref>。 * [[1947年]]（昭和22年） ** 7月23日 - 豪雨による路盤及び橋脚損壊により横荘線全区間運休、当駅も営業休止となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）16-17ページより。</ref>。 ** 7月29日 - 横手駅 - 舘合駅間が復旧、当駅も営業再開となる<ref name="bunkenKW03"/><ref name="bunkenRML061-2"/>。 * [[1952年]]（昭和27年）2月15日 - 鉄道会社名を'''羽後交通'''に改称。それに伴い'''羽後交通横荘線'''の駅となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 * [[1971年]]（昭和46年）[[7月20日]] - 横荘線の廃線に伴い廃止となる<ref name="bunken2008M-02"/><ref name="bunken2010H-1-R"/><ref name="bunkenKW03"/><ref name="bunken2008SH-1"/>。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった<ref name="bunkenRML061">書籍『RM LIBRARY 61 羽後交通横荘線』（著：若林宣、[[ネコ・パブリッシング]]、[[2004年]]9月発行）6-7,10ページより。</ref>。ホームは線路の南側（老方方面に向かって左手側）に存在した<ref name="bunkenRML061"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた<ref name="bunkenRML061"/>。 [[無人駅]]となっており<ref name="bunken2008SH-1-R"/><ref name="bunkenRML061"/>、駅舎は無いがホーム東側出入口附近に[[待合所]]を有した<ref name="bunkenRML061"/>。待合所は比較的新しい作りであった<ref name="bunkenRML061"/>。ホームは横手方に階段を有し<ref name="bunkenRML061"/>駅施設外に連絡していた。 駅名は、羽後交通の資料には「樋'''ノ'''口」「樋'''の'''口」の双方が使用されていた<ref name="bunkenRML061"/>。 == 駅周辺 == 周囲は水田であった<ref name="bunkenRML061"/>。 * 善福寺 == 駅跡 == 1999年（平成11年）時点では、すでに[[区画整理]]がされており、駅跡は残っていなかった<ref name="bunkenTouhoku1999">書籍『とうほく廃線紀行』（[[無明舎出版]]、[[1999年]]12月発行）60ページより。</ref>。 また、当駅跡附近の線路跡は1999年（平成11年）時点では道路に転用されていた<ref name="bunkenTouhoku1999"/>。 == 隣の駅 == ; 羽後交通 : 横荘線 :: [[横手駅]] -
