'''鬼志別駅'''（おにしべつえき）は、[[北海道]]（[[宗谷支庁]]）[[宗谷郡]][[猿払村]]字鬼志別にあった[[北海道旅客鉄道]]（JR北海道）[[天北線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''オニ'''。同線の[[廃線]]に伴い[[1989年]]（平成元年）[[5月1日]]に廃駅となった。 廃止時まで運行されていた[[急行列車|急行]]「[[宗谷 (列車)#宗谷本線・天北線優等列車沿革|天北]]」の停車駅であった。 == 駅構造 == 廃止時点で、[[単式ホーム]]・[[島式ホーム]]（片面使用）複合型2面2線を有する[[地上駅]]で、[[列車交換]]可能な[[交換駅]]であった。互いのホームは駅舎側ホーム中央部分と島式ホーム西側を結んだ[[踏切#構内踏切|構内踏切]]で連絡していた<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）190ページより。</ref>。駅舎側（北側）ホームが上りの1番線、[[待合所]]が設置された<ref name="bunken2011P">書籍『追憶の鉄路 北海道廃止ローカル線写真集』（著：工藤裕之、[[北海道新聞社]]、[[2011年]]12月発行）28ページより。</ref>島式ホームが下りの2番線となっていた<ref name="bunken1983"/>。そのほか1983年（昭和58年）時点では島式ホームの外側に3線を[[停車場#側線|側線]]として有しており、1番線の音威子府方から分岐し駅舎東側の[[切欠きホーム|切欠き]]部分の貨物ホームへの貨物側線を1線有していた<ref name="bunken1983"/>。 [[日本の鉄道駅#直営駅|職員配置駅]]となっており、木造駅舎は構内の北側に位置し1番線ホーム中央部に接していた<ref name="bunken1983"/>。 == 駅名の由来 == 当駅の所在する地名より。地名は、[[アイヌ語]]の「オニウ<small>シ</small>ペッ<ref>[[ラテン文字化|アイヌ語ラテン翻字]]: o-ni-us-pet</ref>」（川尻に・木が・群生している・川）、「オヌウ<small>シ</small>ペッ<ref>[[ラテン文字化|アイヌ語ラテン翻字]]: o-nu-us-pet</ref>」（川口・豊漁・ある・川）のいずれかからとされる<ref name=":3">{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/3140P.pdf|title=アイヌ語地名リスト オニシベ～キタ　P31-40P|accessdate=2017-10-19|date=2007|work=[http://www.pref.hokkaido.lg.jp/ks/ass/new_timeilist.htm アイヌ語地名リスト]|publisher=北海道 環境生活部 アイヌ政策推進室}}</ref>。 == 利用状況 == *
