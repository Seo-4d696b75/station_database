[[福岡県]][[穂波郡]][[幸袋町|大谷村]]大字幸袋に[[筑豊鉄道]]の駅として開業。 * [[1897年]]（明治30年）[[10月1日]] - [[九州鉄道]]に合併。同社の駅となる。 * [[1898年]]（明治31年）[[4月8日]] - 当駅構内で入換え作業中の[[蒸気機関車]]（[[国鉄3300形蒸気機関車#筑豊鉄道|筑豊鉄道13]]）の[[ボイラー]]破裂事故が発生。死傷者7名、民家3軒が破損するという惨事であった<ref>{{cite web|url=http://www6.plala.or.jp/jrcFukui/118.pdf |title=わだち No.118｜13号機関車の正体は？ |publisher=[[鉄道友の会]] 福井支部報 |page=5-7 |format=PDF |date=2008-05 |accessdate=2012-09-09 }}</ref>。詳細は「[[日本の鉄道事故 (1949年以前)#九州鉄道蒸気機関車ボイラー破裂事故|九州鉄道蒸気機関車ボイラー破裂事故]]」を参照。 * [[1899年]]（明治32年）[[12月26日]] - 当駅 - （貨）伊岐須間の貨物支線が開業。 * [[1901年]]（明治34年）[[3月31日]] - 当駅 - （貨）幸袋炭坑間の貨物支線が廃止。 * [[1907年]]（明治40年）[[7月1日]] - [[鉄道国有法|国有化]]により、官設鉄道の駅となる。 * [[1930年]]（昭和5年）[[4月1日]] - 当駅 - （貨）高雄間の貨物支線が開業。 * [[1945年]]（[[昭和]]20年）[[6月10日]] - 当駅 - （貨）高雄間の貨物支線が廃止され、当駅構内扱いとなる。 * [[1969年]]（昭和44年）[[12月8日]] - [[小竹駅|小竹]] - [[二瀬駅|二瀬]]間および当駅 - （貨）伊岐須間の運輸営業廃止に伴い、廃止<ref>1969年（昭和44年）12月6日日本国有鉄道公示第376号「運輸営業の廃止の件」</ref>。 == 駅周辺 == 駅跡にホームがわずかに残っている。 当駅付近の廃線跡は市道となっている。 * 幸袋郵便局 * [[国道200号]] * [[旧伊藤伝右衛門邸]] == 隣の駅 == ; 日本国有鉄道 : 幸袋線 :: [[目尾駅]] -
