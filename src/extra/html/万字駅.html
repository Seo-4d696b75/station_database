'''万字駅'''（まんじえき）は、[[北海道]][[空知郡]][[栗沢町]]万字仲町（現・[[岩見沢市]]栗沢町万字仲町）にあった[[日本国有鉄道]]（国鉄）[[万字線]]の[[鉄道駅|駅]]（[[廃駅]]）である。[[電報略号 (鉄道)|電報略号]]は'''マム'''。万字線の[[廃線]]に伴い[[1985年]]（昭和60年）[[4月1日]]に廃駅となった。 == 駅構造 == 廃止時点で、[[単式ホーム]]1面1線を有する[[地上駅]]であった。ホームは線路の北東側（万字炭山方面に向かって左手側）に存在した<ref name="bunken1983">書籍『国鉄全線各駅停車1 北海道690駅』（[[小学館]]、[[1983年]]7月発行）70ページより。</ref>。地形の関係で駅舎よりかなり低い位置にあった<ref name="bunken1983"/>。[[分岐器|転轍機]]を持たない[[棒線駅 (単式ホーム)|棒線駅]]となっていた。かつて貨物取扱を行っていた時期には[[給水設備#船舶・車両の給水設備|給水タンク]]や[[転車台|ターンテーブル]]も存在したが<ref name="bunken2000H">書籍『鉄道廃線跡を歩くVII』（[[JTBパブリッシング]]、[[2000年]]1月発行）61ページより。</ref>、撤去されていた。 [[無人駅]]（[[日本の鉄道駅#簡易委託駅|簡易委託駅]]）となっていた。[[日本の鉄道駅#有人駅|有人駅]]時代の駅舎は改築され、[[乗車券]]が駅舎内で発券されていた。駅舎は構内の北東側に位置し、ホーム中央部分からシェルター付きの階段を上った<ref name="bunken1983"/>。 == 駅名の由来 == 当駅の所在する地名より。地名および炭鉱名は[[万字炭鉱]]の経営者、朝吹家の[[家紋]]の「[[卍]]」に由来する<ref name="bunken2010HE">書籍『廃線終着駅を訊ねる 国鉄・JR編』（著：[[三宅俊彦]]、[[JTBパブリッシング]]、[[2010年]]4月発行）26ページより。</ref>。 == 利用状況 == * 1981年度の1日乗降客数は16人<ref name="bunken1983"/>。 == 駅周辺 == * [[北海道道38号夕張岩見沢線]] * 万念寺 - 駅附近<ref name="bunken1983"/>。髪の伸びる「お菊人形」が祀られている。 * [[岩見沢市]]営バス「万字バス待合所」停留所 == 歴史 == * [[1914年]]（大正3年）[[11月11日]] - '''[[日本国有鉄道|国有鉄道]]万字軽便線'''志文駅 - 万字炭山駅間開通に伴い開業<ref name="kanpo1">[https://dl.ndl.go.jp/info:ndljp/pid/2952785/4 『官報』 1914年11月04日　鉄道院告示第102号]（国立国会図書館デジタルコレクション）</ref>。[[日本の鉄道駅#一般駅|一般駅]]。 * [[1922年]]（大正11年）9月2日 - 線路名を'''万字線'''に改称、同線の駅となる。 * [[1970年]]（昭和45年）8月17日 - 貨物取扱い廃止。 * [[1978年]]（昭和53年）4月6日 - [[チッキ|荷物]]取扱い廃止。 * [[1979年]]（昭和54年）3月4日 - 駅舎改築。 * 時期不詳<ref group="注" name="ekigyoumu">1981年（昭和56年）4月時点では無人駅であった（書籍『コロタン文庫65 私鉄駅名全百科』（[[小学館]]、[[1981年]]9月発行）49ページより。）。</ref> - [[無人駅|無人]]（[[日本の鉄道駅#簡易委託駅|簡易委託]]）化。 * [[1985年]]（昭和60年）[[4月1日]] - 万字線の廃線に伴い廃止となる。 == 駅跡 == 1999年（平成11年）時点では駅舎が万字仲町[[簡易郵便局]]の局舎として再利用されており、旧駅舎からホームへの階段やホームも残存している<ref name="bunken2000H"/>。駅舎のそばには美流渡駅跡設置と同じ形状の、[[駅名標]]をかたどった「万字駅 跡碑」の看板が建立されている<ref name="bunken2000H"/>。2011年（平成23年）時点でも同様であった<ref name="bunken2011H">書籍『北海道の鉄道廃線跡』（著：本久公洋、[[北海道新聞社]]、[[2011年]]9月発行）28ページより。</ref>。 == 隣の駅 == ; 日本国有鉄道 : 万字線 :: [[美流渡駅]] - '''万字駅''' - [[万字炭山駅]] == 脚注 ==
