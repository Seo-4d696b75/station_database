[[近江今津駅 (江若鉄道)|近江今津駅]]間の開通に伴い1931年（昭和6年）に開業した{{Sfn|江若鉄道の思い出|p=110}}。この区間を建設する際、当駅の位置は[[新旭駅 (江若鉄道)|新儀駅]]と近江今津駅の中間に近い位置に設置する計画であったが、用地の問題などがあったため両駅の中間から南（新儀駅寄り）にずれた位置に設けられることとなった{{Sfn|江若鉄道の思い出|p=110}}。安曇 - 近江今津間の開通は当初1930年（昭和5年）8月末を予定していたが、翌年にずれ込んだのは当駅の設置に手間取ったことが一因であった{{Sfn|江若鉄道の思い出|p=108}}。 饗庭駅が予定より南に設けられたことにより当駅と近江今津駅の駅間距離は必要以上に長くなってしまったため、その隔たりを埋めるように[[北饗庭駅]]がのちに設置されている{{Sfn|江若鉄道の思い出|p=114}}。 江若鉄道は1969年（昭和44年）10月31日をもって営業を終了し{{Sfn|江若鉄道の思い出|p=124}}、当駅も翌11月1日に[[廃駅|廃止]]された{{Sfn|今尾|2008|pp=31&ndash;32}}。 * [[1931年]]（昭和6年）[[1月1日]] - 江若鉄道の安曇 - 近江今津間の延伸開業に合わせて高島郡[[饗庭村]]に開業{{Sfn|今尾|2008|pp=31&ndash;32}}<ref>[{{NDLDC|2957723/4}} 「地方鉄道運輸開始」『官報』1931年3月9日]（国立国会図書館デジタル化資料）</ref>。 * [[1969年]]（昭和44年）[[11月1日]] - 駅廃止{{Sfn|今尾|2008|pp=31&ndash;32}}。 == 駅構造 == {{駅配線図開始|title=饗庭駅線路配置図|left=[[新旭駅 (江若鉄道)|新旭]]}} {{鉄道配線図|0|0|BV|0|0|quai|quai|0|0}} {{鉄道配線図|0|0|courbebg|voie|voie|voie|courbebd|0|0}} {{鉄道配線図|voie|voie|voie|voie|voie|voie|voie|voie|voie}} {{鉄道配線図|0|Dvoie|0|quai|quai|quai|0|Mvoie|0}} {{鉄道配線図|0|0|courbehg|voie|voie|voie|courbehd|0|0}} {{駅配線図終了|right=[[北饗庭駅|北饗庭]]|source={{Sfn|竹内|1967}}|note=本線とつながっていない線路は撤去済みを示す}} 饗庭駅は旅客と貨物の両方を取り扱うことができる[[一般駅]]で{{Sfn|寺田|2010|p=10}}、[[プラットホーム|ホーム]]は1面のみだった{{Sfn|レイル|p=85}}。当初は2本の線路の間に挟まれた[[島式ホーム]]で、[[列車交換]]が可能であったが、交換設備が撤去された結果列車交換のできない棒線駅となっている{{Sfn|竹内|1967}}{{Sfn|寺田|2010|p=10}}{{Sfn|寺田|2010|p=15}}。[[停車場#側線|貨物側線]]や貨物用のホームも有していた{{Sfn|竹内|1967}}{{Sfn|寺田|2010|p=15}}。 晩年は[[委託駅]]であった{{Sfn|寺田|2010|p=15}}。 == 利用状況 == 開業から数年間の年間乗降客数・貨物取扱量の状況は以下の通り。 {| class="wikitable collapsible" style="font-size:85%;white-space:nowrap" |- !colspan=7|年間の旅客および貨物の取扱量 |- !rowspan=2|年!!colspan=2|旅客!!colspan=2|貨物!!rowspan=2|出典 |- !乗車!!降車!!発送!!到着 |- |1931年||17,578人||17,608人||1,616トン||259トン||<ref>[{{NDLDC|1464961/408}} 『滋賀県統計全書』昭和6年版]（国立国会図書館デジタル化資料）</ref> |- |1934年||19,144人||19,121人||469トン||493トン||<ref>[{{NDLDC|1452136/411}} 『滋賀県統計全書』昭和9年版]（国立国会図書館デジタル化資料）</ref> |- |1935年||21,580人||21,368人||707トン||516トン||<ref>[{{NDLDC|1452161/412}} 『滋賀県統計全書』昭和10年版]（国立国会図書館デジタル化資料）</ref> |- |1936年||22,636人||22,171人||836トン||644トン||<ref>[{{NDLDC|1452178/404}} 『滋賀県統計全書』昭和11年版]（国立国会図書館デジタル化資料）</ref> |} == 駅周辺 == 駅前には[[農業協同組合|農協]]の事務所と倉庫が置かれ、周辺の村々で収穫された米が当駅から出荷されていた{{Sfn|江若鉄道の思い出|p=110}}。駅の東側には帝釈天堂（田井帝釋天堂）がある{{Sfn|江若鉄道の思い出|p=111}}。かつては境内に巨大な松の木が生え、滋賀県の天然記念物にも指定されていたが、松は1934年（昭和9年）の[[室戸台風]]により倒壊したとされ現存しない{{Sfn|江若鉄道の思い出|p=111}}。 江若鉄道の廃線後、当駅付近の線路跡は[[湖西線]]の用地に転用され、駅があった場所は空き地に変わった{{Sfn|江若鉄道の思い出|p=112}}。ただその周囲には2015年現在も倉庫が建ち並んでおり{{Sfn|江若鉄道の思い出|p=111}}、米の出荷拠点であった当駅の存在を偲ばせる。 == 隣の駅 == ; 江若鉄道 : 江若鉄道線 :: [[新旭駅 (江若鉄道)|新旭駅]] -
